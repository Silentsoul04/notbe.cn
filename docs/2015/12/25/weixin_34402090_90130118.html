Atom + activate-power-mode震屏插件Windows7下安装

Dec 25, 2015

Atom是Github推出的一个文本编辑器，搜索一下大概是给Web前端用的，最近比较火的是他的一个插件activate-power-mode，可以实现打字屏振效果。

用来装装逼还是挺适合的，本来想试试能不能像sublimetext一样能编译C/C++之类的文件，后来发现根本不行，google到的也只有一些零碎的makefile之类的东西，没什么卵用，

所以现在这个文本编辑器真的只能用来装装逼。

 

话不多说，来看看具体怎么设置。

首先先下载这个文本编辑器

https://atom.io/

大小85.8MB，并且会很流氓的使用chrome风格的自动安装进系统盘目录，有洁癖的可以直接打住了。

 

然后到这里下载插件包

https://github.com/JoelBesada/activate-power-mode

 

解压文件到下面的目录

 

注意文件目录是带.atom的packages目录下，不要和文件运行的目录，很相似的路径搞混了，不然是安装不成功的

 

 

然后在命令提示符（可能需要管理员权限运行）中进入

C:\Users\你的用户名\.atom\packages\activate-power-mode

运行

apm install

 

如图所示看到显示

Installing modules done

就说明安装成功了

 

运行Atom编辑器，按Ctrl + Alt + O（注意是O（欧）不是0（零））

或者在菜单中开启

 

如果只想要粒子效果，不要屏振，可以打开

C:\Users\你的用户名\.atom\packages\activate-power-mode\lib\activate-power-mode.coffee

修改53行的x,y值，我改成了下面这样

写了个振动范围，你也可以写成常数如0.00001这样就几乎看不到振动，也可以增加振幅，如果你不晕的话，比如振动个10像素之类的......

和一般游戏中的屏振没什么区别
