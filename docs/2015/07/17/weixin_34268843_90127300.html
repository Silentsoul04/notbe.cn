jinfo命令的使用

Jul 17, 2015

jinfo命令 该命令可以打印出java进程的配置信息：包括jvm参数，系统属性等
用法： jinfo [ option ] pid jinfo [ option ] executable core jinfo [ option ] [server-id@]remote-hostname-or-IP
参数：

pid   进程号 executable   产生core dump的java executable core   core file remote-hostname-or-IP  主机名或ip server-id    远程主机上的debug server的唯一id
选项： no option  打印命令行参数和系统属性 -flags  打印命令行参数 -sysprops  打印系统属性 -h  帮助
示例： （1）jinfo pid （2）jinfo -flags pid
