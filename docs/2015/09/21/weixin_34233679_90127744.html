sublimeCodeIntel 的配置

智能提示插件，这个插件的智能提示功能非常强大，可以自定义提示的内容库，我的Python智能提示设置（配置文件路径为packages\SublimeCodeIntel-master\.codeintel\config）为：

在项目的根目录目录下建立.codeintel/config

但是在windows 需要进入dos 环境下建立。以点开头的文件夹和文件。资源管理器不允许创建点开头的文件或文件夹，但在命令提示符下是可以的，因此遇到这些情况的时候请切换到命令提示符下工作。

使用mkdir .codeintel，在当前目录建立了一个名为“.codeintel”的文件夹。
1 {
2     "Python": {
3         "python":'D:/Program Files/Python26/python.exe',
4         "pythonExtraPaths": ['D:/Program Files/Python26','D:/Program Files/Python26/DLLs','D:/Program Files/Python26/Lib','D:/Program Files/Python26/Lib/plat-win','D:/Program Files/Python26/Lib/lib-tk','D:/Program Files/Python26/Lib/site-packages']
5     }
6 }

（路径根据你自己安装的python路径编写）

其中“pythonExtraPaths”就是需要智能提示所需要用到的内容库。
