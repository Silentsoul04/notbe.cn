在js中的回调函数的理解

May 11, 2019

一、 回调函数

  1. 回调函数：函数A作为参数(函数引用)传递到另一个函数B中，并且这个函数B执行函数A。我们就说函数A叫做回调函数。如果没有名称(函数表达式)，就叫做匿名回调函数。
  2. 回调函数的实现：作为另一个函数的参数，在另一个函数中作为返回结果，在另一个函数中调用它。当我们作为参数传递一个回调函数给另一个函数时，我们只传递了这个函数的定义，并没有在参数中执行它。当包含(调用)函数拥有了在参数中定义的回调函数后，它可以在任何时候调用(也就是回调)它。这说明回调函数并不是立即执行，而是在包含函数的函数体内指定的位置“回调”它(形如其名)。
    3.回调函数的作用：javascript是单线程的，也就是说js代码的执行顺序是从上到下，有时候你需要一个操作完全结束再到下一个操作，为了确保这一点就必须使用回调函数。
    注意：如果回调函数中传入的参数是一个对象，那么在回调函数内容如果要改变这个对象的值，记得要操作这个对象的某个属性，而不是对象本身，因为对象传参是传地址的。

二、函数的指针调用方式

  1. 回调：回调的机制是通过指针来调用函数。在javascript中，经常使用的调用形式是回调。
  2. 回调函数：按照调用者的约定实现函数的功能，由调用者调用。传递一个参数化的函数，就是将这个函数作为一个参数传到另一个主函数里面，当那一个主函数执行完之后，再执行传进去的作为参数的函数。这个过程中的参数化的函数就叫做回调函数，也可以说是被作为参数传递到另一个函数（主函数）的那个函数就叫做回调函数。
  3. 回调函数相当于是闭包：闭包能够进入包含它的函数的作用域，因此回调函数能获取包含它的函数中的变量，以及全局作用域中的变量。
  4. 回调函数，也被称为高阶函数，是一个被作为参数传递给另一个函数的函数，是属于函数式编程。
  5. 回调函数的应用：
    1）异步调用（例如读取文件，进行HTTP请求，等等）
    2）时间监听器/处理器
    3）setTimeout和setInterval方法
    4）一般情况：精简代码
    5） js的回调函数写组件更方便

三、回调函数的分析

  1. 回调函数：函数A作为参数传递给函数B中，在函数B中执行函数，那么就可以函数A为回调函数
    实例代码：
function f1(a){
      console.log(a);
   }

  console.log(f1);            // 函数名，返回函数本身
  /**f1(a){ console.log(a); }**/
  console.log(f1(12));    //12 调用执行函数，返回值
  1. 函数可以作为参数传递
    实例代码：
     function f2(num1){
     console.log(num1);
 }
 function f3(fun,value){
     fun(value);
 }
 console.log(f3(f2,10));     //10

说明：将f2函数作为参数传递给f3函数

  1. 将匿名函数作为参数传递给f4函数
    实例代码：
   function f4(fun,value){
        fun(value);
    }
    console.log(f4(function(num){console.log(num);}),18);    //18
    console.log(
            f4(function(num){
                console.log(num);
            }),20
    );    //20

说明：之前的f2函数相当于就是匿名函数，作为参数传递的，也称之为回调函数

四、回调函数的传参分析

  1. 将回调函数的参数作为与回调函数同等级的参数进行传递
    实例代码：
   function f5(value){
       console.log(value);
   }
   function f6(fun,value){
       fun(value);
   }
   console.log(f6(f5,30));  // 30

说明：
第一步：回调函数f5需要一个参数value
第二步：函数f6后面的第一个参数应该是一个函数，但确是函数本身，在函数后面加()相当于就是运行这个函数，如果需要参数，那么在这个时候就需要传入该参数
第三步：这里的f5方法相当于就是回调函数，f6方法接收了两个参数，第一个参数是f5，第二个参数是30，那这里的第二个参数相当于也是f5所需要的参数，所以可以理解为将回调函数的参数作为与回调函数同等级的参数进行传递

  1. 回调函数的参数在调用回调函数的内部创建
    实例代码：
   function f7(value){
       console.log(value.name);
   }
   function f8(fun){
        var value = {
            name:"张三"
        };
        fun(value);
   }
   console.log(f8(f7));     // 张三

说明：
第一步：回调函数f7需要一个参数value，这个参数value相当于是一个对象，name相当于是属性
第二步：f8函数只接收一个参数，这个参数就是f7函数本身，回调函数f7的参数在回调函数的内部已经定义过了，直接传递也能够达到回调函数传参的目的，所以可以理解为回调函数的参数在调用回调函数的内部创建
第三步：f8相当于是主函数，f7相当于是参数，也可以称之为回调函数

五、回调函数的案例：

  1. 需求分析：编写一个排序函数，实现数字排序。排序方法由客户函数实现，函数参数个数为两个，两个参数的关系作为排序后的元素间的关系
    实现代码：
//定义排序函数
   function sortNumber(obj,fun){
       //参数验证，如果第一个参数不是数组或者第二个参数不是函数，那么就抛出异常报错
       if (!(obj instanceof Array) || !(fun instanceof Function)){
             // 生成错误信息
             var e = new Error();
             e.number = 10000;
             e.message = "参数无效";
             //抛出异常
             throw e;
       }

       for(n in obj){
           for(m in obj){
               //使用回调函数，规则由用户设置，由客户函数实现
               if( fun(obj[n],obj[m])){
                   //交换数据
                   var temp = obj[n];
                   obj[n] = obj[m];
                   obj[m] = temp;
               }
           }
       }
       //返回排序后数组
       return obj;
  }

说明：在定义排序函数之后，首先要进行参数验证，判断两个参数分别是数组和函数，否则就会报错。然后，实现排序函数，分别设置n和m两个数组，使用回调函数，交换数据，最后返回排序后的数组。
