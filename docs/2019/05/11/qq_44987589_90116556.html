硕彦博创——Linux用户详解

一.用户类型
1.超级用户，就是Root用户，系统管理员，UID为0。
2.系统用户，就是Linux系统中一类特殊的用户，UID一般为1~499。
3.普通用户，由root用户创建的用户，UID为500以上。

二.用户管理*
1.创建用户账号命令useradd
命令格式：useradd 【option】 用户账号名
-c 描述新用户账号 -d 设置用户主目录 -e 设置账号过期日期 -f 设置密码失效时间 -g 指定用户所属私有组 -s 设置用户登录Shell类型 -u 设置用户ID
2.修改用户账号属性命令usermod
命令格式：usermod 【Option】 用户账号名
-d 指定主目录 -e 设置过期时间 -l 新用户账号名 -I 锁定用户账号 -U 解除用户账号锁定
3.删除用户账号命令userdel
命令格式：userdel 【Option】 用户账号名
-r 连同主目录一起删除账号
4.维护用户口令的命令passwd
命令格式：passwd 【option】 用户账号名
-d 删除指定账号的密码 -l 锁定用户账号 -u解除指定用户账号的锁定状态 -s 查看状态
5.设置密码期限命令chage
命令格式：chage 【Option】 用户账号名
-E 设置账号过期日期 -l 设置账号不活跃天数 -m 设置密码最小生存期 -M 设置密码最大生存期
**
三.编辑配置文件管理用户**
1.用户账号配置文件/etc/passwd
【用户账号信息的格式】
root❌0:0:root:/root:/bin/bash
用户账号名:密码:用户ID(UID):组ID(GID):全名:用户主目录:Shell类型

2.用户密码配置文件/etc/passwd
【用户密码信息的格式】
root:*:17230:0::99999:7:::
用户账号名:加密的用户密码:最后一次修改密码的日期:密码的最小生存日期:密码的最大生存日期:更换密码前警告的天数:账号不活跃的天数:账号过期日期:保留字段
注：未使用，以“:”表示
