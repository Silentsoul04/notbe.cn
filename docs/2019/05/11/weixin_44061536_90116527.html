初始化循环变量只会执行一次
	循环条件如果为true
		执行循环体的内容
		改变循环变量

	直到循环条件为false时结束循环


循环中有两个关键字
	break     结束整个循环   跳到循环后面继续执行
	continue  结束本次循环  跳过continue下面的语句 进入下一次循环

无限循环：

for(;;){}

while和for可以完全互换   
	while基本会于循环次数不确定的
	for一般用于循环次数确定的

while和for循环都是要先进行条件判断，如果条件为false，则循环一次也不会进行

do{
	// do while循环是先进行循环，再进行条件判断   至少会执行一次循环体
