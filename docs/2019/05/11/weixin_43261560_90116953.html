Contos7 下安装jdk

卸载

1、先查看自的Centos自带的版本 **rpm -qa | grep java**

2、用这个命令来删除系统自带的jdk rpm -e --nodeps 后面跟系统自带的jdk名

4、删完之后可以再通过 rpm -qa | grep java 或 rpm -qa | grep jdk 命令来查询出是否删除掉

安装

1、用命令跳到 cd /usr/local 目录下创建 mkdir java文件夹


2、去jdk管网下载 https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html


3、使用SecureCRT-8.5工具按快捷键 alt+p 切换 sftp模式 拖到jdk-8u211-linux-x64.tar文件到SecureCRT-8.5工具里

5、将jdk-8u211-linux-x64.tar复制到 你创建的新文件夹 usr/local/java
cp /home/admin/下载/jdk-8u211-linux-x64.tar.gz /usr/local/java
复制命令 下载的jdk安装包路径 复制到的目录

6、使用命令 跳到 ：cd /usr/local/java 文件夹下解压：
tar -zxvf jdk-8u211-linux-x64.tar.gz

7、通过 vim /etc/profile 命令打开 profile 文件盘配置环境变量打开之后按 i 进入 insert （插入）模式，在文件末尾添加上环境变量再按 ESC 退出 vim模式输入 :wq 保存退出
export JAVA_HOME=/usr/local/java/jdk1.8.0_211
export CLASSPATH=.: J A V A H O M E / l i b / d t . j a r : JAVA_HOME/lib/dt.jar: JAVAH​OME/lib/dt.jar:JAVA_HOME/lib/tools.jar
export PATH= P A T H : PATH: PATH:JAVA_HOME/bin


8、退出vim模式输入： source /etc/profile 让变量生效

9、输入javac 或 java -vresion 查看成功
