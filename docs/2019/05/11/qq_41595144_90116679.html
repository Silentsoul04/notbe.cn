Freemarker初步使用Demo

May 11, 2019

一.首先导入依赖:

//二选一，因为我用的是Gradle:
//		Maven依赖:
	<!-- https://mvnrepository.com/artifact/org.freemarker/freemarker -->
		<dependency>
		    <groupId>org.freemarker</groupId>
		    <artifactId>freemarker</artifactId>
		    <version>2.3.28</version>
		</dependency>
		
//	Gradle依赖:
	<!-- https://mvnrepository.com/artifact/org.freemarker/freemarker -->
	compile group: 'org.freemarker', name: 'freemarker', version: '2.3.28'

二:在资源文件下创建模板文件 test.ftl：

 <html>
	<head>
<meta  charset="utf-8">
  		<title>demo</title>
   </head>
   <body>
		<#---   我是一个注释，不会输出与html注释不同，html注释会输出到页面，我不会  -->
	${kname},啦啦啦测试。${message}
	</body>
 </html>

JAVA代码：

  //1.创建一个配置对象，参数传入版本号
    Configuration configuration = new Configuration(Configuration.getVersion());
     
        try {
           //2.设置模板所在的目录
            configuration.setDirectoryForTemplateLoading(new File("D:\\test"));

            //3.设置字符集
            configuration.setDefaultEncoding("utf-8");
            //4.获取模板对象
            Template template = configuration.getTemplate("test.ftl");

            //5.创建数据模型（可是对象，或者Map对象）
            Map map = new HashMap();
            map.put("kname","vale");
            map.put("message","注意每个数据必须传值，否则报错。");	
            //6.创建一个输出流对象。
            Writer out = new FileWriter("d:\\test.html");

            //7.输出
            template.process(map,out);

            //8.关流
            out.close();
