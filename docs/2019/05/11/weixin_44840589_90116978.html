【Java】排序方法系列之选择排序

May 11, 2019

在需要排序的序列中找到最小（大）元素，将其存放到序列的起始位置。
算法如下：
①从待排序序列中，找到关键字最小的元素a[min]；
②如果a[min]不是该排序序列要的首元素，将其和首元素调换；
③从之后的元素中，找出关键字最小的元素，重复①、②步，直到排序结束。
举个例子：

public class 选择排序 {
	public static void main(String[] args) {
		int[]a= {3,44,38,5,47,15,36,26,27,2,46,4,19,50,48};
		for(int j = 0;j<a.length;j++) {
			int min=j;
			for(int i=j;i<a.length;i++) {
				if(a[i]<a[min]) {
					min=i;
				}
			}
			int temp = a[j];
			a[j]=a[min];
			a[min]=temp;
			}
		for(int i=0;i<a.length;i++) {
			System.out.println(a[i]);
		}
	}
}
