3、获取root权限

手机端依次进行如下操作：
设置-我的设备-全部参数-MIUI版本连续点5次以上；
设置-更多设置-开发者选项-设备解锁状态-绑定账号；
设置-授权管理-root权限管理-开启root权限；
重启手机生效

4、 解锁system分区

手机连接电脑的情况下
手机端设置-更多设置-开发者选项-开启USB调试；
USB与电脑的连接模式设为传输文件(MTP)；
电脑端下载并解压adb工具；
在开始处输入cmd打开命令提示符输入以下指令

cd yourpath\platform-tools
adb kill-server
adb start-server
adb devices
adb root
adb disable-verity
adb reboot

下载安装两个app：Syslock，RE管理器:Root Explorer；
可通过电脑端下载好apk通过usb导入手机端进行安装，附下载地址：Syslock，RE管理器:Root Explorer，也可手机端安装酷安app在app内搜索应用进行安装，安装后需要打开一次应用并关闭
设置-授权管理-root权限管理-给Syslock和RE管理器:Root Explorer授权root权限；
打开Syslock，点一下中间那个按钮，稍等会解锁system分区，然后重启手机；
重启后打开RE管理器:Root Explorer，在最顶层目录就可看到system文件夹

5、 卸载系统预装应用

针对出厂内置的第三方app ，删除根目录下/cust/app/customized和/system/data-app所有文件夹；针对系统内嵌app（比如主题浏览器等等），见参考教程
