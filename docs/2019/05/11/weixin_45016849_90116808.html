LeetCode打卡第二天 #53最大子序和

May 11, 2019

因为昨天做的题有关动态规划，就找了相似题目的题刷。
任务：
给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。
示例:

输入: [-2,1,-3,4,-1,2,1,-5,4],
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。

做这题时稍微拐个弯，想到既然是最大子序和，那这段子序从头开始加到任意一个位置，其和始终大于等于0。因为如果其为负，则去掉它，剩下的一段子序和更大。
既然如此，那我们就可以直接从头开始加，并定义一个max通过和每一次假的和比较来来记录最大值，如果加的和小于0，那么前面这一段就没有意义了，直接从下一位重新开始加。这样加到最后，max的值就为最大值。

实现代码：



int maxSubArray(int* nums, int numsSize){
    int i,s,max;
        s=nums[0];
    max=nums[0];
    for(i=1;i<numsSize;i++)
    {
        
        if(s<0){
            s=nums[i];
        }else s+=nums[i];
        if(s>max) max=s;
    }
    return max;
}
