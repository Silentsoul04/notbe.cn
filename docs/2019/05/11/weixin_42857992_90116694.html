JAVA——枚举

May 11, 2019

枚举的概念

枚举是在一定范围内取值,并且这个值必须是枚举类型中的任意一个,并且只能有一个

枚举的特点

1 必须在规定范围内取值
2 这个值只能取一个
3 这个值可以是规定范围内的任意一个
4 枚举的本质就是一个Java类

注意事项

1 枚举中所有的成员，必须出现在枚举对象的下面
2 如果枚举类中有一个成员，那么最后一个枚举对象后面的分号不能省略
3 枚举中构造方法必须私有，防止外界创建对象修改成员变量的值
4 枚举中可以有抽象方法 ————可通过创建匿名内部内并重写抽象方法后描述某个枚举成员的信息，提高程序的可读性
5 枚举也是switch语句中的常量形式之一

枚举中常用的方法

1 String name() 返回此枚举常量的名称，与其枚举声明中的声明的完全相同。
2 int ordinal() 返回此枚举常数的序数（其枚举声明中的位置，其中初始常数的序数为零）。
3 T valueOf(Class enumType, String name) 返回具有指定名称、指定枚举类型字节码的枚举常量。(静态方法，枚举名直接调用)
格式 : 枚举名 枚举常量 = 枚举名.valueOf(枚举名.class,“枚举常量名”);
4 T valueOf(String name) 返回具有自定名称的枚举常量(静态方法，枚举名直接调用)
5 T[] values() 返回枚举常量的数组(静态方法，枚举名直接调用)

代码演示

public class EnumDemo {
    public static void main(String[] args) {
// 枚举名直接访问枚举常量，返回常量本身
        Animals name1=Animals.DOG;
        System.out.println(name1);
                
// String name() 返回此枚举常量的名称，与其枚举声明中的声明的完全相同。
        String name2 = Animals.DOG.name();
        System.out.println(name2);
        
// int ordinal() 返回此枚举常数的序数（其枚举声明中的位置，其中初始常数的序数为零）。
        int index = Animals.PERSON.ordinal();
        System.out.println(index);

// T valueOf(String name) 返回指定名称的枚举常量(静态方法) 
        Animals animals1 = Animals.valueOf("PERSON");
        System.out.println(animals1);
        
// T valueOf(Class<T> enumType, String name) 返回具有指定名称、指定枚举类型字节码的枚举常量。(静态方法)
        Animals animals2 = Animals.valueOf(Animals.class, "PIG");
        System.out.println(animals2);
        
// T[] values() 返回枚举常量的数组(静态方法) 
        Animals[] array = Animals.values();
        // foreach 循环遍历数组 arr
        for (Animals arr : array) {
            System.out.print(arr+" ");    
        }
    }
}

enum Animals {
    DOG, PIG, CAT, PERSON  
}
