4.循环语句（if，while）

May 11, 2019

1.if语句
一般格式为：

 if  条件:
        结果
    else:
    	结果

多选择为：

num = input('请输入你所要选择的套餐:  ')

if num == '1':
	print('你选择了套餐一')
elif num =='2':
	print('你选择了套餐二')
else:
	print('你选择了套餐三')

注：字符串转换成数字：int(str) 条件：str必须为数字，
数字转换成字符串：str(int)

num =int( input('请输入你所要选择的套餐:  '))
        
if num == 1:
	print('你选择了套餐一')
elif num ==2:
	print('你选择了套餐二')
else:
	print('你选择了套餐三')

2.while语句:

while 条件:
	循环体

终止循环：改变条件。break跳出循环
continue：结束本次循环，继续下次循环
