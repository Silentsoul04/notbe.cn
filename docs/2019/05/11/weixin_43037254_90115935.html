职场新人必懂的潜规则

一、哪些话不该说？

  1. 不透露不打听工资

  2. 和领导的私下交情

  3. 不听小道消息

  4. 自己未来的规划

  5. 不好的习惯改掉

a. 不打断别人的话

b. 不马后炮

c. 不说不会

d. 推卸责任

二、不应该做的事

  1. 卷入政治斗争

  2. 不要以为自己很了解公司

  3. 不要以为自己很了解老板

  4. 不要以为自己很了解同事

三、领导和同事的潜台词

  1. “要去约会吗？”穿着

  2. “最近有什么事吗？”工作不投入

  3. “工作能力很强，相信你会做好。”不给予帮助

细节展现潜力，处处留心皆学问

  1. 记住别人的名字

  2. 记录重要人的生日

  3. 记录好每天的工作

  4. 做事超出别人的预期

  5. 做好应急预案

  6. 工作区保持整洁

  7. 职场礼仪

Excel常用操作

八大函数搞定大部分办公需求

一．几个常用的Excel函数

  1. if 判断
    a.条件
    b.满足条件的返回值
    c.不满足条件的返回值

例：当A1>A2，显示合格，否则显示不合格

=if（A1>A2,”合格”，”不合格”）

  1. sum 求和函数 快捷键“Alt加=”

Sum if单个条件求和 条件区域，条件，求和区域

Sum ifs多条件筛选 求和区域，条件区域，条件

  1. V lookup 垂直查找

H lookup 水平查找

  1. Count 计数非空单元格的个数 （数字型）

Counta所有非空单元格的个数

  1. Count if 统计条件区域中符合单个指定条件单元格个数

  2. Count ifs统计条件区域中符合多个指定条件单元格个数

  3. Date 年月日组成日期

  4. Round指定单位四舍五入

    Round between随机数生成函数

  5. If ever 返回错误值 正确—结果 错误—指定值

二．常用的功能

1.数据有效性

2.条件格式

3.数据透视表

4.VBA操作系统

三．快捷键

F1帮助键

F2输入Ctrl+F2打印预览

F4重复上一步操作

F9选中部分转换为结果

F12文件另存为

Ctrl+反单引号 （返单引号在1的左边）

Alt+d+f+f快速筛选

Ctrl+E快速填充

四．7点建议

  1. 标题居中文字左对齐，数值右对齐，加千分符

  2. 数据类表格不要合并单元格

  3. 正文：10.11号字体，汉字：微软雅黑，数字：Arial

  4. 底色统一

  5. 尽量少用隐藏行列，采取创建组折叠展开

  6. 最佳行高18

  7. 标题涂色，去掉表格线

流程图，快速掌握的工具

一．基本构造

横向甬道——部门，角色

纵向甬道——业务的分类

开始，结束——类似横向椭圆

过程——长方形——动宾

判断——菱形——条件选择

（多）文档——要输入或输出的文件

数据库——圆柱形——文件、数据的归档

二．怎样画流程图

  1. 几个软件

Excel；Visio；Axure rp；亿图

  1. 步骤

a. 纵向甬道

n个角色，n+1个甬道

业务流程的标题

b. 添加图形，用线连接

c. 为每一步添加说明

d. 调整格式，颜色

三．注意事项

  1. 一页之内

  2. 从左至右，从上至下

  3. 连接线不交叉

  4. 以开始符号开始

  5. 同一个过程只画一个输入方

  6. 流程名——动宾

怎么处理同事、上下级关系

一．如何与上级相处

  1. 做好自己的本职工作，按时完成上级交付的任务，及时汇报

  2. 有自己的处事原则，不要出言顶撞

  3. 虚心接受意见

  4. 感谢，夸奖

  5. 离职不要闹太僵

二．如何与下级相处

  1. 引领

    氛围，目标，引领他们完成任务

  2. 信任

    工作重点而不是细小的任务

  3. 鼓舞

  4. 分享

经验，知识

三．如何与同事相处

  1. 配合工作

  2. 新同事多帮忙，老同事多交流

  3. 不坑人，主动吃亏

  4. 不说三道四，勇于承担责任

  5. 带小礼物

  6. 给加班同事帮助

  1. 网络搜索：微博、脉脉、在行、天眼查、知乎、领英、微信群

  2. 熟人介绍：高级别保险人员的人脉圈子

同乡，同校，同爱好

内职业生涯规划：自己的性格、天赋、资源，原动力

外职业生涯规划：地点、内容、环境、薪资

  1. 选择一个自己想成为的目标职业

a. 时间

b. 行业，企业

c. 地点

d. 职位

e. 职务

  1. 知道这个岗位具备的能力

  2. 行动和学习计划

    收集行业信息

如何让你的领导赏识你，谈升职加薪

一．几点经验

  1. 待在重要岗位/重要项目组

  2. 做好报联商

  3. 给领导关怀

  4. 提升自己的能力/核心竞争力

二．怎样谈升职加薪

  1. 时机 自己做出成绩，领导心情好

跳槽的正确姿势

一．什么情况下可以考虑跳槽

  1. 企业文化价值观不相符

  2. 没有成长空间

  3. 涨薪无望，升职无望

二．怎样准备
面试内容，个人作品，朋友圈，简历

三．跳槽的思路
城市，岗位，行业

四．如何避免跳坑

  1. （亲戚朋友）打探

  2. 企查查，天眼查 公司底细

  3. 领英、脉脉、职友集、Boss直聘 员工对公司的评价
