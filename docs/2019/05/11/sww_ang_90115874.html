IDEA的破解

博主是按照志哥的方法破解的：见链接
链接：志哥破解IDEA18.1的方法.
需要注意的是：志哥破解19.1 版本用的 license server已经在19年四月底到期了，所以博主只能重新破解18.1.3 的版本了。18版本的破解用的是 activation code破解之后到期的时间会很长。所以建议用18.1.3版本。
PS：以上的破解方法都是针对IDEA Ultimate版(Web开发)
IDEA Community版(Android开发和JVM，开源)

IDEA配置

破解之后，进入创建项目的页面点击右下角config中的settings就进入了设置页面

首先是导包的设置


“Insert imports on paste”
——设置为 “all”，表示当复制粘贴程序代码时，会自动import需要的包。
“Show import popup for”
——勾选表示当输入需import的类名，静态方法和属性，会询问是否要import。
“Add unambiguous imports on the fly”
——勾选表示若有多个匹配的包会询问。
“Optimize imports on the fly”
——勾选表示会动态优化import内容，自动删除多余或者重复的import。

代码提示设置

选择none是为了不区分大小写提示。要不然我们写一个类声明的时候，如果首字母没有大写，就不会提示。

字体设置

show only monospaced fonts 就是只显示等宽字体。
大小设置成16,如果12的话看着有点费劲。
