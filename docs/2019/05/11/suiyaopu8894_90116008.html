Anaconda无法更新包(numpy,tensorflow,opencv)

May 11, 2019

今天折腾了一天Numpy一直是13.1 ，更不到16.1.但Pip可以更新到。

然后看到了某张图片

 

原理很简单，把国内的镜像源去掉就好。

 

举例Windows系统，去掉源

打开

C:\Users\jasper\.condarc（不一定是jasper，这是你当前所在的用户）

 

把所有源去掉 留下default 就好。

channels:
  - defaults
ssl_verify: true
show_channel_urls: true

剩下这么多足以。

因为用的是默认源，国内下载会有点慢。emmm，后面你懂的。
