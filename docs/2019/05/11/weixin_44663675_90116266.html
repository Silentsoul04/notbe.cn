Python每日一记49>>>SQL Server安装

May 11, 2019

今天主要学习了SQL Server简介与SQL Server安装
简介就不再陈述了，直接记录安装啦
首先我们要知道SQL Server有5个版本，分别是企业版，标准版，开发版，Web版，精简版，具体的版本介绍个功能也不多加叙述
下载地址如下，我们选择安装2017
https://www.microsoft.com/zh-cn/sql-server/sql-server-downloads
我们选择精简版，如下

我们直接选择简单安装即可，会自动配置，安装完成后就会提示安装SSMS

我们直接选择安装，并按照安装提示一步步进行即可
这里需要解释的是，SQL Server是作为一个服务由操作系统监控的，而SSMS是作为一个单独的程序运行的，简单理解就是python和pycharm之间的联系。
两者都安装完成后，我们就可以进行SSMS的启动和链接了
在开始菜单中找到程序microsoft server 2017–SQL Server management studio,弹出如下界面

点击浏览更多，选择数据库引擎下的链接即可。


链接成功就进入以下主界面

以上就实现了数据库安装的基本要求了，后续就可以慢慢进行数据库的其他操作了。
