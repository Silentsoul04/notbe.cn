sublime text3 下载及配置

May 11, 2019

 

一、下载

1. 官网的不靠谱，win10系统中文某些符号乱码[方块+问号]

2. 到php中文网去下载（http://www.php.cn/xiazai/gongju/1473，这个是英文版，可以安装插件）

3. 注册码百度一大堆

 

二、配置

1. win键，输入sub，直接调出ST3

2. ctrl+shift+p，输入ip，安装package control

3. ctrl+shift+p，输入ip，正式下载插件：

1）emmet【必备】

2）alignment【必备】

3）brackethighlighter【必备】

4）deleteblanklines【必备】

5）docblockr【必备】

docblockr怎么用：

（1）防止在代码注释之后回车，下一行自动生成注释标记

Preferences -> Package Settings -> Doc Blockr -> Settings - User
 

{
  "jsdocs_extend_double_slash": false
}

（2）自动生成注释 作者名+时间，加上上面的，即：（这个可选）
 

{
  "jsdocs_extend_double_slash": false,

  "jsdocs_extra_tags":["@author 名字","@date {{date}}"]
}

 

4. 其他配置项： 

Preferences -> Settings
//把 tab 转换成4个空格
"tab_size": 4,
//把tab 转换成 空格
"translate_tabs_to_spaces": true,
//在保存时自动把tab 转换成空格
"expand_tabs_on_save": true,
//当某行为空格且无其它字符时, 保存时会去除空白
"trim_trailing_white_space_on_save": true

 

记录一下，防止忘掉！（当然还有其他配置，待续......）
