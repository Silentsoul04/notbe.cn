May 11, 2019

解题思路：选择出三个数相乘，得出的值最大值构成有两种情况：一是 分别选出三个最大（正）、最小（负）和第二小（负）的值，它们的乘积是正的，二是 三个值 分别是数组中 最大值 次大值 和 第三大的值。

 

代码如下：

class Solution(object):
    def maximumProduct(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
       
        a,b,c,d,e,f  = 0,0,0,0,0,0
        
        a = max(nums)
        nums.remove(a)
        
        b = max(nums)
        nums.remove(b)
        
        c = max(nums)
        nums.remove(c)
        
        nums += [a,b,c]
        d = max(nums)
        nums.remove(d)
        
        e = min(nums)
        nums.remove(e)
        
        f = min(nums)
        nums.remove(f)
        
        return max(a*b*c,d*e*f)

思路：众数就是一个数组中出现次数最多的数

根据这个思路代码如下：

class Solution(object):
    def majorityElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        dict_ = {}
        for i in nums:
            dict_[i] = dict_.get(i,0) + 1
            
        x,y = 0,0  
            
            
        for a,b in dict_.items():
            if b > y:
                x,y = a,b 
        else:
            return x

 这个时间复杂度是2n~但是没注意到题目中有一个条件:众数是指数组中出现次数大于[n/2]的元素，意思就是说，从小到大排序之后，中位数一定是众数。

这个思路代码如下：

class Solution(object):
    def majorityElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        return sorted(nums)[len(nums) // 2]

 速度比上面快了不少~

 

底子很薄，看了很多的题，都是树，图等等，我刚看了栈、队列和链表，等周末了把树看完集中的涮一波好了，找了一道没考算法的题：

 

思路：思路一： 因为数组中的元素要么一次要不两次，所以我可以先对它排序，然后如果相邻两个元素相同的话，那么就加到列表中去。

         思路二： 用字典统计数组中出现的次数，如果没有出现就统计为0，如果以前出现过就加到列表中去。

         思路三： 利用 1<= a[i] <= n，所有元素大小都不超过数组a的长度的特点，采用a[i]-1的值做为下标访问数组a，因为a中有些元素出现2次，即当以他们（a[i-1]）为下标会访问这样的下标元素2次。因此在第一次访问时，采用标记（取负），再次访问到时判断正负 。

三种思路代码如下：

class Solution(object):
    def findDuplicates(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        #第一种方法
        d,c = [],0
        
        for i in sorted(nums):
            if i == c:
                d.append(i)
            c = i
        return d


        #第二种方法
        d,c = {},[]
        for i in nums:
            if d.get(i,0) == 1:
                c.append(i)
            else:
                d[i] = d.get(i,0) + 1
        
        return c

        ##第三种方法

        dd = []
    
        for i in range(len(nums)):
            num = abs(nums[i])
            if nums[num - 1] > 0:
                nums[num - 1] *= -1
            else:
                dd.append(num)
        return dd

只有第三种方法完全实现了没有占用多余空间的要求，其次这三种方法都实现了时间复杂度在n以内的要求。
