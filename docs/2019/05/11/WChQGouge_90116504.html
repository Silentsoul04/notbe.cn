为树莓派配置IPV6公网地址

使用6plat服务为树莓派配置IPV6地址

  • 前提是树莓派已安装openvpn且已申请到6plat的个人账号

1.新建文件输入以下内容并保存为client.ovpn

client
dev tun
proto tcp
remote 46.6plat.org 9185
comp-lzo
resolv-retry infinite
nobind
persist-key
persist-tun
setenv CLIENT_CERT 0
auth-user-pass
remote-cert-tls server
verb 3
sndbuf 0
rcvbuf 0
cipher none
