docker 安装

  • 查看centOS版本
    docker 要求centOS版本的内核版本高于3.10
uname -r
3.10.0-957.10.1.el7.x86_64
  • 升级软件包及内核（选做）
yum update
  • 安装docker
yum install docker

如果执行命令报错Delta RPMs disabled because /usr/bin/applydeltarpm not installed.那么执行下面两条命令

yum provides '*/applydeltarpm'
yum install deltarpm
  • 启动docker
systemctl start docker
  • 将docker服务设置为开机启动
systemctl enable docker

常规操作

  • 搜索镜像，mysql为例
docker search mysql

docker hub：https://hub.docker.com/

  • 拉取镜像
ducker pull 镜像名:tag

tag是可选的

  • 查看本地所有的镜像
docker images
  • 删除指定本地镜像
docker rmi image-id

docker 安装 tomcat

docker search tomcat

docker pull tomcat

docker images
操作 命令 说明
运行 docker run --name mytomcat -d tomcat:latest -d 后台运行，-name自定义容器名字，容器名字：标签
列表 docker ps（查看运行时容器） 加上-a 可以查看所有容器
停止 docker stop 容器名字或容器id 停止当前运行的容器
启动 docker start 容器名字或容器id 启动容器
删除 docker rm 容器id 删除指定的容器
端口映射 docker run -d -p 8888:8080 tomcat -p 主机端口映射到容器内部的端口
容器日志 docker logs 容器名字或容器id 查看日志

端口映射时，注意防火墙是否关闭或者开放了相关的端口

更多命令：https://docs.docker.com/engine/reference/commandline/docker/

  • 多个tomcat容器
docker run -d -p 8887:8080 tomcat
docker run -d -p 8888:8080 tomcat
docker run -d -p 8889:8080 tomcat

docker 安装 mysql

实际上是在 docker hub 搜索镜像

docker search mysql

docker pull mysql

docker run --name mysql01 -e MYSQL_ROOT_PASSWORD=123456 -d mysql

docker ps

启动完成之后，使用客户端访问依然连接不到，原因是端口没有映射

docker run -p 8000:3306 -v $PWD/mysql:/var/lib/mysql --name mysql -e MYSQL_ROOT_PASSWORD=123456 -d mysql:5.7
参数 说明
-p 主机端口映射
-v $PWD/mysql:/var/lib/mysql：将主机当前目录下的/mysql挂载到容器的/var/lib/mysql
-e MYSQL_ROOT_PASSWORD=password：初始化root用户的密码

如果报错名字被占用，换一个名字就行了，到这里安装完成

Docker 部署 SpringBoot 项目

先配置文章末尾的相关连接的Docker 创建jdk镜像，因为后面要使用作为基础镜像

~下创建docker文件夹，然后执行

mkdir docker
cd docker
vi dockerfile-boot

实际内容根据自己的名字填写

FROM jdk:1.8
ADD springboot-hello.jar /springboot-hello.jar
EXPOSE 8080/tcp
ENTRYPOINT ["java","-jar","/springboot-hello.jar"]

将自己的springboot应用复制到docker文件夹下，也就是和dockerfile-boot同一个目录

构建容器

docker build -t boot-hello . -f dockerfile-boot

查看容器是否构建完成

[root@izwz9buyhgwtrvz1fhraikz docker]# docker images
REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
boot-hello          latest              480c5b9c1d8c        6 seconds ago       615 MB

运行容器

docker run --name boot-hello -d -p 8002:8080 boot-hello

查看运行日志，部分
