循环中有两个关键字
	break     结束整个循环   跳到循环后面继续执行
	continue  结束本次循环  跳过continue下面的语句 进入下一次循环


猜数字游戏：
	1.生成随机数
	import java.util.Random;		
	在main方法中
	Random rand = new Random();
	rand.nextInt(arg);   //随机生成一个[0,arg)之间的整数

无限循环：

for(;;){}

while和for可以完全互换   
	while基本会于循环次数不确定的
	for一般用于循环次数确定的

while和for循环都是要先进行条件判断，如果条件为false，则循环一次也不会进行
