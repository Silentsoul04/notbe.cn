1.什么是JDBC

JDBC就是Java数据库连接，说白了就是用java语言操作数据库

2.得到Connection对象

1.导入jar：驱动。
2.加载驱动类：class.forName(“类名”);
3.给出：url，username，password
4.使用DriverManager类得到连接对象Connection对象

3.代码演示

4.ClassNotFoundException和SQLException 异常

1.ClassNotFoundException异常：

  • 没有导入驱动包
  • 驱动类有误

2.SQLException异常：

  • 检查3个参数 url，username,password 是否正确
  • 是否开启了mysql服务

5.操作增 删 改

3.调用他的 int executeUpdate(String sql); 他可以发送 DML,DDL（返回的值，代表的是所影响的行数）

6.查询
