Ubuntu18.04 LTS美化、常用软件安装、QQ、百度云

May 11, 2019

先给看看效果图

1、美化
安装gnome-tweak-tool
终端：sudo apt install gnome-tweak-tool
去掉shell上无法修改的叹号
sudo apt-get install gnome-shell-extensions
重启
打开tweak
将“User themes”设置为ON
安装GTK主题
https://www.opendesktop.org/browse/cat/366/ord/top/
找到自己想要的主题下载
解压将其移动到/usr/share/themes/
图标
https://www.opendesktop.org/s/Gnome/browse/cat/132/
解压后把文件放到/usr/share/icons/
2、ubuntu安装qq,微信
终端：
git clone https://gitee.com/wszpkzpk/deepin-wine-for-ubuntu.git
cd deepin-wine-for-ubuntu
./install.sh
安装完成后去阿里云下载： http://mirrors.aliyun.com/deepin/pool/non-free/d/
qq：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.im/
微信：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.wechat/
百度云：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.baidu.pan/
一般最小化图标后，有比较难看的窗口，强烈建议用Tolcons Plus
软件中心下载Toplcons Plus。若安装失败
https://extensions.gnome.org/extension/1031/topicons/
选最新版本下载，解压。终端输入
sudo mv TopIcons@phocean.net.v22.shell-extension /usr/share/gnome/gnome-shell/extensions/
若失败终端输入sudo nautilus,找到文件复制到/usr/share/gnome-shell/extensions/
重启
打开tweak
扩展Toplcons Plus打开
3、安装搜狗输入法
https://pinyin.sogou.com/linux/?r=pinyin
下载，进入下载目录打开终端输入ls
sudo dpkg -i xxxx.deb
(xxxx.deb是你想要安装的软件名，在ls下复制粘贴软件名)
配置输入法框架fcitx
软件中心下载fcitx
或者
终端输入
sudo add-apt-repository ppa:fcitx-team/nightly
sudo apt-get update
将输入法框架选择为fictx
修复依赖sudo apt-get install -f
4、安装wps
先卸载系统自带的office
终端：sudo apt-get remove --purge libreoffice*
官网下载
https://www.wps.cn/product/wpslinux/
下载，进入下载目录打开终端输入ls
sudo dpkg -i xxxx.deb
(xxxx.deb是你想要安装的软件名，在ls下复制粘贴软件名)
下载字体包
链接: https://pan.baidu.com/s/1DAHEwV7kqHPOt8fGZQR9vg 提取码: sarm 
建立字体文件夹
sudo mkdir /usr/share/fonts/wps-office
把字体包解压复制到这里面
然后终端输入依次
sudo mkfontscale
sudo mkfontdir
sudo fc-cache
5、安装网易云音乐
https://music.163.com/#/download
方法同上
6、安装蓝灯
https://github.com/getlantern/download/wiki
找到ubuntu版
方法同上
7、安装SMPLayer,一款支持多种视频格式播放器
sudo add-apt-repository ppa:rvm/smplayer 
sudo apt-get update 
sudo apt-get install smplayer smplayer-themes smplayer-skins 
安装录屏软件sudo apt-get install kazam 
或在软件中心搜索kazam
8、安装GDebi 软件包安装程序
软件中心下载
gdebi 允许您安装本地的 deb 包并解决和安装其依赖软件包
9、状态栏显示系统资源使用情况
推荐用system monitor extension
安装系统库：
终端
sudo apt-get install gir1.2-gtop-2.0 gir1.2-networkmanager-1.0  gir1.2-clutter-1.0
软件中心搜索system monitor extension
