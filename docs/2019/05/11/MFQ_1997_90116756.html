centOS 7修改默认的PHP版本

May 11, 2019

在linux服务器上搭建LAMP或者LNMP环境的时候，在安装PHP的过程中，如果直接使用yum install php 的命令就会默认安装php 5.4.*版本。若想安装php 7.*版本，有两种方法，如下：
（1）使用下载相应的PHP源码包进行编译安装（本次不进行讲解）。
在使用编译安装的时候，注意配置好各个路径。

（2）修改源，使用yum命令进行安装。
1）查看本机是否安装了PHP旧版本，若有则进行移除
1、yum list installed | grep php #查看安装信息
2、yum remove php #卸载PHP
3 、还需要卸载PHP的依赖环境 使用yum remove 软件名字 一个一个进行删除，注意各个依赖顺序。在删除的时候提出该软件依赖其他软件，则先卸载后者。以此类推。

2）更换RPM源

	    rpm -Uvh https://mirror.webtatic.com/yum/el7/epel-release.rpm
	    rpm -Uvh https://mirror.webtatic.com/yum/el7/webtatic-release.rpm

3）开始安装PHP版本
1、安装PHP 7.1.*

yum install -y php71w php71w-cli php71w-common php71w-devel php71w-embedded php71w-fpm php71w-gd php71w-mbstring php71w-mysqlnd php71w-opcache php71w-pdo php71w-xml php71w-ldap php71w-mcrypt

2、安装PHP 7.2

yum install -y php72w php72w-cli php72w-common php72w-devel php72w-embedded php72w-fpm php72w-gd php72w-mbstring php72w-mysqlnd php72w-opcache php72w-pdo php72w-xml php72w-ldap php72w-mcrypt

3查看PHP版本

php -v
