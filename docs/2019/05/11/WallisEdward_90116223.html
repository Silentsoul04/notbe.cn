java运行环境处理及简单程序

May 11, 2019

JDK(Java Development Kit)            Java开发工具包

        javac            Compiler（编译器）

JRE(Java Runtime Environment)    Java运行时环境

JVM(Java Virtual Machine)    Java虚拟机（一次编写，多次执行）

理解

 

简单的源代码格式：

 

public class 类名称{

            //java 的入口函数（方法）

            public static void main（String[] args）{

                    //写代码的位置

                    System.out.println("中文或英文");

    }

}

 

类名称要和文件名一样（规定）

类名称用英文，不要用中文和拼音，要用英文，每个单词的首字母大写（规范）

 

创建与运行方法：

 

先在文件夹中创建一个txt文件，然后确定文件名，将后缀改成.java，用notepad++打开，在其中可以编译

其中写入了，记得保存

public class HelloWorldApp{

    public static void main(String[] args){

        System.out.println("你好中国")；

    }

}

注意这里文件名要和类名相同（即class后的名称·，规范），文件夹写名字最好不要用中文

文件名按照 每个单词首字母大写的方式命名

 

接着在左下输入cmd命令，切换到源代码所在目录

 

cmd一开始是C盘，切到E盘，e: 回车

进到所在目录  cd JAVA学习\Tab键自动补齐

 

 

怎么确保自己进入到正确的文件夹   dir

 

 

 

清屏 cls

 

编译：       javac -encoding UTF-8 源代码的名称（HelloWorldApp.java）

编译后会生成一个HelloWorldApp.class的文件

 

几个概念：

 

面向对象  Object-Oriented

提供的服务放在.中

具体的实现放在.c中

类似C的封装性思想，区分了内和外

1.对象

            状态            （属性）变量

            行为            （方法）函数

2.类

        对象的蓝图       模板

 

java从入口的部分就体现面向对象（class）

 

类生成对象（class）        定义对象的蓝图
