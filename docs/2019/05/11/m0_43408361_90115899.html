3d 数学--矩阵

May 11, 2019

矩阵

在unity 中

  • Matrix4x4.SetRow 和 Matrix4x4.SetColumn 来设置一个 矩阵的某行和某列

  • Matrix4x4.GetRow 和 Matrix4x4.GetColumn 来得到一个 矩阵的某行和某列结果类型为vector4,index从0开始

  • 矩阵的加法，相同位置相加 [ 0,1] +[ 1,1]=[ 1, 2]

  • 矩阵的乘法 ， 8*[1,1]=[8,8]-------------
    -r n 的矩阵 只能和n * c的矩阵相乘 结果为rc的矩阵
    -第一个矩阵的第 i 行 点乘 第二个矩阵的第 j 列 为结果 矩阵的第 i 行第 j 列 (规则)

  • 在unity 中使用Matrix4x4.operator进行矩阵和矩阵的的乘法计算

  • 在unity 中使用Matrix4x4.identity 获取一个4*4的单位矩阵

  • 在unity 中使用Matrix4x4.isiientity判断一个矩阵是否为单位矩阵

  • 在unity 中使用Matrix4x4.zero获取一个4*4全为0的矩阵

转置矩阵

  • 将原矩阵的 i 列 换位 i 行 转换后的矩阵为转置矩阵
  • 在unity 中使用Matrix4x4.transpose获取矩阵的转置矩阵

逆矩阵

  • A B=BA=I // i 为单位矩阵(A ,B必须为方块矩阵 ) 那么AB互为逆矩阵
  • 如果A矩阵的行列式 不为0 那么 A矩阵可逆
    • |A| = 如果为2阶 直接对角相乘 后相减
    • 如果为3阶 — a11 * a11的代数余子式 - a12a12的代数余子式 +a13a13的代数余子式 (行列都可以计算)
      -代数余子式 ----为删除相应行列剩下矩阵 (a11 删除第一行第一列)

    4阶矩阵的逆矩阵求法

    • a11c 为a11的 代数余子式
    • 在unity 中使用Matrix4x4.inverse获取一个4*4矩阵的逆矩阵
