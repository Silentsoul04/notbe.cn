前端讲义34_javascript使用HTML元素事件属性

May 11, 2019

比如当用户点击一个HTML元素时，事件发生了可以执行JavaScript代码，简单点代码

<p onclick="this.innerHTML='谢谢!'">点击我!</p>

或者
<p onclick="changetext(this)">点我!</p>
<script>
    function changetext(id){
        id.innerHTML="谢谢!";
    }
</script>

也可执行JavaScript函数

<p>点击按钮执行showDate()函数。</p>
<p id="demo"></p>
<button onclick="showDate('demo')">显示时间</button>	
<script>
function showDate(id)
{
    document.getElementById(id).innerHTML=Date();
}
</script>

也可向HTML元素分配事件

<p>点击按钮执行showDate()函数。</p>
<button id="BtnDemo">显示时间</button>
<p id="demo"></p>
<script>
function showDate()
{
    document.getElementById("demo").innerHTML=Date();
}
document.getElementById("BtnDemo").onclick=function(){showDate()};
</script>

此处我们把showDate()函数分配给HTML元素 id="BtnDemo"

经常使用的几个事件的一般用途：
onchange 事件常用于输入字段的验证
onmouseover 和 onmouseout 事件可用于在鼠标指针移动到或离开元素时触发函数。
onmousedown、onmouseup 以及 onclick 事件是鼠标点击的全部过程。
当鼠标按钮被点击时，触发 onmousedown 事件，接着当鼠标按钮被松开时，会触发 onmouseup 事件，最后，当鼠标点击完成时，触发 onclick 事件。
当用户进入或离开页面时，会触发onload和onunload 事件，可用于检查访客的浏览器类型和版本，用来加载不同版本网页，还可检验浏览器Cookie可用状态等
