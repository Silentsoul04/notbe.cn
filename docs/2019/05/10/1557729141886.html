第二题

标题：五星填数

如【图1.png】的五星图案节点填上数字：1~12，除去7和11。
要求每条直线上数字和相等。

如图就是恰当的填法。

请你利用计算机搜索所有可能的填法有多少种。
注意：旋转或镜像后相同的算同一种填法。

请提交表示方案数目的整数，不要填写任何其它内容。

解法：给五角星上每个点编号然后按编号顺序从小到大填充好第一种排列方式(全排列的第一种)，然后用全排列函数进行判断是否符合五条线相等，至于去重，因为旋转起来的旋转前后以及镜像对折起来的对折前后都不会有重合的点，一种排列方案会有10种表示形式(旋转5种，每种旋转又有2种镜像对折，所以是5*2=10种)，答案也就是总的结果除以10。

代码：

#include<bits/stdc++.h>
using namespace std;
int num[10] = {1, 2, 3, 4, 5, 6, 8, 9, 10, 12};
int main() {
  int sum = 0;
  do {
    if(num[1]+num[2]+num[3]+num[4] == num[0]+num[2]+num[5]+num[8] &&
        num[1]+num[2]+num[3]+num[4] == num[0]+num[3]+num[6]+num[9] &&
        num[1]+num[2]+num[3]+num[4] == num[1]+num[5]+num[7]+num[9] &&
        num[1]+num[2]+num[3]+num[4] == num[4]+num[6]+num[7]+num[8]) sum++;
  } while (next_permutation(num, num+10));
  cout << sum / 10 << endl;
  return 0;
}
