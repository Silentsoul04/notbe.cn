Java-接口

May 10, 2019

一、接口 interface
定义：固定格式 public abstract 返回值类型 方法名字（参数列表）;
代码：

1 public interface AMyInterface {
2 public abstract void function();
3 }

二、实现类

1实现类实现
关键字implements
　　class 类 implements 接口{
　　　　重写接口中的方法
　　}
2多接口实现
　　实现类，重写全部的接口的所有抽象方法，才能建立实现类的对象
3.在继承类时还能多接口实现
　　public class D extends C implements A,B{
　　}

代码：
public class BMyInterfaceImpl implements AMyInterface,CMyInter{
　　public void function(){
　　　　System.out.println(“接口的实现类，重写接口中的方法”);
　　}

public void work(){
　　}
}

三、接口的成员变量与方法

接口成员：
1.成员变量
　　固定格式：public static final 数据类型 变量名 = 值;
　　　　public 权限
　　　　static 直接被类名调用
　　　　final 固定变量的值
2.成员方法
　　public abstract 返回值类型 方法名（参数列表）；
3.实现类实现接口，需要重写全部的抽象方法，创建实现类对象

代码：
1 public interface CMyInter {
2 public static final int x = 3;
3
4 public abstract void work();
5 }

四、接口与接口关系

接口与接口之间，继承关系

Java支持接口之间多继承

代码：

1 public interface DmoreInterExtends extends AMyInterface,CMyInter{
2 public abstract void D();
3 }

代码：D继承A,C；E实现类，重写A,C,D所有抽象方法

public class EAchieveD {

public void function(){}

public void work(){}

public void D(){}

}
