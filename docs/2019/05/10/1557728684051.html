python 解决汉诺塔问题---递归函数

May 10, 2019

汉诺塔：汉诺塔（又称河内塔）问题是源于印度一个古老传说的益智玩具。大梵天创造世界的时候做了三根金刚石柱子，在一根柱子上从下往上按照大小顺序摞着64片黄金圆盘。大梵天命令婆罗门把圆盘从下面开始按大小顺序重新摆放在另一根柱子上。并且规定，在小圆盘上不能放大圆盘，在三根柱子之间一次只能移动一个圆盘。


要解决这个问题,思路最重要,思路有了,问题就迎刃而解了.
对游戏玩法可以简单的分为三个步骤:
1,将前63个盘子从x移动到y上,确保大盘在小盘之下;
2.将最底下的第64个盘子从x移动到z上;
3.将y上的63个盘子移动到z上.
但是步骤1和3如何完成呢?
4.将x上的63个盘子借助z移动到y上,
5.将y上的63个盘子借助x移动到z上,
但是要完成4,5步需要以下三个步骤:
6,将前62个盘子从x移动到y上,确保大盘在小盘之下;
7.将最底下的第63个盘子从x移动到z上;
8.将y上的62个盘子移动到z上.
有没有发现123与678相似呢?
这就用到了递归函数了

num = 0


def hanoi(n, x, y, z):
    global num
    if n == 1:
        num += 1
        print("步数:", num, "将圆盘", n, "从", x, "移动到", z, )
    else:
        hanoi(n - 1, x, z, y)
        num += 1
        print("步数: %d 将圆盘 %s 从 %s 移动到 %s" % (num, n, x, z))
        hanoi(n - 1, y, z, x)


n = int(input("请输入汉诺塔层数:"))
a = 2 ** n - 1
print("预计需要步数", a)
hanoi(n, "x", "y", "z")
