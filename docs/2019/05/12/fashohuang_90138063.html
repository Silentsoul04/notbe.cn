图中由上到下是初级，中级，高级人员的设计思路。

初级人员用了三个div，一个整个的div下包含left和right两个div。left中是图片，right中是h6标题，p段落，span来表示时间。

中级人员省去了left，用定位来确定图片的位置。

高级人员只用了一个大的div，其余img，h6，p，都是用css来确定其布局。大的demo3的div同样包含了所有的元素，但是设置pic的margin为：-20px 0 0 -100px，将img移出了div的范围。

可以看出，高级人员的HTML显得十分地简洁。

写HTML的时候，不要过多地关注页面地布局，这些都可以用css来调整。只要注重页面本身的文本
结构，力求HTML结构简单化。简而言之，就是将结构和样式分离的思想。
