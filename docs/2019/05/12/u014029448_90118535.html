交叉编译生产jansson库方法笔记

May 12, 2019

1、下载jansson源码：

git clone https://github.com/akheron/jansson

 

3.2、安装autoreconf

apt-get install autoconf

 

3.3、安装LIBTOOL

apt-get install LIBTOOL

 

3.4、安装jansson源

    4.4.1、cd jansson

    3.4.2、生成编译工具

autoreconf -i     #反解析configure.ac文件，生成编译工具

    在生产的configure文件中做如下修改：

cross_compiling=yes #（修改）

CC=arm-hisiv300-linux-gcc #（增加）

3.4.3、执行configure生成makefile文件

./configure

make

make install

    库会安装在/usr/local/lib

export PKG_CONFIG_PATH=/usr/local/lib

 

3.5、生成的libjansson.a库路径：

jansson-master/src/.libs/libjansson.a
