抽象类

May 12, 2019

概念：

被这个单词abstract修饰符修饰这个类就是抽象类

抽象类写法格式：

public  abstract class say {
	
	
}

抽象函数定义：

public  abstract class say {
	
	public abstract void taecher() ;
}

使用它abstract修饰的方法是抽象方法，抽象方法没有方法体(方法体 就是{})

Public abstract void say();

抽象方法的访问修饰符不能是private ,因为不能被继承更不能被重写

如果抽象方法的访问修饰符是默认的，则要求子类必须和抽象父类在同一包下面并由子类重写父类的抽象方法

抽象方法的访问修饰符(public ,protected)时候，不做限制,子类重写后就可以抽象方法不能是静态的。

抽象类和抽象方法的关系

有抽象方法的类必须是一个抽象类。但抽象类中可以没有抽象方法

子类和抽象类的关系

    如果父类是一个抽象类，则子类必须重写父类所有的抽象方法，否则子类必须是一个抽象类

特点；

  1. 如果一个类定义了一个抽象的函数，那么该类必须是抽象的类
  2. 如果子类继承一个抽象的父类，那么子类必须实现父类中的所有抽象行为
  3. 抽象的类是不能直接被初始化的一般的抽象类都作为父类出现抽象类不能实例化，不能创建对象,
  4. 子类集成抽象的父类，可以不用重写父类的非抽象函数，因为子类通过继承可以得到行为
  5. 抽象类去继承抽象类，不需要实现抽象父类中的抽象函数。抽象的子类可以获取到抽象的父类中的抽象函数
  6. 抽象类中可以写非抽象的函数

 

一个类中如果有1个或多个的抽象方法，那么该类必须定义成抽象类

如果父类是抽象类，那么子类继承父类时可以选择以下两种方式：

1）将父类的抽象方法实现（方法的重写）

2）将子类继续定义成抽象类
