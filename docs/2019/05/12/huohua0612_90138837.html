js闭包--理解

May 12, 2019

最近通过《JavaScript高级程序设计》学习了js的闭包知识。在js中经常把闭包和匿名函数搞混。闭包就是指：有权访问另一个函数作用域中的变量函数。创建闭包的常见方式，就是在一个函数内部创建另一个函数。

下面来看一个小栗子：

		function anonymousdemo(propertyname){
			
			return function(object1,object2){
				var value1 = object1[propertyname];
				var value2 = object2[propertyname];
				
				if(value1 > value2){
					return 1;
				}else if(value1 < value2){
					return -1;
				}else{
					return 0;
				}
			}
		}
		var object1 = {"age":21};
		var object2 = {"age":24};
		//调用
		var anonymous = new anonymousdemo("age");
		var result = anonymous(object1,object2);
		alert(result);

理解：在上面的例子中，value1 = object1[prpertyname];value2 = object2[prpertyname];这两行是匿名函数中的代码,这两行代码访问了外部函数anonymous(propertyname){};中的变量propertyname。之所以能访问这个变量，是因为内部函数的作用域中包含了外部函数的活动对象（也可以说外部函数的作用域）。看下图所示：

 从图中看出，匿名函数包含了外部函数anonymousdemo()的作用域链，所以匿名函数可以使用anonymousdemo中的全部变量。

这就就是“闭包”的概念。
