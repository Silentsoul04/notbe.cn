如何强制修改电脑密码

May 12, 2019

通过cmd net user命令修改电脑密码-前提：没有使用微软账户登陆

首先win+r 打开cmd窗口

输入：net user          会出现你当前电脑的账户

输入命令：net user LW 1111  回车执行即可

net user ---是修改密码的命令

LW---是电脑账户的名字

1111--- 是要修改的密码

在你不知道电脑密码的情况下就可以更改，是不是很方便呢，哈哈

但是，如果是微软账户登陆的电脑怎么办呢？

执行这个命令就会报错

微软账户登陆只能修改微软账户的密码了，个人建议，为了自己电脑的安全，使用微软账户还是相对比较安全的，否则，电脑密码被改了，也很心累的哈。
