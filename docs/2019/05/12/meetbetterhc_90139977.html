什么是函数

函数是指一段在一起的，可以做某一件事的程序。也叫子程序、（OOP）方法 

函数的声明

function  函数名（）{

函数体

}

例如：

function demo () {  

alert('测试函数');

  } 

匿名函数：就是没有函数名的函数；例如：

setTimeout(function () {
    window.close();
},2000)


函数的调用

函数名（）

函数的参数

调用方法时需要传递给方法的一些信息

形参：形式上的参数，定义的，无值的

实参：实际传入的值

注意：定义参数列表时，不需要写类型，直接写名字就可以

          调用方法时，参数可传可不传，但实参与形参的位置是对应的（不能说跳过一个）

arguments

arguments是方法自带的一个数组，利用arguments做方法的重载

return

一些方法把处理的结果返回给调用的地方

注意：只能返回一个值

           当遇到return后马上停止
