nginx编译安装

May 12, 2019

1、去官网下载最新的stable版本的nginx二进制文件

wget http://nginx.org/download/nginx-1.16.0.tar.gz

2、然后解压，放到自己想要的地方

tar zxvf nginx-1.16.0.tar.gz
mv nginx-1.16.0 ../

3.执行 configure，指定nginx的安装目录，以及需要编译的模块（我这里是使用默认的），生成编译文件在objs目录下可以看到

./configure --prefix=/data/nginx.1.16.0

4.make，执行编译，生成nginx运行需要的文件，也是在objs目录下

make

5.将编译生成的文件，拷贝到nginx指定的安装目录

make install

6.启动nginx，进入到nginx的安装目录，然后到sbin目录下

cd /data/nginx.1.16.0/sbin
./nginx

7.然后nginx就启动成功了，一个最简单的nginx的helloword已经实现啦
