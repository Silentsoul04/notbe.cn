MySQL学习笔记

May 12, 2019

查询和更新指令构成了 SQL 的 DML 部分：

SELECT - 从数据库表中获取数据
UPDATE - 更新数据库表中的数据
DELETE - 从数据库表中删除数据
INSERT INTO - 向数据库表中插入数据

SQL 中最重要的 DDL 语句:

CREATE DATABASE - 创建新数据库
ALTER DATABASE - 修改数据库
CREATE TABLE - 创建新表
ALTER TABLE - 变更（改变）数据库表
DROP TABLE - 删除表
CREATE INDEX - 创建索引（搜索键）
DROP INDEX - 删除索引

SELECT语句

从表中选取数据

语法
1.SELECT 列名称 FROM 表名称
从表中选取数据。

2.SELECT * FROM 表名称
从表中选取所有的列
提示：星号（*）是选取所有列的快捷方式。

3.SELECT DISTINCT Company FROM Orders
列出不同（distinct）的值。
提示：关键词 DISTINCT 用于返回唯一不同的值。

4.SELECT 列名称 FROM 表名称 WHERE 列 运算符 值
有条件地从表中选取数据
例：SELECT * FROM Persons WHERE City=‘Beijing’
注：引号的使用
SQL 使用单引号来环绕文本值（大部分数据库系统也接受双引号）。如果是数值，请不要使用引号。

5.ORDER BY 语句
ASC顺序；DESC逆序
例1：以字母顺序显示公司名称
SELECT Company, OrderNumber FROM Orders ORDER BY Company, OrderNumber

例2：同时满足以字母顺序显示公司名称（Company），并以数字顺序显示顺序号（OrderNumber）
SELECT Company, OrderNumber FROM Orders ORDER BY Company, OrderNumber

例3：以逆字母顺序显示公司名称
SELECT Company, OrderNumber FROM Orders ORDER BY Company DESC

例4：以逆字母顺序显示公司名称，并以数字顺序显示顺序号
SELECT Company, OrderNumber FROM Orders ORDER BY Company DESC, OrderNumber ASC

INSERT INTO 语句

用于向表格中插入新的行。
语法
INSERT INTO 表名称 VALUES (值1, 值2,…)
我们也可以指定所要插入数据的列：
INSERT INTO table_name (列1, 列2,…) VALUES (值1, 值2,…)

Update 语句

用于修改表中的数据
语法
UPDATE 表名称 SET 列名称 = 新值 WHERE 列名称 = 某值

DELETE 语句

DELETE 语句用于删除表中的行。
语法
1.DELETE FROM 表名称 WHERE 列名称 = 值
2.删除所有行
DELETE FROM table_name
或者：
DELETE * FROM table_name

TOP 子句

用于规定要返回的记录的数目
语法
SELECT TOP number|percent column_name(s)
FROM table_name LIMIT number
例：选择Persons头两条记录
SELECT TOP 2 * FROM Persons
例：从的 “Persons” 表中选取 50% 的记录。
SELECT TOP 50 PERCENT * FROM Persons

LIKE 操作符

在 WHERE 子句中搜索列中的指定模式
语法
SELECT column_name(s)
FROM table_name
WHERE column_name LIKE pattern
例1：从 “Persons” 表中选取居住在以 “g” 结尾的城市里的人
SELECT * FROM Persons
WHERE City LIKE ‘%g’
例2：从 “Persons” 表中选取居住在以 “g” 结尾的城市里的人
SELECT * FROM Persons
WHERE City LIKE ‘%g’
例3.从 “Persons” 表中选取居住在包含 “lon” 的城市里的人
SELECT * FROM Persons
WHERE City LIKE ‘%lon%’
例4.从 “Persons” 表中选取居住在不包含 “lon” 的城市里的人
SELECT * FROM Persons
WHERE City NOT LIKE ‘%lon%’
例5查找“user_info”表中含有name字符串的名字
SELECT * FROM user_info WHERE name like ‘%"+name+"%’
