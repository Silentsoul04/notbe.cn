二十三章 – XML

XML

变量 数组 对象 对象数组 集合 文件/IO流 XML 数据库

一、理解XML的概念个优势

XML （eXtensible（扩展 ） Markup（标记，收藏） Language）可扩展的标记语言

文件格式： .xml 目前采用1.0版本

W3C(世界万维网联盟)世界性的中立技术标准组织 ，给一系列的内容制定标准，包含着XML

作用：

1.用来存储数据

2.数据传输格式

3.应用程序配置文件

4.AJAX的基石

二、会编写格式良好的XML文档

XML由两部分组成：

1.声明 放在XML文档的第一行 用来标注文档的一个配置格式

version采用的版本规范 encoding编码

2.主体 的描述内容

主体的描述内容由标签（元素/节点/标记）组成

一定有且只有一个根标签

XML的标签

语法：

<标签名    属性名 = “属性值”...> 标签内容 </标签名>

标签的分类：

<books></books> 双标签：由开放性标签和闭合性标签组成的 中间可以添加内容 标签内可以添加属性信息

<books/> 单标签/自动闭合标签  只能添加一些属性信息

三、理解XML中特殊字符的处理方式

有时需要在文件内存储符号 但是在XML中有些地方不允许出现< > “” 等，我们可以用转义解决

&lt;      <
&gt;      >
&apos;    '
&quot;    "
&amp;     &

<![CDATA[无需考虑特殊符号的任意内容]]>

四、了解解析器和DOM树节点构造

DOM： Document Object Moder （文档对象模型）

DOM解析：他是将XML内容加载到内存中生成对应的文档结构树（DOM树）

在文档结构树中 ，它是由一系列的节点构造的 ，接地吗有不同的类型

例如：标签转换为元素节点

注释转换为了注释节点

标签内容转换为文本节点

标签属性转换为属性节点
