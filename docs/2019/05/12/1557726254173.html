无重复字符的最长子串

May 12, 2019

LeetCode的题：
给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入: “abcabcbb”
输出: 3
解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。
示例 2:

输入: “bbbbb”
输出: 1
解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。
示例 3:

输入: “pwwkew”
输出: 3
解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。
请注意，你的答案必须是 子串 的长度，“pwke” 是一个子序列，不是子串。

思路：一开始想暴力解题，但是看到别人说用滑动窗口时间复杂度和空间复杂度都很低，于是看网上的思路，但是很多大佬只有代码没有详细解释，我在此贴出我的理解。

首先很多同学说用map来写，这里由于是char型，只有有限个字符类型，所以开了一个flag数组来记录扫描过程中是否出现了相同的字符，如果没有，则将相应的值置为true，并且right右移,如果出现过，则将flag置为false,left右移，这样left和right就像一个窗口一样框住了最长的无重复子串。

class Solution {
public:
int lengthOfLongestSubstring(string s) {
int left=0;
int right=-1;
int max=0;
bool flag[256]={false};
while(left<s.length()&&right+1<s.length()){

    //循环条件我不知道为什么有right+1<s.length,我写的是right<s.length,可是在leetcode里进不了循环，在其他编译器可以，有同学知道的话可以留言告诉我
        if(flag[s[right+1]]==false){
            flag[s[++right]]=true;//s[i]表示一个字符，而flag[s[i]]就表示相应ascii码位置处，类似于一个字典
        }
        else{
            flag[s[left++]]=false;
        }
        if(right-left+1>max)
            max=right-left+1;
    }
    return max;
}

};
