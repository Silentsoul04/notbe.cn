多人同演＋即时互动Virtual Cast化身VR虚拟角色做直播

May 12, 2019

原文链接：

https://www.pcmarket.com.hk/2018/04/16/%E5%A4%9A%E4%BA%BA%E5%90%8C%E6%BC%94%E5%8D%B3%E6%99%82%E4%BA%92%E5%8B%95-virtual-cast%E5%8C%96%E8%BA%ABvr%E8%99%9B%E6%93%AC%E8%A7%92%E8%89%B2%E5%81%9A%E7%9B%B4%E6%92%AD/

最近的动画中偶然都会看到节目主持和嘉宾以虚拟角色的姿态，在不同的地方跨地域在虚拟空间的录像室进行互动直播。现在，只要你有部VR Ready的电脑，加上一套HTC Vive VR装置，要开间虚拟直播室就轻而易举了，因为日本推出了一个免费的虚拟角色直播服务「Virtual Cast」。

Virtual Cast可以将直播主的肢体动作忠实呈现在直播中

这个由营运日本著名视频平台Nico Video的DWANGO，和北海道一间VR技术研发公司infinite Loop共同推出的虚拟空间直播及交流服务「Virtual Cast」，直播主只要戴上HTV Vive拿起手掣，就可以化身成虚拟角色进行直播，又或者加入别人的直播里共演。由于是VR角色，所以比起以前介绍的只改变样子的手机直播App更加彻底，更有互动性。

Virtual Cast可以让异地的人在同一个虚拟直播室中出现共演互动

Virtual Cast本身没有直播功能，直播主需要透过《Niconico Live Encoder》或《Open Broadcaster Studio》的画面撷取机能来撷取显示在PC上的画面才能直播出去。虽然听起来有点阳春，不过它也提供了一些直播上很有用的功能。

首先直播主可以在虚拟环境表演，比起在家中狗窦直播卖相好得多。直播主可以汇入自己的角色，随意调校镜头角度，又可以以虚拟画笔在虚拟直播室画画写字，还可以将PC桌面的画面在虚拟空间中显示出来，适合用来直播游戏画面。

软件附有很多虚拟道具让直播主与共演者进行互动

互动功能方面，平台提供加入其他人直播的「凸机能」，而「凸」进来的嘉宾可以跟直播主一同互动。平台又提供一些虚拟物件让虚拟直播室的人互相互动，例如用胶锤来搞个VR《鬼马双星》节目都没有问题。而观众的留言也可以以对话框或字体物件方式出现在虚拟空间，让直播主拿来把玩。不过留言变物件机能似乎只适用于在官方营运的「Niconico生放送」中使用，没有说明其他直播平台都能用。

Niconico生放送的观众留言的话，会在虚拟直播室以文字物件或留言框方式出现，虚拟直播室的人可以跟留言「互动」。

Virtual Cast支持的直播平台包括Niconico生放送、YouTube Live、Twitcast、Twitch、OPENREC.tv。而启动直播之后也同时会在Virtual Cast网站发布。大家可以在官网免费下载，个人或商用都可以使用，不过就不能用来进行未经许可的产品宣传、选举相关、宗教相关和异性交友相关的直播。
