May 12, 2019

1.background

  • background-color /* 背景色 */
  • background-image /* 背景图片 */
  • background-repeat /* 背景图平铺方式 */

2.border

  • border-width /* 边框宽度 */
  • border-style /*边框样式 */
  • border-color /*边框颜色 */

    PS：只有颜色可以被省略

3.font

  • font-style：italic /* 斜体 */
  • font-weight：bold /* 加粗 */
  • font-family：arial，sans-seril /* 字体种类 */
  • font-size：20px /* 字体大小 */
  • line-height：35px /*行高 */（可用于设置字体垂直居中）

    PS：可省略一些属性，但需要注意顺序问题
