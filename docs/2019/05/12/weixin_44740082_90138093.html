leetcode 231. 2的幂(python)

May 12, 2019

题目链接

题目描述：

给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

示例 1:

输入: 1
输出: true
解释: 20 = 1
示例 2:

输入: 16
输出: true
解释: 24 = 16
示例 3:

输入: 218
输出: false

解题思路：
用n对2取余，余数不为0说明不是2的幂，返回false，余数为0，则将n除以2，继续进行取余。

class Solution:
    def isPowerOfTwo(self, n: int) -> bool:
        if n==0:
            return False
        while n!=1:
            if n%2!=0:
                return False
            n=n//2
        return True
