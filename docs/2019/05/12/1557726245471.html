什么是 vim？

Vim是从 vi 发展出来的一个文本编辑器。代码补完、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用。

简单的来说， vi 是老式的字处理器，不过功能已经很齐全了，但是还是有可以进步的地方。vim 则可以说是程序开发者的一项很好用的工具。连 vim 的官方网站 (http://www.vim.org) 自己也说 vim 是一个程序开发工具而不是文字处理软件。

几个比较实用的命令

插入，删除，查找，复制，粘贴和替换。

20o    #向下插入20行
20dd   #向下删除20行
:n1,n2/word1/word2/g #在n1到n2行之间查找
:1,$s/word1/word2/gc   #从首行到末行查找word1用word2替换并询问用户
/word      # 向下查找word
?word   # 向上查找word
nyy  #n 为数字。复制光标所在的向下 n 行，例如 20yy 则是复制 20 行(常用)
p,P  #小写p 为将已复制的数据在光标下一行粘贴，大写P 则为贴在游标上一行！
