javascript 代码运行顺序说明

May 12, 2019

一. 步骤说明

  1. 预解析  --- 把所有的函数定义提前，所有的变量声明提前，变量的赋值不提前。
  2. 执行  --- 从上到下执行  （setTimeout，setInterval,ajax中的回调函数，事件中的函数需要触发执行）。

二. 示例

原代码：

<!DOCTYPE html>
<html>
<head>
	<title>代码运行说明</title>
</head>
<body>

</body>
<script type="text/javascript">
	
	fn();//此时调用未定义的方法：fn，可以输出。
	fn1();//此时调用未定义的方法：fn1，会报错，不能输出。

	function fn(){
		console.log(1111);
	}

	var fn1 = function(){
		console.log(2222);
	};

</script>
</html>

运行阶段代码：


<!DOCTYPE html>
<html>
<head>
	<title>代码运行说明</title>
</head>
<body>

</body>
<script type="text/javascript">
	function fn(){
		console.log(1111);
	}；
	var fn1；


	
	fn();//此时调用未定义的方法：fn，可以输出。
	fn1();//此时调用未定义的方法：fn1，会报错，不能输出。

	fn1 = function(){
		console.log(2222);
	};

</script>
