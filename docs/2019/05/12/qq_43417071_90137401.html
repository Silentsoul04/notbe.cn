L1-003 个位数统计 (15 分)

给定一个 k 位整数 N
请编写程序统计每种不同的个位数字出现的次数。例如：给定 N=100311，则有 2 个 0，3 个 1，和 1 个 3。
输入格式：
每个输入包含 1 个测试用例，即一个不超过 1000 位的正整数 N。
输出格式：
对 N 中每一种不同的个位数字，以 D:M 的格式在一行中输出该位数字 D 及其在 N 中出现的次数 M。要求按 D 的升序输出。
输入样例：
100311
输出样例：
0:2
1:3
3:1

思路：

  1. 输入正整数N，并将每个数字存储到A数组中
  2. 新定义一个数组B，数组B中存放0-9
  3. 将A数组与B数组相比较，得出各个数字的个数（出现次数为0的不输出）

难点：输入的整数数字位数范围在1000位以内，说明不能用普通的变量直接定义（如int或long int）
只能将输入的数字一个个存储到数组中，这里所使用的方法是从键盘输入字符，将字符转换位数字
例如：字符’7’->数字7 ，只需进行操作'7' - '0';
所以上述的输入数字，在键盘其实本质是输入字符（‘0’ - ‘9’），在字符输入的同时转化成数字并存储到数组A中，可通过scanf()函数实现：
例如：

int i = 0;
while(scanf("%d", &m) != EOF){
	a[i] = m;
	i++;
}

代码如下：

#include<stdio.h>
#include<stdlib.h>

int main(void){
    char ch[1000], t;
    int a[1000];
    int k = 0, i, j, cishu;
    
    while((t = getchar() )!= '\n'){
        a[k] = t - '0';
        k++;
    }
    
    for (i = 0; i <= 9; i++) {
		cishu = 0;
		for (j = 0; j < k; j++) {
			if (a[j] == i) {
				cishu++;
			}
		}
		if(chishu != 0)
			printf("%d:%d\n", i, cishu);
	}
    return 0;
}
