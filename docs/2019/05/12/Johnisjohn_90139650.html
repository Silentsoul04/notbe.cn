tensorflow代码实战3_Variable使用_简单例子

May 12, 2019

说明：资料来源于莫烦老师教程：

https://www.youtube.com/watch?v=jGxK7gfglrI&list=PLXO45tsB95cKI5AIlf5TxxFPzb-0zeVZ8&index=11

 

一、代码

import tensorflow as tf

state = tf.Variable(0,name='counter')
#print(state.name) #打印出：counter:0
one = tf.constant(1)

new_value = tf.add(state,one)#变量加常量还是变量
update = tf.assign(state,new_value)#把new_value的值加载到state上

init =  tf.initialize_all_variables()# 必须要初始化
with tf.Session() as sess:
	sess.run(init)
	for _ in range(3):
		sess.run(update)
		print(sess.run(state))#必须要run一下，才能出现state的结果
