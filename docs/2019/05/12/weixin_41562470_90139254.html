Ubuntu基本操作

     常用命令

 

1、ls:查看目录下有什么

2、ls -al:查看目录属性

3、pwd:查看当前位置

4、whoami:查看当前登录的用户

5、sudo passwd:给root用户修改密码

6、echo '' > sou.txt:清空文件内容

7、tar xzvf:解压文件

(x 解压z gzip压缩格式v 显示详细信息f file.tar 的参数. man一下.tar

zxvf 目标压缩包z 将要解压的包压缩格式为 gzip格式x 要执行的是解压动作（与压缩 c 相对）

v 显示解压过程中的信息f 代表file ，固定要带的)

然后，login进入系统之后，开始进行用户界面的安装。首先输入如下命令：

 

sudo apt-get install xinit。安装时的界面如下所示。

 

上述安装完毕之后，再安装环境管理器。本人亲测安装的是GNOME。使用如下命令安装：

sudo apt-get install gdm

 

然后，安装桌面环境。本人亲测安装的是KUbuntu。安装命令如下：

 

sudo apt-get install kubuntu-desktop/ubuntu-desktop->这两种界面不一样

或

------------------------------------------------------------------------------------------------

如果你只想安装必要的桌面管理软件而不想安装一些诸如 Evolution 和 OpenOffice 的不需要的应用软件，可使用下面的命令：

 

sudo apt-get install --no-install-recommends ubuntu-desktop

 

安装轻量级桌面 xfce 使用下面的命令：

 

sudo apt-get install xubuntu-desktop

 

安装 KDE 桌面，使用下面的命令：

 

sudo apt-get install kubuntu-desktop

---------------------------------------------------------------------------------

上述安装完毕之后，直接重启。重启完成后，再进入系统，便是图形界面了。

 

sudo dpkg -l | grep ubuntu-desktop:查看是否有桌面程序

 

sodu apt-get purge ubuntu-desktop:删除软件

 

 

安装中文输入法（要在在英文系统下安装）

1、

安装IBus框架，在终端输入以下命令：sudo apt-get install ibus ibus-clutter ibus-gtk ibus-gtk3 ibus-qt4

2、启动IBus框架，在终端输入命令：im-switch -s ibus。

3、安装拼音引擎，在终端输入命令：sudo apt-get install ibus-pinyin，即可安装拼音。

4、设置IBus框架，在终端输入命令：ibus-setup，在出现的窗口选择

Input Method，之后选择拼音输入法即可。

注意：Shift键可切换中英文
