输出DNA螺旋

标题Description

瓜皮集齐了7颗龙珠，准备许愿，但是他却不知道召唤神龙的咒语，他看见箱子里还有一张纸，上面写着：如何你能画出DNA的双螺旋结构，你就会知道召唤神龙的咒语，瓜皮一看这是最后一步了，又厚着脸皮，死缠烂打的让你帮他解决难题，你快被他烦死了，于是决定帮他最后一次。

Input

输入包含多组测试数据。第一个整数N（N<=15）,N表示组数，每组数据包含两个整数a,b。a表示一个单位的DNA串的行数，a为奇数且 3<=a<=39。b表示重复度(1<=b<=20)。

Output

输出DNA的形状，每组输出间有一空行。

Sample Input

2
3 1
5 4

Sample Output

HINT

#include<stdio.h>
int main(){
	int n,i,j,k,t;
	scanf("%d",&n);
	int a[n],b[n];
	for(i=0;i<n;i++){
		scanf("%d%d",&a[i],&b[i]);
	} 
	for(i=0;i<n;i++){
		for(j=1;j<=b[i];j++){
			for(t=1;t<a[i];t++){
				  for(k=1;k<=a[i];k++){
					 if(t= =k||t+k==(a[i]+1)){
						 printf("X"); 
					 } 
					 else printf(" ");
				   }
				   printf("\n");
		      }
		}
		for(k=1;k<=a[i];k++){
			if(t= =k||t+k==(a[i]+1)){
				printf("X");
			}
			else printf(" ");
		}
		printf("\n");	
		printf("\n");
	}
	return 0;
}
