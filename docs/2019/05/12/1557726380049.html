写给自己的话

May 12, 2019

编程也已经快接近三年了，由小白到现在这个算的上入行的从业者吧，开始学习的c++ ，后来入公司却是wpf + .net ，感谢刚进公司的陈师傅，第一次接触了wpf图形框架，感觉好神奇，好在有好的项目，就是图像处理的，说是图像处理，其实大部分集中在对原始数据的数字处理，期间接触了 opengl  Aforge 等好的开源知识集合....在项目勉强写好之后，就要将一些算法集成到com组件中，好在c++的知识没忘记，踩过了很多坑，终于写了一个调用com+wpf的软件.....可惜，由于个人的一些不成熟想法，提了离职..感觉挺不好意思的.后来进入了一家新创业公司，由于c++使用经验比较少，所以新公司，推介的是 c++ 写底层DLL ，+ wpf写上层应用的框架结构来开发，这个需要踩很多坑，接着就大量的使用了opencv ，以及opengl 的东西了（其实很多东西都是看资料和百度的），同时对opencv 的算法进行cuda加速，（惭愧，虽然当时想写，但是毕竟一个人要从底层写到上层，所以，cuda写的比较少，主要是经理写的）。接着陆续使用osg  Qt 方面的东西了，到目前为止，接触了大半年的时间了。长路漫漫，加油，总感觉自己能力不强，唉，学这些东西，需要仔细琢磨和试错多次才能明白一些知识点的使用，心累..现在平时很多积累写在了oneNote上面，现在，是时候将以前的很多知识写成博客的形式，留给自己观看.(wpf   Qt   opencv  opengl  osg pcl agorge 等等.)
