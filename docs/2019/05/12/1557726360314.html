JAVA数据结构与算法（二）队列

队列是一种比较特殊的线性结构。它只允许在表的前端（front）进行删除操作，而在表的后端（rear）进行插入操作。进行插入操作的端称为队尾，进行删除操作的端称为队头。
队列中最先插入的元素也将最先被删除，对应的最后插入的元素将最后被删除。因此队列又称为“先进先出”（FIFO—first in first out）的线性表，与栈(FILO-first in last out)刚好相反。
一个重要公式：队列长度=（front+rear+length）%length

下面我们来实现一个队列

public class MyQueue<T> {
private int size;
private int rear;//队尾
private int front;//队首
private Object[] data;
private int lengh;

public MyQueue(){    //也可以自己传一个参数长度，不过要进行判断
	this.size=10;
	this.rear=-1;
	this.front=0;
	data=new Object[size];
	this.lengh=10;
}

添加队列

public boolean add(T t){
	if(size==lengh){
		return false;
	}else{
		rear++;
		data[rear]=t;
		return true;
	}
}

删除队列

public T  offer(){
	if(size==0){
		return null;
	}else{
		T t=(T) data[front++];
		if(front==rear){
			front=0;
			rear=-1;
		}
		return t;
	}
}

看一下队列头

public boolean peek(){
	if(size==0){
		return false;
	}else{
		T t=(T) data[front];
		System.out.println(t);
		return true;
	}
}
