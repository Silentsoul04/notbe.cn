MyEclipse代码因为JDK版本不同导致报错解决方法

May 12, 2019

JDK未对应在MyEclipse中引用导致代码报错

很多小伙伴在初学JAVA时，不知道怎么在MyEclipse(JAVA开发工具)中引用JDK(JAVA开发工具包)
导致项目代码因JDK版本原因报错，并且无法有效的解决报错，那么本篇将介绍如何的引用JDK来解决代码报错。

一般在安装Java时，在它安装的目录中有一个JDK的文件夹(jdk1.7.0_80)
我的JAVA版本为1.7，那么它对应的JDK版本也为1.7，目录为C:\Program Files\Java\jdk1.7.0_80

(PS:安装目录自己找找,一般都在自己安装Java的盘里面，一般格式为jdk.1.x.x_xx)

那么接下开始引用步骤

  1. 在MyEclipse的主页面中,上面有一排导航栏
  2. 需要引用JDK的话，我们选中Window
  3. 弹出下拉框，选中Preferences(自定义设置)

顺序为(MyEclipse - Window - Preferences - Java - Installed JREs-Add-(默认)-Next-Directory-JDK文件夹位置-确定-Finish-勾选添加的JDK-OK)

1 在弹出Preferences窗口中，选择到Java
2. 点击Java,选中Installed JREs(这里只是选中，并不点击打开)

3.弹出Add JRE窗口,默认为Standard VM,这里直接点击Next进入到下一步

4.点击Directory，在进入到下一步
5. 选中jdk所在文件夹位置(我的为:C:\Program Files\Java\jdk1.7.0_80)，点击确定
6. 点击确定后返回至Add JRE，点击Finish

7.勾选刚刚添加的jdk.1.7.0_80，然后点击OK

最后在新增Java Project时，选中Use a project specific JRE即可。

以上就是解决当前JAVA代码因JDK版本不同而引起报错的解决方法
