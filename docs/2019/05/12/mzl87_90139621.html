WSL Arch Linux 已在 Microsoft Store 上可用

May 12, 2019

在前两天的 Build 2019 大会上，微软宣布 WSL 2 将采用全新的架构，使用真正的 Linux 内核。微软对于 Linux 等开源项目的全面拥抱给开发者带来了很多惊喜，继在微软商店（ Microsoft Store）提供 Ubuntu、SUSE Linux 和 Fedora 等发行版后，深受欢迎的 Linux 发行版 Arch Linux 也于近日在微软商店上架了。

获取地址：https://www.microsoft.com/zh-cn/p/wsl-arch-linux/9p2s3qr49vnn

当然，如果希望在 Windows 10 中运行 Linux 发行版，需要先在系统中启用 Windows Subsystem for Linux（WSL）设置。

原文地址：https://www.oschina.net/news/106630/wsl-arch-linux-to-the-ms-store
