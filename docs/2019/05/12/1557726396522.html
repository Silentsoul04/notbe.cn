leetcode131. 分割回文串

May 12, 2019

题目

给定一个字符串 s，将 s 分割成一些子串，使每个子串都是回文串。

返回 s 所有可能的分割方案。

示例:

输入: “aab”
输出: [ [“aa”,“b”], [“a”,“a”,“b”] ]

解题思路

判断字符串s从[0,i)是否是回文串：
如果是，则放入list，然后继续深度遍历s[i,s.len]，在深度结束时移除。
如果不是，则不做处理。

代码如下

class Solution {
    public List<List<String>> partition(String s) {
        List<List<String>> res = new ArrayList<>();
        List<String> list = new ArrayList<>();
        dfs(res, list, s);
        return res;
    }

    public void dfs(List<List<String>> res, List<String> list, String s) {
        //如果s为空，则说明所有的子串都已经加入list,此时是深度的出头
        if (s.equals("")) {
            res.add(new ArrayList<>(list));
            return;
        }
        char[] chars = s.toCharArray();
        int len = chars.length;
        //从[0,1)判断到[0,len)
        for (int i = 1; i <= len; i++) {
            if (helper(chars, i)) {
                list.add(s.substring(0,i));
                dfs(res, list, s.substring(i));
                list.remove(list.size()-1);
            }
        }
    }
    //判断chars[0]到chars[len]是否是回文串
    public boolean helper(char[] chars, int len) {
        for (int i = 0; i < len / 2; i++) {
            if (chars[i] != chars[len - i - 1])
                return false;
        }
        return true;
    }
}
