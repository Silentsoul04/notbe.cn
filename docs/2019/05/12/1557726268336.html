Python利用adb命令操作安卓手机批量打电话

May 12, 2019

python版本3.7
编辑器 pycharm
电脑配置完成adb
adb 官网下载：http://adbshell.com/upload/adb.zip
数据线连接手机，并打开usb调试，勾选usb（安全设置）允许通过usb调试修改权限或模拟点击

代码如下
#操作前先在设置里打开power键可以结束通话按钮，否则会导致代码报错 from time import sleep import os import random import sys numbers=[] with open('C:/Users/sun/Desktop/phone.txt','r') as f: for line in f: numbers.append(list(line.strip('\n').split(','))) for number in numbers: remain=len(numbers)-numbers.index(number)-1 remain = str(remain) #拨打电话 call = os.popen('adb shell am start -a android.intent.action.CALL -d tel:{}'.format(number)) teltime=random.randint(20, 36) sj="随机等待" print("还剩" + remain + "个"+sj+str(teltime)+"秒") sleep(teltime) #挂断电话 Hangup = os.popen('adb shell input keyevent 6') sleep(2)
注意事项：时间设置太短太速度，号码容易被电信公司拉黑。
