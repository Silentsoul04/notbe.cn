GPS导航(7)：系统框架

May 12, 2019

系统模块

系统 OS 层使用 Linux3.6 系统，底层调用多种驱动，其中包括：USB 键盘、串口驱动，触摸屏驱动等。软件层 API 使用 QT5 自由版提供的API 。应用层为电子地图应用程序。

主要功能

  • 地图显示：显示所在位置周边地理信息，可对当前进行缩放，拖动，自动切换地图。
  • 路径导航： 自定义起点终点，利用最优路径算法计算出 一条最优路径，并且在地图上面显示。
  • 历史记录：系统会自动保存用户最近使用的 10 条路径导航记录，以方便用户在再次需要时浏览。
  • 卫星定位：在地图对应位置显示当前所在位置。
