Vue-----二，vue-cli开发

May 12, 2019

一，安装vue-cli

1、先安装node.js http://nodejs.cn
先下载，然后跟着node.js一路默认点下去安装完成即可。
2、安装nvm https://github.com/coreybutler/nvm-windows/releases上下载nvm-setuo.zip

3、第三步：修改settings.txt
在你安装的目录下找到settings.txt文件，打开后加上
node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

4、打开cmd命令行，下载npm和cnpm
5、输入vue init webpack [项目名称]创建一个项目
6、输入npm run dev启动项目
7、根据给出的地址（8080）进入创建的项目首页
8、将项目放入vscode中，之后每次修改保存后不用刷新，直接进入首页就自动更新了

二、开始开发

1、这个项目中的index.html是主要呈现的页面，src/components这个路径里的.vue文件是web的一个个组件，main.js是进行引入组件的js文件。

2、在组件中：
①需要注意在模板<template></template>中只能有一个子元素，不然就会报错。如果要添加多个子元素就需要先加一个子元素咋template下，然后再在这个新的子元素下面添加需要的子元素。
②在script的export default中写data,这里的data应该是一个函数而不是一个对象了。
data:function(){
return {},
}
可以简写为 data(){
return{}
}
③在methods中，最后一个函数后面不能写 ,（逗号）

④全局样式与局部样式
在子组件的style中，若加了scoped则写的是局部样式，只能在子组件中起效，在别的组件中无法生效
如果去掉scoped则是全局样式了

一般都趋向于添加一个scoped，这样可以独立模板
