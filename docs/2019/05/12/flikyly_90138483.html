不完美破解四川电信中兴B860A高安机顶盒

May 12, 2019

目标

家里有个闲置的中兴B860A盒子，又有下载些电视剧什么的，在线看你懂的，又有广告又要钱。但是客厅的电视接的是电信原始的直播盒子，访问不了家里局域网的资源，所以想用这个闲置的盒子播放局域网上自己下载的电视剧，同时用某奇异，某讯和某酷看看带广告的在线视频，就有了这次折腾。

失败的经历

最开始的想法是能不开盒就不开盒，结果按网上的方法各种连接，开ADB是开了，但是各种软件，卡刷，线刷，试了个遍，都不行，最后发现，这个盒子是网上传说的无解的四川电信高安版。也就是说到现在网上也没有免费的完美破解方案。付费的嘛，那就没意思了。

IBHR-ANDROIDSTBGA-068为高安
IBHR-ANDROIDSTB-068为非高安高安
最后参考某论坛的贴子： 最简单办法破解B860A高安版（高手请飘过），达到最初设定的目标

安装过程

开盒子，接TTL线

盗一张网上的图：

下载需要的APK包

当贝桌面，当贝市场（因为没有破解，所以有安装APK的限制，所以想用当贝市场安装APP的时候就这样了，反正安不了。）

所以要什么APP，只能自己下载，再开盖子用命令安装。
下载其它要用的APP，比如某奇异，某讯和某酷等。
将上面的APP拷到TF卡或者U盘上，再插入盒子。

putty命令

在windows环境下，运行putty，设置串口参数：115200，同时，接上盒子，并把盒子接入局域网，再接上电视，OK。开电。

此时putty开始跑码，如果没跑码，对换上面图中的rxd线和txd线；如果还不行，说明USB转ttl模块有问题，先检查这个模块。

输入命令

start adbd                          注释：打开adb
cd /storage/external_storage/sda1   注释：进入U盘
cp *.apk /data/app                  注释：复制apk
cd /data/app                        注释：进入盒子应用目录
ls                                  注释：看看文件在不在
am start com.dangbei.tvlauncher     注释：安装所有apk，并启动当贝桌面

等一会儿，电视上应该就出来当贝桌面，并已经安装好所有APP了。好了，我的目标达到了。

缺点

能力有限，因为没有破解APK的安装限制，所以并不能用当贝市场安装新的APP，期待后面有办法吧！

我们的目标是玩儿盒子，别让盒子玩儿！
