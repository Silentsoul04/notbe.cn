LeetCode 27.移除元素

May 12, 2019

给定一个数组 nums 和一个值 val，你需原地移除所有数值等于 val 的元素，返回移除后数组的新长度。

不要使用额外的数组空间，你必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

示例 1:

给定 nums = [3,2,2,3], val = 3,

函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。

你不需要考虑数组中超出新长度后面的元素。

示例 2:

给定 nums = [0,1,2,2,3,0,4,2], val = 2,

函数应该返回新的长度 5，并且 nums 中的前五个元素为 0,1,3,0,4。
注意这五个元素可为任意顺序。

你不需要考虑数组中超出新长度后面的元素。
注意这五个元素可为任意顺序。

你不需要考虑数组中超出新长度后面的元素。

说明:

为什么返回数值是整数，但输出的答案是数组呢?

请注意，输入数组是以“引用”方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

你可以想象内部操作如下:

// nums 是以“引用”方式传递的。也就是说，不对实参作任何拷贝
int len = removeElement(nums, val);

// 在函数里修改输入数组对于调用者是可见的。
// 根据你的函数返回的长度, 它会打印出数组中该长度范围内的所有元素。
for (int i = 0; i < len; i++) {
    print(nums[i]);
}

思路1：

题目要求去除指定的元素，并且不考虑数组的元素顺序，我们可以考虑交换的方式，利用双指针将所有等于val的元素同不等于val的交换。当 nums[j] 与给定的值相等时，递增 j 以跳过该元素。只要 nums[j]≠valnums[j] ，我们就复制 nums[j] 到 nums[i]并同时递增两个索引。重复这一过程，直到 j 到达数组的末尾，该数组的新长度为 i。

代码：

class Solution {
    public int removeElement(int[] nums, int val) {
        int len=0;
        for(int i=0;i<nums.length;i++){
            if(nums[i]!=val){
                nums[len++]=nums[i];
            }
        }
    return len;
    }
}

思路2：

依旧是交换的思想，定义一个头指针 i 和尾指针 j ，如果 nums[i] == val，并且 nums[j] != val，我们就将nums[j]同nums[i]交换，然后递增 i，递减 j ，当 i==j 时，所有等于val的元素都被交换到 i 后面，返回 i+1 即可。这个方法有点像快速排序。

如图：

尾指针 j 找到第一个不等于val的的元素，然后如果头指针 i 指向的元素等于val的话，则交换，nums[i] != val 则i++，直到找到一个等于val的元素：

交换完后 i 自增1， j--，找到一个不等于val的元素，判断nums[i]是否等于val，如果等于则交换：

最后当i>=j时，所有等于val的元素都被交换到 i 后面，返回 i+1 即可。

代码：

class Solution {
    public int removeElement(int[] nums, int val) {
        if(nums.length==0)    return 0;
        
        int i = 0;
	int j = nums.length - 1;
	int temp;

	while (i < j) {
		if (nums[j] == val) {
			j--;
			continue;
		}
		if (nums[i] == val) {
			temp = nums[i];
			nums[i] = nums[j];
			nums[j] = temp;
		}
		i++;
	}

        if(nums[i]==val)    i--;
	return i + 1;
    }
}
