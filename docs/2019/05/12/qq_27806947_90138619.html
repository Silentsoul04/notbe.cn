推测是：pyqt5安装的版本太多啦：

解决方案

卸载全部的pyqt5，然后重新安装一个。

pip3 uninstall pyqt5

如下图：有两个版本的

再安装一个：

sudo pip3 install pyqt5==5.10

安装后，启动spyder3成功！！
