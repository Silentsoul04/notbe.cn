Scanner

什么是键盘录入？
我们前期在使用变量空间时，都是先声明好变量空间并赋值后在进行操作。也就是说在程序运行前，所有的变量空间中的数据都已经确定。而有时我们需要在程序运行的过程中，通过键盘录入的方式去给变量空间进行赋值，就好像在软件的使用过程中我们手动通过键盘去录入帐号密码一样。而想要完成这个功能，就需要使用java中的键盘录入技术来完成。
Java中的键盘录入是通过Scanner类来完成，因为整个录入过程是面向对象的知识点，我们此时只需要能够记住代码使用即可。
书写代码时的注意事项:
1.创建键盘录入器
2.在录入赋值的过程中需要注意，录入的数据类型需要和接收数据的空间类型一致.
比如:
(1)通过键盘录入器调用next()方法给String类型空间赋值
(2)通过键盘录入器调用nextInt()方法给int类型空间赋值
(3)通过键盘录入器调用nextDouble()方法给double类型空间赋值
(4)通过键盘录入器调用next().charAt(0)方法给char类型空间赋值
具体代码如下:

// 导入键盘录入工具包
import java.util.Scanner;
// 演示键盘录入
public class Demo {
    public static void main(String[] args) {
        // 创建键盘录入器
        Scanner r = new Scanner(System.in);

        System.out.println("下面演示，通过键盘录入器给变量空间使用键盘录入的方式赋值");
        System.out.println("在录入赋值的过程中需要注意，录入的数据类型需要和接收数据的空间类型一致。");

        // 通过键盘录入器调用next()方法给String类型空间赋值
        System.out.print("请输入姓名："); // 键盘录入提示
        String name =r.next(); // 录入的数据会赋值给name变量空间
        System.out.println("我的名字叫：  " + name); // 输出打印键盘录入的数据


        System.out.print("请输入年龄：");// 键盘录入提示
        int age =r.nextInt();// 录入的数据会赋值给age变量空间
        System.out.println("我今年： " + age + "岁！");// 输出打印键盘录入的数据

        // 通过键盘录入器调用nextDouble()方法给double类型空间赋值
        System.out.print("请输入身高：");// 键盘录入提示
        double height =r.nextDouble(); // 录入的数据会赋值给height变量空间
        System.out.println("我的身高是： " + height); // 输出打印键盘录入的数据

        // 通过键盘录入器调用next().charAt(0)方法给char类型空间赋值
        System.out.print("请输入性别：");// 键盘录入提示
        char sex = r.next().charAt(0); // 录入的数据会赋值给sex变量空间//(0)代表的是录入的第几位的值.
        System.out.println("我的性别是： " + sex); // 输出打印键盘录入的数据
    }
}
