汇编常用dos指令

May 12, 2019

1号指令：

mov ah，1

int 21h

作用：从屏幕中读取当个字符放入al

2号指令：

mov ah, 2

int 21h

作用：将dl对应的ASCII码输出到屏幕

9号指令：

lea dx， buf

mov ah，9

int 21h

作用：将以$结尾的字符串buf输出到屏幕上。

0a号指令：

buf db 10, 0, 10 dup('$')

mov dx，offset buf

mov ah,0ah

int 21h

作用：将屏幕上的字符串读入到缓冲区dx，从第三个位置开始读入。
