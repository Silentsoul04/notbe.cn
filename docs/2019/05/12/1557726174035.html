halcon 发布注意事项

May 12, 2019

在编译和发布Halcon过程中，需要完成以下两步：

1.Halcon代码编译调试。加入Halcon头文件和在Release/Debug 包中加入相应的DLL文件。参考下面的文章：

https://www.cnblogs.com/love6tao/p/6574798.html   halcon发布

2.Halcon 代码的发布。当发布到未安装Halcon的电脑上时，会报出错误代码“Halcon error #2036 :could not find license.dat in operator”，或“error = 5207”无法找到help文件。处理方法如下：

（1）打包原机器的help文件夹，实现方法如下

新建一个bat文件 和一个lic文件夹， lic文件夹下面是MVTec/halcon/help ，help文件夹里面的东西去安装halcon中 help文件夹里面找到复制即可

cd /d %~dp0
SET regpath=%cd%
xcopy %cd%\lic "C:\Program Files" /e

 运行上述 bat文件 ，就可以将lic下面的文件夹 拷贝到 C:\Program Files\下了

（2）配置环境变量：

变量名：HALCONROOT

变量值：C:\Program Files\MVTec\HALCON（保持原电脑一致）。
