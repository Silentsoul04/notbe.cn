SpringBoot入门之快速搭建SpringBoot项目

May 12, 2019

一.简介
    网上关于SpringBoot的介绍已经很详细了这里不多说了，直接开始搭建。

二.开发环境

    JDK:1.8

    Intellij IDEA:2018.1.6 

    Maven:3.6.1

一.开始搭建

    搭建有两种方式：

    第一种方式：通过官网创建

    1.访问官网https://start.spring.io/

点击生成项目后会直接将项目下载到你本地，找到下载的文件解压它

2.解压完毕后，打开idea，选择File-->Open

找到你解压的文件，一路Next，完毕之后，把依赖下载一下

依赖下载完毕之后，直接运行就好了。

 

第二种方式：通过idea直接创建

1.打开idea，选择File -> new ->Project

2.输入project的基本信息

3.更新Maven依赖，完毕后直接运行

 

三.测试是否成功

1.在DemoApplication的同级目录下，新建一个controller包，此包下创建一个HelloController，如下图

注意：此HelloController的位置一定要在DemoApplication的同包或者同包的子集目录下，否则是扫描不到的

package com.example.demo.controller;

import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @RequestMapping("/hello")
    public String hello() {
        return "Hello SpringBoot!";
    }
}

2.创建完成后，启动项目

如果显示成这样则说明启动正常，注意tomcat的端口默认的是8080

3.打开浏览器，在项目中输入：localhost:8080/hello

创建完毕！
