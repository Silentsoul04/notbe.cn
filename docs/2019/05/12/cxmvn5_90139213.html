在手机浏览器中点击链接跳转到手机QQ聊天界面的实现方式

May 12, 2019

有个朋友问我，QQ好友发说说，他点击了查看全文-然后就会自动跳转到添加好友页面，无论是安卓手机端还是PC手机端，都行。 
这是怎么做的。

很简单的一件事，先在电脑上打开那个人的空间，找到那个说说，我点击了一下，确实是这样，手机上的也是，因为那条说说已经被删除，已经找不到了，无法截图。

 

QQ的
try {
    //加人的
    String url = "mqqwpa://im/chat?chat_type=wpa&uin=1203415047";//uin是发送过去的qq号码
    //添加群的 
    //String url = "mqqwpa://im/chat?chat_type=group&uin=670943305&version=1"; //uin是发送过去的qq群号码
    startActivity(new Intent(Intent.ACTION_VIEW, Uri.parse(url)));
} catch (Exception e) {
    e.printStackTrace();
}
