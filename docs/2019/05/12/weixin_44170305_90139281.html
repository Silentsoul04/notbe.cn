YTU 3085 指针调用函数实例

题目描述

该题目主要考察大家对通过指针调用函数的掌握情况，给出两个数，输出它们之间的最大值与最小值。

 

部分代码已给出，请补全缺少的部分。


#include <stdlib.h>

int max(int a,int b)
{
    return a>b?a:b;
}

int min(int a,int b)
{
    return a>b?b:a;
}

int main()
{
    int a,b;
    int (*p)(int,int),(*q)(int,int);
    scanf("%d%d",&a,&b);

/* 缺少的部分 */


    printf("max:%d min:%d\n",p(a,b),q(a,b));
    return 0;
}

输入

输入数据只有一行，包含两个整数a与b。

输出

输出只有一行，即这两个数的最大值与最小值，中间用空格分隔。

样例输入

copy

3 4

样例输出

max:4 min:3

提示

 

只需提交程序缺少部分。

p=max;
    q=min;
