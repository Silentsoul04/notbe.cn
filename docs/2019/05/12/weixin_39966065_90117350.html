一：变量

1.变量的声明与赋值
Java：两个阶段可分步进行
int i; i = 1 
scala:两个操作必须同时
val i = 1 

2.scala中的不可变（final，val，mutable） 与 java中的不可变（final）
（1）final：对变量进行修饰后，该变量的引用不能改变，但是其值可以更改
（2）scala中对不可变的定义进行了扩展
1》final
    final 在拥有继承关系的两个类中，表示所修饰的 field 或者 method 不可以被覆盖
2》val
    运用在变量中与final类似
    在拥有继承关系的两个类中，子类可以覆盖父类的field和method，拥有各自所独有的特性
3》mutable
mutable/immutable 关心的是对象（即可）的状态值
例子：    
    var map1 = Map("Tom"->21,"Tony"->23);
    val map2 = Map("Tom"->21,"Tony"->23);

二：函数

1.return 的使用
scala中不推荐使用 return 
2.函数调用时：在不传递参数的情况下，可以省略括号
3.函数声明时：方法块儿 {}  与  “=” 的互用
4.scala 中 能够将一个函数当成简单的变量进行传递，它是处于一等公民的地位
5.main 测试方法书写：
    java中：public static void main(String[] args){}
    scala中：用object类
    （1）原生方式：
    object HelloWorld {
      def main(args: Array[String]) {
        println("Hello World!!!")
      }
    }

（2）继承App类： 
object HelloWorld extends App{
 if(args.length > 0 ) println("hello, " + args(0))
 else println("Hello World !!!")
}
 

 

三：数据类型

（1）scala的数据类型自动判断：
方法，变量的声明

限制范围：
1.迭代函数
2.带名函数中的匿名函数声明时
def greeting(name: String) = {
  def sayHello(name: String):String = {
    return "Hello, " + name
  }
  sayHello(name)
}

（2）基本数据类型与与之照应的引用数据类型的转换：
scala中通过隐式转换可直接使用加强类型的方法
例子： 1.to(10)  // int ——》 RichInt
java中必须显式

 

四：集合

整体：Iterable、Seq、Set、Map
结构划分： Iterable是所有集合trait的根trai
           Seq下包含了Range、ArrayBuffer、List等子trait
Tuple 与 Map 类似，比Map更加强大的键值功能

1.运算符本身当成函数去使用
 例子：1.+(1)

例子： 1.to(10) ————》 1 to 10

4.yield 在 scala 的作用 
与for 进行配合，而进行值的返回
之前的用法汇总：： 
（1）for(i<-1 to 10) yield i    会返回一个 Vector 的集合 

（2）val b2 = for (ele <- b) yield ele * ele

（3）val keepIndexes = for (i <- 0 until a.length ) yield {
  if (a(i) < 0) foundFirstNegative = true
  i
}

作用：与for联用，使（for-yield）表达式有返回值
      与if 一样

七：极致的简写

（1）匿名函数当成参数传递的简写规则：
(参数名称：参数类型) => {函数体}     ————》 在就是各种省略 
(参数名称：参数类型) => 函数体
(参数名称) => 函数体
(参数名称) => 函数体
例子：
def greeting(func: (String) => Unit, name: String) { func(name) }
greeting((name: String) => println("Hello, " + name), "leo")
greeting((name) => println("Hello, " + name), "leo")
greeting(name => println("Hello, " + name), "leo")

（2）“_”特殊符号的使用
例子：
def triple(func: (Int) => Int)=func(3) 
func: (Int) => Int  ：  定义了我要传递 传入参数入Int类型，返回值为Int类型的 函数
func(3)  是 triple函数的返回值

变形；
=》triple(num=>num)
=》3*triple(num=>num)
=》triple(3 * _)

（3）函数调用时：在不传递参数的情况下，可以省略括号
（4）调用apply函数时，省略 apply
"Hello World"(6)，实际上是"Hello World".apply(6)
