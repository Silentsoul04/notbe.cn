ImportError: DLL load failed: 找不到指定的模块。

May 12, 2019

1） 针对numpy和pandas进行的报错

我也遇到过这类相似的问题，我是通过一下几行代码解决问题。

    pip uninstall pandas
    pip uninstall numpy
    pip install pandas
    pip install numpy

这样问题就解决了。
