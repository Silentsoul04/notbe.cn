May 12, 2019

c#学习之旅

相关规范说明

1、C# 是大小写敏感的。
2、所有的语句和表达式必须以分号（;）结尾。
3、程序的执行从 Main 方法开始。
4、与 Java 不同的是，文件名可以不同于类的名称

从一个小例子开始：输出hello world

using System;  // 系统
namespace HelloWorldApplication  // 进行命名空间的声明
{
   class HelloWorld  // 进行类的声明
   {
      static void Main(string[] args)  // 定义main方法，是所有c#程序的入口
      {
         /* 我的第一个 C# 程序*/
         Console.WriteLine("Hello World"); // 指定行为，进行输出打印
         // WriteLine 是一个定义在 System 命名空间中的 Console 类的一个方法。该语句会在屏幕上显示消息 "Hello, World!"。
         Console.ReadKey(); // 使程序会等待一个按键的动作，防止程序从 Visual Studio .NET 启动时屏幕会快速运行并关闭。
      }
   }
}

关键字说明

1、using

2、class

class关键字通常进行类的声明

3、注释说明

/*...*/ 多行注释说明

数据类型

在C#中数据类型分为以下三种

值类型(value types)
引用类型(references type)
指针类型（pointer types）
值类型

可以直接进行数据的存储，具体数值可分为（整型，字符，浮点…）

引用类型

核心在于的引用，不包含存储在变量中的实际数据
内置的引用类型有：object, dynamic, string

object(对象类型)

所有对象类型的终极基类，可以被分配任何其他类型，

dynamic(动态类型)

可以存储任何类型的值在动态类型变量中
与object类型类似，不同在于object类型变量的类型检查发生在编译时，而dynamic类型的值在运行时

String(字符串类型)

运行你给对象分配任何类型的字符串，是从object派生而来
字符串可以通过两种形式，进行分配：““和@符号

指针类型（Pointer typers）

指针类型变量，存储另一种类型的类型地址
