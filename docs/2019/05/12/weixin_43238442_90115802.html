May 12, 2019

梯度下降

梯度下降是一种适用面很广的优化参数方法，理由偏导数对多元函数的参数迭代更新使得损失函数取得最小值。

损失函数-Loss Function

损失函数一般都使用方差来评判模型的好坏。

理论基础

比如有一个线性预测函数： f ( x ) = b + w x f(x)=b+wx f(x)=b+wx
我们就用方差作为Loss Function： L ( f ) = ∑ i = 1 n ( y i − f ( x i ) ) 2 L(f)=\sum_{i=1}^{n}(y_{i}-f(x_{i}))^2 L(f)=i=1∑n​(yi​−f(xi​))2
让这个函数取地最小值的参数 b , w b,w b,w 就是我们想要的参数，所以最终可以把Loss Function看作以原函数系数为自变量的函数： L ( b , w ) = ∑ i = 1 n ( y i − ( b + w x i ) ) 2 L(b,w)=\sum_{i=1}^{n}(y_{i}-(b+wx_{i}))^2 L(b,w)=i=1∑n​(yi​−(b+wxi​))2
这样就成了求极值的问题。

具体步骤
  • 随机选择起点，即随机选择预测函数的参数（Loss Function的自变量）；
  • 选择学习率 η \eta η；
    在这里插入一个小技巧：在做梯度下降时寻找适合的学习率很关键；建议将不同learning rate的训练结果可视化在一张图上

    这样就知道自己的学习率是不是合适的。
  • 计算Loss Function在这一点的偏导数 ∂ L ∂ w \frac{\partial L}{\partial w} ∂w∂L​ 和 ∂ L ∂ b \frac{\partial L}{\partial b} ∂b∂L​；
  • 更新参数 w 1 = w 0 − η ∂ L ∂ w w_{1}=w_{0}-\eta \frac{\partial L}{\partial w} w1​=w0​−η∂w∂L​
    b 1 = b 0 − η ∂ L ∂ b b_{1}=b_{0}-\eta \frac{\partial L}{\partial b} b1​=b0​−η∂b∂L​
    在微积分中梯度 ∇ f \nabla f ∇f 就是函数值增加最快的方向，其反方向就是函数值下降最快的方向; 我们使得Loss Function自变量向这个方向移动，即将预测函数Hypothesis Function的参数向这个方向移动，反复迭代就能得到最小值/局部最小值。这就是梯度下降。
  • Loss Function的求导
    对于每一个Loss Function我们都需要对其每个变量求偏导这样也能使得我们的程序更加简单：
    L ( b , w ) = ∑ i = 1 n ( y i − ( b + w x i ) ) 2 L(b,w)=\sum_{i=1}^{n}(y_{i}-(b+wx_{i}))^2 L(b,w)=i=1∑n​(yi​−(b+wxi​))2
    ∂ L ∂ w = 2 ∑ i = 1 n ( y i − ( b + w x i ) ) ( − x i ) \frac{\partial L}{\partial w}=2\sum_{i=1}^{n}(y_{i}-(b+wx_{i}))(-x_{i}) ∂w∂L​=2i=1∑n​(yi​−(b+wxi​))(−xi​)
    ∂ L ∂ b = 2 ∑ i = 1 n ( y i − ( b + w x i ) ) ( − 1 ) \frac{\partial L}{\partial b}=2\sum_{i=1}^{n}(y_{i}-(b+wx_{i}))(-1) ∂b∂L​=2i=1∑n​(yi​−(b+wxi​))(−1)
    一定要注意在这里 x i x_{i} xi​ 是一个参数。
数学解析

L = ( b , w ) L=(b,w) L=(b,w) 的梯度gradient就是 ∇ L \nabla L ∇L：
∇ L = [ ∂ L ∂ b ∂ L ∂ w ] \nabla L= \left[ \begin{matrix} \frac{\partial L}{\partial b}\\ \\ \frac{\partial L}{\partial w} \\ \end{matrix} \right] ∇L=⎣⎡​∂b∂L​∂w∂L​​⎦⎤​
是一个指向 L ( b , w ) L(b,w) L(b,w) 上升最快的方向的vector，我们就让函数向其反方向移动，移动的幅度就是学习率。更新方式如下：
[ b n w n ] = [ b n − 1 w n − 1 ] − η [ ∂ L ∂ b ∂ L ∂ w ] \left[ \begin{matrix} b_{n}\\ \\ w_n \\ \end{matrix} \right] = \left[ \begin{matrix} b_{n-1}\\ \\ w_{n-1}\\ \end{matrix} \right] -\eta\left[ \begin{matrix} \frac{\partial L}{\partial b}\\ \\ \frac{\partial L}{\partial w} \\ \end{matrix} \right] ⎣⎡​bn​wn​​⎦⎤​=⎣⎡​bn−1​wn−1​​⎦⎤​−η⎣⎡​∂b∂L​∂w∂L​​⎦⎤​

存在的问题
  • 全局最小值和局部最小值
  • 函数也会在saddle point收敛

探索模型的方法

  • 选择其他的模型

我们不知道线性的Hypothesis Function是不是最佳的，某个feature的高次方是否也有着很好的拟合效果，所以我们就要试一试其他的模型。无论预测函数的是怎样的，Gradient Descent都一样可以使用。

  • 选择其他的feature用于Hypothesis Function
    我们也不知道其他的feature是不是也对预测值有着影响；就可以胡乱作图看一看是不是有什么关系，然后redesign我们的model；

  • 采用更多的数据
    随着数据量的增加很多hidden的情况就会接连出现；

正则化 Regularization

通过对某个参数增加惩罚项来控制这个参数的大小:
L ( b , w ) = ∑ i = 1 n ( y i − ( b + w x i ) ) 2 + λ ∑ i = 1 n w i 2 L(b,w)=\sum_{i=1}^{n}(y_{i}-(b+wx_{i}))^2+\lambda\sum_{i=1}^{n}w_i^2 L(b,w)=i=1∑n​(yi​−(b+wxi​))2+λi=1∑n​wi2​
最后一个 w i w_i wi​的平方和就是用于正则化的惩罚项，这样会使得Loss Function偏向于选择 w w w比较小的结果，我们最后得到的预测函数就会比较smooth。 λ \lambda λ是需要我们自己确定的，到底多大比较合适需要自己确定。
一般的，我们认为平滑的结果更有可能是正确的，震荡强烈的预测函数虽然也有可能是正确的。
