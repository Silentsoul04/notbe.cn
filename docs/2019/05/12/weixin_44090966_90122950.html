Part01:jQuery省市联动案例

1、需求：

  • 在左侧下拉框选择省份，右侧下拉框就会出现对应省份的城市

2、jQuery遍历：

  • 第一种方式：jQuery对象.each(function(index,ele ){
    //this 遍历后的结果，是js对象
    //ele 遍历后的结果，是js对象
    //index 索引
    })
//获取所有checkbox的value值并依次打印
$(function(){
    //绑定点击事件
    $("#btn001").click(function () {
        //属性选择器获取对象，遍历
             $("[type=checkbox]").each(function (index,ele) {
                 //打印value值,转换为jQuery对象调用val()方法获取值
                 alert($(this).val());
                 //打印索引值和value值，调用js对象的value属性获取值
                 alert(index+":"+ele.value);
             })
         })
});
  • 第二种方式：$.each(jQuery对象，function(index,ele ){
    //this 遍历后的结果，是js对象
    //ele 遍历后的结果，是js对象
    //index 索引
    })
//获取所有不可见的input元素的value值并依次打印
$.each($(“[input:hidden]"),function (index,ele) {
   // alert(this.value)
    alert(index+":"+ele.value)
})

3、jQuery对value、html、text对操作

  • value：
    • 获取：jQuery对象.val( )
    • 赋值：jQuery对象.val(“值”)
  • html：
    • 获取：jQuery对象.html( )
    • 赋值：jQuery对象.html(“内容”)
    • 如：$(“div”).html( );//获取div标签里面的内容
  • text：
    • 获取：jQuery对象.html( )
    • 赋值：jQuery对象.html(“内容”)
  • html和text的区别：
    • 赋值：
      • html会把标签直接解析到浏览器页面上
      • text会把标签作为文本的形式展示
    • 获取值：
      • html会把html标签获取出来
      • text只会获取文本，不会获取html标签

4、文档操作

  • 内部插入：
    • append：a.append©；//把c插入到a内部后面位置
    • prepend：a.prepend©；//把c插入到a内部前面位置
    • appendTo：a.appendTo©；//把a插入到c内部后面位置
    • prependTo：a.prepend©；//把a插入到c内部前面位置
  • 外部插入：
    • after：a.after©；//把c插入到a外部的后面位置
    • before：a.before©；//把c插入到a外部的前面位置
    • insertAfter：a.insertAfter©；//把a插入到c外部的后面位置
    • insertBefore：a.insertBefore©；//把a插入到c外部的前面位置
  • 删除节点：
    • empty：清空，保留父节点
    • remove：移除 ，连同父节点一起删除

5、省市联动实现的主要代码：

//获取省份对象，绑定change事件
$("#pro").change(function () {
    //清空市的下拉选择框,保留父节点
    $("#city").empty();
    //获取省份的value值
    var index = $("#pro").val();
    //遍历市的数组
    $(arr[index]).each(function () {
        //把遍历后的市添加到市的下拉选择框中,追加
        $("#city").append("<option>"+this+"</option>");
    })
})

Part02:jQuery左右选中案例

1、需求：

  • 点击第一个按钮，把左侧选中的第一个数据移到右侧
  • 点击第二个按钮，把左侧选中的所有数据移到右侧
  • 点击第三个按钮，把左侧所有的数据都移到右侧
  • 后三个按钮功能类似

    2、HTML中select下拉选择框的属性：
  • multiple=“true”;//可以让下拉选择框选择多个
  • size=“10”;//设置下拉选择框显示的个数

3、jQuery表单对象选择器：

  • :enabled：可用的
  • :disable：不可用的
  • :checked：选中的（单选框和多选框）
  • :selected：选中的（下拉选择框）

4、鼠标悬停：

  • jQuery对象.hover（function( ){ 方法体1(鼠标移入）},function( ){ 方法体2(鼠标移出)})

5、事件切换：用于绑定两个或多个处理器函数，以响应被选元素的轮流click事件

  • jQuery对象.toggle(function( ){方法体1},function( ){方法体2});//点击奇数次实现第一个方法，点击偶数次实现第二个方法

6、实现代码：

$(function () {
    //给第一个按钮绑定click点击事件
             $("#toRight1").click(function () {
                 //把左侧选中的第一个数据移动到右侧
                 $("#left option:selected:first").appendTo($("#right"));
             });
             //给第二个按钮绑定click事件
             $("#toRight2").click(function () {
                 //把左侧选中的所有数据移到到右侧
                 $("#left option:selected").appendTo($("#right"));
             });
             //给第三个按钮绑定click事件
             $("#toRight3").click(function () {
                 //把左侧所有数据移到右侧
                 $("#left option").appendTo($("#right"));
             });
             //给第四个按钮绑定click事件
             $("#toLeft1").click(function () {
                 //把右侧选中到第一个数据移到到左侧
                 $("#right option:selected:first").appendTo($("#left"));
             });
             //给第五个按钮绑定click事件
             $("#toLeft2").click(function () {
                 $("#right option:selected").appendTo($("#left"))
             });
             //给第六个按钮绑定click事件
             $("#toLeft3").click(function () {
                 $("#right option").appendTo($("#left"))
             })
 })

Part03:jQuery表单校验

1、jQuery校验器插件的使用：

  • 导入jquery.js文件
  • 导入jquery-validate.js文件
  • 导入messages_zh.js国际化文件
  • 校验格式：
    (在页面加载成功之后)表单对象.validate({
    rules:{ }, //校验规则
    messages:{ } //自定义提示信息
    })
  • rules写法：
    要校验的name属性:{
    校验器1:取值,
    校验器2:取值
    }
  • messages写法：
    要校验的name属性:{
    校验器1:”自定义提示信息1",
    校验器2:”自定义提示信息2"
    }
  • 常用校验器
    • required （必填校验(是否可以为空)）
      • 取值：true，false
    • number （数字校验）
      • 取值：true，false
    • min (最小值校验）
      • 取值：数字
    • max (最大值校验）
      • 取值：数字
    • range (数值区间校验）
      • 取值：[最小值，最大值]
    • maxLength|minLength|rangeLength (最大长度／最小长度／长度区间）
      • 取值：数字|数字|[最小值，最大值]
    • email (简单邮箱校验）
      • 取值：email
    • equalTo (重复性校验）
      • 取值：jQuery对象
   $(function () {
          $("#formid").validate({
              rules:{
                  //用户名
                  username:{
                      //必填校验
                      required:true
                  },
                  //密码
                  password:{
                      required:true,
                      //数值校验
                      number:true
                  },
                  //确认密码
                  repassword:{
                      required:true,
                      //重复性校验
                      equalTo:$("[name=password]")
                  }
              },
              messages:{
                  username:{
                      required:"用户名不能为空"
                  },
                  password:{
                      required:"密码不能为空",
                      number:"密码必须为数字"
                  },
                  repassword:{
                      required:"确认密码不能为空",
                      equalTo:"两次密码不一致"
                  }
              }
          })
      })
  • 自定义校验器：
    • 格式：
$.validator.addMethod(“校验器名称”,function(value,ele,param){
   //校验规则
    //value:用户录入的值
    //ele:当前录入的值所在的js对象
    //param:校验器的取值
    //return true 表示符合规则
    //return false 表示不符合规则，默认
 },”提示信息”)
  • 自定义身份证格式校验器
$.validator.addMethod("cardLength",function (value,ele,param) {
    //判断用户录入的值是否为15位或者18位
