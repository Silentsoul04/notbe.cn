文件

文件模式

模式 含义
ios::in 打开文件读取数据
ios::out 打开文件写入数据
ios::app 在文件末尾写入数据
ios::ate 定位到文件末尾
ios::trunc 清空文件原有内容
ios::binary 打开二进制文件

与ifstream文件输入流对象关联的文件，默认情况下以ios::in模式打开。如果文件不存在，则打开失败。

inFile.open("example.txt");
//等价于
inFile.open("example.txt",ios::in);

与ofstream文件输出流对象关联的文件，默认情况下以ios::out | ios::trunc模式打开。如果文件不存在，则创建新文件；如果文件存在，则清空文件原有内容。

outFile.open("example.txt");
//等价于
outFile.open("example.txt",ios::out|ios::trunc);

关闭文件 :

inFile.close();

输入输出

get从文件中读取一个字符，put向文件中写入一个字符。

char ch;
inFile.get(ch);
outFile.put(ch);

getline函数用于从文件中读入一行字符。

string line;
getline(inFile,line);
outFile<<line<<endl;

vector

1.实例化的方式

vector<int>vec1;
vector<int>vec2(vec1);//拷贝构造vec2
int intArray[]={0,1,2,3,4};
vector<int>vec3(intArray,intArray+5);//用已存在的数组创建vec3

2.常用方法

成员 作用
size() 返回向量中的元素个数
front() 返回向量第一个元素
back() 返回向量最后一个元素
push_back() 在向量尾部插入一个元素
pop_back() 在向量尾部删除一个元素
insert() 插入一个元素并返回新元素的位置
erase() 删除某一个元素并返回下一个元素的位置

3.遍历方式

for(int i=0; i<vec.size(); i++)
	cout<< vec[i] << " ";
//使用基于范围的for语句
for(auto value: vec)
	cout<< vec[i] << " ";
//使用迭代器
for(auto p=vec.begin(); p!=vec.end(); ++p)
	cout<< *p << " ";

容器的选择

  1. 如果你需要高效的随即存取，而不在乎插入和删除的效率，使用vector
  2. 如果你需要大量的插入和删除，而不关心随即存取，则应使用list
  3. 如果你需要随即存取，而且关心两端数据的插入和删除，则应使用deque

容器适配器

容器适配器让一种已存在的容器类型采用另一种不同的抽象类型的工作方式实现。

stack

默认情况下采用deque容器作为基础容器.。
初始化：

stack<int>intStack;
//采用vector容器
srack<int, vector<int>>intStack;

用法如下

成员函数 含义
empty() 判断是否为空栈
size() 返回栈中元素数量
push() 添加元素到栈顶
pop() 从栈顶删除元素
top() 获取栈顶元素

priority_queue

优先队列中每个元素都有一个优先级，所有元素根据优先级的某种排序规则自动排序，默认情况下是降序。优先级最高的元素最先被访问。
默认情况下采用vector容器。
初始化：

priority_queue<int>intPriorityQueue;

用法与queue相同。

迭代器

Iterator模式是运用于聚合对象的一种模式，通过运用该模式，使得我们可以在不知道对象内部表示的情况下，按照一定顺序（由iterator提供的方法）访问聚合对象中的各个元素。

string迭代器

//正向迭代器
string::iterator ite;
//对象被const修饰时
string::const_iterator ite;
//正向迭代
//begin()返回一个迭代器,指向字符串的第一个元素
//end()指向字符串最后一个字符的下一个位置
for(ite=s.begin();s!=s.end();++s)
	cout<<*ite;
//或者
ite=s.begin();
while(ite!=s.end())
	cout<<*(s++);

//逆向迭代器
string::reverse_iterator ite;
//对象被const修饰时
string::const_reverse_iterator ite;
//rbegin()指向字符串的最后一个字符
//rend()指向字符串第一个字符的前一个位置
//注意逆向迭代器移动方向相反，所以从尾部仍然通过++来移动
for(ite=s.rbegin();s!=s.rend();++s)
	cout<<*ite;
//或者
ite=s.rbegin();
while(ite!=s.rend())
	cout<<*(s++);
