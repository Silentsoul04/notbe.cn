什么是MySQL Installer？

MySQL Installer是一种msi文件，可以一键安装、集成、配置多种MySQL产品，这种向导式服务使得繁琐的数据库搭建变得简单。目前MySQL Installer只支持windows系统。
如果单单是为了满足个人学习需求，安装配置MySQL Server就行了，但对于要进行数据库服务的公司而言MySQL Server远远不够。
过去的我们都在为下载哪些MySQL产品而发愁，为怎么安装而发愁，为怎么配置管理而发愁。“自从有了MySQL Installer，妈妈再也不用担心我的搭建”。

MySQL Installer官网下载

输入下列网址，进入MySQL官网：
https://www.mysql.com/
依次点击DOWNLOADS，Community，MySQL on Windows，MySQL Installer

然后将页面拉到最底下，有两个软件供下载，我们选择下载第二个，版本8.0.16，大小373.4M。（第一个只用于web）
点击第二个Download。

不用注册，直接点最底下的No thanks，just start my download

选择下载路径，点击下载。
下载完成，到下载路径找到软件。

MySQL for Visual Studio官网下载

一会使用MySQL Installer下载时还要有MySQL for Visual Studio 1.2.8，我们先下载安装。
输入下列网址，进入MySQL官网：
https://www.mysql.com/
依次点击DOWNLOADS，Community，MySQL on Windows，MySQL for Visual Studio

将网页拉到底，有两个软件，一个是msi格式一个是压缩包格式。我们当然选择下载与MySQL Installer配套的msi格式。

同上，不用注册，直接点最底下的No thanks，just start my download
直至下载完成。

安装Visual Studio 2019

Visual studio是微软公司开发的IDE，每两年发布一次新版本，发布年的3月份发布，目前有2012，2013，2015，2017，2019。每个版本三个类型，社区版、专业版、企业版，社区版免费。
输入下列网址登陆。
https://visualstudio.microsoft.com/zh-hans/
点击网页上方的下载进入下图网页。

下载的是安装器。双击安装器在线安装Visual Studio 2019，注册一个微软账号，启动Visual Studio 2019。

安装Visual Studio 2015

不推荐大家使用Visual Studio 2015，有上面的VS 2019就够了。2015版有3.7个G，不同于其他VS，安装麻烦。

直接双击Visual Studio 2015 . iso，以DVDFab方式打开。

安装VS 2015至少需要45min。
安装完成重启电脑。
从开始菜单中打开VS 2015，使用微软账户启动VS 2015。

安装MySQL for Visual Studio

双击mysql-for-visualstudio-1.2.8，开始安装。
点击Next。
进入第二页，选择安装类型。我们点击Compelete，Install。
安装完成，点Finish

使用MySQL Installer安装全套数据库

安装前建议清除电脑中已有的MySQL产品，实在有重要数据就不清。双击mysql-installer-community-8.0.16.0，过一会弹出窗口。

MySQL Installer自动检测到了MySQL for Visual Studio，点击右边的add…。
第二页的Filter选择你的安装产品，点击Edit，我们选All。


选择好下载类型，我们点Filter。左框中显示可安装产品，你需要将你要的东西添加到右边的框。选好了点Next。

第三页是准备安装，点Execute继续。

全部安装完成，点Next
