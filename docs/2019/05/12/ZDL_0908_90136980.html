ACM校赛网络赛 问题 A: LS的数学题

May 12, 2019

题目描述

又是一年山东科技大学ACM校赛！欢迎各位。

今年是2019年！虽然LS数学不好，但他还是给你们出了一道数学题。

给你a,b,c,d四个数，有多少对(x,y)，满足a≤x≤b，c≤y≤d，且x与y的乘积是2019的倍数呢？ 

提示：2019只有3和673两个质数因子哦。

输入

输入包含多组数据，每组数据包含一行，四个整数a,b,c,d。1≤a≤b≤109,1≤c≤d≤109。

输出

对于每组数据，输出一个整数，表示答案，以回车分隔。

样例输入

1 3 1 2019
1 2019 1 2019
1 1000000000 1 1000000000

样例输出

5
6725
1649754749959584

思路：

要使[a，b]区间（记为A）与[c, d]区间（记为B）的乘积为2019的倍数，则有以下四种情况：

  • A区间内3的倍数的个数 × B区间内673的倍数个数（注意：这里3的倍数中不可包含673的倍数，对于673而言也不可包含3的倍数）
  • A区间内673的倍数的个数 × B区间内3的倍数的个数（注意内容同上）
  • A区间内2019的倍数的个数 × B区间内数字的个数（同上）
  • A区间内非2019倍数的个数 × B区间内2019倍数的个数（原因是上一步骤已经包含A区间内2019倍数与B区间内2019倍数相乘）

当然，也可以这样：

  • A区间内3的倍数的个数 × B区间内673的倍数个数（注意：这里3的倍数中不可包含673的倍数，对于673而言也不可包含3的倍数）
  • A区间内673的倍数的个数 × B区间内3的倍数的个数（注意内容同上）
  • A区间内2019的倍数的个数 × B区间内非2019倍数的个数 
  • A区间内非2019倍数的个数 × B区间内2019倍数的个数
  • A区间内2019倍数的个数 × B区间内2019倍数的个数

代码：

第一种：

#include<iostream>
#include<cstdio>
typedef long long  LL;
using namespace std;
int digit_bei(int a, int b, int beishu)
{
    return (b/beishu - (a-1)/beishu); /*a-1的原因是如果b，a都不是beishu的倍数，那么只能返回0了*/           /*二者之差为与倍数有关的倍数的个数*/
}
int main()
{
    int a, b, c, d;
    while(cin >> a >> b >> c >> d)
    {
        /*ab区间内3的倍数乘以cd区间内673的倍数*/
        LL s1 = digit_bei(a, b, 2019), s2 = digit_bei(c, d, 2019);      /*仅仅是2019的倍数*/
        LL t1 = digit_bei(a, b, 673) - s1, t2 = digit_bei(c, d, 673) - s2;  /*如果2019的倍数也存在，那么673的倍数中也包含2019的倍数，应减去*/
        LL r1 = digit_bei(a, b, 3) - s1, r2 = digit_bei(c, d, 3) - s2;      /*同上*/
        cout << r1 * t2 + t1 * r2 + s1 * (d - c + 1) + s2 * (b - a + 1 - s1) << endl;
    }
}

using namespace std;

{
    return (b/beishu - (a-1)/beishu); /*a-1的原因是如果b，a都不是beishu的倍数，那么只能返回0了*/           /*二者之差为与倍数有关的倍数的个数*/
}
int main()
{
    int a, b, c, d;
    while(cin >> a >> b >> c >> d)
    {
        /*ab区间内3的倍数乘以cd区间内673的倍数*/
        LL s1 = digit_bei(a, b, 2019), s2 = digit_bei(c, d, 2019);      /*仅仅是2019的倍数*/
        LL t1 = digit_bei(a, b, 673) - s1, t2 = digit_bei(c, d, 673) - s2;  /*如果2019的倍数也存在，那么673的倍数中也包含2019的倍数，应减去*/
        LL r1 = digit_bei(a, b, 3) - s1, r2 = digit_bei(c, d, 3) - s2;      /*同上*/
        cout << r1 * t2 + t1 * r2 + s1 * (d - c + 1 - s2) + s2 * (b - a + 1 - s1) + s1 * s2 << endl;
    }
}
