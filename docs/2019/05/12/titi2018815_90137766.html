基础练习 十六进制转八进制

May 12, 2019

题目：


　　给定n个十六进制正整数，输出它们对应的八进制数。
输入格式
　　输入的第一行为一个正整数n （1<=n<=10）。
　　接下来n行，每行一个由0~9、大写字母A~F组成的字符串，表示要转换的十六进制正整数，每个十六进制数长度不超过100000。
输出格式
　　输出n行，每行为输入对应的八进制正整数。
　　【注意】
　　输入的十六进制数不会有前导0，比如012A。
　　输出的八进制数也不能有前导0。
样例输入
　　2
　　39
　　123ABC
样例输出
　　71
　　4435274
　　【提示】
　　先将十六进制数转换成某进制数，再由某进制数转换成八进制。

思路：

这道题，你想要做对，那么你必须知道如何把十六进制数转换成八进制数，我们首先就会想到找一个过度数字：先将十六进制数转换成某进制数，再由某进制数转换成八进制。那么这个数字是几呢？嗯呐，二进制是比较方便的。

十六进制（一位变成四位）---->二进制（三位变一位）--->八进制

 十六进制如何转换成八进制：

把十六进制的数变成二进制的数，就是把十六进制数一位变成四位二进制，不够四位的前面补0（如1(16)--->1(2)--->0001(2) );

 把二进制的数变成八进制的数，那么就是从低位开始三位二进制变成一个八进制数，最后的不够三位就是前面补0变成三位（如1010101（2）--->001 010 101(2)--->125(8) );

例子：39（16）--->0011 1001(2)--->0 0011 1001(2)--->000 111 001(2)--->071(8)--->71(8);

123ABC(16)--->0001 0010 0011 1010 1011 1100(2)--->0001 0010 0011 1010 1011 1100(2)--

                   ->000 100 100 011 101 010 111 100(2)-->04435274(8)--->4435274(8);

代码如下：

#include<stdio.h>
#include<algorithm>
#include<string.h>
#include<string>
#include<iostream>
using namespace std;

string s,ss;
int n;
string ans;

void bian()//二进制转换成八进制；
{
    string b="";//初始化；
    if(ss.size()%3==1)//判断不够三位的要添加的0；
        b="00";
    else if(ss.size()%3==2)
        b="0";
    b=b+ss;
    ans="";//初始化；
    for(int i=0; i<b.size(); i=i+3)//每三位转换成一位；
    {
        int sum=0;
        for(int j=0; j<3; j++)
        {
            sum=sum*2+b[i+j]-'0';
        }
        char e=sum+'0';//注意；
        ans+=e;
    }
    int flag=0;
    for(int i=0; i<ans.size(); i++)//输出，记得去前导零；
    {
        if(flag==0)
        {
            if(ans[i]!='0')
            {
                cout<<ans[i];
                flag=1;
            }
        }
        else
            cout<<ans[i];
    }
    cout<<endl;
}

void huan()//十六进制转换成二进制；
{
    reverse(s.begin(),s.end());//记得反转；
    ss="";//初始化；
    for(int i=0; i<s.size(); i++)
    {
        int k;
        if(s[i]>='0'&&s[i]<='9')
            k=s[i]-'0';
        else
            k=s[i]-'A'+10;
        for(int j=0; j<4; j++)//一位转换成四位，即十六进制转换成二进制；
        {
            int t=k%2;//一定要转换四次，这样不够四位的就制动加0；
            k=k/2;
            char w=t+'0';
            ss+=w;
        }
    }
    reverse(ss.begin(),ss.end());//记得再反转回来；
    bian();
    return ;
}

int main()
{
    while(~scanf("%d",&n))
    {
        for(int i=0; i<n; i++)
        {
            s="";//初始化；
            cin>>s;
            huan();//十六进制转换成二进制；
        }
    }
    return 0;
}
