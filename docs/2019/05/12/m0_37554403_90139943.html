May 12, 2019

svn操作步骤之创建仓库

1. 下载SVN。下载地址：https://tortoisesvn.net/downloads.html
英文不好的可以下载中文语言包

2.建立SVN仓库。 可以选择任意磁盘来创建文件夹作为容器。
这里我在G盘中创建文件夹‘’SVN仓库‘’，进入文件夹后右键-----》选择TortoiseSVN-----》选择Create repository here。并选择Create folder structure(会自动在仓库中创建tags(标签),trunk(主干),branches(分支)文件夹，用于项目不同情况下的存储)。

在此文件夹下右键-----》选择TortoiseSVN-----》选择Repo-borwser可以打开SVN仓库的虚拟目录。
这些文件在电脑盘中是无法看见的，只能通过Repo-borwser来访问。

3.将项目加入到仓库中。
3.1.创建一个项目，然后使用Repo-borwser来访问SVN仓库，右键trunk—》选择Add folder,选择项目所在文件夹即可，但是使用时需要重新checkout出来才能管理。

3.2.在使用Visual studio创建项目后，可以选择界面右下角的‘添加到源代码管理’并选择SVN。Next—》Exisiting Repository----》Start repository browser… 选择所创建的SVN仓库虚拟目录，选择trunk即可将项目导入到此文件夹下管理。

注：trunk存储项目开发的主干线。
branches存储项目临时分支。
tags存储上线版本或特定版本。
