nodejs快速搭建项目

May 12, 2019

  • 创建一个node文件夹,下载安装nodejs,傻瓜式操作
  • 打开命令行检查node版本 node -v

  • 快速搭建前端项目框架

  • 全局下载vue vue-cli (npm install vue vue-cli -g)
  • 创建前端项目 vue init webpack 项目名称(不要用中文)
  • 进入到项目根目录下   cd 项目名称
  • 安装依赖包  npm install 
  • 运行项目 npm run dev 
  • 运行成功浏览器访问地址 localhost:8080查看

  • 快速搭建后台项目框架

  • 全局下载 express express-generator (npm install express express-generator -g)
  • 创建后台项目 express  项目名称(不要用中文)
  • 进入到项目根目录下   cd 项目名称
  • 安装依赖包  npm install 
  • 运行项目 npm start
  • 运行成功浏览器访问地址 localhost:3000查看
