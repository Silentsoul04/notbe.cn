最长递增子序列问题LCS

May 12, 2019

题目描述：
字符序列的子序列是指从给定字符序列中随意地（不一定连续）去掉若干个字符（可能一个也不去掉）后所形成的字符序列。令给定的字符序列X=“x0，x1，…，xm-1”，序列Y=“y0，y1，…，yk-1”是X的子序列，存在X的一个严格递增下标序列<i0，i1，…，ik-1>，使得对所有的j=0，1，…，k-1，有xij=yj。例如，X=“ABCBDAB”，Y=“BCDB”是X的一个子序列。
输入：
两个字符串
输出：
最长公共子序列的长度
最长公共子序列
**思路：**我们定义两个字符串a[],b[],然后各自的长度为len1和len2,
定义一个int型的二维数f[][]，f[i][j]表示a[1…j]和b[1…j]的最大公共子序列，
即递推公式为：
{ f [ i ] [ j ] = f [ i − 1 ] [ j − 1 ] + 1 ; ( a [ i − 1 ] = b [ j − 1 ] ) f [ i ] [ j ] = m a x ( f [ i − 1 ] [ j ] , f [ i ] [ j − 1 ] ) ; ( a [ i − 1 ] ! = b [ j − 1 ] ) \left\{ \begin{array}{c} f[_i] [_j]=f[_i-_1][_j-_1]+1; (a[_i-_1]=b[_j-_1])\\ f[_i][_j]=max(f[_i-_1][_j],f[_i][_j-_1]) ;(a_[i-_1]!=b[_j-_1])\end{array}\right. {f[i​][j​]=f[i​−1​][j​−1​]+1;(a[i​−1​]=b[j​−1​])f[i​][j​]=max(f[i​−1​][j​],f[i​][j​−1​]);(a[​i−1​]!=b[j​−1​])​
然后最长公共子序列即为f[len][len]。
求最长公共子序列就是从后往前推。
直接上代码 不多BB
求长度的代码：

    for (i=1;i<=len1;i++)
            for (j=1;j<=len2;j++)
        {
            if (a[i-1]==b[j-1]) f[i][j]=f[i-1][j-1]+1;
            else f[i][j]=f[i-1][j]>f[i][j-1]?f[i-1][j]:f[i][j-1];
        }

求序列的代码：

for (i=len1,j=len2;i>0&&j>0;)
        {
            if (a[i-1]==b[j-1])
            {
                c[k++]=a[i-1];
                i--;
                j--;
            }
            else if (f[i-1][j]>=f[i][j-1]) i--;
            else j--;
        }

要注意的是这个 if (f[i-1][j]>=f[i][j-1]) i–; 语句里面>=是否能换成>的问题，如果仅存在一个最长的子序列，那么加于不加”=“无影响的，如果存在多个，那么结果是不一样的。
例如 ABCBDAB
BDCABA
