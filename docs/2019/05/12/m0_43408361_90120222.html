3d数学--矩阵变换

May 12, 2019

变换

  • 齐次坐标

    • 将一个n维的向量用n+1维来表示(x,y,z)=>(x,y,z,0)
    • 点的齐次坐标为(x,y,z,1)
    • 向量的齐次坐标为(x,y,z,0)
      -[ a(1,0,0,0),b(0,1,0,0),c(0,0,1,0) o(0x,0y,0z,1)] * [vx,vy,vz,0或1] 表示向量和点在坐标系上的运算
      • [ a(1,0,0,0),b(0,1,0,0),c(0,0,1,0) o(0x,0y,0z,1)]//为下图的3d坐标系F
    • 点与点相减 为向量
    • 向量与向量的加减还是向量
    • 点与向量的加减，为移动了向量距离的点

    [矩阵的表示形式]

仿射变换

  • 线性变换//变换前是直线，变换后还是直线，直线比例不变，原点不变
  • 仿射变换=线性变换+平移//变换前是直线，变换后还是直线，直线比例不变 ，平移选择缩放都是仿射变换
  • 3维的变换 表示如下 , 第一个矩阵为变换矩阵
  • 平移变换矩阵
  • 缩放变换矩阵

  • 旋转变换矩阵 分别为 绕x，y，z的 旋转变换矩阵
平移变换矩阵的逆矩阵为相反平移的变换矩阵（-tx，-ty，-tz）其他不变
缩放变换矩阵的逆矩阵为相反缩放的变换矩阵（1/sx，1/sy，1/sz）其他不变

坐标系变换与物体变换

  • F (MC)//为物体变换 F 为上图的坐标系 M为变换矩阵 C 为需要变换的点或向量

  • (F*M)*C// 为坐标系变换

复合变换

  • 将平移选择缩放结合起来的变换 但因为矩阵不满足交换律，所以不同的变换顺序得到的结果可能不同

在unity中的矩阵变换

  • Matrix4x4.Translate //创建一个平移矩阵
  • Matrix4x4.Rotate//创建一个旋转矩阵
  • Matrix4x4.Scale//创建一个缩放矩阵
  • Matrix4x4.TRS //创建一个复合矩阵
  • Matrix4x4.MuitIplyPoint 或者 Matrix4x4.MuitIplyPoint3x4 //变换一个点的位置
  • Matrix4x4.MuitIplyVector//变换一个向量
    public  float rotAngle;
    public float stretch;
    MeshFilter mf;
    Vector3[] origVerts;
    Vector3[] newVerts;
    {
     mf = GetComponent<MeshFilter>();
        origVerts = mf.mesh.vertices;
        newVerts = new Vector3[origVerts.Length];
    }


    void Update()
    {
        Quaternion rot = Quaternion.Euler(rotAngle, 0, 0);
        Matrix4x4 m = Matrix4x4.TRS(Vector3.zero, rot, Vector3.one);

        Matrix4x4 inv = m.inverse;

        for (var i = 0; i < origVerts.Length; i++)
        {
            var pt = m.MultiplyPoint3x4(origVerts[i]);
            pt.y *= stretch;
            newVerts[i] = inv.MultiplyPoint3x4(pt);       
        }
        mf.mesh.vertices = newVerts;
    }
