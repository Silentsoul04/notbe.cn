leetcode-237 删除链表中的节点

May 12, 2019

题目描述

请编写一个函数，使其可以删除某个链表中给定的（非末尾）节点，你将只被给定要求被删除的节点。
现有一个链表 – head = [4,5,1,9]，它可以表示为:

输入输出样例

样例1

输入: head = [4,5,1,9], node = 5
输出: [4,1,9]
解释: 给定你链表中值为 5 的第二个节点，那么在调用了你的函数之后，该链表应变为 4 -> 1 -> 9.

样例2

输入: head = [4,5,1,9], node = 1
输出: [4,5,9]
解释: 给定你链表中值为 1 的第三个节点，那么在调用了你的函数之后，该链表应变为 4 -> 5 -> 9.

说明

链表至少包含两个节点。
链表中所有节点的值都是唯一的。
给定的节点为非末尾节点并且一定是链表中的一个有效节点。
不要从你的函数中返回任何结果。

算法分析

乍一看题目很懵逼，仔细审完题目之后还是很懵逼。确实这个题目的描述和输入输出样例很容易误导大家是遍历整个链表，找到链表中那个特定值的节点然后删除，但其实看完评论后发现并不是这样。题目的意思是给你链表中一个节点，然后删除这个节点。我们知道，常规的单链表删除节点的方法，是将某一节点的前一个节点的next指针域指向被删除节点的下一个节点，然后将被删除的节点free掉。但是在此题中，由于是单链表，我们无法得到需要被删除节点的上一个节点。这一点也是这一题出题人的关键意思所在。那么我们如何来删除题目所给的节点呢？这里我们就要用到一个非常非常非常非常巧妙的方法如下：

常规做法（此题无法使用）

本题做法

我们无法访问我们想要删除的节点 之前 的节点，我们始终不能修改该节点的 next 指针，但是我们可以将需要删除节点的值替换为下一个节点的值，然后删除下一个节点，进而间接地完成了删除“所给节点”地目的。



根据题目的说明，我们知道要删除的节点不是列表的末尾，所以我们可以保证这种方法是可行的。
本题的代码也非常简单。

C语言代码实现
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
void deleteNode(struct ListNode* node) {
    node->val=node->next->val;
    node->next=node->next->next;
}
