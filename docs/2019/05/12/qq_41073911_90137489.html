Docker-Portainer可视化工具

第一步：配置环境

  1. 因为portainer运行时需要依赖daocker的docker.sock配置文件,但是docker.sock配置文件本身是不可用的
    可以把它删除,如果你想保险起见的话可以先移走
    rm -rf /var/run/docker.sock
或
mv /var/run/docker.sock /var/
  2. 重启Docker服务
service docker restart
  1. 重启之后，执行如下代码
docker volume create portainer_data
docker run -d -p 9000:9000 -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer
  1. 此时portainer镜像及容器已经创建完并运行了，你可以访问并测试一下
    我本机地址为 192.168.26.134:9000
  2. 进入portainer网页之后会让你先初始化一个账户,输入账户密码
  3. 登录之后会显示这样一个页面,这个页面是让你选择你的docker容器的
此时只需点击第一个 local 然后点击connect即可连接成功
