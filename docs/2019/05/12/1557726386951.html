centos7 七步教你安装搭建 LAMP 服务

May 12, 2019

先说说LAMP是什么

LAMP是指一组通常一起使用来运行动态网站或者服务器的自由软件名称首字母缩写：

  • Linux，操作系统

  • Apache，网页服务器

  • MariaDB或MySQL，数据库管理系统（或者数据库服务器）

  • PHP、Perl或Python，脚本语言

软件环境

CentOS 7.2 64 位

1.搭建MySQL 数据库

使用yum安装

yum install mysql-server -y

安装完毕 启动

service mysqld restart

设置MySQL密码

/usr/bin/mysqladmin -u root password '123'

安装Apache

yum install httpd -y

启动Apache

service httpd start

安装PHP和MySQL支持工具

安装PHP

yum install php php-mysql -y

我们在/var/www/html 目录下创建一个info.php 来检查php服务是否安装成功

示例代码为

<?php phpinfo(); ?> //这句代码的意思是显示服务器信息

然后重启服务

service httpd restart

http://机器ip/info.php

后出现以下图片  代表安装成功
