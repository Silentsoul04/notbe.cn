Centons6.5中安装与配置Maven

May 12, 2019

安装环境:Centos-6.5

安装方式:源码安装

1.下载地址:http://mirrors.hust.edu.cn/apache/maven/maven-3/3.2.5/binaries/apache-maven-3.2.5-bin.tar.gz

2.在usr目录下建立soft文件夹

cd /usr

mkdir soft

3.下载安装包到soft目录下

[root@iZwz945po7bqabsr6de22fZ soft]# wget http://mirrors.hust.edu.cn/apache/maven/maven-3/3.2.5/binaries/apache-maven-3.2.5-bin.tar.gz

4.解压到当前目录

[root@iZwz945po7bqabsr6de22fZ soft]# tar -zxvf apache-maven-3.2.5-bin.tar.gz

5.移动文件

[root@iZwz945po7bqabsr6de22fZ soft]# mv apache-maven-3.2.5 /usr/local/maven

6.编辑配置文件,配置环境变量

[root@iZwz945po7bqabsr6de22fZ /]# vim /etc/profile(进入编辑模式,对所有用户都有效)

在最后一行添加以下内容(注PATH只能有一个,比如之前已配置了jdk,直接在后面继续增加即可)

export MAVEN_HOME=/usr/local/maven

export PATH=$PATH:$JAVA_HOME/bin:$PATH:$MAVEN_HOME/bin

ESC(退出编辑模式)

:wq(保存退出)

7.重启机器或执行生效命令

[root@iZwz945po7bqabsr6de22fZ /]# source /etc/profile

或sudo shutdowm –r now

8.查看安装情况:

[root@iZwz945po7bqabsr6de22fZ /]# mvn -version /mvn -v

Java -version
