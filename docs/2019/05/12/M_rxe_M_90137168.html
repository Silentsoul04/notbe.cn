创建django的框架
1、打开pycharm点开file，然后创建New project

2、点击左边的django项目

3、找到django环境以及创建文件夹

4、创建完成之后，修改settings.py文件

NAME’: ‘day05_pm’, MySQL数据库名
‘USER’: ‘root’,，MySQL用户名
‘PORT’: ‘3306’,MySQL默认端口
‘HOST’: ‘localhost’,本机地址
‘PASSWORD’: ‘******’,MySQL密码

修改语言，以及修改时间，我们默认用上海的时间
5、在MySQL中创建数据库，用来与pycharm进行交互
可以在pycharm控制台直接进入数据库，创建与settings.py文件中连接的数据库，创建完成之后，点击右边的databases连接MySQL



6、连接完成之后，导入MySQL驱动，在__init__.py中添加以下指令

完成之后在控制台输入迁移指令

python manage.py makemigrations

生成迁移文件

python manage.py migrate

执行迁移

完成以后查看MySQL中会生成迁移表格
7、完成django框架的创建以及与数据库的连接操作
