Git学习

May 12, 2019

命令行操作
5.4.1查看状态
git status
no branch // 没有分支
No commit // 没有任何提交的文件
根据状态提示，往仓库中创建一个demo01.txt的文件，文件内容(按i插入内容，按: wq保存文件，按:q!不保存强制退出)

vim demo01.txt // 回车
按i 进入编辑状态
输出完成后按CTRAL + C 退出编辑状态
输入 :wq 保存并退出/ 输入 :q! 退出并不保存刚才编辑的内容
使用 cat demo01.txt 查看文件内容
将demo01.txt文件加入到暂存区
git add demo01.txt //输入Enter
// 提交文件
git commit -m “日志内容” demo01.txt //没有日志信息系统会帮你默认生成。

总结==》 1、创建文件 2 、通过git add 命令把文件添加到暂存区 3、git commit 把文件提交到本地库

查看本地仓库中的文件：git log / git reflog --pretty=oneline / git reflog --oneline / git reflog
后退版本：git reset --hard HEAD^ (^表示后退一个版本) 想要后退多个版本需要使用：git reset --hard HEAD^n(n代表后退n个版本)
使用索引值回滚到指定版本：git reset --hard <索引值>；
删除工作区的文件： rm 文件名

对比文件差异：
将工作区中的文件和暂存区进行比较
命令：git diff <文件名>
将工作区和本地仓库的历史记录进行比较：
命令：git diff <本地库中历史版本> <文件名>

Git分支操作：
查看分支：git branch -v
创建分支：git branch <新分支名>
删除分支（不能删除当前正在使用的分支）: git branch -d <分支名>
切换不同分支：git checkout <需要接受的分支名>
合并分支：
第一步：切换到接受修改的分支上：git checkout <需要接受的分支>
第二步：执行merge命令：git merge <需要合并的分支>
查看当前分支下的文档：ls
