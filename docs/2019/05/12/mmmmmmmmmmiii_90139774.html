电脑微信多开营销号养成技术详细教程

May 12, 2019

首先讲一下简单在电脑上多开微信的方法：在电脑上安装一个安卓手机模拟器，会自动安装一个模拟器多开器的，点开多开器后，添加模拟器窗口，直接在多开器上打开多开窗口，可以根据您电脑配置任意添加新的窗口，打开窗口后再里面下载安装微信即可。

微信如今已经算是国内最大的一款社交软件了，其实社交软件的运用有很多普遍的就是商业，营销，引流等等的使用方法，所以腾讯对这方面打压的很多，也就是通过封号。
怎么做到养号的方法让自己在微信营销商业这一块站稳脚步呢，今天小编详细介绍一下，电脑上通过模拟器等工具来多开微信的账号。

微信有规则变非常之快、个人号监管最严、打击营销力度最大、封号率高，微信解封条件从刚开始的短信秒解，到现在无比苛刻的辅助认证条件实在是让人很无语，一旦被封，几乎无解！

微信封号因素四个方面：硬件设备、IP 、行为和微信账号 ID

⊙硬件：指微信号登陆的设备，几个关键标识是 MAC、IMEI、序列号等等；

⊙IP：就是设备联网时被分配的网络地址，流量 IP 和 WIFI 的 IP 还是有不同的。流量基于基站，WIFI 基于路由器的发射模块，微信是可以分辨的；这里可以所以使用【兔子IP】解决IP问题，全国城市IP随意切换。

⊙行为：就是从上次登陆时间和地理位置来判断是否移动速度异常，另外还有被举报、被自动检测到等等情形。

⊙微信账号 ID：微信是通过监测微信号的行为，来判断是否将一个账号 ID 或者 IP 关进行限制的。目前来说封 ID 和设备的情况较多，因为一个 IP 下往往有很多用户，不可能因为一两个 ID 违规，就将整个 IP 下的设备都封掉。避免封IP选择兔子IP把IP切换一下。

所以封 ID 是首选，其次严重违规了，会封设备。所以经常有这种情况，就是一个机子上，某个微信的功能被限制了，再上其他号，也一样被限制。
封IP的情况也不是没有，就是一个 IP 下有大量违规 ID，那这个 IP就很危险。最常见的就是模拟器养号的，不死就都活，一死一片。封IP 的时候一般也会连 ID 也一起封了。所以尽量让多开的微信IP地址都改成不一样的，可以通过兔子IP.

如果被微信检测到一个 wifi 下登录大量微信，你的整个 ip 段都会被记录然后封掉。如果被微信检测到一个微信客户端频繁切换账号进行营销，那么你的这个客户端登过的微信都会被封，主号和小号都不例外。

现在微信检测机制比较全面，腾讯都是通过机器人检测的，并非人工。只要你有一点违规都会以各种理由封你。



总的来说，官方对于一些没有实名认证、虚拟运营商注册的微信号、资料雷同、涉嫌机器批量操作，频繁添加好友、从不互动聊天社交、群发广告等、已经在采取限制与打击。

微信号的注册准备阶段

首先得准备好以下工具：

1、一台能正常接收短信的手机

2、一个能提供注册新微信的手机卡

注册好之后马上进行实名认证，现在微信的实名认证可以通过银行卡或者身份证认证，一张身份证只能认证 5 个微信号。

在养新号的前 15 天，不要做任何的营销操作，每天互动。个人微信号平时干嘛，养的号就干嘛。比如主动或被动添加好友，对方是老的微信号，通过后聊两句。

和已有的好友进行语音，文字，表情聊天。订阅公众号，阅读和分享文章，注意停留时间。加几个微信群修改群名片，然后聊几句。下载微信表情包，收藏一点表情。

与朋友圈点赞或评论，自己也发几个朋友圈。没事抢抢红包发发红包。阅读腾讯新闻并分享到朋友圈带地理位置，给好友分组备注、微信钱包放点零钱，也可以买个一元的理财。

注意事项：

注册微信号一定要用官方版微信， 最好不要用 170 号段的电话卡、腾讯屏蔽的，不要在一个时间，一个地点开号，可以换下位置（兔子IP改个IP），密码不要一样的，建议用变量字符+手机号这样的方式。

头像图片尽可能也不要一样，15 天内不要使用摇一摇，附近加好友功能，登陆之后就一直保持稳定登陆，不要退出或者切换其他号，注意朋友圈发送频率和数量。

归属地的定位为实际定位。就是说在注册微信号的时候，比如说你这个手机号码归属地是广东啊，那你注册的时候 ip 一定要是属于广东的才行，你使用的 ip 与你的位置要对应。（可以在兔子IP软件选择手机号码归宿地的城市即可。）

比如说，你在福州注册微信号，微信能定位到福州具体到某条街某栋楼，而你 ip 却是上海的，立马判断作弊，这点最容易被忽视，注册多个微信号 ，大家都会想到用新手机，新手机号，换 ip，唯独会忘记定位。切记！

养营销号：

养营销号避免不了，就是内容重复。用一个主号先养个十五天，刚开始前三天到五天的话，一天发七八条朋友圈。一个星期后就可以减少到四条五条这样子，这个数量你们自己去衡量。

然后十五天后的话，其他的小号跟着。其他的小号另外一个小号跟着这个主号开始发起来，这样子的话，它是有一个时间的错开，养号就会避免一个内容重复，哪怕重复的话，他也不是说检查的时候同一个时间发同一个内容，这样子的话。这样子可以避免开封号。
