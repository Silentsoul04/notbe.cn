catkin_make 时遇到 “对‘inflateValidate@ZLIB_1.2.9’未定义的引用” 解决办法

May 12, 2019

问题现象：

/home/username/anaconda3/lib/libpng16.so.16：对‘inflateValidate@ZLIB_1.2.9’未定义的引用

问题原因：


ROS  和 Anoconda 同时安装导致的

 

解决办法：

step1

在~/.bashrc中注释掉所有和Anoconda有关的内容

step2

确认在~/.bashrc内有export PATH="/home/username/anaconda3/bin:$PATH"

step3

$ conda remove libpng

此时已经可以使用catkin_make了

step4

$ sudo apt-get install libpng16-16
