封装

May 12, 2019

概念：

将类的某些信息隐藏在类内部，不允许外部程序直接访问，而是通过该类提供的方法而实现功能

封装:

类的封装，方法封装，

操作

在属性前，添加private关键字

效果

通过private关键字封装的属性，不能在类的外部通过对象名.属性名来

直接访问。

类的内部，依然可以正常访问。

 私有属性的访问

通过在类中添加公有的方法，来访问私有的属性。

通常这个公有方法的名字有一定的命名规范：

获取值：getXxx（Xxx表示属性名，首字母大写）

设置值：

setXxxget获取 set设置

代码如下

public class Anima {
	
 
   private String name;
   private int age;
   private char sex ;
public String getName() {
	return name;
}
public void setName(String name) {
	this.name = name;
}
public int getAge() {
	return age;
}
public void setAge(int age) {
	this.age = age;
}
public char getSex() {
	return sex;
}
public void setSex(char sex) {
	this.sex = sex;
}
