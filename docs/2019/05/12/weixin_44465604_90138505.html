Terminology

我们迎来了一大批学生，学生数据中包括GPA，高中排名，和家庭收入。那么这一堆学生的数据就是数据集(data set)，每个学生就是一个样本（instance或者sample）。而GPA, high school ranking, and family income就是他们的属性（attribute）。GPA 4.0, HS ranking #100, income $30000这些数据就是属性值（attribute value) 。其他一些比较直白的定义还包括learning, training, training data, learning data, training set, training sample。

  1. Supervised Learning：训练数据有标记信息，其中分类与回归属于监督学习 input x and output y and you use an algorithm to learn the mapping function from the input and the output
    1. Two parts: Training and testing, prediction (regression problem)
    2. Linear regression
      1. To estimate the real values
    3. Logistic regression
      1. To estimate T/F values
      2. Sigmoid Curve
    4. Decision Tree
      1. For classification 这是个好学生还是个坏学生？
      2. Capable for both continuous dependent and categorical variables
      3. Internal nodes represent test on an attribute
      4. Each branch represents outcome of test and each leaf node represents class label (decision taken after computing all attributes)
      5. A path from root to leaf represents classification rules
      6. Single outcome
    5. Random Forest
      1. Better accuracy than decision tree
    6. Naive Bayes Classifier
      1. Based on the Bayes Theorem
    7. Model Fitting
      1. Making your algorithm learn the relationship between predictors and outcome so that you can predict the future values of the outcome.
  2. Unsupervised Learning 训练数据没有标记信息，代表有聚类
  3. Reinforcement Learning

Linear regression: “Line of the best fit”

  • x- explanatory variable (independent)
  • y- dependent variable
  • 基本形式: f(x) = w^t x + b 其中W=(w1; w2; w3; …; wd) The model will be defined once the w and b are defined from learning progresses.
Example: Predicting house price in a city

Procedure: Training set -> Learning Algorithm - > Hypothesis
Definition of hypothesis: A function which takes the size of the house as variable (x) and output the estimated price (y).
How do we represent the hypothesis?
h theta x = theta0 + theta1 x （也写作H thetax）aka Linear Regression with one variable(unvariate linear regression)
目的：
需要寻找theta0 和 theta1：
Choose theta0 and theta1 so that h(X) is close to y for our training examples(x,y). We want (h(x) - y)^2 to be minimized. Predicted的值和真正值的差距应该降到最小。
不同的theta1, theta0会得出不同样的线性方程，theta1的值也影响着J(theta1)MSE Cost Function。可以利用level curve去visualize最合适的theta1，theta0的value。

MSE Cost function:
Most commonly used in Linear Regression. MSE Cost Function的值越小，象征着线性回归的准确性越高。
Reading: http://mccormickml.com/2014/03/04/gradient-descent-derivation/

To minimize the cost function J(theta0, theta1): Gradient Descent 算法

We have J(theta0, theta1), we want an algorithm for minJ(theta0, theta1).
从某个theta0， theta1开始，一直更换他们的值去寻找J(theta0, theta1)的最小值。一直寻找最大的descent.
:= 是Assignment operator， take the value of b and assign it to a.
当人们谈论Gradient Descent的时候，他们是在谈论Simultaneous Update.


关于公式里的alpha （learning rate）和Derivative part的理解：
如果导数是负数，那么theta1会在原来的值上减去一个正数，所以相当于theta1在递减。（在向最低处靠近，如果已经达到一个optima的值，theta1不会再更新了，因为导数成0）。
alpha则决定了每挪一步的大小，但是我们没有必要一直减小alpha的值。但是当我们向最小值靠近的时候，gradient descent会自动缩减每一步的大小，因为即使alpha是固定的，derivative的变化也可以影响step的大小(derivative is getting smaller)。

Batch Gradient Descent:
“Batch” means each step of gradient descent uses all the training examples.

Linear Algebra knowledge for Linear Regression Model:
Matrices, Vectors,Addition (should be the same dimension), Subtraction, multiplication with matrices and vectors, matrix inverse, transpose

Matrix: rectangular array of numbers, dimension of matrix will be row x column (similar to 2D array)
Prediction = Data Matrix * Predictor
