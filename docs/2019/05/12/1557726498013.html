Excel 界面和基础操作

基础单元格操作
输入数据 数据格式 合并单元格 自动填充 选择性粘贴 去重 分列 排序 筛选 条件格式 插入下拉列表 行高列宽设置 冻结首行首列 边框 单元格换行

任务一
给Excel截图并标注功能。 建议将开始/插入等功能区里的具体内容模块也标注。

任务二
生成一个行高30，列宽15（第六列列宽45），名为“Excel组队学习”的表，要求如下： 1、第一列为职位ID，背景色为浅蓝； 2、第二列为职位类型，字体颜色为红色； 3、第三列为学历，每个单元格有下拉列表，选项为大专、本科、硕士、博士； 4、第四列为行业方向，单元格边框为红色虚线； 5、第五列为薪资水平，数据类型为货币，保留两位小数； 6、第六列为职位标签。


对于行宽与列宽的设置(这里以第1行与第D列为例)，直接点第1行/D列，右键设置行列宽即可，在学历一列，数据 -> 数据验证 -> 数据验证 ,然后如下图设置(注意学历之间的逗号是英文格式的)：

然后对行业方向一列进行处理，选择G1~G6，ctrl+1,，选择边框一栏进行调整。

任务三
在任务二表格的基础上进行如下操作：

  • 冻结窗格1-3行，效果如下：

    操作：点击A4行，视图 -> 冻结窗口 -> 冻结窗口，这样就完成了前三列的冻结，也就是所你选择哪一行，那么就会冻结在这一行之前的其他行。

  • 将第六列分列，效果如下：

    操作：选择F列1~6行，数据 -> 分列 -> 分隔符号 -> 逗号打勾 ->确定。

  • 利用条件格式，将薪资列大于8000的收入填充为深绿色，并将它们筛选出来，效果如下

    操作：选中E行2~6行，条件格式->突出显示单元格规则->大于->输入8000并且设置为填充深绿色文本；筛选，这里按颜色筛选或者数字筛选都可以。

  • 第二列将重复值删除，只保留唯一值，效果如下：

    操作：选中B列2~6行，数据 -> 删除重复项 -> 删除扩展区域 -> 只勾中职业类别即可。

完。
