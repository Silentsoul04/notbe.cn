centos7 配置静态ip

May 12, 2019

简介

centos7无界面配置静态ip

配置原理

网络地址相关的配置在 · /etc/sysconfig/netowrk-script

ifcfg-*这种命名方式的文件一般为某个网卡或者某个无线网络的配置：ip地址，网关，子网，dhcp等。实际上有界面配置，最终也将修改这个文件

下面查看的信息非必须，只是了解下系统的基本情况

查看本机网络情况

ip addr #(or ifconfig {yum install net-tools})
#查看每个网卡对应的地址分配情况

查看网卡信息

lspci #不存在 yun install pciutils
or
nmcli # 可以查看网卡信息，也可用来连接无线wifi

理解ifcfg-文件

文件中有个uuid，与网卡对应，用于唯一确定配置对应的网卡

静态ip配置

新装系统的情况配置静态ip

不贴具体的配置参数，因为根据理解每个配置参数，就不用每次都看教程了。下面说下配置静态ip都步骤

1: 启动网卡，BOOT=yes

2: 关闭dhcp设置为静态 dhcp->static

3: 静态ip IPADDR=192.168.1.130

4: 子网掩码 NETMASK=255.255.255.0

5: 网关 GATEWAY=192.168.1.1

6: DNS配置 DNS1=192.168.1.1

7: shell执行 systemctl restart network

上面的配置步骤，根据情况参数，在文件中修改，有就改，没有就增加，如果不配置DNS1那么无法ping通域名

上面都是手打，具体一些参数记不到，回家了补上
