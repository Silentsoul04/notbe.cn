4.字符串格式化问题
Python 支持格式化字符串的输出 。尽管这样可能会用到非常复杂的表达式，但最基本的用法是将一个值插入到一个有字符串格式符 %s 的字符串中。
例如：
print ‘My name is %s and weight is %d kg!’%(‘xiaoming’,42)

#打印结果：My name is xiaoming and weight is 42 kg!
