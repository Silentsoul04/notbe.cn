通过创建四个类，大致讲一下继承（适合初学者）（如有其他见解，欢迎增删改查）

概念：
/**

  • 继承：
  • 作用：代码的重用，复用。
  • 单继承：
  • 1.一个子类只能有一个父类。
  • 2.一个父类可以有多个子类。
  • 子类不能继承：
  • 1.父类的构造方法(只能通过super调用，这样可以妥善初始化继承来自父类的成员变量);如果子类的构造方法中没有调用父类的构造方法，java
  • 编译器会自动加入对父类无参构造的调用（如果该父类没有无参的构造器，会出现编译错误）;若子类的构造方法调用了父类的其他构造器，则子类
  • 不再默认提供super();通过super()调用父类的构造方法必须位于子类构造的第一位。
  • 另一种表达：子类如果没有构造方法,系统先默认super调用父类的无参构造(前提是父类得有一个无参构造,默认的不算,得定义出来),不然子类会报错,如果父
  • 类没有无参构造,只存在有参构造(一般父类都会定义参数,所以肯定也会定义有参构造,这两个一般是共存的),子类需要手动super调用父类的有参构造方法.
  • 2.父类的私有成员(private)
  • 子类对象：
  • 先新建父类对象–再新建子类对象–两个对象绑在一起，整体作为一个对象–调用成员时，先找子类再找父类
  • 构造子类之前，必须先构造父类：
  • 用法：
  • super.成员变量—调用父类的成员变量
  • super.方法()----调用父类中的方法
  • super()-------调用父类的构造方法
  • 向上造型：
  • 1.一个子类可以向上造型为父亲的类型，即：父亲类型的引用可以指向子类对象。
  •  	格式：父亲类型 变量名 = new 子类类型

  •  		Person p = new Student();
  • 向下造型：
  • 2.父亲的引用可以指向子类对象,但通过父亲的引用只能访问父亲所定义的成员，不能访问子类扩展的部分.
  •  	格式：子类类型  变量名 （子类类型）父类类型的变量;

  •  		Student stu = (Student)p; //变量p指向Student对象

*重写(Override):
*1.发生在两个类中，而且是子父类的关系，子类方法与父类方法方法名相同，方法体不同。
*2.当子类重写的方法被调用时（无论是通过子类的引用调用还是通过父亲类的引用调用），运行的的是子类重写后的版本
*3.子类在重写父类的方法时，可以通过super调用关键字调用父亲的版本。
*注意：
*1)重写方法被调用时，看对象，
*2)两同两小一大原则：

  • 2.1)两同:方法名相同，参数列表相同
  • 2.2)两小:1.子类返回值类型小于父亲的 2.声明的异常（被重写的）小于等于父类的。

*重写和重载的区别：

  • 重载：发生在一个类中，同名不同参，“编译期绑定”，看参数绑定的是哪个方法；
  • 重写：发生在两个类中（父子关系），方法名相同，参数列表相同，“运行期绑定”，看对象绑定的是哪个方法；
  • ctrl+t 光标放在类上，ctrl+t，可以看到类继承结构

    Person类（父类）
/**
 * Person类具体框架：
 * Person={
 * 		String name,
 * 		String gender,
 * 		int age,
 * 		Person(),
 * 		Person(String name,String gender,String age),
 * 		toString()
 * }
 */
public class Person {
	String name;
	String gender;
	int age;
	public Person() {//无参构造
		this("无名", "不明", 16);//调用底下全参构造
		System.out.println("Person无参");
	}
	public Person(String name, String gender, int age) {//全参构造
		this.name = name;
		this.gender = gender;
		this.age = age;
		System.out.println("Person有参");
	}
	@Override
	public String toString() {//重写toString()方法
		return name+","+gender+","+age;
	}
}

Student类（子类）

/**
 * Student类框架：
 * Student={
 * 		String school,
 * 
 * 		Student();
 * 		Student(String name,String gender,int age,String school),
 * 		toString()
 * }
 */
public class Student extends Person {
	/*
	 * Student继承了Person的toString,但是Person的构造方法不能继承。
	 * Student有一个默认的无参的构造方法
	 */
	String school;
	public Student() {
		//具有默认隐含调用，super();调用父类的无参构造，写不写都有
	        //super();
		System.out.println("学生无参");
	}
	public Student(String name,String gender,int age,String school) {
		/*
		 * super();默认的隐含调用，调用默认的参数，括号里写的时候就是手动调用，需要填参数
		 * this,super 都要求是首行代码，两者不能并存，都是调用方法，目的是减少代码重复。
		 */
		super(name,gender,age);
		this.school = school;
		System.out.println("学生的有参");
	}
	/*
	 * 重写toString()方法
	 * 如果 父类的代码不满足子类的需要，在子类中需要重新定义，重新编写。重写可以理解成"覆盖","改写"
	 */
	@Override
	public String toString() {//本类默认继承toString方法，但是如果要添加属性，需要重写以及返回调用
		/*
		 * 如果重写属性，代码量太大，name,gender,age可以直接调用父类，为了减少代码重复，用super.toString()
		 */
		return super.toString()+","+school;
	}
}

Employee类（子类）

/*
 * Employee类框架：
 * Employee{
 * 		double salary,
 * 
 * 		toString()
 * }
 * 
 * 光标放到类上面ctrl+t查看类的集成结构
 */
public class Employee extends Person{
	double salary;
		
	@Override
	public String toString() {
		return super.toString()+","+salary;
		
	}
}
