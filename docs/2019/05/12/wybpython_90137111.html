Django的安装

安装虚拟环境

1. sudo apt install virtualenv

2. sudo apt install virtualenvwrapper

3. 在当前目录下创建一个 .virtualenvs 的文件夹

4. sudo vim ~/.bashrc 

                        在文件最下面添加：

                                                 export WORKON_HOME=/home/huguanyu/.virtualenvs
                                                 source /usr/share/virtualenvwrapper/virtualenvwrapper.sh

5. 激活环境  source ~/.bashrc

环境的使用

               创建虚拟环境：mkvirtualenv 环境名称 -p 路径

               退出当前虚拟环境：deactivate

               删除虚拟环境：rmvirtualenv 环境名称

               进入虚拟环境：workon name

安装Django

在虚拟环境中输入命令：

                              pip install django

创建django项目：django-admin startproject name
启动项目：python manage.py runserver
创建app：python manage.py startapp name
