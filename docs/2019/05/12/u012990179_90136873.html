Linux学习_010_在 CentOS6.8 上安装 Eclipse

May 12, 2019

一、准备工作

安装环境：CentOS 6.8 桌面版（其他发行版本安装过程基本一样）
所需软件：（下载时选择 Linux 版 tar.gz 格式）
  1、Oracle jdk 各个版本下载地址： https://www.oracle.com/technetwork/java/javase/archive-139210.html
  2、Eclipse 各个版本下载地址：https://www.eclipse.org/downloads/packages/release

本博主使用的 jdk 版本是 1.8(64位)：http://download.oracle.com/otn/java/jdk/8u192-b12/750e1c8617c5452694857ad95c3ee230/jdk-8u192-linux-x64.tar.gz
Eclipse 版本是 Eclipse Mars2：https://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/mars/2/eclipse-jee-mars-2-linux-gtk-x86_64.tar.gz

二、安装过程

1、Oracle jdk 的安装:
  参考链接：https://www.cnblogs.com/chenmingjun/p/9931593.html

2、Eclipse Mars2 的安装：
（1）将下载好的 eclipse-jee-mars-2-linux-gtk-x86_64.tar.gz 上传至 Linux 系统的 /opt/sofeware/ 目录下
（2）解压至 /opt/module/ 目录下

[root@hadoop102 software]# tar -zxf eclipse-jee-mars-2-linux-gtk-x86_64.tar.gz -C /opt/module/

三、配置环境变量

1、java 环境变量在文件 /etc/profile 中配置
2、在 profile 文件中添加如下代码

#JAVA_HOME
export JAVA_HOME=/opt/module/jdk1.8.0_144
export JAVA_JRE=/opt/module/jdk1.8.0_144/jre
export PATH=$PATH:$JAVA_HOME/bin:$JAVA_JRE/bin
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar

3、让 proflie 文件生效

source /etc/profile

4、进入桌面模式，在 /opt/module/eclispe/ 文件夹中打开 Eclipse

$ /opt/module/eclipse/./eclipse

四、Eclipse 开发环境配置

  Linux 下 的 Eclipse 开发环境配置可以参考 Windows 下的配置。
  参考链接：https://www.cnblogs.com/chenmingjun/p/10036038.html
