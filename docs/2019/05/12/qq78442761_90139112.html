基本概念

Arduino中有一排是模拟输入：

Analog in。

这里不要在setup上面设置；

 

还有一个是电位器，他是长成这样的：

左右分别接地或者5v，可以转过来，因为到时候转动就可以了

中间那个连一个模拟输入，这样Arduino就可以获得硬件的数据了；

 

另外一个知识点是打印：

Serial.begin(9600);

为：设置比特率9600这样就可以得到串口数据了！

Serial.println(value);这样就可以打印了！

源码如下：

void setup(){

  pinMode(3, OUTPUT);
  Serial.begin(9600);
}

void loop(){

  int value = analogRead(A0) / 4; // 0~1023
  analogWrite(3, value); //value is 0~255
  Serial.println(value);
  delay(200);
}
