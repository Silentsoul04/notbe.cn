Oracle CDC撤编，SUN嫁给Oracle后还剩哪些家伙

May 12, 2019

1989 年，Oracle 正式进入中国市场，是第一家进入中国的世界软件巨头。2002年该公司在深圳建立第一家研发中心，随后在北京、上海、苏州、南京相继成立研发中心。最近惊现Oracle中国区研发中心(CDC)进行裁员撤编。Oralce裁员然而给我们并没半毛钱关系，但是Orace收购SUN后成为JAVA语言的拥有者，这却关系到我们这一类主要以JAVA混饭吃得码农们。Oracle收购SUN已经过去快十年了，借此机会来盘点一下曾经SUN拥有的产品的命运。

1、Solaris操作系统已死

Solaris操作系统已死，Solaris也称SunOS，Oracle收购SUN后就自然拥有Solaris。云服务大行其道的今天，用的都是CentOS Linux，一些刚入职场的小伙伴并不知道Solaris Unix操作系统。但是Solaris Unix以前却是与AIX、HP-UX齐名的三大UNIX操作系统之一。2016年，Oracle停止了Solaris的开发，至少50%的团队将在短期内被裁掉。将来不会有Solaris 12，Solaris 11.4成为绝唱，也不再会发布重大版本。

2、MySQL数据库发行老牛拉慢车

在2008年的时候，Sun宣布已经与MySQL AB达成协议，以大约10亿美元收购MySQL AB，其中8亿美元现金收购MySQL AB的全部股权，另外的2亿美元作为期权。本来 SUN公司是想发展自己的数据库的，但是事以愿违；收购MySQL两年后，SUN就被Oracle收购了，这样的话MySQL就变成了Oracle的产品了，因为Oracle数据库是闭源，而MySQL是开源的，因为两种文件的冲突，注定Oracle公司不会大力发展MySQL数据库，目前虽然MySQL数据库也在维护，但是发行版本特别慢，另外成立于2009年的MariaDB，MySQL之父Michael Monty Widenius用他的新项目MariaDB完成了对MySQL的反戈一击。开发这个分支的原因之一是甲骨文公司收购了MySQL后，有将MySQL闭源的潜在风险，因此社区采用分支的方式来避开这个风险。 过去一年中，大型互联网用户以及Linux发行商纷纷抛弃MySQL，转投MariaDB阵营。MariaDB是目前最受关注的MySQL数据库衍生版，也被视为开源数据库MySQL的替代品。

3、从此再无JavaEE

在Java科技社交数年怀疑与猜测之后，2018年Oracle公司终于宣布将把 Java EE捐赠给开源组织，换人来领导Java EE开发。

4、JDK开启收费模式

2018年4月Oracle就宣布，自2019年1月起Java SE 8公开更新将不向没有商用许可证的业务、商用或生产用途提供。你就收你的费，反正大把的代替品。像一些大公司都有自己的JDK，例如HP、SAP 都有自己的JDK。

Oracle 撒手不管，宣布对个人用户 (Personal Users) ， Java 8 官方支持时间持续到 2020 年 12 月；对商业用户（Commercial Users），2019 年 1 月之后不再提供免费更新。

红帽喊话，计划在 2023 年之前继续提供对 OpenJDK 8 的支持。

AWS 加入，正式推出 OpenJDK 长期支持版本 Amazon Corretto。

阿里巴巴也宣布重磅开源 OpenJDK 长期支持版本 Alibaba Dragonwell

Java 生态圈碎片化趋势越来越明显。对开发者而言，既不希望冒风险即刻更换，又希望可以使用具有持续安全更新和完备功能的 JDK。此时，阿里巴巴宣布开源 OpenJDK 长期支持版本。这是否可以打动坚守的 Java 8 群体并解决其燃眉之急？从阿里开源个大量项目中看，阿里是使用Java语言的大厂毋庸置疑。有理由相信阿里开源的Alibaba Dragonwell可以满足中小型企业的需求。

5、Java语言王者地位不再牢固

现在Java语言虽然还是处于榜单首位，但是并不像以前那样一直遥遥领先。最近两年，人工智能的发展，使得Python语言受欢迎的长度非常高；再加上Google因为Java侵权案要赔Oracle公司88亿美刀，加速了Google发展自己的编程语言，Google在Android方面大力推广使用Kotlin语言替代Java语言。Go语言的幕后大佬也是Google，目前国内也开始有不少公司使用Go语言了，例如上个月B站泄露的代码就是使用Go语言开发的。
