电商团购悬浮窗

May 12, 2019

哎，学个H5好难啊，这个东西里面的标签太多了；有点记不住了；

这个代码主要是抄的，但是全是自己打的；

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<title>电商团购悬浮框</title>
<link href="example13.css" rel="stylesheet" type="text/css" />
<style type="text/css">
	body,li,ul{
		padding: 0;
		margin: 0;
		list-style: none;
	}
	/*因为浏览器不同，所以要统一格式*/
	ul{
		width: 200px;/*表格宽度*/
		height: 270px;/*高度*/
		margin: 20px;/*外边距*/
		border: 3px solid #613e72;/*边框的 厚度，实线，颜色*/
		padding: 10px;
		/*内边距*/
		
	}
	body{
		font-family: "微软雅黑";/*全体字体*/
		font-size: 18px;/*字体大小*/
		
	}
	li{
		width: 142px;
		height: 40px;
		line-height: 40px;/*行高和li的高度相同时，可以使文字居中*/
		background: url(../img/clock.png) no-repeat left center;
		/*
		 * 这个地方触碰到了我知识的盲区，然后我通过反复试验，才弄懂了这行代码的大体意思
		 * 背景图片   平铺的方式    background-position: 水平左对齐， 垂直居中；
		 */
		padding-left: 40px;
		margin: 0 auto 5px;
		/*
		 * 这个其实是有4个参数的；
		 * 方向问  上  右  下   左  ，
		 * 3  个参数
		 *  上   左右    下  
		 * 2个参数  
		 * 上下   左右
		 * 这个和padding差不多的
		 */
		color: #613E72;/*字体的颜色*/
	}
	.item{
		background: url(../img/icon.png) no-repeat ;
		color: #fff;
		background-color: #613E72;
		background-position: 5px center; 
	}
	.back{
		background: url(../img/back.png) no-repeat left center;
	}
</style>
</head>
