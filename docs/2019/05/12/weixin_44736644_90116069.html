Linux基本命令

May 12, 2019

pwd查看当前位置
ps查看当前终端运行的进程
ps-ef显示所有进程
kill结束一个进程
kill-9强制结束一个进程
&程序在后台运行
bg %l后台运行程序
fg %l前台运行程序
jobs查看后台运行的程序
jobs-l详细信息
sleep睡眠n秒
halt关机
reboot重启
su切换用户，默认切换管理员
exit退出当前用户
ls显示当前目录下的文件
“ls -l”==“ll”显示文件的详细信息
ls -a显示隐藏文件
ls -al显示所有信息及隐藏文件
cd切换目录
cd .当前位置
cd …返回上一层目录
cd-切换至最近访问的目录
cd~切换到家目录
cd+相对路径/绝对路径
touch创建一个普通文件
mkdir创建一个目录文件
rmdir删除一个空目录
rm删除一个普通文件
rm -r删除一个目录
rm -f删除时不进行交互提示
cp拷贝普通文件　　----------------“源路径+名字　目标路径+名字"
cp -r拷贝目录
mv剪切/重命名
more查看文件内容
less反复查看文件内容
head默认显示文件前10行-n查看n行
tail默认显示文件后10行-n查看n行
cat打印文件内容至屏幕上
cat file1 file2 > file 合并至file中
cat >file创建file，并输入内容，ctrl+d结束
ctrl+c终止程序
ctrl+z停止程序
