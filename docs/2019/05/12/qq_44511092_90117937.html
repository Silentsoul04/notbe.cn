ubantu常用操作

May 12, 2019

1.移动
cd /usr/local
cd ~
cd …
2.创建
mkdir a
mkdir -p a1/a2
-mkdir -p参数是能直接创建一个不存在的目录下的子目录
3.删除空文件夹
rmdir a
rmir -p a1/a2/a3
-p级联删除
4.复制
sudo cp -r /tmp/test /usr
-r ：递归持续复制，用于目录的复制行为；
5.移动
sudo mv /usr/bashrc1 /usr/test
sudo mv /usr/test /usr/test2
6.删除
sudo rm /usr/test2/bashrc1
sudo rm -r /usr/test2
-r删除此目录下的所有文件
7.创建
touch hello
8.授权
sudo chown root /tmp/hello
查看:
ls -al
ls -l
pwd
cat、tac、more、head -n 20、tail -n 20
find ~ -name .bashrc
grep -n ‘examples’ ~/.bashrc
解压缩
sudo tar -zcv -f /test.tar.gz test
sudo tar -zxv -f /test.tar.gz -C /tmp
