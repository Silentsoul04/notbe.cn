获取短信验证码倒计时

May 12, 2019

开发工具与关键技术：后端
作者：汪利燕
撰写时间：2019年5月12日
嗨喽，可爱的我又来啦！
最近在做一个项目，项目里面肯定不能少了注册登录这一块的，所以我今天要说的是注册的时候获取短信验证码倒计时，因为注册的时候要用手机号注册，用手机号注册肯定要获取验证码，输入验证码之后才能进行登录。下面我就把这个方法写出来。（效果图）

点击获取验证码就开始倒计时，倒计时60秒开始一直到0秒结束，结束之后回到重新获取。
一、HTML

<body>
<form> 
  <label style="float:left;">验  证 码</label> 
     <input type="text" class="c_code_msg" style="margin-right: 10px;"> 
        <span class="msgs">获取短信验证码</span>
  </form>
</body>

二、CSS

<style type="text/css">
 form{ 
   margin:100px auto; 
   width:500px; 
 }
 label{ 
   font-size:16px;
   line-height:40px; 
   margin-right:10px; 
 }
 input{ 
   width:212px;
   height:38px; 
   padding:0 4px; 
 }
 .msgs{ 
   display:inline-block; 
   width:104px; 
   color:#fff; 
   font-size:12px; 
   border:1px solid #0697DA; 
   text-align:center; 
   line-height:30px;
   background:#0697DA; 
 }
 form .msgs1{//倒计时的秒数
   background:#E6E6E6; 
   color:#818080; 
   border:1px solid #CCCCCC;
 }
</style>

三、JavaScript

<script src="../js/jquery-3.2.1.min.js"></script>//js插件
<script>$(function () {
 //获取短信验证码
  var validCode=true; 
  $(".msgs").click (function  () {  
  var time=60;//秒数 
  var code=$(this);
     if (validCode) { 
        validCode=false;  
        code.addClass("msgs1"); 
   var t=setInterval(function  () {  
          time--;  
          code.html(time+"秒"); 
      if (time==0) {    clearInterval(t);  
           code.html("重新获取");    
           validCode=true;  
           code.removeClass("msgs1"); 
          }  
        }，1000)
      }
   })
 })
</script>

注意：一定要引用<script src="../js/jquery-3.2.1.min.js"></script>这个插件，如果不引用它就会报错。
