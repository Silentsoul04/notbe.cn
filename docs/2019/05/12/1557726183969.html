安装之前

在安装Jupytext前默认大家都装好了Jupyter notebook，这个很简单安装Anaconda很容易的，对了还有以下内容都是在win10下玩的~

安装Jupytext

安装Jupytext有两种方法：

A方法：

pip install jupytext -upgrade

B方法：

conda install -c conda-forge jupytext

建议使用A方法，因为在命令行中直接运行完了就能在Jupyter中看到了

当然如果使用B方法就需要先进入安装Anaconda的环境下，可以使用conda env list查看，如下图
进入到框出来的目录下，就可以使用B方法安装了，不然Jupyter里面不会出现Jupytext
