2.安装完后打开软件出现以下界面，点击Activate激活，显示激活失败后关闭软件；

3.重新打开软件后，再次点击Activate激活，出现下界面，选中“I have an activation code from Autodesk”；

4.打开下载的安装包中的注册机，64位电脑选中x64，右键选择以管理员权限运行程序；

5.打开后点击Patch，显示成功后点击确定；

6.接着复制步骤3界面中的申请码；

7.粘贴至以下栏目中，点击Generate；

8.复制生成的激活码;

9.粘贴至步骤3界面中以下框内，点击Next即可；

10.可以看到已经激活成功，点击Finish。
