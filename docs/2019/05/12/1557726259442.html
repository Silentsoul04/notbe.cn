May 12, 2019

一、ros空间的创立
1.创建一个工作空间
mkdir -p catkin_ws/src
2.编译
catkin_make
3.配置工作空间，刷新环境
source ~/catkin_ws/devel/setup.bash
注：每次catkin_make之后，都要刷新环境（source），很麻烦，以下方法可以一劳永逸
gedit ~/.bashrc
打开bashrc文件
source ~/catkin_ws/devel/setup.bash
将上面的话添加到文件末尾（因为每开一次终端，会刷新一次这个文件，所以可以直接在末尾加入这个命令）
或者直接输入命令：echo “source ~/tutorial_ws/devel/setup.bash” >> ~/.bashrc
二、ros基础指令
1.rospack
作用：查找某个pkg的作用
rospack find package_name
列出本地所有pkg
rospack list
2.roscd
作用：跳转到某个pkg路径下
roscd package_name
3.rosls
作用:列举某个pkg下的文件信息
rosls package_name
4.rosed
作用：编辑pkg中的文件
rosed package_name file_name
5.catkin_create_pkg
作用：创建一个pkg
catkin_create_pkg <pkg_name> [deps]
deps是包的依赖，比如rospy,roscpp
6.rosdep
作用：安装某个pkg需要的依赖
rosdep install [package_name]
