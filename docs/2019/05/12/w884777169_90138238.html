git学习之路

May 12, 2019

本文以码云为例讲解一下git的用法。

首先我们需要在码云上建立一个库，这部分我不讲解了，大家可以自己尝试一下。

下面我直接讲git的使用：

  1. 首先我们需要在自己的硬盘中创建一个文件夹，作为存储我们数据的本地库。
  2. 打开可以运行git 命令的软件。我这里使用的是git shell
  3.  
  4. 输入指令到达“本地库目录” 
    cd d:
cd D:\specimenBig\

  5. 输入指令 git init 初始化 本地目录,运行后我们可以看到在specimenBig目录下出现了一个.git 文件（这个文件默认是隐藏的）

  6. 本地代码仓库与远程代码仓库进行同步，执行代码：

    git remote add super626 git remote add super626 https://gitee.com/super626/specimen.git
    其中远程库的名字就是super626(我自己取得名字)，也可以改成别的。以后远程链接库就是名字就是super626.
https://gitee.com/super626/specimen.git 后面的地址是你要同步的远程连接库的url。如下图所示：

  7.  运行 git clone https://gitee.com/super626/specimen.git    再查看原本目录你会发现它有变化。但是这样并不会把所有的分支下载下来。

8.如果还存在分支，请运行  git pull super626 dev 这样你就可以在specimenBig中，看见需要下载的文件了。

 

dev 是一个分支的名字。
