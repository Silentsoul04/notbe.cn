程序员2号

控制面板.{ED7BA470-8E54-465E-825C-99712043E01C}

放在桌面，至今没有出过车祸。

懂的都是十年以上的司机了。

有人问该怎么打开，可以直接输入路径，也可以用压缩文件，或者ie打开。

补充：这是win7的一个彩蛋，把文件夹加上后缀能变成一个控制面板一样的东西，该文件夹无法通过双击等正常方式打开。所以适合藏东西。你自己试试就知道了。
