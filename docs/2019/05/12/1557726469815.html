Ubuntu16如何安装32位兼容库,即ia32-libs

May 12, 2019

Ubuntu16如何安装32位兼容库,即ia32-libs ，安装遇到一些问题

1.sudo apt-get update

2.sudo apt-get install lib32stdc++6 lib32z1 lib32ncurses5 lib32bz2-1.0

ubuntu使用”apt-get install“安装程序时遇到错误提示

   You might want to run 'apt-get -f install' to correct these:

The following packages have unmet dependencies:

 lib32asound2 : Depends: libc6-i386 (>= 2.7) but it is not going to be installed

使用如下命令修复安装包

sudo apt-get --fix-broken install

3.继续安装

sudo apt-get install lib32stdc++6 lib32z1 lib32ncurses5 lib32bz2-1.0

之后就可以了。
