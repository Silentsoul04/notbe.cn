必备工具篇

  1. 系统工具
    安全优化: 360和电脑管家两个都用过，360用习惯了选装360.(但软件管理更喜欢电脑管家的)
    输入法: 搜狗和qq输入法(无广告确实算亮点选qq输入法)
    下载器: 迅雷精简版，freeDownloader(一个下torrent,一个下磁力链)
    解压工具: bandzip(免费好用，用习惯了)
    视频播放器: potplayer(优秀)
    音乐播放器: 网易云,酷我(都用)
    浏览器: chrome,firefox(国内版不然登录不了)(默认用谷歌，fq浏览网页用火狐快捷)
    云盘: 百度云，微云(都用)
    文档三宝: 无MSoffice前提下用wps纯净版
    记事本: notepad++(就用这一个就够了)
    聊天工具: TIM,微信电脑版
    虚拟机: vmware14.0
  2. 开发工具
    数据库: mysql,pgsql,navicat,studio3T,mongodb(vm),redis(vm)
    IDE: idea(默认),pycharm,eclipse(备用)
    版本控制: git
    远程: teamview,xshell6,winSCP
    搜索: everything
    其他工具: tomcat,maven&nexus,jdk,nginx(vm),kafka(vm),zk(vm)
    系统镜像: ubuntu16.04,centos7(默认用centos)
