虚拟机 Ubuntu安装gcc和g++

May 12, 2019

默认安装Ubuntu是没有带gcc和g++的，需要自己安装。

  1. 首先需要知道root的密码,方法为在普通用户的客户端下：sudo passwd root  ,重新设置root密码，因为Ubuntu刚安装时不知道root的密码的，所以需重置。
  2. 使用su 切换到root登录使用终端。
  3. Ubuntu 是使用apt 该命令下载安装程序的，该过程需要联网。用apt install gcc 和apt install g++ 安装gcc 和g++。
  4. 用gcc -v 和g++ -v。来判断gcc和g++的版本是否安装成功。
  5. 扩展：其他编译命令如vim 也是这样的操作处理a安装。(apt install vim )
