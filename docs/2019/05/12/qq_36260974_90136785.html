MyEclipse 集成 Tomcat

● MyEclipse 配置服务器

  • 使用 MyEclipse 配置服务器后，就可以使用 MyEclipse 来启动和停止服务器了。当然，你需要先安装好服务器（Tomcat），才能配置。
  • MyEclipse 自带了一个 Tomcat，但我们不会使用它。所以，我们需要先把 MyEclipse 自带的 Tomcat关闭，然后再来配置我们自己的 Tomcat。
    • 关闭 MyEclipse 自带 Tomcat。
      • 在 Servers 窗口当中选择 MyEclipse Tomcat，右键选择 Configure Server Connector。
      • 选择 disable，最后先应用再 ok！

    • 配置我们安装的 Tomcat。
      • Server 窗口空白处右键。
      • 按下图说明操作

    • 使用 MyEclipse 启动 Tomcat
      • 右键选择 Run Server 来启动服务
      • 此时我们可以观察到 Console 输出了一些日志信息
