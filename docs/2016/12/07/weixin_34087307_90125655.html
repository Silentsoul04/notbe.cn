打印楼梯，同时在楼梯上方打印两个笑脸

Dec 7, 2016

题目：打印楼梯，同时在楼梯上方打印两个笑脸。
1.程序分析：用i控制行，j来控制列，j根据i的变化来控制输出黑方格的个数。
2.程序源代码：

 1 #include<stdio.h>
 2 int main()
 3 { int i,j;
 4   printf("\1\1\n");     //打印两个笑脸
 5      for(i=1;i<11;i++)
 6      { for(j=1;j<i;j++)
 7        printf("%c%c",23,23);   //23这个数字可以是任意数，代表该数所对应的符号
 8        printf("\n");
 9       }
10   return 0;
11 }
