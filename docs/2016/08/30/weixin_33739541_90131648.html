AutoCAD Map 3D 2011 Update 1 – released!

Aug 30, 2016


In our continuing effort to improve quality, performance and respond to our customer’s needs, we have released AutoCAD Map 3D 2011 Update 1.

Downloads

§ http://usa.autodesk.com/adsk/servlet/ps/dl/item?siteID=123112&id=15741597&linkID=9240858

Changes Included in this Update

AutoCAD Map 3D 2011 Update1 includes AutoCAD 2011 Update 1 and also resolves the following Map 3D 2011 related issues:

Display Manager

  • Loading a .layer file created by Topobase and then closing the drawing crashes AutoCAD Map 3D.
  • Running the REGEN command after undoing the delete of a topology element crashes AutoCAD Map 3D.
  • FDO raster layers containing elevation data do not respect scale ranges.

FDO Create / Edit

  • Selecting a large number of features stalls AutoCAD Map 3D.

FDO Oracle

  • Crash occurs when memory runs out due to continued use of previews. 

FDO Raster

  • Transparency fails with PNG 8 files. 

FDO WMS

  • Enabling the “Cache WMS data” option in the Data Connect dialog after connecting to a WMS service crashes Map.
  • Connecting to a WMS service fails with an unhandled exception while trying to read the WMS schema.

Install/Uninstall

  • An error message is displayed when installing AutoCAD Map 3D 2010 Update 1 on a non-English version of AutoCAD Map 3D.

Raster Design

  • AutoCAD Map 3D crashes upon exiting when Raster Design is invoked within the session. 

OSNAP 

  • OSNAP sometimes fails to work when digitizing a line by snapping to points. 
  • Grip stretch is cancelled when snapping to features symbolized by blocks.  
  • OSNAP sometimes fails to snap to features symbolized by blocks.

Topobase Interoperability

  • Memory not released when TBexporter.arx is loaded and used in AutoCAD Map 3D.

Topology

  • The Attribute option of the MAPTHEMTOPO command doesn't work when running from the CLI.

Transform/Rubbersheet

  • The ADERSHEET command mishandles MultiLeader and MLine entities.

UI

  • Selection stops working after selecting features repeatedly within a session.  

Vector Symbolization

  • Crash occurs when adding custom hatch styles to a Map. 

Please check Readme for installation instructions and other Update 1 details.
