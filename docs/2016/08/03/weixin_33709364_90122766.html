MVC==>Model View Controller 

Model ====>模型就是要去显示，保存，创建，更新和删除的对象（实体）

View =====>视图是用户看到并与之交互的界面

Controller==>控制器是对数据的处理以及为View提供输出数据（MVC模型中控制器是真正核心，每个请求必须通过控制器处理【有些是不需要模型和视图的】）
