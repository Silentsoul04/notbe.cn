PDF.NET 开发框架之 SOD框架 Ver 5.2 正式版开源源码发布

Jan 19, 2018

PDF.NET 开发框架之 SOD框架 Ver 5.2.1.0307 正式版发布，包含以下部分：

SOD_Pwmis.Core

--包括下列数据提供程序

  1. SqlServer
  2. SqlServerCe
  3. Access
  4. OleDb
  5. ODBC
  6. Oracle

--包含框架的核心类库

PDF.NET SOD All

--包括框架的全部类库和数据提供程序，目前有

  1. SQLite
  2. MySQL
  3. PostgreSQL

PDF.NET SOD AllSource

--包括全部类库源码和示例程序源码，包括超市管理系统源码

PDF.NET SOD 文档

--包括下列文档：

  1. _NET ORM 的 “SOD蜜”--零基础入门篇 - 深蓝医生 - 博客园
  2. PDF_NET SOD 开源框架红包派送活动 && 新手快速入门指引 - 深蓝医生 - 博客园
  3. PDF实体类效率测试
  4. PWMIS.Core.chm
  5. 版本说明
  6. 解决方案说明
