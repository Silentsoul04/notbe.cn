Shortcut 常用快捷键

Dec 4, 2017

多行注释：

VS2010:　　　　　  　/

NotePad++:　　　 　Ctrl Q

Xcode:　　　　   　　CMMND /

回到光标所在之前位置

VS2010: 　　　　　　Ctrl +/-

关闭当前页：

VS2010:　　　　　　鼠标中键

NotePad:　　　　　　Ctrl W

选中当前词：

VS2010:　　　　　　Ctrl W

复制当前行

VS2010：　　　　　　Ctrl C

NotePad++:  　　　　Ctrl D

快速查找

VS2010：　　　　　　Ctrl F3　　　　/　　　　Shift F3

跳转到指定行

VS2010:　　　　　　Ctrl G

删除当前行的开头到光标处的所有字符:

VS2010:　　　　　　　　Ctrl Backspace

Notepad++：　　　　  Ctrl Shift Backspace

删除光标处到当前行末尾的所有字符

VS2010:　　　　　　　　Ctrl Delete

Notepad++：　　　　  Ctrl Shift Delete

格式化代码

VS2010:　　　　　　　　Edit -> Advanced -> Format Selection

Notepad++:　　　　　TextFX -> TextFX Edit -> Reindent C++ Code

缩进：

VS2010/Notepad++　　tab / shift+tab

Xcode　　　　　　　　　cmd+] / cmd+[
