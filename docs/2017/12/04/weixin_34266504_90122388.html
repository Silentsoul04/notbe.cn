解决方法：

 菜单 ---> Windows ---> Preferences ---> MyEclipse ---> WebLogic8 ---> JDK ---> Optional Java VM arguments

添加以下参数： -ms256m -mx512m

说明：分配最小内存256M，最大512M，即可解决问题,当然你也可以设置的更大
