开源的Spread表格控件介绍

Dec 8, 2017

Spread COM 表格控件功能介绍

作为开源的Spread表格控件来说，这款控件实在是凝聚了许多汗水和心血

以下就让我帮大家介绍一下这款优秀的表格控件好了。 

帮助更多用户能够了解它并且在实际开发过程中使用到它。

从而得到方便和实惠。

1. 查询语句建立

Spread表格控件有12中数据类型帮助用户建立有效的查询语句, 根据数据库字段类型,Spread可以设置相应的列类型, 让用户仅可以输入相应类型的数据。

2. 树浏览

spread表格控件在数据展示上有极大的灵活性, 用户可以用非常简单的方式展示各种定制数据。

3. 打印/打印预览

Spread表格控件 有很多内置的打印功能, 能够帮助用户轻松的预览和完成打印。

4. 添加数据

每个应用程序的核心需求都是需要从不同的数据源读取数据。无论数据源是来自ODBC数据库（Microsoft Access, SQL Server, 或其他）， 数据文件，Microsoft Excel 97/2000文件， 或者其他数据源， Spread都能够读取并展示。

5. 绑定数据模式

Spread表格控件能够从多个数据库中自动读取数据， 并通过使用ActiveX Data Objects (ADO) 2.0， OLE DB technology，Data Access Objects (DAO)，或Remote Data Objects (RDO)与之连接。 当数据源被连接后， Spread可以展示用户所需的信息；包括读取和展示字段名称来建立列标题；建立所需的列；根据数据创建对应的字段类型，来确保数据在修改时的匹配； 展示对应格式的需求数据； 无需写任何代码， 便可以让用户通过程序屏幕更改数据；使用标准的导航查看数据；添加新的列或行。

6. 非绑定数据模式

Spread表格控件在连接数据库的功能上非常强大， 同样Spread在运行时从其他数据源上载入数据的功能也同样很高效。
除了常见的主要数据源外， Spread还可以通过简短的程序加载其他数据文件， 包括：Microsoft Excel 97 worksheets， Microsoft Excel 2000 worksheets，.SS2/3文档，Comma-delimited文档，Tab-delimited files，Custom-delimited文档。

7. 可视化模式

假设用户有300000行数据需要展示在表格中， 在内存中读取与展示需要较长的时间， 这样会让不停的等待用户。
使用Spread虚拟模式， 可以加快数据展示的时间， Spread表格控件支持两亿行和两亿列数据。

8. Spread设计器

Spread Designer用来设计并快速创建一个表单原型。通过其直观、易用的界面，在设计阶段对表单的外观进行定制，从而缩短开发时间。Spread Designer可为表单创建一个快照。当所有的更改完成后，所有可直接应用于工作表单。Spread Designer支持打开已有设计文件并将设计更改保存为文件。
