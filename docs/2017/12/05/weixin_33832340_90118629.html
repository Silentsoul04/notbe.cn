微软将在夏季公开发行Internet Explorer 8 Beta 2

Dec 5, 2017

尽管现在任何人都可以下载测试版IE 8,微软对它的定位则是面向Web开发人员.微软IE浏览器部门的总经理哈查莫维奇向CNET News.com表示,但是,计划于今年夏季发布的β2版IE 8则面向更大范围的用户.
据国外媒体报道称,哈查莫维奇在谈到本周三发布的β1版IE 8时说,它是公开的,被放在了微软网站上,任何人都可以下载它.他表示,尽管Web Slices等特性可能对消费者有吸引力,当前的β版IE 8是面向开发人员的.

微软没有披露正式版IE 8的发布时间.哈查莫维奇也没有披露IE 8是否会与微软的新一代操作系统Windows 7同时发布.他指出,XP版IE 7的发布时间领先于Windows Vista,因此从技术上来讲IE 8的发布时间也可能领先于Windows 7.

β版IE 8能够在32位和64位版本的Windows Vista、Windows Vista SP1、Windows XP SP2、Windows Server 2003、Windows Server 2008上运行.

哈查莫维奇表示,在设计IE8时,微软同时兼顾了与标准的兼容和增添新功能.
