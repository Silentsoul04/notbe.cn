知方可补不足~数据库名称和数据库别名不同了怎么办

Dec 7, 2017

当你的数据库安装成功后，你修改了计算机名称，这时，你的sqlserver名称与别名可能就不相同了，当这两个名称不相同时，你再使用sql的发布与订阅功能时，将会被提示出错，“请使用服务名称，不能使用别名和ＩＰ地址”，下面是具体解决方法

use master
go
select @@servername;
select serverproperty('servername')

得到的结果表明，这两个名称确实不一样

然后我们把老的名称删除，换成现在的别名

sp_dropserver '2011-20131105UM'
go
sp_addserver '2011-20130123DR','local' 
sp_helpserver

然后重启ＳＱＬＳＥＲＶＥＲ，然后再进行查看就ＯＫ了

这两个名称相同后，我们的发布/订阅服务就可以用了，呵呵
