活字格企业Web应用生成器V3.0发布更新，支持插件管理和多人协作开发

Dec 7, 2017

可视化设计，快速生成企业Web应用！日前，活字格发布V3.0 更新版（V 3.0 Update 1）。本次更新增加了插件管理，支持多人协作开发，丰富了表格中的单元格类型，扩展了开发功能，对活字格的易用性和扩展性进行了提升。点击此处，下载最新版本

活字格是一款企业 Web 应用生成器，使用了类 Excel 的设计界面，只要你会 Excel，通过简单的拖拽操作，就能快速制作出一个 Web 信息管理系统。无需专业软件知识，没有任何技术门槛，轻松实现各行业的数据信息管理，降低你的管理成本，提升你的工作效率。

这是一款真正面向业务人员的、“无技术门槛”的企业信息系统创建工具。使用活字格，企业可对现有数据进行集中管理，摆脱用Excel管理数据的种种限制；还可根据业务需要，高度自定制业务逻辑，实时应对需求的变化，不依赖第三方软件开发商；并能对系统的数据权限做精准控制，灵活配置不同的业务角色。

大到业务全流程管理系统，小到办公用户管理系统、快递收发系统、面试系统，企业都可即做即用，一键发布，实现多人协作，降低管理成本，提升工作效率。

伴随本次更新，活字格还发布了5个全新的演示Demo，包括图书管理、设备管理、员工管理、教务管理、会员管理等，用户可在线查看演示，快速体验活字格易用、灵活的产品特性；也可下载源文件，根据企业业务情况进行修改，即可使用。点击此处，查看演示

 

主要更新内容：

  • 新增插件管理：可添加插件，扩展系统功能，提供菜单、树结构、地图，微信消息等插件

  • 支持多人协作开发，工程文件可以进行版本管理

  • 丰富了表格中的单元格类型，支持图片、二维码和附件等复杂单元格类型。

  • 增强数据安全性，页面默认用户必须登录才能访问。

  • 扩展开发功能，在系统内可直接创建JS、CSS的文件，并进行管理。

  • 提升易用性

　　　　-- 外联数据库支持工作流

　　　　-- 公式单元格值可以进行编辑

　　　　-- 命令可以进行临时禁用，方便调试

　　　　-- 支持重置自动增长编号

　　　　-- 普通单元格式支持条件设置

　　　　-- 权限设置更细化

 

全新发布的演示系统：

系统分为“图书管理员”和“读者”（移动端）两个角色。图书管理员可以进行系统基础信息设置，图书的信息及借阅管理，读者的信息和充值情况的管理。读者通过移动端，可查看自己的借阅信息，包括当前借阅图书、历史借阅情况等。


设备管理系统

可对企业所有的设备进行统筹管理，包括设备的基本信息管理、维修管理、备件管理、文档管理等，并提供设备折旧等一系列报表，为企业管理和利润核算提供参考。

 

员工管理系统

通过本系统，可实现企业员工信息的电子化管理，不仅可对员工的基本信息进行分类记录，还可进行员工信息的精准或模糊查询，并提供多张统计报表，便于管理者随时掌握企业人力资源情况。
