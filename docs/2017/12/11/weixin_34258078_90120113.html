sql 语言中 when case 用法

Dec 11, 2017

sql语言中有没有相似C语言中的switch case的语句？？

没有,用case when 来取代就行了.

比如,以下的语句显示中文年月

select getdate() as 日期,case month(getdate())
when 11 then '十一'
when 12 then '十二'
else substring('一二三四五六七八九十', month(getdate()),1)
end+'月' as 月份
