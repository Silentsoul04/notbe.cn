box(装箱)消耗大

box在堆栈中创建一个新的对象，性能消耗大

int i = 123;
// Boxing copies the value of i into object o.
object o = i;

示意图

 

 unboxing(拆箱)

  1. 检查对象实例
  2. 将该值从实例复制到值类型变量中

 

int i = 123;      // a value type
object o = i;     // boxing
int j = (int)o;   // unboxing

示意图

 

 总结

装箱和折箱都会引用性能的消耗，如：ArrayList使用泛型集合：(List<T>) 来避免box
