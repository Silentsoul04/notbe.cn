Autodesk Infrastructure Modeler (原Galileo伽利略项目)已经正式发布

Jan 3, 2017


如果你还记得以前的Galileo伽利略项目，现在有个好消息，伽利略项目已经发布，正式命名为Autodesk Infrastructure Modeler，简称AIM，注意不要和AIMS（Autodesk Infrastructure Map Server）弄混了  

 更新2013-05-10 ：  Autodesk Infrastructure Modeler(AIM) 已经更名为Autodesk InfraWorks(AIW).

AIM的官方站点是http://www.autodesk.com/infrastructuremodeler 大家可以到这个站点下载试用版来玩玩儿，注意操作系统要求vista或者win 7， 还在用XP的童鞋们得升级了。

 

Infrastructure Modeler 到底是干什么的？  和Autodesk的比如Navisworks，3ds Max等其他其他三维产品有啥区别？

Autodesk Infrastructure Modeler（AIM）是基础设施项目的概念设计工具，能帮助你在基础设施的设计时以自然的思维环境下进行。其他产品比如Revit和Civil 3D可用于在AIM的初步概念设计的基础上进行进行更精细的设计。3ds Max可以用于为在AIM中的设计结果创建更逼真的渲染效果和动画， Navisworkd可以提供详细的可视化表达，碰撞检测和项目管理工具等。Autodesk Infrastructure modeler还提供了从2D数据生成3维场景的能力。

更多的介绍可以参考http://www.autodesk.com/infrastructuremodeler 和常见问题解答FAQ (pdf - 370Kb)
