2017帝都租房攻略：昌平通州租金涨幅高达25%

Jan 3, 2017

帝都去年房价涨得太吓人，对于我们这些北漂，就算买不起房，也要租到称心如意的房子！

沙漠之鹰在2016年春节之后写过一篇《2016年北京租房攻略》（强烈建议阅读），反响还不错。一年过去了，是时候写一篇2017年的攻略了！为了避免重复，先划出2016年文章的重点，除了平均价格上涨15%，其他结论今年基本适用：

  • 整租比单间性价比高很多，单间15平性价比最高。整租60-90平米平均租金最便宜
  • 中关村，金融街，小西天，东直门是北京租房最贵之地，如果价格敏感则远离大学和商业区
  • 单间南向比北向平均贵400，楼层2层最贵，1层最便宜，3-6层基本无区别
  • 1公里之内，65平房子每离地铁近100米，租金涨100元。

2016年北京五环内的房租价格热力图（这张图依然能用是因为五环内租金变化不大）

房租涨了多少？？

这一定是你关心的问题，这样房东涨价时心里就有谱。哪些地方涨得多？下面的两张图能够回答你：

对比两年的数据我们可以发现，通州以租金上涨28.8%位于榜首，其北京副中心的影响表现得非常明显。而昌平也紧跟其后位于第二，主要利好是昌平线于2016年年初正式运行，且海淀北部的高科技园区吸引了大量码农。五环外活跃度明显提升，最热门区域是回龙观、北苑和上地，尤其是回龙观，2016年租房交易量增长了接近44.6%，量价齐涨。反而是中心城区的租房活跃度下降，公主坟等区域的租房量甚至持续下跌。东西城，朝阳海淀四个区的供给百分比已经从2016年的68%，降到了60%左右。

上面的数据意味着，昌平区原先4000元的房子涨到了5000元，涨后均价已经接近朝阳海淀。也许你觉得25%的涨幅太多了，但若知道全北京房价普涨59%(参考本文)，租金的涨幅就表现得非常温柔了。

下面的图可以直观地看到租房总价(元/月)和存量的关系：

因此，租单间可选价位最多的区间在1800-2200之间，一室一厅整租在4000-4500之间，小户型两室一厅(65平米左右)在5100-5700之间。这样就能对预算有比较好的判断。

到底该住市区还是郊区呢？这是个问题。

主要地铁线的价格分布?

上班族基本是地铁党，能尽快坐上地铁乃是人生幸事，我们先看几条主要地铁线沿线的租房价格。

数据都提取自地铁站周围，步行距离1公里之内的小区的平均租房单价(元/平米/月)（北京最怕过天桥，沙漠君为了算出步行距离也是拼了），由于数据稀疏，数据仅供参考，不是每个地铁站边小区都有房子出租的。

先看大家最需要的13号线：

五道口果然是宇宙中心，知春路周边大量公司都推高了房价。你可能疑问为何望京西也这么贵，因为它周边有几个全北京都很靠前的高端盘。柳芳和光熙门可认为是价格洼地，那边有大片的住宅小区。

接着是非常拥堵的六号线：

从慈寿寺到呼家楼的价格都很贵（毕竟是三环内，而且路过CBD），一出金台路，十里堡之后，有大量成片住宅区，租金就掉了下来，不过据说这一段地铁挤得相当酸爽...

2号线则比较平均，依然是朝阳门价格最高(CBD加成太强了)，由于不连接居住辐射区，多数人仅用作换乘，也不拥堵，因此二号线就闭着眼睛租吧：

再看大家最喜闻乐见的10号线，一定远离海淀黄庄，如果公司在北边，安贞门是个不错的选择，西土城站租金低是因为附近几乎没出租屋，尤其是内环车，几乎没人和你抢，南边上班则首选西局泥洼首经贸（锻炼大家的颈椎~）

最后再看看4号线,整个北京的南北差异表现地淋漓尽致，该在哪里租就不需要沙漠君说什么了吧。

居住条件or通勤距离

另外放在每个上班族同学面前的问题，居住条件和通勤距离怎么选择
？
而北京地铁每站相隔运行时间大概为4分钟，考虑到距离多远一站耗费的不只是运行时间，这里按照6分钟计算，多远一站意味着每个月需要多花5220=200分钟。假设在路程上的时间都是无谓损失，这意味着多远一站每个月将在无形中损失0.52*200=104元。上面这条经验规律正是距离和时间成本在租金上的反映

经验规律：在核心区的地铁辐射线上（如八通线），每靠近核心区一站在同等条件下月租金会多150元/月。这条规律非常符合通过计算得出的结果。北京大学生白领平均月薪为5000元左右，按每天工作8小时算每周工作5天算，每分钟的时间价值为5000/(86020)=0.52元。

不少同学可能忽略了一个事实，时间也是金钱：假设月薪八千元，则时薪接近五十元。地铁每站间隔五分钟(加上停站时间)，每天多花一个小时在通勤，则相当于支付时间成本1千元。当然，每靠近核心区一站，房租会多花150元。但是考虑到花在地铁和通勤上的时间，基本只能通过看剧发呆来打发，尤其是拥挤不堪地到了公司，对一天的工作状态都是有严重影响，到了周六日，也不愿意再挤地铁进城。

如果想要感受北京这座城市，就最好不要住在郊区。如果你下班之后主要是娱乐和睡觉，那么在通勤时间上多花一些时间也无可厚非，不过若希望能花更多时间用在自我提升，比如学习健身，笔者更建议选择靠近工作单位的租房位置，牺牲一部分面积获得更高质量的时间分配。这样能极大程度地提升幸福感哦~

建议

你可以在公众号后台留下邮箱，沙漠君会发给你全北京2017年所有小区的租金/租售比/房价/租房供给等详细信息。

从全国数据来看，北京房租最高，上海次之。因此在北京奋斗，就天然地开启了hard模式。下面是沙漠君给出的一些建议：

  • 远离黑中介，警惕二房东，不要贪图便宜，尽量选择靠谱中介，费用可控，后期退还押金也更有保障
  • 在豆瓣等自主发布租房信息的平台上，注意区分中介和真实房东，这里遇到的中介可能会不靠谱
  • 在与房东谈价时，适时证明职业优越性和稳定性，更能赢得房东信任，获取更优价格
  • 优先选年轻房东，年长者喜欢斤斤计较(亲历)
  • 从经济角度看，买房显然比租房更合适，帝都房价抗跌性不容置疑
  • 不要太惦记免佣金，丁丁租房是链家旗下的租房平台，仅仅一年时间就倒闭了。
  • 不论如何，交钱前一定要查看房主产权证明，房主身份证。以及两证的名字是否一致。

北京租房市场是相当有周期性的，过年时租房市场陷入低谷，之后紧接着便是租房最热季：研究生毕业，租房换房者，务工返京潮都推动租房市场量价齐涨。

从全年的数据来看，三月初租房选择性最多，到了三月底，可选择的合适房子就变少了，房价也最高，而八九月份的房租和供给都是全年最低的。如果你目前需要换房，就趁着现在去看房子吧。

后记：估算全北京有多少人在租房？

北京五环内有450万套住房，五环外有150万套。常住人口2200万，外来人口800万。中介网站常年有一万套左右的房屋出租，半衰期（一半被出租后下架）的时间为5-7天，二手房交易有一半不通过中介。通过以上数据，我们大概估计：有一半的出租不通过中介，估算有200万左右的出租房，五环内120万套，五环外约80万套。平均每4-5人住一套房子。北京除了个别高端盘不出租之外，绝大多数房子应该都住人，空置率应该非常低，估算接近1-2%左右，接近5-8万套左右。（你觉得我猜得对吗？）
