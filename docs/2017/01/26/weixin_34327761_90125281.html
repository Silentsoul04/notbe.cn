[Mac OS] Homebrew简介及安装wine

Jan 26, 2017

Homebrew官网 http://brew.sh/index_zh-cn.html

Homebrew是神马

linux系统有个让人蛋疼的通病，软件包依赖，好在当前主流的两大发行版本都自带了解决方案，Red hat有yum，Ubuntu有apt-get

 神马，你用mac os，不好意Mac os木有类似的东东，泪奔中几经折腾总算找到了第三方支持：Homebrew，Homebrew简称brew，是Mac OSX上的软件包管理工具，能在Mac中方便的安装软件或者卸载软件，可以说Homebrew就是mac下的apt-get、yum神器

Homebrew安装

Homebrew的安装非常简单，打开终端复制、粘贴以下命令，回车，搞定(请放心使用，原汁原味的官方安装方法搬运）

-------用于安装homebrew-------

1. 打开Terminal.app;

2. 输入：

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

3. 回车键执行...  

 

-------homebrew自检-------

1. 输入：

brew doctor

2. 回车键执行... 

 

-------使用homebrew安装wine-------

1. 输入：

brew install wine

2. 回车键执行...

 

-------安装缺失组件-------

1. 浏览器访问https://xquartz.macosforge.org/landing

2. 下载安装Xquartz

 

-------重试：使用homebrew安装wine-------

1. 输入：

brew install wine

2. 回车键执行...  (网速不好的同学可以先洗洗睡了，its a long time.)

 

检查wine安装状态

wine --version

使用wine打开exe程序

wine filename.exe
