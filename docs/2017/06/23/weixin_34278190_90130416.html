Ubuntu离线安装VSCode（附带前期准备工作）

Jun 23, 2017

跨平台系列汇总：http://www.cnblogs.com/dunitian/p/4822808.html#linux

补充一个小知识点：Ubuntu的root默认是没有密码的，你切换到root也是需要密码的，这时候不要慌

sudo passwd root

设置一下密码即可

在说正式步骤前先把准备工作做到位：

1.IP设置，这个因为是GUI的，手动设置一下就好了，如果想知道命令行的方式看这篇文章：（http://www.cnblogs.com/dunitian/p/6658578.html）

2.SSH远程连接的问题

先安装一下SSH，之前说过2遍了，这边就不说了

然后XShell就可以连接了

成功

3.FTP远程连接的问题

选SFTP，TFP的模式Unubtu是连接不上的，可能是出于安全考虑

设置一下编码，不然连接后中文会乱码

成果了

-----------------------------------------

准备工作到位了，下面就步入正规

下载安装包

 

ftp上传到Unubtu上

安装deb包，基础命令不了解可以去这看：（http://dnt.dkill.net/dnt/）

dpkg -i xxx.deb 安装deb的包

apt-get install -f 安装相关依赖

apt-get install --fix-broken 尝试自动修复

这时候就收工了，真的收工了，搜索一下vs

固定一下位置，我这边已经固定了

收工，要是要装C#，Python，Go，C++的扩展就在VSCode里面直接点击安装即可

 扩展：

dpkg -r 不删除配置文件的卸载方式

dpkg -P 删除配置文件的卸载方式
