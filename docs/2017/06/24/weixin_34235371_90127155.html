Python3之Django框架搭建详细步骤

Jun 24, 2017

安装Django

自行下载的pip,可执行如下命令:

pip install django

下载python3版本可以自带pip3 ,命令如下:

pip3 install django

此命令会下载django最新版本,若想指定版本,请执行如下命令:

pip3 install django==版本号

或者自行下载django,通过命令行进入到django文件目录下,执行如下命令:

python setup.py install

开始创建项目

按顺序执行如下命令

编辑myTestDjango文件夹中的settings.py文件,找到该对应代码,在最下方追加’myDjango’

INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'myDjango'
]

通过命令行启动
1、通过命令行进入到myTestDjango文件夹下
2、执行如下命令

python manage.py runserver
