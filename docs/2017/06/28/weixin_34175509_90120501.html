如何申请Autodesk ReCap 360 photo的云币（Cloud Credit)

Jun 28, 2017


在之前的博客中我介绍过Autodesk的照片建模云服务—Autodesk ReCap 360 photo，通过Autodesk ReCap 360 photo，你可以非常方便的通过照片生成三维模型。如果你感兴趣可以到https://recap360.autodesk.com 上免费注册一个Autodesk ID试试看。免费的Autodesk ID只能用于生成预览（Preview）级别的模型，精度不是很高。如果你需要高精度的模型，那你需要在生成时选择“ultra”，每生成一个模型，需要耗费5个Cloud Credit（约合5美元）。

 

那么Cloud Credit从哪儿来呢？ 有聪明人所了，购买！对了，你可以从Autodesk商店上购买，不过这不是重点。

 

还有一个好消息就是，对于ADN用户，可以免费得到2500个Cloud Credit，这可是不小的一笔钱呢。ADN会员可不要错过啊。那ADN会员如何申请Cloud Credit呢？

 

1. 登录到下面网址申请ADN的软件和web服务（需要使用你的ADN账户登录） 。 没有账户？问你们老板。老板也不知道？那你们公司多半还不是ADN会员。一会儿介绍怎么加入ADN。

http://adn.autodesk.com/adn/servlet/index?siteID=4814862&id=22817699

然后点下面红框中的链接。下面的说明也说了，有些web 服务不需要填套数和安装方法，你只需要把你的需求用英语写清楚就行了，会有人做人工审核。

2. 点完链接，又是一个按钮，唉，怎么能这样呢，直接进不就好了吗，不过好吧，为了免费的2500刀，忍了吧。对了，上面说了，对于2013年9月后的所有Autodesk产品的license申请，都可以在这里做哦~

3. 然后弹出一个窗口，第一项就是选产品，我们想用ReCap 360，那么就选Autodesk ReCap 360.

4. 然后填套数和安装方法，上面说了可以不填，但有人说不填过不去，那就随便填一下吧。

5. 下面是填你的联系方式，姓名，电话，电子邮件，公司名什么的，如实填写就行了。

6. 最重要的来了，在下面的comments窗口里要详细描述一下，你需要哪个服务，想用来干什么，你的ADN ID和Autodesk ID是什么等等。

比如我想要Autodesk ReCap 360 Photo，那我可以这么写，

Hi,

I  would like to try "Autodesk ReCap 360 Photo" web service with "ultra" option, which requires some cloud credits. Would you please grant me some cloud credits as ADN member?

My ADN ID is: DEVRxxxx 
My Autodesk ID is :   xxxx@company.com

What I am trying to do with ReCap 360 is .....

Regards, 
You name

 

其中ADN ID是你们公司在申请ADN时的ID，一般是DEVRXXXX的形式。 Autodesk ID就是你登录https://recap360.autodesk.com时用的那个，一般是一个email地址。然后详细描述一下，你想用这个服务干什么，详细描述一下，越详细越好，最好把用途啊、前景啊什么的，能说的都说一下。因为ReCap 360现在还是technical preview，产品经理们要判断你是不是有些认真的想法来用这个服务。如果他们认为你只是玩玩儿试试看，那你就用上面的preview选项i就行了，估计就不会给你Cloud Credit了。所以，如果你真的有什么想法，要在这里说清楚。

7. 最后，在提交之前，把上面comments写的东西也给我发个邮件daniel.du@autodesk.com，这样我也能知道你的意思，我们以后方便交流。

 

 

 

 

8. 最后的最后，要用英文啊，English，审核的人是老外，不懂中文。如果你用中文写半天，那边不懂，还是瞎耽误功夫。

 

好了，点提交按钮，静候佳音吧。

 

看到这里了，你还不是ADN会员？那赶紧鼓动你老板申请一个，看这里http://www.autodesk.com.cn/adsk/servlet/index?id=10378032&siteID=1170359， 如果还有不明白的，联系唐献华 xianhua.tang@autodesk.com .
