Autodesk Infrastructure Map Server(AIMS) 2013开发帮助文档在哪里？离线包下载

Jun 16, 2017


AIMS2013已经发布，你可能发现他的开发帮助文档不见了，并没有随安装程序安装，安装光盘中也没有。其实从AIMS2012开始，相关的开发文档就不随光盘提供了，而是集中以Wiki的方式放在了网上。这样方便开发部随时对文档进行更新，大家在参考的时候也能随时反馈。而且放在网上的另一个好处就是便于搜索引擎搜索，有些问题通过Google就能找到相关答案了。

 

下面是AIMS 2013的帮助文档地址：

http://wikihelp.autodesk.com/infr._map_server/enu/2013

当然你也猜得到AIMS 2012的地址：

http://wikihelp.autodesk.com/infr._map_server/enu/2012

 

在线文档的好处刚才说了，当然也有不好的地方，就是不方便，每次都得联网。这里我做了个离线版，如果你需要离线版帮助，可以从这里下载（注意，这个离线包生成于2012年4月16日）：

http://dl.dropbox.com/u/48146765/wikihelp_mapserver2013.zip

 

另外还有详细的API Reference参考文档离线包下载
http://images.autodesk.com/adsk/files/autodesk_infrastructure_map_server_2013_api_references.zip 

You can also download a copy of AIMS API Reference documents which includes Infrastructure Studio API references, Web Extension API references, and Ajax Viewer API references.
