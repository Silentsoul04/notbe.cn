iOS:授权用户定位NSLocationManager的使用

Jun 25, 2017

请求用户批准定位：
在iOS8，要想获得用户的位置，必须经过用户批准授权
开发者可以在Info.plist中添加两个配置项
–NSLocationAlwaysUsageDescription
–NSLocationWhenInUseUsageDescription
 
通过两个实例方法获取授权
–requestWhenInUseAuthorization
–requestAlwaysAuthorization

定位管理器的代理方法：

当用户的位置发生改变时，就会不断调用代理方法，比如

- (void)locationManager:(CLLocationManager *)manager didUpdateLocations:(NSArray *)locations•

当定位失败时，会调用

- (void)locationManager:(CLLocationManager *)manager didFailWithError:(NSError *)error

前期准备：

下面就是代码的操作了：

1、声明相关的属性并实现定位协议

#import "ViewController.h"
#import <MapKit/MapKit.h>
#import <CoreLocation/CoreLocation.h>
@interface ViewController ()<CLLocationManagerDelegate> //实现定位管理者协议
@property (strong,nonatomic)CLLocationManager *locationManager; //声明定位管理者 @end

2、创建定位管理者,设置代理，开始定位

- (void)viewDidLoad {
    [super viewDidLoad];
    //创建位置管理者
    self.locationManager = [[CLLocationManager alloc]init];
    
    //获取用户授权
    [self.locationManager requestAlwaysAuthorization];
    
    
    //定位授权状态(始终定位、当使用时定位)
    CLAuthorizationStatus status = [CLLocationManager authorizationStatus];
    if (status == kCLAuthorizationStatusAuthorizedAlways || status == kCLAuthorizationStatusAuthorizedWhenInUse)
    {
        NSLog(@"授权通过");
    }
    else
    {
        NSLog(@"授权不通过");
    }
    
    //设置定位管理器的代理
    self.locationManager.delegate = self;
    
    //设置精度
    self.locationManager.desiredAccuracy = kCLLocationAccuracyBest;
    
    //开始定位
    [self.locationManager startUpdatingLocation];
}

3、实现定位协议的代理方法，从地图上获取任意一个位置的地理信息，以及定位失败时的处理

#pragma mark -实现定位管理者的代理方法
-(void)locationManager:(CLLocationManager *)manager didUpdateLocations:(NSArray *)locations
{
    NSLog(@"%@",locations);
    
    //取出一个位置信息
    CLLocation *location = [locations lastObject];
    
    //定义地理信息编解码
    CLGeocoder *geocoder = [[CLGeocoder alloc]init];
    
    //转换地图信息(将地图信息转换为具体的地理信息)
    [geocoder reverseGeocodeLocation:location completionHandler:^(NSArray *placemarks, NSError *error) {
        
        //取出位置标记
        CLPlacemark *placeMark = placemarks[0];
        
        NSLog(@"%@,%@",placeMark.country,placeMark.locality);
    }];
}

//定位失败
-(void)locationManager:(CLLocationManager *)manager didFailWithError:(NSError *)error
{
    NSLog(@"%@",[error localizedDescription]);
}

4、实现触摸开始事件,将两个地理位置添加到地图中并设置大头针，调用系统地图，最终显示两个地点的线路

//触摸开始处理
-(void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event
{
    //定义地理信息编解码
    CLGeocoder *geocoder = [[CLGeocoder alloc]init];
    
    //转换地理信息(将地理信息转换为具体的地图信息)
    [geocoder geocodeAddressString:@"东三旗" completionHandler:^(NSArray *placemarks, NSError *error) {
        if ([placemarks count] == 0)
        {
            NSLog(@"没找到相应的信息");
        }
        else
        {
            //取出位置标记
            CLPlacemark *placeMark = placemarks[0];
            NSLog(@"%@%@",placeMark.country,placeMark.locality);
            
            
            //位置标记被封装在地图上的具体信息(包括国家、城市等等)
            MKMapItem *mapitem = [[MKMapItem alloc]initWithPlacemark:(MKPlacemark*)placeMark];
            
            //转换地理信息(将地理信息转换为具体的地图信息)
            [geocoder geocodeAddressString:@"八达岭长城" completionHandler:^(NSArray *placemarks, NSError *error)
             {
                
                //当前位置在地图上的具体信息
                MKMapItem *mapitem2 = [[MKMapItem alloc]initWithPlacemark:((MKPlacemark*)placemarks[0])];
                
                //调用系统地图显示当前设置的位置
                [MKMapItem openMapsWithItems:@[mapitem,mapitem2] launchOptions:nil];
            }];
        }
    }];
}
