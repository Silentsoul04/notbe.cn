JavaScript/WebGL前端开发工程师

岗位职责：
1、负责开发基于浏览器的三维模型展示软件
2、负责对开发的软件进行测试并编写自动化测试脚本
3、负责系统性能监控和系统优化

任职要求：
1、熟练掌握JavaScript
2、熟悉DirectX/OpenGL或WebGL， 如Three.js、Babylone.js, 或Unity等游戏引擎，熟悉Shader编程者优先
3、熟悉计算机图形学
4、熟悉敏捷开发流程，2年以上编程经验
5、学习能力强
6、掌握任意一门web开发语言
加分项：
1、Comfortable with communication in English.
2、熟悉node.js, Express.js, Ruby on rails, Meteor等任意一个框架者优先。
3、熟悉GLSL或HLSL 优先

前端开发工程师

岗位职责：
1、参与公司Web前端产品的设计和开发，负责前端功能的开发和优化；
2、开发基于HTML5技术的可灵活定制、可扩展的前端UI组件；
3、负责前端HTML、CSS、JS的性能调优，及浏览器兼容性;
4、负责Web前端产品的交互性的设计以及实现;
5、配合团队完成项目前端开发

职位要求：
1、统招本科以上学历,2年以上Web项目经验;
2、精通HTML5/CSS3/JQuery/Bootstrap等相关技术，具有HTML5实际项目开发经验；
3、精通JavaScript/Ajax等Web开发技术，对页面性能和浏览器兼容有丰富的实践经验；
4、熟悉HTML/XHTML/CSS等网页制作技术，熟悉页面架构和布局；
5、熟练使用DreaWeaver、PhotoShop 、Fireworks、Illustrator等相关软件；
6、至少熟悉一种服务器端开发语言，熟悉node.js优先；
7、热爱前端技术、有高度的技术敏感度、广阔的技术视野；
8、具有较强的团队合作精神、进取心和求知欲，热爱学习，勇于挑战

node.js全栈工程师

岗位职责：
1)负责产品网站页面的前端开发和页面重构；
2)参与产品后台功能和开放接口的设计与研发；
3)参与产品性能优化,安全性优化和服务器部署维护；

任职要求：
1)统招本科211及以上学历,计算机及相关专业毕业；三年以上开发工程师经验；
2)对于新技术充满热情, 如 html5, css3, react等, 用过至少一个前端构建工具（Grunt, gulp, …）；
3)精通至少另外一种服务器端的语言(比如C++、Java、Python、Ruby、Perl等)，具有丰富多面的前后端开发经验，懂得如何实现先后端交互的具体细节；
4)熟练运用数据库和SQL (Structured Query Language，结构化查询语言)，了解关系型数据库设计, 非关系型数据库(MongoDB)等；
5)掌握构建项目的全方面技术，知晓如何从头开始设计和实施一个完全可扩展的Web应用，并使用云服务提供商将应用部署在Linux主机上；
6)优秀的英语阅读理解能力；
7)熟悉基于SVN或Git的协作开发流程。
8)良好的文档书写能力和代码注释习惯，对代码整洁度、可维护性有较高追求。

加分项：
1)有大型 web app 的开发经验；
2)有数据统计或商业分析方面的经验；
3)Comfortable with communication in English.

Unity3D开发工程师

岗位职责：
1.使用Unity3d开发沉浸式交互式应用和虚拟现实软件
2.根据项目需求文档，提出项目解决方案，设计合理的程序架构
3.具有严谨的逻辑思维能力，编写高效、稳定、安全的代码和Unity脚本
4.熟悉Direct3D或OpenGL之一，熟练掌握计算机图形学基础知识，有Shader编程经验者优先
5.与团队成员进行沟通合作，总结分享开发经验知识

任职要求：
1.本科及以上学历，计算机相关专业，优秀的开发者可以适当放宽条件。
2.2年以上Unity3d开发经验，精通Unity3d引擎，熟悉3DMAX建模软件。
3.具备独立研发和解决问题的能力。
4.具备团队合作精神，有创新精神。
5.主动性强，能积极参与公司的各项任务。
6.有一定的英文文档阅读能力。
熟悉3D Max开发者优先。
