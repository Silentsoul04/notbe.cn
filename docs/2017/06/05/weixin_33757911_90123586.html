安装Vault Professional Server的一些问题

Jun 5, 2017


我在安装Vault Professional Server（ADMS）时由于我的IIS设置的问题导致安装程序不能继续，下面是我遇到的一些问题和解决办法，也许会对你有帮助，当然如果对你没有效果，你的问题也可能是其他原因引起的。

 

第一个：

Failed to read IISWebs table (-2147024893)

 

 

我的解决办法：

开始 –》 运行 输入inetmgr打开IIS管理器，我用的是Win7，IIS 7.5 右键Default Web site,选Edit Binding，

其中的http绑定应该是这样的，端口80，IP地址为全部未分配，主机名为空：

 

第2个问题：

Vault的系统诊断包括提示：  Accessing an http static file failed at URL http://localhost:80/TestASP/TestASPNetStatic.txt.

 

我的解决办法：进入到IIS管理器，选中default web site，右边的Authentication

把ASP.NET Impersonation 改成disabled。

 

 

另外，Vault的系统诊断总提示重启，请看安装Autodesk Vault Server 总提示需要重启计算机？
