Ubuntu常用软件安装（附带地址和卸载自带软件）

Apr 21, 2017

跨平台系列汇总：http://www.cnblogs.com/dunitian/p/4822808.html#linux

上次说了安装VSCode（http://www.cnblogs.com/dunitian/p/6661644.html）和sogou（http://www.cnblogs.com/dunitian/p/6662374.html）拼音，其实基本指令一样，剩下的就不单独列出了，比如安装谷歌浏览器等等（火狐性能太差）

Deb包地址：http://pan.baidu.com/s/1kVI8eft

这些是我觉得不错的，其他的要么功能太low要么性能太差，有更好的可以推荐

先说安装谷歌浏览器吧

dpkg -i xx.deb 安装deb包

apt-get install -f 修复安装依赖

卸载谷歌浏览器：dpkg -P google-chrome-stable 不保留配置文件的卸载

说到浏览器我发现了一款更牛叉的谷歌内核浏览器(我个人比较喜欢，性能和谷歌浏览器差不多,内容却丰富很多)，vivaldi==》网址：https://vivaldi.com/download/

打开后可以锁定到“任务栏”

下面安装音乐，QQ音乐之前也有Linux版本，夭折了，用的是网易音乐，其他音乐都太low了

文档要是用不惯自带的，可以用WPS的，体积比自带Office小，WPS比较符合中国人审美

固定一下就可以了（wps字体出问题的可以看这个：https://www.cnblogs.com/EasonJim/p/7146587.html）

思维导图可以用XMind或者百度脑图在线版本（包链接里面有分享，官方也有下载）

 

卸载软件可以用两种：

一般用 sudo dpkg -P 软件名 即可（dpkg -r 不删除配置文件的卸载方式 dpkg -P 删除配置文件的卸载方式）

彻底删除，包括依赖项：sudo apt-get --purge remove 软件名

 

附录：卸载自带软件：

Ctrl+Alt+T调出终端，执行一下代码：

卸载libreOffice
sudo apt-get remove libreoffice-common

删除Amazon的链接
sudo apt-get remove unity-webapps-common

删除不常用的软件
sudo apt-get remove thunderbird totem rhythmbox empathy brasero simple-scan gnome-mahjongg aisleriot onboard deja-dup

sudo apt-get remove gnome-mines cheese transmission-common gnome-orca webbrowser-app gnome-sudoku landscape-client-ui-install
