windows下如何安装和启动MySQL

Apr 30, 2017

1.下载，解压到自己喜欢的目录

2.配置环境变量。MYSQL_HOME,值为mysql的根目录；在path中添加%MYSQL_HOME%/bin目录。

3.向windows注册mysql服务。必须用管理员权限打开命令行，然后切换到mysql的bin目录下，输入命令：mysqld.exe --install MySql --defaults-file="d:/lovejava/mysql-5.6/my-default.ini",【d:/lovejava/mysql-5.6/my-default.ini】改为自己的mysql安装目录。

4.启动mysql服务。在命令行输入：net start mysql，成功会提示：MySql 服务已经启动成功。

解决密码为空问题

  1. cmd模式进入mysql的bin目录,输入d:/lovejava/mysql-5.6/bin>mysqld --defaults-file="d:/lovejava/mysql-5.6/my-default.ini" --console --skip-grant-tables,回车,此命令行会挂起。
  2. 另开一个命令行，同样切到mysql bin目录下，d:/lovejava/mysql-5.6/bin>mysql -u root mysql
  3. 修改密码， update user set password=PASSWORD('123456') where user='root';
