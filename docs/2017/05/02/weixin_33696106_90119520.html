Map 3D中常见的用户界面和使用习惯修改方法

May 2, 2017


 

AutoCAD命令行

按CTRL + 9，关闭或开启命令行。

Map 3D经典视图区（Map 3D Classic Workspace）

激活经典或传统Map 3D用户界面

1. “命令”行中，输入WORKSPACE。

2. 输入C或setCurrent。

3. 输入Map Classic。

Map任务面板（Map Task Pane）

显示Map Workspace（Map视图区）/Task Pane（任务面板）

1. “命令”行中，输入MAPWSPACE。

2. 输入O或ON。

隐藏Map Workspace（Map视图区）/Task Pane（任务面板）

1. “命令”行中，输入MAPWSPACE。

2. 输入F或OFF。

AutoCAD菜单栏

隐藏下拉菜单或菜单栏。

1. “命令”行中，输入MENUBAR。

2. 输入0（零）。

显示下拉菜单或菜单栏

1. “命令”行中，输入MENUBAR。

2. 输入1（一）。

AutoCAD Ribbon（功能区）

“命令”行输入RIBBON，即可显示AutoCAD Ribbon（功能区）。

“命令”行中输入RIBBONCLOSE，即可隐藏AutoCAD Ribbon（功能区）。
