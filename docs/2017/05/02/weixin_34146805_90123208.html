AutoCAD Map 3D 2013新功能视频中文版

May 2, 2017


此版本的 AutoCAD Map 3D 包括以下新要素：

准备布局图纸以供公众查看

观看视频 (3:25)

  • 将显示地图与视口关联。
  • 使用“显示管理器”控制可见性。
  • 为每个视口/地图添加图例。
  • 使用动态比例尺。
  • 通过旋转指北针控制方向。

有关详细信息，请参见使用显示地图。

将数据转换为行业模型

观看视频 (3:33)

  • 转换基于文件的 GIS 数据，如 SDF、SHP
  • 转换数据库（如 Oracle 或 SQL Server）中的 GIS 数据。
  • 转换 DWG 对象和属性，如对象数据。
  • 使用预定义的映射文件轻松刷新大型数据集。

有关详细信息，请参见输入数据。

创建和编辑符号

观看视频 (3:32)

  • 定义符号并将其编录在图形中。
  • 直接使用 AutoCAD 的“块编辑器”编辑符号。
  • 从一个图形到另一个图形轻松输入符号。

有关详细信息，请参见使用符号。

使用编辑集更新数据库

观看视频 (3:53)

  • 在“直接编辑”模式或新的“编辑设定”模式中编辑。
  • 解决由多用户编辑相同数据而导致的所有冲突。
  • 脱机编辑行业模型数据。
  • 在脱机工作之后将更改与数据库合并。
  • 使用任何支持的数据库或数据源。

有关详细信息，请参见脱机工作。

定义线性参考系统（需要编程）

观看视频 (4:50)

  • 基于现有直线段和其他要素定义线性参考系统的框架。
  • 使用管理员将资源和属性添加到 LRS。
  • 使用 AutoCAD Map 3D API 对线性参考系统的功能进行编程。

有关详细信息，请参见设置线性参考系统。

将 DWG 文件发布到 Autodesk Infrastructure Map Server

观看视频 (4:29)

  • 发布单个或多个 DWG。
  • 发布通过 FDO 连接的 DWG 对象和 GIS 数据。
  • 配置要从每个图形发布的内容，如图层或对象数据。
  • 使用过滤器和表达式精确定义要发布的数据。

有关详细信息，请参见配置要发布的 DWG 数据。

在 Microsoft SQL Server 中维护和编辑行业模型

有关详细信息，请参见设置新的基于 SQL Server 的系统。
