如何在Infraworks中创建多树种组成的森林

May 2, 2017


在Infraworks 2014中，你可以有shp文件导入生成树木和森林，也可以直接在模型中规划一片区域作为森林。美中不足的就是，这些充其量叫树林不能叫森林，因为他们的样式都是一个树种，而真正的森林肯定是由多种植物组成的，高高低低，大大小小，那才叫森林。在Infraworks中怎么做呢？

 

先看看效果：

 

如果是自己画的森林区域，选择树木两次从而选中区域中的全部树木，如果是从shp等文件中导入生成的森林区域，那么在数据源面板中右键树木，然后“选中全部”

切换到属性面板，给TAG属性赋值，比如Forest

然后切换到样式规则面板 Home – 》 Style Rule， 在Tree tab下创建一个新的样式规则，比如起个名字叫Forest，在Expression中输入（或者用表达式生成器） TAG=‘Forest’，在样式中随机加入几种树木的样式，可以多加几种：

 

弄完了 Refresh 样式：

 

切换到Properties面板，为选中的树木设定样式规则。 删除Manual Style 设置Rule Style为 Forest，然后 更新Update。

 

好了，大功告成，如果你没弄错的话，应该能看到上面的森林了。
