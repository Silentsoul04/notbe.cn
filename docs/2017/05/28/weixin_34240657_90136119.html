五分钟搭建 Flash 视频直播站

May 28, 2017

想在家里对全世界直播网络视频节目吗?
如今视频网站是多如牛毛,但能让你玩直播的估计没几个吧?
看完这篇教程就能帮你实现网络主持人的梦想.
不花钱,不懂编程,不用写代码也行哦~


首先是最低机器要求:
Windows XP sp2 + 1GHz 的奔腾4处理器 + 1G 内存

附加硬件:
摄像头 + 耳麦

必备软件:
Adobe Flash Media Server 3.0
Adobe Flash Media Encoder 2.0

第一步:安装 Flash Media Server 3.0 并将默认端口指定为 80
（这样可以保证最多人可以看到你的视频,因为80端口一般不会被封）
如果机器上安装 IIS 或者 Apache,记得先关闭或者改成其他端口.
 (点击看大图)

第二步:安装并启动 Flash Media Encoder 2 进行设置,如下图(注意在localhost后面添加80端口号,看下图)
设置OK后按 START 键

第三步:启动 Flash CS3 Professional, 将 Flash 视频组件添加到舞台,并按照下图进行设置.

最后,将编译生成的 SWF 放到你自己的站点下,就可以让别人看到你的直播了.

如果没有网站也没关系,通过下面的地址访问:
http://www.dengjie.com/live/
(把 localhost 改成你机器的IP地址然后按 GO 即可)

直播完成后,会自动生成一个FLV视频文件,可以上传到空间继续回放:
http://www.dengjie.com/live/live_record.html

注: Flash Media Server 3.0 开发版只能提供 10 人连接的支持.

分类:  IIS, C# Asp.net
