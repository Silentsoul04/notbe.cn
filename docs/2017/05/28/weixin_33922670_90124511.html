asp.net 2实用技术汇总

May 28, 2017

1、创建虚拟服务器 C:\WINDOWS\Microsoft.NET\Framework\v2.0.50727\WebDev.WebServer.EXE /port:8555 /path:"D:\NewProject\Ihttphandle\Web" /vpath:"/myApp" 

     启动2008中的WebDev.WebServer 在。net2.0里面有一个异常好用的工具就是WebDev.WebServer，位于c盘下的microsoft.net的framework中呢！但是在2008中却不在这个目录中！而是位于：C:\Program Files\Common Files\Microsoft Shared\DevServer\9.0\WebDev.WebServer.EXE
     使用这个东东可以让你指定web服务器，自定义端口调试！很是方便！

     将下面代码复制到一个一reg扩展名结尾的文件内： 

Windows Registry Editor Version 5.00 

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Folder\shell\VS2005 WebServer] 
@="ASP.NET 2.0 Web Server Here" 

[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Folder\shell\VS2005 WebServer\command] 
@="C:\\Windows\\Microsoft.NET\\Framework\\v2.0.50727\\Webdev.WebServer.exe /port:8080 /path:\"%1\"" 

运行添加到注册表内，这样一个以8080端口为默认服务器的菜单命令就添加到右键菜单内了。右击想设置成虚拟站点的目录，立马就好了，并且目录名称就为虚拟站点目录。

2、设置SQL2005 占用的最大内存，防止他无限制的占用内存导致系统死机
       
      

--  打开SQL高级选项设置
USE  master
 EXEC  sp_configure  ' show advanced options ' ,  1
RECONFIGURE   WITH  OVERRIDE

 --  设置最大内存为 512M
USE  master
 EXEC  sp_configure  ' max server memory (MB) ' ,  512
RECONFIGURE   WITH  OVERRIDE

 

3、 不要相信using的try catch功能 使用using时也要手动释放资源，他释放的那个时差足以使系统崩溃N次！

4、实现div自适应高度的方式：
      

Code
html,body{
    margin:0px;
    heigth:100%;
}

.divUse{
    height:100%;
}
设置body & html 的高度为100% 这是高度自适应的关键所在。一个对象的高度是否可以使用百分比显示，取决于对象的父级对象！

 


5、当上下两个对象都是用了margin且没有使用float时，他们的上下margin会叠加：如.a1{margin:10px;} .b1{margin:10px;} 那么a1&b1之间的上下距离应该是20px; 但是由于没有使用float 他们之间的实际距离却是10px; 当我们使用了float 的时候.a1{margin:10px; float:left;} .b1{margin:10px; float:left;}这个时候就出现了我们希望的它们的上下间距是20px; 但是在ie6下面又有一个问题就是使用了margin & float的时候 margin的left & right 就会加倍显示，此时使用.a1{margin:10px; float:left; display:inline;} .b1{margin:10px; float:left; display:inline;}来解决！ display属性用于强制对象按照某一种显示模式进行解析！
