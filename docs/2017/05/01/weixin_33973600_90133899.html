Windows 7 Tips! --透明缓存(transparent caching)技术

May 1, 2017


当今应用环境中，用户经常分散在各地，经常需要通过网络才访问资源，很多情况下可能是相当慢速的连接，比如在外地或者家里。通常情况下，用户每次访问一个文件，都要重新下载一遍，即使这个文件从未改变过，这样造成了漫长的等待和用户体验的低下。

在Windows 7中引入了一个新的特性--透明缓存技术来改善这样的情况。透明缓存技术在读取文件时主动吧文件作缓存，如果下次用户又需要这同一个文件，系统就可以从本地缓存中读取而不是重新从远程服务器上去下载，从而节省了带宽和下载时间。

注意这不是脱机应用的解决方案，用户还是需要联网，因为这个过程中透明缓存还是需要和服务器进行联系获取相关信息，以便判别本地缓存中的数据是否已经过时了。如果缓存中的已经和服务器上的不一样了，当然就得重新到服务器上去读取并更新缓存了。
