Autodesk Topobase 2011 Update 1 – released!

In our continuing effort to improve quality, performance and respond to our customer’s needs, we have released Autodesk Topobase 2011 Update 1.

Downloads

§ Topobase Web: http://usa.autodesk.com/adsk/servlet/ps/dl/item?siteID=123112&id=15954288&linkID=9242779

§ Topobase Client: http://usa.autodesk.com/adsk/servlet/ps/dl/item?siteID=123112&id=15952777&linkID=9242779

Changes Included in this Update

Autodesk Topobase 2011 Update 1 includes AutoCAD 2011 Update 1 as well as AutoCAD Map 3D 2011 Update 1 and also resolves the following Topobase 2011 related issues:

Topobase Administrator

Data Model

  • Open a workspace with a Swiss Land Management which has more than 200 jobs causes the program to fail unexpectedly
  • Default values for some table attributes are not set during document update.
  • The default value of system table TB_JOB.Data_Created has been corrected.

Topobase Client

Export to DWG (TBEXPORTDWG)

  • Export to DWG does not consider the scale ranges for enhanced line styles.
  • Features in some view feature classes are missing in export result.

Interlis Import

  • Default values for Horizontal and Vertical alignment stored in tbm-Files are not recognized.

Label

  • The position of automatically created label is not updated when its parent feature is moved.

Coordinate File

  • Screen refreshes for every imported point when using field code distribution.

Jobs

  • Delete job hangs when an attributive dialog is open.
  • Setting a job to pending state sometimes fails unexpectedly in Swiss Land Management document.
  • Change maintenance job status causes the program to fail unexpectedly with a configured topology checker.

COGO

  • Only Feature Classes of one Document are shown in CoGO's save calculated point list box if you have more than one document in one workspace.
  • Missing Topic names in COGO's save calculated point list box.

Other

  • SNAPs don't work on some boundary points.
  • Stretch command doesn't work with OSNAPS in map space.
  • Checkboxes in style editor for points are partly hidden.

Topobase Electric

  • Move an arc segment which has cross section causes the program to fail unexpectedly.
  • Failed to move 2 segments with consecutive conductor.
  • Failed to split conductor with device which is located in halfway of ducts.
  • Wrong translation of options for ELClearContainer command in German.

Topobase Wastewater

Import/Export

  • ISYBAU .H file import: picture info not in the first block cannot be detected and imported.

Other

  • Move point feature will lead to crash when WW_ComputeNrOfSectionsTopoChange feature rule is enabled.
  • Reverse direction of section doesn't switch values for input_z and output_z of this section.

Topobase Web

  • User Interface Settings don't take effect in TBWeb.
  • Can browse only first document in TB Web2011 if the workspace contains multiple documents.
  • Parent feature id is not filled in Form automatically when digitizing child feature of compound feature.

MapGuide

  • FDO WMS – Better quality when requested image height or width > 2047

. 

Please check Readme for installation instructions and other Update 1 details.

Our sincere thanks to SWD, QA, UX, Support, AC and anyone else involved in helping us to release this important Update.
