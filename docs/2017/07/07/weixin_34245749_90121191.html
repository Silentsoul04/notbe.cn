ASP.NET MVC5+EF6+EasyUI 后台管理系统（51）-系统升级

Jul 7, 2017

系统很久没有更新内容了，期待已久的更新在今天发布了，最近花了2个月的时间每天一点点，从原有系统

MVC4+EF5+UNITY2.X+Quartz 2.0+easyui 1.3.4无缝接入 MVC5+EF6+Unity4.x+Quartz 2.3 +easyui 1.4.3.

并以easyui 1.4.3的gray皮肤为基础，升级10个扁平化皮肤 皮肤查看地址

更新的主要目的：新的MVC5特性和更好的性能

记录一下升级过程。

1.除了web层，其他全部提取。

2.新建解决方案。以前命名空间为App.现在更名为Apps.

3.新建MVC5web层。并复制所有MVC4 View Areas和配置信息

4.引用所有提取的类库，并直接安装unity 4.x nuget包

5.成批替换命名空间，从新创建EF6.1并更新数据库，和复制原EF5中的复杂类型

6.运行系统只存在一处前端的tab页不兼容，很容易解决

7.重构少量的前段代码和后台代码，更加优雅

完美过度到新版本。以后文章标题将更改为构建ASP.NET MVC5+EF6+EasyUI 1.4.3+Unity4.x注入的后台管理系统

等待完成工作流版本之后，以后系统将以样例为主导。对系统添加新的插件和使用方法，让系统以后做开发直接应用
