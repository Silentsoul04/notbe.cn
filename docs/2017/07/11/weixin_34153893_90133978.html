eclipse的myeclipse插件下载安装方法

Jul 11, 2017

1，第一种方法直接通过网站更新。

MyEclipse 10.0 (Update Site)

To install using an Eclipse update site:

  1. Install Eclipse Classic 3.7.   使用的eclipse是3.7 classic的
  2. Edit the eclipse.ini file in the eclipse folder and change the -Xmx384m value to -Xmx512m.  修改eclipse的配置文件eclipse.ini文件，改变其中的参数Xmx384m 改为-Xmx512m（是堆的占用内存修改参数）
  3. Start Eclipse and click Help > Install New Software > Add the following update site: 
    http://downloads.myeclipseide.com/downloads/products/eworkbench/indigo/enterprise-stable/

     打开eclipse然后找到菜单help进入如下路径：Help > Install New Software >输入如下网站：http://downloads.myeclipseide.com/downloads/products/eworkbench/indigo/enterprise-stable/

  1. Make sure "Group items by category" is selected and select the "MyEclipse Enterprise Workbench" group. 

  在列表中勾选“Group items by category”和“MyEclipse Enterprise Workbench”

  1. Click Next twice, accept the licenses presented and click Finish to start the installation. Restart when prompted.

点击“next”两次，接受出现的协议，然后点击“finish”，开始安装。按照提示进行重启eclipse即可；
