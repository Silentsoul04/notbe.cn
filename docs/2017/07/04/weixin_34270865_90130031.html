欧特克Autodesk发布基础设施2012系列产品

Jul 4, 2017


下一个版本的MapGuide Enterprise的正式名称将是Autodesk Infrastructure Map Server 2012(AIMS 2012)。

 

 

点击这里观看Autodesk 2012系列产品发布会视频。 

 

 

Autodesk 2012基础设施系列产品包括：

AutoCAD Civil 3D 2012,为土木工程设计提供智能化的设计模型，现在还包含了Autodesk Storm and Sanitary Analysis 2012来帮助土木工程师对暴雨排水管网的分析和设计。

AutoCAD Map 3D 2012, 提供了面向基础设施的智能化模型，现在还包含了面向天然气，上水，下水和电力等行业的数据模型。

Autodesk Infrastructure Map Server 2012 不仅包含了MapGuide Enterprise的全部功能，还包含了原有的Autodesk Topobase Web模块，即面向天然气，上水，下水和电力等行业的专有模块，此外还有用于移动设备（iPad、 iPhone）的iViewer浏览器以及GeoREST扩展模块。

Autodesk基础设施设计套装将在今年晚些时候发布，包含三个版本：标准版（Standard），Premium 版和 Ultimate终极版. 标准版包括AutoCAD Map 3D 2012，Autodesk Navisworks Simulate 2012和AutoCAD。 Premium 版增加了AutoCAD Civil 3D 2012和 Autodesk 3ds Max Design。Ultimate终极版包括Premium 中的所有内容，并且把Autodesk Navisworks Simulate 2012提升为Autodesk Navisworks Manage 2012。

 

如果你好奇想看E文，原文请见http://geospatial.blogs.com/geospatial/2011/03/autodesk-announces-2012-products-for-infrastructure.html（请FQ访问）

 

这里我想提醒大家注意的就是在Map 3D 2012 和AIMS 2012中增加的面向行业的模块，比如天然气，上水，下水，电力等行业模块，这些模块定义了不同行业的完整的数据模型和业务规则及其常见的工作流。如果你在做相关行业的开发，充分利用这些现在模块，相信将很大的减轻你的重复劳动。而且，这些模块都是可以定制的，你还可以完全定制你自己的行业模块。相关产品ADN网站上已经有beta版下载了，待正式版下载链接出来后我会及时更新。
