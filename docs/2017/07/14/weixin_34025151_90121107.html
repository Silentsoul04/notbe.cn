Mac-OSX下Ruby更新

Jul 14, 2017

Mac下是自带Ruby环境的，在有些情况我们是需要更新Ruby的，安装和更新Ruby环境可以通过rvm命令进行操作，rvm在安装过程中通过HomeBrew安装依赖包，如果之前没有装过HomeBrew，顺序是HomeBrew→Rvm→Ruby。

HomeBrew安装

Homebrew 使 OS X 更完美。使用 gem 来安装 gems、用brew 来搞定那些依赖包，安装命令:

1
ruby -e  "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

参考网站http://brew.sh/index_zh-cn.html.

 

Rvm的安装

Rvm是用来更新和管理Ruby版本的，安装命令:

 

1
curl -L https: //get.rvm.io | bash -s stable

 

查看当前可用的版本:

1
rvm list known

检测是否安装正确:ruby -v

1
rvm 1.26.11 (latest)  by  Wayne E. Seguin <wayneeseguin@gmail.com>, Michal Papis <mpapis@gmail.com> [https: //rvm.io/]

Ruby安装　　

安装命令:

 

1
rvm install 2.0.0

 

这里如果有梯子的话请用一下梯子，如果没有梯子也不要紧，万能的淘宝提供了Ruby下载源，需要将Rvm中的地址更新一下：http://ruby.taobao.org/

1
$ sed -i .bak  's!cache.ruby-lang.org/pub/ruby!ruby.taobao.org/mirrors/ruby!'  $rvm_path/config/db

设置默认Ruby版本:

1
rvm 2.0.0 -- default

查看Ruby版本信息（ruby -v）：

1
ruby 2.0.0p643 (2015-02-25 revision 49749) [x86_64-darwin14.1.0]
