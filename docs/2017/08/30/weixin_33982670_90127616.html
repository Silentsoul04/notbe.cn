Remote Desktop Organizer – 管理组织远程桌面

Aug 30, 2017

在远程控制上的软件不少，有赛门铁克的pcanywhere，dameware，VNC等等，各有优缺和使用范围，今天介绍一种新的软件Remote Desktop Organizer 。他与以上软件同属远程控制软件但是也有所不同吧。或者可以说他是一个集成远程桌面的软件，使远程桌面使用更方便。

Remote Desktop Organizer 是一款 Windows 远程桌面管理软件，让你在同一个窗口内浏览到多个远程桌面的信息，方便 Windows 远程管理。[via nirmaltv] @Appinn

或许很少有普通用户同时连接 N 多的远程桌面的需求，Remote Desktop Organizer 就是这样一款帮助你管理大量远程桌面的软件，适合于拥有众多电脑的工作人员。当然如果家里有几台电脑也可以连上玩玩。Remote Desktop Organizer 可以建立不同的分类，并且能同时打开远程桌面。

Win7 用户可以在这里打开远程桌面（被控制端）

 http://www.appmazing.com/remote-desktop-organizer/
