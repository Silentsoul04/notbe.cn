Mac下的Parallel Windows忘记密码怎么办？

Aug 14, 2017


Mac机上安装了Parallel Windows，日久年深不登录结果忘记了登录密码，百爪挠心，想破脑壳试了n个密码都不行，放了一个多月也没想起来。

今天没事网上溜溜，肯定也有和我同病相怜的弟兄，果然，这个链接救了我，不敢独吞，给大家共享： http://blog.chrisknost.com/resetting-your-password-for-parallels-7-using-mac-os-x-terminal/

1. 启动parallel windows

2. Mac中启动终端（terminal） (Finder > Applications > Utilities > Terminal)

3. 敲如下命令：

prlctl list --all 注意了两个减号

输出应该是类似：

UUID STATUS IP_ADDR NAME

{959c9839-f4d5-48a8-93e2-4ebc9c98b023} running - Windows 7

拷贝上面的UUID下面的命令要用.

prlctl set 959c9839-f4d5-48a8-93e2-4ebc9c98b023 --userpasswd <yourWindowsName>:123

authentication tokens updated successfully.

替换上面的UUID 和你的windows 用户名，这里把密码设置成了123. 注意也是两个减号

好了，再登录到windows，哈哈！成功了。 我高兴的把儿子抛到半空直到精疲力尽以示庆祝！
