几个有用的JSON工具

Aug 7, 2017


好久没写博客了，这里都要长草了：） 这几天研究PLM360 REST API和Infraworks REST API，一天到晚和JSON打交道，发现这几个小工具非常好用，推荐给大家。

 

第0个，大名鼎鼎的Fiddler，这是必备的，不再赘述。

 

第1个，http://json2csharp.com ，从云端服务获得json字符串后，都要反序列化为不同语言中的对象才能用，所以首先肯定要定义好和json对应的类。对于C#，这个网站非常好用，可以由json字符串自动生成C#的数据模型类：

 

第2个， http://jsonviewer.stack.hu/ 如果你拿到一堆json字符串，但没有格式化，可以用这个网站。当然其实fiddler也可以做格式化，所以这个重要性一般，供参考。

 

补充一个Json2Java  ：http://jsongen.byingtondesign.com/
