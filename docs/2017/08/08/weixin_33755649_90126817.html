自定义.NET CF控件，美化Windows Mobile程序界面

Aug 8, 2017

虽说.NET CF开发高效，在企业级开发方面是方便、强大，可是他的界面一直却让我很心痛，面对iPhone我还是低下了我高贵的头。所以整理了一些常用的界面控件及处理方法。

提示信息控件，用于动态的提示一些信息给用户！

背景渐变，通过P/invoke实现背景色的渐变填充，封装了一个背景色渐变的按钮

图片按钮，按钮的背景为一张图片，按下时交替显示另一张图片，在一些导航处这样的效果挺好。

绘制背景透明的图片，在有背景色的Form上显示透明图片，这是.NET CF本身提供的方法，关键是最后一个参数。

 DrawImage(Image image, Rectangle destRect,  int  srcX,  int  srcY,  int  srcWidth,  int  srcHeight, GraphicsUnit srcUnit, ImageAttributes imageAttr);

demo下载，里边有控件源码及调用示例。部分参考自MSDN。
