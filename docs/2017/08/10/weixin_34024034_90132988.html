ArcGIS Server 9.3 beta 体验三 -- 简阅 ADF 框架

Aug 10, 2017

  ADF整体框架图，Web Browser和Web Controls都包含了JavaScript Library，两者是完全相同，还是有其他的关系以后分析代码的时候就明白了。


    下面这张图是对ADF应用开发接口的划分，相比9.2文档中的图示，不同颜色表示更加清晰易懂。
    

     Web ADF所有控件和API实现都包含在<Server Install>\DotNet文件夹下。


    还有一些ArcObjects .Net Primary Interop Assemblies和COM代理。


    JavaScript库和客户端支持文件，所有文件都在IIS root目录下的aspnet_client文件夹下，9.3Web ADF的支持文件在ESRI文件夹，9.0/9.1的在esri_arcgis_server_webcontrols文件夹，这和9.2是类似的。
