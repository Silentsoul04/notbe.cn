QTP的那些事--Excel导入QC没反应的解决方法

Aug 16, 2017

用excel导入需求、用例或bug到QC中，到第五步的时候，choose a map option，无论create a temporary map，还是Type a new map name,点下一步，没有反应的解决方法：

在 C:\Program Files\Microsoft Office\OFFICE12\XLStart\TDAddin 文件夹下新建一个Tasks文件夹即可！
