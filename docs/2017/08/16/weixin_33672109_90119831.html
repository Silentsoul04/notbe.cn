一、准备软件

VNC Server (MAC OS X已支持)
RealVNC/TightVNC

二、MAC OS X设置

注：Mac OS X 10.5 已经支持了VNC Viewer访问的功能，设置如下:

1、在 系统偏好设置-共享-勾选“远端管理”,点击电脑设置

2、勾选“VNC显示程序可以使用密码控制屏幕“在文本框中输入8位密码,点击”好“，这样就可以通过其它PC或Mac上的VNC Viewer来连接这台Mac进行远程控制了。

3、选择可以远程登录的用户

方法一：RealVNC连接

官网:https://www.realvnc.com/download/

1、安装 RealVNC，如果仅仅用作连接其它电脑，可以不勾选VNC Server

2、安装完成后，打开，输入远程计算机名后，点击Connect

3、输入设定的密码，点击OK

4、第一次连接时，会提示你连接未加密，是否信任;勾选下次不再提醒，就可进入MAC OS的界面啦

方法二：TightVNC连接

在Windows机器上也可以安装其它连接工具，例如TightVNC

TightVNC官网：http://www.tightvnc.com/download.php

1、安装时选择默认设置就可

2、输入主机名，点击 Connect

3、输入密码，点击 OK

4、进入MAC 的桌面

其它连接方式

还可以通过在MAC OS及Windows上安装 TeamViewer 来进行远程桌面
