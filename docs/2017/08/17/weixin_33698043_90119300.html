QTP的那些事--连接oracle的简单方法（附源码）

Aug 17, 2017

'-----------------------------------------------------------------
'function：数据库连接
'call method：在调用文件中先定义Res,Cmd
'For example：DB(connectstr)
'
'连接字符串，你可以新建一个tld文件，然后设置相关参数就可以直接复制这个字符串即可
'
'-----------------------------------------------------------------
function DB(connectstr)
      Dim dbcon
      Set dbcon=createobject("adodb.connection")
      dbcon.Open connectstr
      If dbcon.State=0 Then
           reportProgress "你的连接对应的数据库失败了，请检查你的连接字符串配置再试",4
      else
'reportResult(teststep, expectvalue, actualvalue)
             reportResult "连接数据库操作","正确连接数据库成功","正确连接数据库成功"          
      End If
       DB=dbcon
End function
'-----------------------------------------------------------------
'function：执行SQL语句
'Input parameter: StrSql,dbcon
'For example：executeSql(dbcon,strsql)        
'---------------------------------------------------------
function  executeSql(dbcon,strsql)
      Dim rs
      Set rs=CreateObject("adodb.recordset")
       rs.Open strsql,dbcon
       set executeSql=rs
End function

'---------------------------------------------------------
'function：关闭数据库
'For example：DBClose(dbcon,rs)
'---------------------------------------------------------
function DBClose(dbcon,rs)
     rs.Close
     Set rs=nothing
     dbcon.Close
     Set dbcon=nothing
End function
