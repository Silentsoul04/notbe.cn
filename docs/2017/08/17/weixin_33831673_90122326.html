Ubuntu 12.04安装Mac OS X Lion 主题

Aug 17, 2017

主题文件

下载地址：Mac OS X theme for Ubuntu 12.04 LTS（可下载V2）

该文件包括 Plymouth 主题、壁纸、图标主题、GTK/窗口主题

 

预览

右键点击，在新标签页中打开查看大图。

设置步骤

A：Plymouth 主题

打开终端，执行下面命令：

sudo cp -R Paw-OSX/ /lib/plymouth/themes #复制 Paw-OSX 主题

sudo update-alternatives --install /lib/plymouth/themes/default.plymouth default.plymouth /lib/plymouth/themes/Paw-OSX/paw-osx.plymouth 100 #配置Paw-OSX优先级

sudo update-alternatives --config default.plymouth #在列表中选择 paw-osx 

sudo update-initramfs -u #更新当前kernel的initramfs

如果不喜欢，可回滚到 Ubuntu Plymouth 主题：

sudo update-alternatives --config default.plymouth
sudo update-initramfs -u

B：安装 Cario Dock 或 Docky

任选其一安装即可。

sudo apt-get install cairo-dock #安装Cario-dock
sudo apt-get install docky #安装Docky

当然，也可以安装 Elementary OS 团队开发的 Dock：

C：主题安装

-图标主题

优秀的主题有 Faience 系列、FaenzaMac 等。上述下载文件中包括一款图标主题：Lion By MBOSS。请以管理员权限将其复制到 /usr/share/icons 目录下。

-GTK/窗口主题

文件夹下的 themes 包括了系统主题 Lion By MBOSS。可将其复制到~/.themes（为隐藏文件，Ctrl+H 显示该目录，若没有则手动新建该目录即可）或以管理员权限复制到 /usr/share/themes 目录下。

-壁纸

文件夹中包括 12 张 Mac Lion 系列壁纸。

如果不嫌PPA安装速度慢，可添加该主题PPA来安装鼠标主题、图标主题和窗口主题：

sudo add-apt-repository ppa:noobslab/themes #添加到仓库
sudo apt-get update #更新软件列表
sudo apt-get install mac-os-lion-cursors #安装鼠标主题
sudo apt-get install mac-os-lion-icons #安装图标主题
sudo apt-get install mac-os-lion-theme #安装GTK/窗口主题

D：其它配置

如 Marlin 文件管理器

sudo add-apt-repository ppa:elementary-os/daily #可能不稳定或有错误发生
sudo apt-get update
sudo apt-get install marlin

Elementray OS 团队目前开发的应用有 Maya（日历）、Scratch（文本编辑器）、Noise（音乐播放器）、Dexter（通讯簿）等具有 Mac 风格的应用。

DONE!
