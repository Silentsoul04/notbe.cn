Autodesk 为其云技术发布新品牌－ Autodesk Forge

Aug 22, 2017


近些年来Autodesk陆续发布了不少云解决方案和服务，比如BIM 360， Infraworks 360，Autodesk 360/A360， AutoCAD 360等等，这些众多叫做360的产品或服务来势汹汹，有时候也让人有点摸不着头脑，Autodesk最近为其云服务技术发布了新品牌－－ Autodesk Forge。http://forge.autodesk.com/

Autodesk Forge不是什么新鲜事物，只是Autodesk 云服务的总称，上面提到的众多云产品和服务都包含在Autodesk Forge之下。一个类比就是亚马逊云服务叫AWS，其下包含S3， EC2，Elastic Search等等众多云服务；和亚马逊AWS对应Atuodesk Forge也是Autodesk的云服务的总称。

Autodesk Forge包含三部分，Forge 计划（program）、Forge 平台（platform） 和 Forge基金（ fund）。

Forge Program 致力于提供社区支持计划，通过代码示例、博客、论坛等多种方式支持用户使用Autodesk 云服务扩展自己的业务。

其中一个重要的活动就说6月中旬将在旧金山举办developer conference。http://forge.autodesk.com/conference/


Forge platform即一些列云服务的总称，即一系列为设计、制造、使用、协作和可视化提供的云服务和API。当然其中包括我们熟悉的View and Data API。


Forge Fund基金提供1亿美金的基金投资于创业公司，以便促进Autodesk云技术的应用。目前重点关注制造业相关项目。
