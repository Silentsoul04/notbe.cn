Visual Studio开发工具(4):Visual Studio 2010简体中文版会在6月中旬发布

Nov 15, 2017

  Visual Studio 2010本地化语言版本发布日程公布，简体中文版会在6月中旬发布 .
     Visual Studio 2010于2010年4月12日正式发布了。但是目前还只有英文版本，最近微软在紧锣密鼓地开发本地化语言版本。今天微软发布第一批本地化语言版本，分别为法语、德语和日语。
   
今后一段时间微软将陆续提供其它语言版本，具体发布日程如下：
 
    (1)5月下旬发布以下语言版本：西班牙语、意大利语、俄语；

    (2)6月中旬发布以下语言版本： 简体中文、 繁体中文、韩语；

    (3)今年夏季陆续发布以下语言版本：捷克语、波兰语、土耳其语、巴西葡萄牙语。
 
    (4)此外，微软会陆续提供以下语言包：阿拉伯语、希腊语、希伯来文、印地文、匈牙利语、马来西亚语、奥里雅语、泰米尔语、泰国语。
