传Windows 7 正式版明年6月发布

Nov 15, 2017

熟悉微软内部情况的人士周五表示,为及时消除Vista操作系统给微软带来的诸多负面影响,该公司将于今年10月底推出Windows 7首个测试版,且其正式版发布时间并不是2010年初,而是2009年6月3日.

美国知名IT网站internetnews.com援引消息人士的表述称,虽然微软对 Windows 7发布时间的公开说法是2010年初,但实际上微软内部已制定了Windows 7发布的时间表.该时间表显示,Windows 7的确切发布日期为2009年6月3日.


微软此前称,在今年10月底于洛杉矶市举行的“微软开发者大会”上,将公布Windows 7的更多技术详情.消息人士却表示,实际上在这次大会上,微软将发布Windows 7的首个测试版.一家不愿透露身份的硬件厂商也透露,微软已经把Windows 7内部测试版交给各大硬件厂商测试,以检测Windows 7与硬件之间的兼容性,但目前只有同微软存在合作关系的硬件厂商才能拿到该测试版.

对于上述最新传闻,微软一位发言人表示,Windows 7仍在按原定开发计划进行,至于其他方面的信息,微软目前还不方便对外透露.

美国市场研究公司Endpoint总裁罗杰·凯(Roger Kay)表示:“据我所知,微软正努力加大Windows 7开发进度.”独立市场研究公司Directions on Microsoft分析师迈克·切利(Mike Cherry)也认为,如果微软在今年10月底公布Windows 7首个测试版,届时会引起外界强烈关注.

上月期间,微软Windows部门高级副总裁史蒂文·辛诺夫斯基(Steven Sinofsky)正式启动了名为“Windows 7工程”的官方博客,称将通过该博客及时披露Windows 7开发的最新进展情况,并欢迎外部开发者和普通网民就Windows 7开发事宜积极提出建议.

辛诺夫斯基今年5月曾表示,Window 7将基于Windows Server 2008内核,而Windows Server 2008内核又是在Vista基础上改进而来,因此Window 7可以兼容所有支持Vista的驱动和软件.
