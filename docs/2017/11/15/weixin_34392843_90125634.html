【Bug笔记】The superclass "javax.servlet.http.HttpServlet" was not found on the Java Build Pat...

Nov 15, 2017

        因为今天下载了一个eclipse se的版本号。所以想把原本eclipse ee这个软件外面的目录eclipse名字该成eclipse ee，方便以后的区分和管理。改了后又一次打开eclipse ee目录下的eclipse.exe。就出现了这个错误（例如以下图a）。 究其原因发现曾经配置的eclipse信息全然初始化了，就像第一次打开elicpse一样。

所以又一次配置tomcat服务器。Window->Preferences->server->Runtime Environment->add->比方我是tomcat7.0这个版本号（如图b），就选择apache tomcat 7.0 ->Browser->比方我把软件放在D盘tools目录以下。路径就是 D:\tools\apache-tomcat-7.0.27...finish就好了（如图c）。
