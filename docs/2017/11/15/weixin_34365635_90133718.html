Java中String类的format方法使用总结

Nov 15, 2017

一.String的format类介绍

关于String的format() 方法还可以可参考如下博文:

http://www.cnblogs.com/fsjohnhuang/p/4094777.html  和  http://kgd1120.iteye.com/blog/1293633

String类的format()方法用于创建格式化的字符串以及连接多个字符串对象。熟悉C语言的读者应该记得C语言的sprintf()方法，两者有类似之处。format()方法有两种重载形式。

二.两种重载方法

改方法的重载方法有两种:

//1. 使用当前本地区域对象（Locale.getDefault()）格式化字符串
String String.format(String fmt, Object... args);

//2. 自定义本地区域对象格式化字符串
String String.format(Locale locale, String fmt, Object... args);

 每种重载方法的解释:

1.第1种:

该方法使用指定的字符串格式和参数生成格式化的新字符串。 新字符串始终使用本地语言环境。例如当前日期信息在中国语言环境中的表现形式为“2007-10-27”，但是在其他国家有不同的表现形式。

语法:

String.format(format,args...)

format：字符串格式。

args...：字符串格式中由格式说明符引用的参数。如果还有格式说明符以外的参数，则忽略这些额外的参数。参数的数目是可变的，可以为0。

 2.第2种 

该方法使用指定的语言环境、字符串格式和参数生成一个格式化的新字符串。新字符串始终使用指定的语言环境。

语法：

String.format(locale,format,args...)

locale：指定的语言环境。

format：字符串格式。

args...：字符串格式中由格式说明符引用的参数。如果还有格式说明符以外的参数，则忽略这些额外的参数。参数的数目是可变的，可以为0。

format()方法中的字符串格式参数有很多种转换符选项，例如：日期、整数、浮点数等。这些转换符的说明如

三.上代码:

  1 import java.util.Date;
  2 import java.util.Locale;
  3 
  4 import org.junit.Test;
  5  
  6 public class StringFormatUtil {
  7     public static void main(String[] args) {
  8         System.out.println(Locale.getDefault());
  9     }
 10     /*
 11     String.format()用法
 12     1、转换符
 13     %s: 字符串类型，如："ljq"
 14     %b: 布尔类型，如：true
 15     %d: 整数类型(十进制)，如：99
 16     %f: 浮点类型，如：99.99
 17     %%: 百分比类型，如：％
 18     %n: 换行符
 19      */
 20     @Test
 21     public void test1(){
 22      String str=null;
 23          //格式化字符串
 24         str=String.format("Hi, %s", "林计钦");
 25         // 输出字符串变量str的内容
 26         System.out.println(str);//Hi, 林计钦
 27         System.out.printf("3>7的结果是：%b %n", 3>7);//3>7的结果是：false 
 28         System.out.printf("100的一半是：%d %n", 100/2);//100的一半是：50 
 29         System.out.printf("50元的书打8.5折扣是：%f 元%n", 50*0.85);//50元的书打8.5折扣是：42.500000 元
 30         System.out.printf("上面的折扣是%d%% %n", 85);//上面的折扣是85% 
 31     }
 32     
 33     /*
 34     2、常见日期时间格式化
 35     c: 包括全部日期和时间信息 星期六 十月 27 14:21:20 CST 2007
 36     F: "年-月-日"格式，如：2007-10-27
 37     D: "月/日/年"格式，如：10/27/07
 38     r: "HH:MM:SS PM"格式(12时制)，如：02:25:51 下午
 39     T: "HH:MM:SS"格式(24时制)，如：14:28:16
 40     R: "HH:MM"格式(24时制)，如：14:28
 41      */
 42     @Test
 43     public void test2(){
 44         // 创建日期对象
 45         Date date=new Date(); 
 46       //格式化输出日期或时间
 47         System.out.printf("全部日期和时间信息：%tc%n", date);//全部日期和时间信息：星期四 八月 04 11:12:56 CST 2016
 48         System.out.printf("年-月-日格式：%tF%n", date);//年-月-日格式：2016-08-04
 49         System.out.printf("月/日/年格式：%tD%n", date);//月/日/年格式：08/04/16
 50         System.out.printf("HH:MM:SS PM格式(12时制)：%tr%n", date);//HH:MM:SS PM格式(12时制)：11:12:56 上午
 51         System.out.printf("HH:MM:SS格式(24时制)：%tT%n", date);//HH:MM:SS格式(24时制)：11:12:56
 52         System.out.printf("HH:MM格式(24时制)：%tR%n", date);//HH:MM格式(24时制)：11:12
 53     }
 54     
 55     /*
 56     3、格式化日期字符串
 57     b或者h: 月份简称，如
 58     中：十月
 59     英：Oct
 60      
 61     B: 月份全称，如
 62     中：十月
 63     英：October
 64       
 65     a: 星期的简称，如
 66     中：星期六
 67     英：Sat
 68       
 69     A: 星期的全称，如：
 70     中：星期六
 71     英：Saturday
 72      
 73     C: 年的前两位数字(不足两位前面补0)，如：20
 74     y: 年的后两位数字(不足两位前面补0)，如：07
 75     Y: 4位数字的年份(不足4位前面补0)，如：2007
 76     j: 一年中的天数(即年的第几天)，如：300 
 77     m: 两位数字的月份(不足两位前面补0)，如：10
 78     d: 两位数字的日(不足两位前面补0)，如：27
 79     e: 月份的日(前面不补0)，如：5
 80      */
 81     @Test
 82     public void test3(){
 83         // 创建日期对象
 84         Date date=new Date();
 85         //格式化日期字符串
 86         String str=String.format(Locale.US,"英文月份简称：%tb",date);
 87         //输出字符串内容
 88         System.out.println(str);//英文月份简称：Aug
 89         System.out.printf("本地月份简称：%tb%n",date);//本地月份简称：八月
 90         
 91         str=String.format(Locale.US,"英文月份全称：%tB",date);
 92         System.out.println(str);//英文月份全称：August
 93         System.out.printf("本地月份全称：%tB%n",date);//本地月份全称：八月
 94         
 95         str=String.format(Locale.US,"英文星期的简称：%ta",date);
 96         System.out.println(str);//英文星期的简称：Thu
 97         System.out.printf("本地星期的简称：%tA%n",date);//本地星期的简称：星期四
 98         System.out.printf("年的前两位数字(不足两位前面补0)：%tC%n",date);//年的前两位数字(不足两位前面补0)：20
 99         System.out.printf("年的后两位数字(不足两位前面补0)：%ty%n",date);//年的后两位数字(不足两位前面补0)：16
100         System.out.printf("一年中的天数(即年的第几天)：%tj%n",date);//一年中的天数(即年的第几天)：217
101         System.out.printf("两位数字的月份(不足两位前面补0)：%tm%n",date);//两位数字的月份(不足两位前面补0)：08
102         System.out.printf("两位数字的日(不足两位前面补0)：%td%n",date);//两位数字的日(不足两位前面补0)：04
103         System.out.printf("月份的日(前面不补0)：%te",date);//月份的日(前面不补0)：4
104     }
105     
106     /*
107     4、格式化时间字符串
108     H: 2位数字24时制的小时(不足2位前面补0)，如：15
109     I: 2位数字12时制的小时(不足2位前面补0)，如：03
110     k: 2位数字24时制的小时(前面不补0)，如：15
111     l: 2位数字12时制的小时(前面不补0)，如：3
112     M: 2位数字的分钟(不足2位前面补0)，如：03
113     S: 2位数字的秒(不足2位前面补0)，如：09
114     L: 3位数字的毫秒(不足3位前面补0)，如：015
115     N: 9位数字的毫秒数(不足9位前面补0)，如：562000000
116      
117     p: 小写字母的上午或下午标记，如：
118     中：下午
119     英：pm
120       
121     z: 相对于GMT的RFC822时区的偏移量，如：+0800
122     Z: 时区缩写字符串，如：CST
123     s: 1970-1-1 00:00:00 到现在所经过的秒数，如：1193468128
124     Q: 1970-1-1 00:00:00 到现在所经过的毫秒数，如：1193468128984
125      */
126     @Test
127     public void test4(){
128         // 创建日期对象
129         Date date=new Date();
130         System.out.printf("2位数字24时制的小时(不足2位前面补0):%tH%n",date);//2位数字24时制的小时(不足2位前面补0):11
131         System.out.printf("2位数字12时制的小时(不足2位前面补0):%tI%n",date);//2位数字12时制的小时(不足2位前面补0):11
132         System.out.printf("2位数字24时制的小时(前面不补0):%tk%n",date);//2位数字24时制的小时(前面不补0):11
133         System.out.printf("2位数字12时制的小时(前面不补0):%tl%n",date);//2位数字12时制的小时(前面不补0):11
134         System.out.printf("2位数字的分钟(不足2位前面补0):%tM%n",date);//2位数字的分钟(不足2位前面补0):32
135         System.out.printf("2位数字的秒(不足2位前面补0):%tS%n",date);//2位数字的秒(不足2位前面补0):26
136         System.out.printf("3位数字的毫秒(不足3位前面补0):%tL%n",date);//3位数字的毫秒(不足3位前面补0):982
137         System.out.printf("9位数字的毫秒数(不足9位前面补0):%tN%n",date);//9位数字的毫秒数(不足9位前面补0):982000000
138         
139         String str=String.format(Locale.US,"小写字母的上午或下午标记(英)：%tp",date);
140         //输出字符串变量str的内容
141         System.out.println(str);//小写字母的上午或下午标记(英)：am
142         System.out.printf ("小写字母的上午或下午标记(中)：%tp%n",date);//小写字母的上午或下午标记(中)：上午
143         System.out.printf("相对于GMT的RFC822时区的偏移量:%tz%n",date);//相对于GMT的RFC822时区的偏移量:+0800
144         System.out.printf("时区缩写字符串:%tZ%n",date);//时区缩写字符串:CST
145         System.out.printf("1970-1-1 00:00:00 到现在所经过的秒数：%ts%n",date);//1970-1-1 00:00:00 到现在所经过的秒数：1470281546
146         System.out.printf("1970-1-1 00:00:00 到现在所经过的毫秒数：%tQ%n",date);//1970-1-1 00:00:00 到现在所经过的毫秒数：1470281546982
147     }
148 }
