常见存储的串口连接管理方法

Nov 15, 2017

各大存储厂商的存储大多有一个COM接口，作为存储工程师提供一些重要配置的接口，COM接口的功能非常强大，可以直接与设备连接，实现对设备的配置，调试，数据传输等一系列操作。本文我们主要讲解通过COM口对设备IP进行管理与配置。下面以常见IBM的DS4700存储和戴尔MD3200存储为例，讲解一下通过COM接口管理IP的步骤。

DS4700串口管理IP配置

1、  串口线（console线）连接，COM线连接存储，其另一端是公头的9针接口，需要一条9针母头的接口转USB接口的COM线，连接到管理电脑。如下图：

2、  在驱动管理中，检查串口驱动是否正常（驱动安装略过），记录识别的COM口标号（当前为COM4），如下图：

3、  启动超级终端，新建连接，起名，继续选择对应COM口（当前COM4）如下图：

4、  连接配置：（波特率：9600  数据位：8  奇偶校验：无  停止位：1  数据流控制：无）如下图：

5、  进入到连接状态下，按Ctrl+Break（笔记本键盘Ctrl+Fn+B）键激活管理界面，当界面出现“Press within 5 seconds: for Service Interface,for baud rate”时按“ESC”键。会出现“VxWorks login:”提示输入用户名，这里输入“shellUsr”(注意大小写)，回车继续下一行出现“Password：”提示输入密码，这里密码输入“wy3oo&w4”回车确认后，出现“Serial Port shell started.”，提示shell模式已经启动，可以进行命令操作。如下图：

 若重复操作，或控制器继续上次操作，出现“Shell task 'tShell0' restarted...”提示，说明已经进入shell模式，可以进行命令操作。如下图：

 6、输入命令“netCfgShow”(注意大小写)，显示出当前设置，如下图：

7、  输入命令“netCfgSet”,可以进入更改这些配置，更改时每项逐一列出更改，回车确认。

更改成功后出现“Network Configuration successfully written to NVSRAM.”提示，为了确认更改，我们可以再输入“netCfgShow”查看配置。如下图：

注：1、若串口终端提示设置波特率，请按空格键，之后再发中断信号Ctrl + Break。2、不要输入S或Ctrl + S，否则无法进入shell命令行。如果已经错误的进入了Service interface界面，可多次按回车键退出该状态，并重新发送中断信号激活串口。

 

实际上，华为设备的市场占有量也很大，而华为的存储设备S3100、S3200就是OEM的IBM DS4700，所以连接配置中的参数与DS4700一致，所不一样的是用户名和密码，用户名：“root”,密码：“infiniti”。

MD3200串口管理IP配置

　　1、  串口线（console线）连接，COM线连接存储，其另一端是公头的9针接口，连接到管理电脑需要一条9针母头的接口转USB接口的COM线，连接到管理电脑，如下图：

2、在驱动管理中，检查串口驱动是否正常，记录识别的COM口标号（当前为COM4），如下图：

3、启动超级终端，新建连接，起个名字，继续选择对应COM口（当前COM4）如下图：

4、连接配置：（波特率：115200  数据位：8  奇偶校验：无  停止位：1  数据流控制：无）比特率首选115200，如不能连接成功尝试使用9600连接。如下图：

5、  进入到连接状态下。按Ctrl+Break（笔记本键盘Ctrl+Fn+B）键激活管理界面，当界面出现如下图：

6、在5秒钟内按“S”(注意大写)键，出现提示输入密码，输入密码“supportDell”。

　　　Enter Password to access Service Interface<60 sec="" timeout="">:supportDell

　　回车确认后，界面会显示出服务菜单。

　　　Service Interface Main Menu

　　1>     Display IP Configuration

　　2>     Change IP Configuration

　　3>     Reset Storage ArrayPassword

　　4>     Quit Menu

　　Enter Selection:

7、  此时输入数字进入选项操作，输入“1”，查看当前Ip设置。如下图：

 

8、  服务菜单界面输入“2”，进入IP设置，如下图：

9、  服务菜单界面输入“3”，重置密码，如下图：

 

通过COM接口连接，不仅可以实现IP的配置、重置密码，还可以通过命令行实现配置、调试、数据传输等操作，当然，我们通过COM接口连接得到管理IP，就可以连接管理软件进行您需要的管理操作了。
