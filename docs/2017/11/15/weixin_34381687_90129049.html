腾讯微博 - Windows Mobile版本

Nov 15, 2017

腾讯微博for windows mobile客户端，支持OAuth验证。

现有功能：

1) OAuth登录(首次使用需要)，授权时如手机浏览器提交没反映，可复制其url在pc端完成验证，再输入得到的验证码。

2) 支持发图片或文字微博。

3) 浏览我的首页，可分页。

 

在我的HTC D2上验证OK，主要还是方便自己。:)
