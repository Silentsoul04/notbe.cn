javascript专题：如何构建自己的js库

Nov 15, 2017

首先看看这个：

(function(){

           //运行的代码

                  })();

红色括号里面是一个匿名函数，红色括号是分割，表示里面的函数是一个部分，绿色的括号表示一个运算符，表示红色括号里面的函数要运行。

相当于定义完一个匿名函数后让它直接运行。

构建自己的js库就是用以下模板代码：

(function(){

        function $(){

          //代码

       }   

        window['myNameSpace']={}         //定义自己的命名空间     //   window['IC']={}

         window['myNameSpace']['$']=$;    //将自己的$函数注册到window的命名空间上

                  })();

在页面上引用自己库中的函数如下：

<body onload=" IC.$() "></body>        //IC为定义的命名空间，就可以调用到自己构建的js库文件中的函数

也可以这样写<body onload=" window.IC.$() "></body>     //window 可以省略。
