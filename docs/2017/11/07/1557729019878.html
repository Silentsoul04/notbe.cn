SSH远程快速登录Linux

       使用SSH管理linux服务器，通常要使用ssh，然后输入用户，密码，其实只要配置一个文件就可以方便登录。
假设要登录server域名是 www.internetcto.com.cn上一个名为cto的账号。若多次登录系统输这么长的名字比较麻烦，所以用cto作为 www.internetcto.com.cn的简称。此外，为了获得较高的传输速度，还要启用ssh的数据压缩功能。下面就可以使用如何设置ssh客户端的配置文件ssh_config:
#cp /etc/ssh/ssh_config   ~/.ssh/config
#vi ~/.ssh/ssh_config
然后在vi中按照如下内容编辑自己的配置文件
Host cto
HostName  www.internetcto.com.cn 
User cto
ForwardAgent yes
compression yes
然后保存退出。输入”ssh cto“命令，ssh会自动地从配置文件中找到主机的全名，使用cto的用户名登录。
如果管理多台linux服务器，则还以admin账号管理域名为 www.internetcto.bj.cn的服务器，同样要启用数据压缩功能能，就可以在上述配置下添加如下内容：
Host cgweb
HostNmae  www.internetcto.bj.cn
User admin
ForwardAgent yes
Compression yes
      保存退出，便可以用”ssh cgweb“来快速登录了，如果管理更多的服务器，还可以继续添加，但注意”Host“名称不可重复。
