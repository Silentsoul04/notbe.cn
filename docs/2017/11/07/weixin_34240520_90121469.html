对于spark源码的目录结构

　　1、编译相关    : sbt 、assembly、project

　　2、spark核心  ：core

　　3、Spark Lib  ： streaming 、 sql 、graphx 、mllib

　　4、运行脚本和配置 ： bin  、sbin 、conf

　　5、虚拟化 : ec2 、docker 、dev

　　6、式例  ： examples 、data

　　7、部署相关： yarn

　　8、python支持 ： python

　　9、repl ： repl

　　10、 3pp : externals
