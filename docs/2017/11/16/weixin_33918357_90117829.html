停下来，等等灵魂（三)

Nov 16, 2017

每一篇会写一些自己经历过的一些现实，不是抱怨，不是Hater，希望看的人能停下来，从自己的灵魂深处想一想，我们自己的某些地方，是不是也有这些烙印，我们是不是需要花点时间，思考一下自己，在我们每日低头往前狂奔的时候，我们是否和我们的灵魂在十字路口走算了。

此文接 停下来，等等灵魂（二），如果没有看之前的，请猛击这里!

前言

一般情况下我都会和面试者直接聊，聊做过的项目，自己解决过哪些问题，自己感兴趣的方向，平时都学习过什么东西等等，遇到问题时怎么解决等等，然后我会根据具体情况定一个起始的薪水。

上一篇里说的那些面试的种类里面，后来我还发现了，真的是"物以类聚，人已群分"。比如：“嘴上说的厉害，一写代码就支支吾吾的”，“比如项目里做了很久都一直是.Net2.0的”，“还有，对任何新的框架都保持恐惧的”, "还有一提ORM,就觉得性能底下，坚决不用的"。

我发现基本上上面的人说相同的话基本都是都是来自相同的公司，而且来自人数相对还比较多的公司，我为他们感觉深深的同情。如果你现在在这样的公司，我强烈建议你赶紧离开那个公司，也就是我说的，你赶紧停下来，先辞职，好好思考，好好学习，让你自己的灵魂赶上来，程序员不写代码，或者对新技术比较恐惧，想想是多么可怕的一件事，如果你愿意做出改变，我也乐意你能联系我。

发Offer之前，我们会和求职的人谈薪资，一般我们会问一下自己的期望，然后我们会给出自己的评价以及对应的薪水，但是经常有人不给自己的期望，然后问你：“你们一般的程序员薪水是多少？”，还有人直接问我个人工资多少钱的，我想说工资这一块，应该自己认认真真了解一下自己，然后了解一下市场，给自己定一个合适的价位，这样成功率就比较高，在西安，比如那些应聘ASP.NET工程师连HttpModule, 连cookie都不知道的人，你就不要开口说8000了。

发Offer

有些相对不错的，在以下几种情况，我们会发Offer:

  1. 态度积极，技术，沟通，英语都不错的且喜欢学习的，我们一般会很快的发Offer.
  2. 态度积极，技术基础扎实，但是一些框架没用过的，比如.Net基础扎实，非常熟悉ASP.NET Webform，一般会不会ASP.NET MVC已经不重要，JavaScript基础扎实，会不会jQuery就更不重要，英语能够基本读写的。
  3. 态度积极，技术一般，英语不错的。
  4. 态度积极，技术不错，英语相对差的，我们就会问他愿不愿意自己努力花时间把英语提上来，当然我们平时与客户的沟通都是英文，读写为主。
  5. 态度积极，学习能力不错，有一定的技术基础，有一定的英文基础，但是非常想来我们这提高自己的（这一种，如果你一开始就想要比较高的薪水，基本上也没戏）。

面试者对Offer的处理

话说我们发出Offer后，有很多人能够按时到岗，由于我们是一个人员规模不大的公司，我们希望建立的都是一个个比较精的小团队（为什么要小团队，后面会说), 也就是每一个发出Offer的人，我们都是花了很大的精力来甄选简历，认真的面试，讲公司的管理以及文化，谈薪资，但是今年我发现以下一些情况，或者说一股邪风。

  1. 等到入职的那天，也就是说你把一切都配好后，他不来，打电话问，说不来了，所以后来我都让人事提前三天确认一下。
  2. 拿着Offer，到别家公司去谈，要求别家公司给更高的薪水。
  3. 拿着Offer, 让自家的公司涨薪的。

我对上面几种情况感觉还是比较气愤的，我想说的是，如果你有上面几种想法的时候，你可以手里拿着Offer,但是拜托请你不要确认，因为一旦你确认后，公司这方就不会再在这个职位上招人了，等着你入职时，没人，往往会对公司，对客户造成很大的影响，更重要的事，这样的行为会让公司也有了戒心，这样对求职者就不好，所有的公司都有戒心后，对这个行业就会有负面的推动作用。

所以，当你接到Offer后，请从灵魂深处想一想，这个公司是不是你想要的公司，文化是不是你愿意的，更重要的是，不要只看工资上那么一点差距，比如500块的差距，你可以再和公司谈判，换句话说：别人多给你几百，你是不是要加班？你是不是没法规律的生活，你是不是累的每天趴到地上，每天搞到12点，然后你还要花几千块做个“大宝剑”来放松你，工资很重要，但是这个不是唯一的衡量，尤其在差别很少的情况，比如500块的差距，500块现在能干个啥？

小结

这片文章说了几个重点吧：

放弃现在
  1. 如果你公司有很多靠嘴写代码的，手不会写的人，请你赶紧离开。
  2. 如果你公司有很多对新技术保持恐惧的人，请你赶紧离开。
  3. 如果你公司的人很多时间都在玩，请你赶紧离开。
  4. 如果你公司的产品偷别人代码的人，请你赶紧离开。
珍惜机会，言出必行，合理处理Offer

当你拿到Offer的时候，不要觉得是一张纸，里面有你自己付出的汗水，有公司付出的汗水，更应该有对你原来的公司的一份感激，当你确认Offer的时候，请你认真思考，这里面有你的青春，这里面有你的责任，这里面也有你的承诺。 请记住，接Offer不是别人问你吃什么饭，而你说：“随便”。
