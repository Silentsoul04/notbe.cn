三、操作步骤

1、下载好doxygen tool的 unity package

地址： https://www.assetstore.unity3d.com/en/#!/content/13962 或 http://jacobpennock.com/unityplugins/DoxygenForUnity.zip

2、把Asset Package导入Unity后，找到此配置文件BaseDoxyfile.txt

4、回到Unity，点击 Window/Documentation with Doxygen

5、会弹出一个Documentation的窗口，红色信息提示你需要配置doxygen

配置项目信息

6、根据自己的项目情况，配置信息；配置完成后，切换到Generate Document选项卡，点击Run Doxygen

7、弹出Doxygen的窗口，切换到Run选项卡，运行Doxygen生成文档
