10个帮助网页设计师创建品牌形象的工具

Nov 16, 2017

许多设计师似乎都对自由职业有兴趣，因为这种情况下他们无须遵从老板的指示，也不必为工作暴走，因为他们可以选择自己喜欢的。不过，这必（并）不意味着你能随便得到最好的项目。事实上，你必须在你的专业能力上利用市场元素，以此向客户推销自己的产品。下面是10个帮助网页设计师创建品牌形象的工具。

Ping.fm

到这里分享最新的作品，或是你正专注的东西，通过平台通知人们比单个通知容易多了。无论新闻状态是“更新”还是新博客诞生，都会迅速传播，不同圈子的人们在此都能沟通。

LinkedIn

一个优秀的设计师能在这里很好地推销自己，因为这里有庞大的注册用户。作为一名设计师，你可在这联系用户或扩展你的社交圈。你的客户可以推荐你的作品，这对你的事业非常有利。

Tweetdeck

twitter爱好者很喜欢的站点。通过adobe air程序你能将关注的人分类。比如创建设计师组，用户组或是灵感组。筛选能提升你的品牌形象的tweets非常方便。

Retaggr

这个网站就是你提升工作的在线名片。你放在上面的简介可以变做e-mail签名，可以整合进博客，也可以整合到你正浏览的网站。通过利用这个网站，你能有效和自己的人际或专业网络联系。

seomoz Tools

自由职业者需要强大的在线展示来赢得客户，你可以使用网站提供的工具对自己的网站进行搜索引擎优化。

Google Analytics

用详细的数据优化你的网站，从中发现目标客户。

Shozu Desktop

网站针对图片和视频，移动终端上的内容可以轻松上传和发布。点击一下，就可以在三十多个社交网络上分享你的视频图片。不须单个站点上传就可轻松更新一个logo设计。

Pictobrowser

网站帮你把你Flickr上的图片放入你的最具影响力的文件夹。

Mars Edit

这个网站令博客发布更容易，同时也能有效管理媒体文件。它能节约自由设计者们的时间，因为它允许人们写不含内容的日志。

Yelp iphone App

网站帮你在你所在地区或别的地区发现客户。
自由设计师需要有效的市场营销手段来挖掘用户。上述工具网站能提高他们的工作，同时能与现有的和未来的用户保持联系。
