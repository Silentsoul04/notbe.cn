JAVA Web开发环境配置

Nov 16, 2017

假设JDK安装到C:\JDK6, TOMCAT安装到c:\Tomcat6

配置JAVA

1.       系统变量PATH变量后添加C:\JDK6\bin

2.       系统环境变量添加

JAVA_HOME: c:\JDK6

CLASSPATH: c:\JDK6\lib\tools.jar; c:\JDK6\lib\dt.jar;

TOMCAT配置

1.       环境变量添加TOMCAT_HOME: c:\Tomcat6

2.       将c:\Tomcat6\common\lib\servlet-api.jar添加到CLASSPATH

3.       Eclipse配置Tomcat

连接JDBC

1.       把sqljdbc.jar拷贝到Tomcat的Lib, JAVA的CLASSPATH要添加此驱动

2.       示例代码
