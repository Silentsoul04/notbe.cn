练习目标

在我之前的文章 Unity 2D Sprite Lighting ，讲到在2D Sprite中可以使用灯光，非常高兴的是，在Unity的新UI系统中我们也可以使用灯光

步骤

1、创建一个Panel，默认是没有材质的

2、为Panel选择 Default-Diffuse 材质

3、选择完材质之后，在Game视图，我们创建的Panel不见了，选择Canvas 修改Render Mode

4、创建一个Point Light，调整好位置

5、尝试修改 Light的参数
