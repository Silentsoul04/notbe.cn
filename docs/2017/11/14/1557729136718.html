Offie、Lync、Exchange 2013 RTM已发布！

Nov 14, 2017

当我们还在为Windows 8感到惊喜，为Surface的V5感到不知所措，就在此时此刻微软已经悄悄的发布了Office 2013从客户端到服务端的全部版本，这也是在Beta中被命名为“Office 15”、“Lync Server 15”的产品，这也正是新一代微软统一沟通解决方案所需要的产品，没错就是Lync Server 2013、Exchange Server 2013、SharePoint Server 2013三大微软重要的服务端以及办公平台最重要的Office套件2013版本。如果你是TechNet或微软MSDN订阅用户，现在已经可以在订阅网站下载到RTM正式版本，包括中文版！

不仅如此，我们的Project、Visio同样也放出了RTM的下载，暂时没有中文版，只有英文版。
