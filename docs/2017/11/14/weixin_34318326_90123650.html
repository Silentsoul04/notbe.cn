超强JavaScript编辑器WebStorm上安装zenCoding

Nov 14, 2017

给力的webStorm上缺少zencoding支持，那可真是编码不知码味，吃饭没有盐味，泡妞没有香味，人生没有趣味。

先下载zencoding for IntelliJ.IDEA

address:

http://code.google.com/p/zen-coding/downloads/list

WebIDE and IntelliJ IDEA Zen CSS 1.0 Sep 2009 7.3 KB 820   WebIDE.Zen.CSS.1.0.zip  
WebIDE and IntelliJ IDEA Zen HTML 1.0 Sep 2009 4.7 KB 917   WebIDE.Zen.HTML.1.0.zip  
WebIDE and IntelliJ IDEA Zen Coding 1.0   Featured

下载三个后解压。

然后找到以下目录：

C:\Documents and Settings\username\.WebIde10\config\templates (xp 系统 webStorm目录)

IntelliJIdea90\config\templates  (IntelliJ 目录)

把解压后文件拷贝进去。

重启webStorm，然后ctlr + alt + s中找live templates中就可以看到zencoding 、zen css、zen HTML，这说明已经安装成功。

然后有人会问，这个触发扩展的快捷键是哪一个呢？

webStorm只提供三个 space tab and enter，这个可以自己选。

在live templates的最下方，如果是宽屏的话，要仔细看哦~！！双击规则也可以看到。

by default expand with space tab and enter
