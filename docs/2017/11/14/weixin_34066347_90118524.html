2.2 Consumer API

2.2、消费者API

 

 

随着0.9.0版本，我们已经增加了一个新的Java消费者替换我们现有的基于zookeeper的高级和低级消费者。

这个客户端还是测试版的质量。为了确保用户平滑升级，

我们仍然维护旧的0.8版本的消费者客户端继续在0.9集群上工作，

两个老的0.8 API的消费者（ 高级消费者 和 低级消费者）。

 

 

The Consumer API allows applications to read streams of data from topics in the Kafka cluster.

 

Examples showing how to use the consumer are given in the javadocs.

 

 

To use the consumer, you can use the following maven dependency:

 对于使用生产者，你可以使用以下的maven依赖
		<dependency>
			<groupId>org.apache.kafka</groupId>
			<artifactId>kafka-clients</artifactId>
			<version>0.10.2.0</version>
		</dependency>
