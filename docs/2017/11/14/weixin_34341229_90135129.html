TFS二次开发系列：二、TFS的安装

Nov 14, 2017

本系列的实例将采用TFS 2012+Sql Server2012编写。

　　TFS的完整版本安装最好是在Windows server2008 64位以上版本中，其包括64位的SQL SERVER 2012，SharePoint 2012,Team Foundation Server 2012和Sql Server Reporting Services。

　　在Windows 7 版本中安装的为非完全版本的。本系列将采用Windows 7中安装进行二次开发。不包括SharePoint组件。

　　在安装过程中注意SQL Server选择为混合模式，在安装TFS的时候使用Windows身份认证的方式。

　　然后我们可以看到以下的TFS控制器，在这里可以管理团队集合、日志和生成配置等，完全版还有报告配置和团队门户配置。　　　　　　      

　　然后在VS2012中我们可以看到团队管理器如下，首先进行连接：

　　

　　点击团队资源管理器，选择“新建团队项目”

　　

　　选择MSF(Microsoft Solution Framework)模板，你可以选择敏捷解决方案或者CMMI解决方案。

　　

　　最后我们开始创建工作项试试。

　　工作项根据MSF模板的不同而不同，大致分为BUG、任务、产品积压工作项、测试用例等。可以根据自身项目需求而定。
