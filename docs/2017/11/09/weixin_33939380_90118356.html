Windows建立Cucumber和Ruby测试环境

Nov 9, 2017

1. 下载安装Ruby1.9.3, 不要用RubyInstall 一键安装，下载zip然后解压到c:\Ruby193 (不要用2.0,用2.0安装不成功，不要怪我)

2. 环境变量配置RUBY_HOME=”C:\Ruby193“

3. Path里添加 %RUBY_HOME%\Bin 或者直接添加 c:\Ruby193\bin

4. 下载Ruby DevKit (注意对应Ruby 版本)

5. 解压Ruby Devkit 到c:\RubyDevKit

6. cd c:\RubyDevKit

运行 Ruby dk.rb init

7. 打开c:\RubyDevkit\config.yml, 在最后加俩行 “- C:/Ruby193” 记住路径前一定要”横线“+空格 （我勒个擦，这个太坑人了)

8. 运行 ruby dk.rb install

9. gem install json

10. gem install httparty

11. gem install cucumber

12. gem install rspec

13. 下载ANSICON 解压

14. 运行ANSICON下x64下的ansicon.exe

15. 在ansicon.exe控制台里进入BDD文件夹  运行 cucumber features —format html -out test.html

大功告成!
