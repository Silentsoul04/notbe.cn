Ruby+Cucumber公司内训PPT

Nov 9, 2017

　最近正在对公司内部进行Ruby+Cucumber的培训(主要是测试人员)，我们的目标是让整个项目的人都掌握如何使用Ruby+Cucumber编写TestCase. 并没有很深入的去研究Ruby的知识，只是为满足写Case的要求。大概分为6课，课程结束后，相信大家都可以自己动起手来写case了。这里就分享下我们培训的PPT，至于详细知识有需要的朋友可以看看我的其它关于Ruby和Cucumber的文章吧。

 

课程安排：

　　1. Ruby&Cucumber Introduction

　　2. Basic knowledge of Ruby

　　3. Conditional Statements and Loops

　　4. Method、Class and Module

　　5. Cucumber

　　6. Watir
