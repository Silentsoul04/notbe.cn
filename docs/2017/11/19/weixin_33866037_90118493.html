Linux/Ubuntu 安装/卸载 软件

Nov 19, 2017

1、安装.deb文件

以安装Chrome为例：

（1）到官方网站下载Chrome浏览器相应版本，比如google-chrome-stable_current_i386.deb；

（2）Ctrl+Alt+T打开终端，进入deb文件所在目录（默认“Downloads”，中文操作系统的话就是“下载”）；

cd ～/Downloads

（2.5）安装之前，一般都会更新/升级

sudo apt-get update
sudo apt-get upgrade

（3）Deb包安装命令：

sudo dpkg -i google-chrome-stable_current_amd64.deb

可能会遇到下图中的问题，其中显示：

dpkg: dependency problems prevent configuration of ...

    ...

    Package libxss1 is not installed.

原因是依赖包的问题，安装一下依赖包“libxss1”

（4）安装依赖包，解决问题：

sudo apt-get install -f install

（5）完成上述操作，提示安装的时候输入“y”，回车，然后就安装好了。

（6）启动程序（终端启动或者Alt+F2搜索程序）

google-chrome

 （*）其他deb包操作相关命令

sudo dpkg -i package.deb //安装
sudo dpkg -r package_name //卸载
sudo dpkg -P package  //删除包（包括配置文件）
sudo dpkg -L package  //列出与该包关联的文件
sudo dpkg -l package   //显示该包的版本
sudo dpkg -l  //列出当前已安装的包
sudo dpkg –configure package  //配置包

2、使用apt-get安装软件

apt-cache search package  //搜索包
apt-cache show package //获取包的相关信息，如说明、大小、版本等
sudo apt-get install package  //安装包
sudo apt-get install package - - reinstall  //重新安装包
sudo apt-get -f install  //修复安装"-f = ——fix-missing"
sudo apt-get remove package  //删除包
sudo apt-get remove package - - purge  //删除包，包括删除配置文件等
sudo apt-get update  //更新源
sudo apt-get upgrade  //更新已安装的包
sudo apt-get dist-upgrade  //升级系统
sudo apt-get dselect-upgrade  //使用 dselect 升级
apt-cache depends package //了解使用依赖
apt-cache rdepends package //查看该包被哪些包依赖
sudo apt-get build-dep package //安装相关的编译环境
apt-get source package  //下载该包的源代码
sudo apt-get clean && sudo apt-get autoclean  //清理无用的包
sudo apt-get check  //检查是否有损坏的依赖
