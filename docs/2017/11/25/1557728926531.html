校园宿舍ADSL共享上网方案浅析

Nov 25, 2017

【 网友提问 】 你好!我们宿舍要上网,打算接ADSL请问用Adsl路由器+交换机的方式共享上网，是否可以?16个人同时上网的时候,速度是否会慢?
【问题解答】 采用带有路由功能的ADSL  Modem+ 交换机的方案共享 ADSL 上网的方法当然是可行的。主要是要弄清楚所用的 ADSL Modem 是否带有路由功能，并且正确地对它进行虚拟拨号路由设置（这里假设所采用的是 PPPoE 拨号方式上网的），当然不同品牌和型号的 ADSL Modem 路由功能设置方法不完全一样。更少不了的是正确进行整体共享网结构的设计和配置。下面我们就针对以上一个个问题向这位网友及所有有这方面需求的朋友进行介绍。
一、 ADSL Modem 路由功能 的检测
至于所用 ADSL Modem 是否支持路有功能，就要看所用 ADSL Modem 硬件和软件（软件也称“ Firmware ” ，固件）是否支持路由功能了。这方面我们主要是通常在相应品牌的官方网站查找相关信息，当然也可通过相应的固件配置进行验证，这方面将在本文后面介绍。
因为现在的 ADSL 用户在首次安装时所用的 ADSL Modem 通常是由 ADSL 服务商（ ISP ）提供的，所以用户一般是没有多大选择余地的。当然如果硬是想自己买，也完全可以，但这样做笔者认为不值得。因为现在安装 ADSL 时 ISP 所提供的 ADSL Modem 通常是免费的。
目前国内使用的 ADSL Modem 品牌基本上都具有路由功能，如伊泰克、华硕、全向、实达等。但要注意的是如果是早期的型号，通常只是从硬件上支持路由功能的，需要从其官方网站上下载相应型号 ADSL Modem 的最新版本升级 Firmware 软件（注意不是驱动程序），按相应的帮助文件说明升级，才能具有路由功能的。如笔者所用的伊泰克 TD-2001 ADSL Modem 原来在软件配置上并不支持路由功能，但伊泰克的官方网站（ [url]http://www.etek-td.com/[/url] ）却提供了这款 Modem 的固件升级软件 Firmware 3.08 beta2 （这是最新版本）。笔者也到上述几个主要品牌的官方网站上查看了有关信息，看到了用于用户自由下载的各自产品的固件升级软件。为了方便大家下载，现把主要型号的固件升级软件下载链接提供如下（都是指最新版本）：
伊泰克 TD2001   Firmware 3.08 beta2 固件下载网址为： [url]http://www.etek-td.com/intro-frame.htm[/url] 。 TD-2018 、 TD-2028 型号 ADSL Modem 均已具有路由功能，不用升级固件。
全向 QR8014 、 QR8021 、 QR8041 的最新固件下载网址都为： [url]http://www.qxcomm.com/download.asp[/url] 。 QL1680、QL1688、QL1880型号ADSL Modem已具有路由功能，不用升级固件。
华硕AAM6000EV、AAM6005HW的最新固件下载网址都为： [url]http://www.asus.com.cn/products/[/url] communication/download.asp 。ASUS AAM6000系列新版ADSL Modem产品均均已具有路由功能，不用升级固件。
实达网络（现为“锐捷网络”）的官方网站暂时打不开，所以不能提供相关信息。
要说明的是现在支持路由功能的 ADSL Modem 基本上都是外置以太网接口类型的， USB 和内置 PCI 接口的暂还没有支持路由功能的，在选择时要注意。不过现在 ADSL 服务商所提供的基本上都是外围以太网接口类型的 ADSL Modem ，不用太担心。
关于如何升级这些原不具有路由功能的 ADSL Modem ，各官方网站上基本上都提供了相应的说明文档，下载了解即可得知。它们基本上都是通过一个小文件传输软件 TFTP 进行升级的（这个软件也可能不同，到相应品牌的官方网站也有下载），具体在此就不再赘述了。
二、 ADSL Modem 路由功能的开启
升级了固件的 ADSL Modem 就支持路由功能了（或者所购的是较新款的 ADSL Modem ，原来就支持路由功能），接下来要做的事就是路由功能的配置了，实际上就是在 ADSL Modem 固件软件中把路由功能打开。因为在这些 Modem 的出厂默认配置中是没有打开路由功能的，而是采用普通的桥接模式（RFC1483 Bridged）。因为不同品牌和型号的 ADSL Modem 路由打开方法不完全相同，又不可能进行全面介绍，在此只能以笔者所用 TD-2001 为例进行介绍了，其实不同型号的配置方法还是有许多相似之处的，其它型号的参照即可。
因路由功能的具体配置又因不同的上网方式不同而不同，目前路由功能的配置主要有两种应用方式：（ 1 ）专线上网方式的路由配置，即RFC1483 Routed；（ 2 ）虚拟拨号（ PPPoE ）方式的路由配置。因篇幅关系，在此只能选择目前应用最广的虚拟拨号方式的路由配置方法为例进行介绍了。在 PPPoE 上网模式下， TD-2001 ADSL Modem 的路由功能开启方法如下（因为它是比较早的产品，需要事先经过固件升级）。
第 1 步，可以把 ADSL Modem 通过网卡直接与一台主机相连，配置好与 ADSL Modem 连接的计算机网卡 IP 地址为“ 192.168.1.x ” （ x 为 2~254 任一整数），这是因为伊泰克的 ADSL Modem 默认的 IP 地址为“ 192.168.1.1” ，与它相连的计算机必须与它在同一网段。不同品牌产品的默认 IP 不一样，有的是 192.168.1.0 ，有的是 192.168.0.1 ，可以参考相应公司的说明文档。
第 2 步，在 IE 地址栏输入“ 192.168.1.1” ，首先弹出的是一个用户身份验证对话框。在此只需要在密码处输入原始“ 12345 ” （不同品牌的原始密码也不一样，查找相关说明即可得知），按回车即可进入如图 1 所示配置界面。注意，必须先正确配置了第一步，否则将进入不了如图 1 所示的配置界面，因为不同网段是不能直接访问的。
图 1
第 3 步，这一步设置是整个路由配置的关键。单击如图 1 所示对话框左边主设置菜单中的“ ATM PVC ” 选项在弹出如图 2 所示对话框中，首先选择“ Connection List ” 列表中的其中一个选项。主要是选择一个对应的 VPI 和 VCI 值。
VPI 的英文称为“ Virtual Path Identifier ”，中文名为“虚拟路径标识”； VCI 的英文全称为“ Virtual Channel Identifier ”，中文名为“虚拟通道标识”。之所以要选择这对标志号，那是因为 ADSL 网络传统电话网线路（从局端到用户端之间的线路）都是采用 ATM 网络进行传输的， ATM 所利用的就 PVC 这种虚拟电路进行的。
在这款 ADSL Modem 中， VPI/VCI 对值 通常有两组，如 VPI （ 0 ） /VCI （ 35 ），和 VPI （ 8 ） /VCI （ 35 ），任选一组即可。不过这里所提供的 VPI/VCI 对是否符合你所用 ISP 提供的 ADSL 线路就不一定了，最好向 ISP 询问。基本上在以前采用单机虚拟拨号方式使用过的 ADSL Modem 会留下正确的配置的，所以在 Modem 中现有的 VPI/VCI 对配置一般是可用的。
选择了 VPI/VCI 对后，即会自动在对话框顶部的相应栏中出现，不会再输入了。然后选择配置界面中的“ Enable NAPT ”复选项，就是要允许进行 NAPT （ Network Address Port Translation ，网络地址端口转换）服务。因为要实现多个用户共享一条线路上网，实际上就相当于所有用户共享一个合法的公网IP地址（只不过此处的公网IP地址是动态的），所以要开启NAPT服务。
最后 就要上网方式的选择了，因为这里假设用户使用的是 PPPoE 虚拟拨号方式，所以要选择 PPPoE 单选项，如果是采用专线方式，则要选择“ RFC1483 Routed” 单选项 。在下面的相应栏中输入虚拟拨号用户帐户名和密码。至于身份验证方法（ Authentication ） 的选择可以默认即可。然后单击“ Modify ” ( 编辑 ) 按钮对所选择的虚拟线路配置进行修改，使其生效。
以上全部配置好后，即可直接单击对话框底部的“OK”按钮（图中未显示），完成本步操作。
以上介绍的是 ADSL Modem 中已有 VPI/VCI 对选项的情况下的配置方法。如果在“ Connection List ” 列表中没有可选项，则可直接在对话框中添加。方法是直接在对话框顶部输入你要添加的 VPI/VCI 对值（对应输入），其它选项的配置与上一样，在单击“ OK ” 按钮前先要单击对话框中间的“ Add ” （添加）按钮，把所添加的 VPI/VCI 对及其配置加进去。然后再单击对话框底部的“ OK ” 按钮完成本部配置。
前面提到如何验证所用的 ADSL Modem 是否支持路由功能可以从其固件配置中年历出，的确如此，我们只需在如图 3 所示对话框中查看是否有 “ RFC1483 Routed” 和PPPoE选项，即可知道是否支持路由功能了。如果仅有 “ RFC1483 Routed” 由表示此ADSL Modem只支持专线方式下的路由，不支持PPPoE方式的路由。
图 2
第 4 步，单击对话框中的“ OK ” 按钮后即出现“ Configuration OK.Use "Save Configuration" to save new settings to flash memory ” 的提示信息，要求对所做的配置进行保存。依提示信息单击对话框左边主设置菜单的“ Save Configuration ” 选项，并在所出现的对话框中单击“ Save ” 按钮保存所做的设置，当出现“ Configuration saved ” 提示时表明已经保存了所做的设置 . 。
第 5 步，单击配置界面左边主菜单（ Setup Menu ）中的重新启动（ Reboot ），然后单击弹出对话框中的重启（ Reboot ）按钮即可完成整个内置 PPPoE 共享的基本设置。重新启动后或通过重新上电即可实现 Modem 自动拨号从而实现局域网共享。
利用 ADSL Modem 自动拨号上网的配置方法是在配置主界面中左边主菜单的“ PPPoE ” 选项，在如图 3 所示对话框中选择前面所配置的 VPI/VCI 标志对值，并输入 PPPoE 虚拟拨号帐户名称和密码。“ Disconnect after idle minutes ” 项中可以设置连接的最长时间，为“ 0 ” 表示永不断线。然后单击“ Connect ” 按钮即可利用 ADSL Modem 自己进行拨号上网了，成功后在如图 3 所示对话框中顶部的“ PPPoE current status ” 栏中显示“ Connected ” ，否则显示“ Disconnected ” ，从中可以得知是否已成功连接。 在“PPPOE”里手动拨号成功后，单击配置主界面左边菜单的“Save configuration”菜单，然后在弹出的对话框中单击“Save”按钮保存，这样下次重起后就可以上电自动拨号了，省去了重新输入帐户的麻烦。
图 3
经过以上配置后，用户在上网时，只需单击 IE ， ADSL Modem 如果已连接成功的话，则可直接共享上网；如果在这之前没有用户连接上网，则 ADSL Modem 会自动拨号上网，非常方便。
如想用了一段时间后想禁用自动拨号功能，改为手动登录MODEM拨号，可在拨号成功后，进入如图3所示“PPPOE”对话框中填入用户名密码后单击“Disconnect”按钮断开，在进入“Save configuration”界面后单击“Save”按钮保存，这样下次重起后就为手动拨号了。
如果还想由 Modem 来为用户自动分配 IP 地址，则可打开 Modem 的 DHCP 服务，其配置对话框如图 4 所示。在这个对话框中首先要选择“ DHCP server ” 项后面的“ Enable ” 单选项，如果想要设置一个 IP 地址池，则可选择“ Assign address pool range ” 复选项，并在下面的“ Beginning IP address ” 项中输入 IP 地址（只需输入最后一位），在“ End IP address ” 项中输入最后一个可用的 IP 地址（也只需输入最后一位）。因为拨号用户通常不知道 ISP 的 DNS 服务器地址，所以也就不需要在“ DNS server IP Address ” 栏中输入 DNS 服务器的 IP 地址。其它项可保持默认，单击“ OK ” 按钮完成配置。同样要使配置生效，需要保存设置，单击主设置菜单的“ Save Configuration ” 选项，然后单击“ Save ” 按钮即可完成。注意在此配置了 DHCP 后，在其它系统中不要再配置 DHCP 服务了。
图 4
最后要说的是，如果是采取固定私有 IP 地址设置方法，而不是采用 DHCP 自动分配服务，则一定要保证所有 IP 地址均要与 ADSL Modem 的 IP 地址在同一网段，当然不能重复。 ADSL Modem 的 IP 地址和进入配置界面的密码都可以重新设置，只需分别单击配置界面左边主菜单的“ LAN IP ” 和 Change Password “ 菜单进入相应的配置界面重新设置即可。
以上介绍的是伊泰克 TD-2001 型 ADSL Modem 的路由功能开启方法，其它品牌、型号的 ADSL Modem 路由功能开启方法参照即可，同时在相应品牌的官方网站上也有相应的说明文档。
三、共享网络结构
介绍完了 ADSL Modem 的路由配置方法后，下面继续介绍这种网络共享的网络结构。既然要实现路由共享，就是想彻底摆脱各用户受某一主机的限制，所有用户只要在路由器打开的情况下直接上网，不用经过其它主机拨号。所以在这种共享方式中， ADSL Modem 就不能直接与任何主机相连了，只能与所用的交换机连接，作为一个独立的终端。具体网络结构配置方案如图 5 所示。
图 5
在这个网友的实际应用中有 16 个用户，加上 ADSL Modem 也必须直接连接交换机端口，所以交换机的端口必须在 17 个以上，最好选择 24 口的，以方便扩展。在速率上当然只需选择 10/100Mbps 的即可。因为它的端口带宽完全可以满足实际网络应用，同时也远大于每个用户的实际共享上网速率。所用双绞线也都可以是直通式的，就是不用跳线，包括 ADSL Modem 与交换机的连接，购买 ADSL Modem 的那条网线就不能用了，因为那是交叉式的，主要是应用于单机与计算机直接连接。
