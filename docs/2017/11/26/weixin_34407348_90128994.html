一款不错的网站压力测试工具webbench

Nov 26, 2017

webbench最多可以模拟3万个并发连接去测试网站的负载能力

1、适用系统：Linux

2、编译安装：

wget  http : //home.tiscali.cz/~cz210552/distfiles/webbench-1.5.tar.gz
tar zxvf webbench-1.5.tar.gz
cd webbench-1.5
make && make install
 

Unbuntu下非root，使用命令：sudo make install (Unbuntu桌面下进入打开命令：ctrl+alt+t)
3、使用：

webbench -c 500 -t 30 测试网站的 url

参数说明：-c表示并发数，-t表示时间(秒)
