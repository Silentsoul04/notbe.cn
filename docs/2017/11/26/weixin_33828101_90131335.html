Windows Phone 7 学习笔记1：Hello, Windows Phone 7

Nov 26, 2017


　　Windows Phone 7 的开发环境安装这很简单，只需要从微软官方下载到SDK（目前最新版本是7.1，适用于芒果7.5的设备），安装后就可以了。如果你的机器没有安装vs2010或者vs2010不是英文版，在安装sdk的时候会附带安装一个 visual studio 2010 express for windows phone，其实就是一个精简版本的vs，只能用作开发windowsphone应用程序。

　　安装完成后，运行 vs 2010 express for windows phone， 新建一个“Windows Phone Application”，取名为HelloWindowsPhone7，选择保存位置后点击确定。

　　Windows Phone 支持的分辨率有两种，分别是320*480和480*800。

　　这个东西采用的是XAML语法，所以熟悉WPF或者Silverlight的话很容易上手。

　　修改/Properties/WMAppManifest.xml中App节点的Title属性，可以设置在设备中显示的名称，如下图：

　　修改MainPage.xaml文件，将ContentPanel修改如下：

        <Grid x:Name="ContentPanel" Grid.Row="1" Margin="12,0,12,0">
<TextBlock x:Name="tbkMessage" Text="Welcome to windows phone 7" HorizontalAlignment="Center" VerticalAlignment="Center"></TextBlock>
</Grid>

　　运行程序，可以看到，模拟器屏幕中央会出现一行文字。

　　我们在debug目录中找到.xap文件，这个文件就是在模拟器中运行的程序了，也是wp7的安装文件。其实这个文件是一个.zip的压缩包，当你把后缀名改为.zip后，用解压缩工具可以看到里面包含的内容。
