WIN8 默认以管理员启动程序

Nov 26, 2017

方法1：

 注册表中HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\AppCompatFlags\Layers

新建一个字符串值，名称设置为程序的路径（不带引号），数据为“RUNASADMIN”，

 

 

方法2： 

WINDOWS 8 默认就是以管理员身份启动VS。

1、定位到VS程序文件（devenv.exe）:
这个文件一般在：安装盘:\Program Files\Microsoft Visual Studio 10.0\Common7\IDE；
也可以直接在任务管理器中右击 VS 的进程名，然后点击打开文件位置，来查找VS所在路径

定位到VS程序文件（devenv.exe）

2、 右击 devenv.exe 选择兼容性疑难解答

右击 devenv.exe 选择兼容性疑难解答

3、 选择 “疑难解答程序 (按钮)”  (按钮)选项 ——>下一步

点击 疑难解答程序

4、 选中 “该程序需要附加权 ”复选框 ——>下一步

选中 该程序需要附加权限 复选框

5、 点击“测试程序” 按钮  ——>下一步：点击以后系统会以管理员身份打开VS ，关不关闭都可以，没有什么影响的

点击测试程序 按钮

6、 点击 “是，为程序保存这些设置” ——>下一步

点击 是，为程序保存这些设置

7、最后，关闭 程序兼容性疑难解答  对话框，现在，直接打开VS看一下上面标题是否带有（管理员），如果有恭喜你成功了
