第01篇：C#星夜拾遗之如何开始C#学习

Nov 26, 2017

从现在开始学习吧。

正确认识C#

它的读音为C Sharp，这个很重要。百度百科上面有C#这个词条，可以详细的了解一下，这对正确认识C#有很大帮助：http://baike.baidu.com/view/6590.htm

配置开发环境

C#的开发环境很简单，安装一个Visual Studio（以下简称VS）就可以了。VS 目前的最新版本是VS 2010，笔者用的仍然是2008版。

开发工具的版本不是最主要的，这里建议初学者使用VS 2008。对于VS 2008 的安装，可以参考这里：Visual Studio 2008安装过程详解。

在安装完成后建议安装Sp1补丁包，并安装Microsoft SQL Server Management Studio Express（VS 2008 默认已经安装了Sql Server Express Edition版，只用单独安装一个管理工具就可以了）。

这些东西搞完后，就可以开始编程了。

控制台版Hello World

打开VS 2008，新建一个控制台应用程序的项目，并在Main方法中添加下面两行代码：

Console.WriteLine("Hello world");

Console.ReadKey();

按 CTRL + F5 运行程序即可。

WinForm版 Hello World

打开VS 2008，新建一个Windows应用程序的项目，在Form1的编辑窗口中双击进入From1的Load事件，并在事件处理函数Form1_Load中添加如下代码：

MessageBox.Show("Hello world");

按 CTRL + F5 运行程序即可。

Asp.net 版 Hello World

打开VS 2008，新建一个Web应用程序的项目，在Default.aspx页面的源编辑界面，在div标签中插入如下内容：

<span>Hello world</span>

按 CTRL + F5 运行程序即可。
