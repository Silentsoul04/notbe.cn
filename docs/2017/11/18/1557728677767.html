“英伦配”收视牛过本山，网管软件如何配奇兵

Nov 18, 2017

选出网络中的奇兵

  “ 2009 年春晚，一个突出的创新，就是大规模应用影象技术，带来美轮美奂的视觉效果。这将改变观众对春晚的固定思维模式：原来春晚也可以这样做。”据郎昆介绍，今年春晚的灯光和舞美都由奥运会开幕式的原班人马来设计，通过投影、三维等技术，影象与节目将实现互动呼应，具有超强的视觉冲击力和感染力。舞蹈《城市变奏曲》将以巨大的多媒体效应，跨越从清末到今天的上百年历史，在城市雕塑、芭蕾、探戈的转换中，实现传统与现代的对话；舞蹈《蝶恋花》，也将利用影像营造出一种美妙绝伦的背景，以舞蹈演员的优美动作相呼应，为观众带来视觉上的优美享受。
  从总导演郎昆的披露看， 2009 年春节晚会将是一台令人耳目一新并有很大创新的春节晚会，除了回归大联欢的主题和性质，强化演员、舞台、现场观众席的互动是一个特色外，打组合牌、大胆亮剑也是一大特点，让宋祖英与周杰伦在春晚上实现“英伦组合”就是非常有新意的创意组合。郎昆透露，“这次春晚，将会把很多观众喜欢的演员和节目进行组合，宋祖英与周杰伦会同台合唱。这次春晚宋祖英与周杰伦将合唱《本草纲目》，在春晚上实现“英伦组合”，两岸歌手同唱“中国风”格外令人期待关注。
但是网管软件如何才能配出网络中的奇兵，创造出网络中的最高收视，换句话说如何为你的企业搭配最佳组合！
 
用户和厂商共同探讨需求
      不同的网管软件之间的功能不一样，有的是桌面管理系统，有的是网络监控系统，还有的是邮件、页面管理系统。用户必须在种类众多的网管软件中挑选出最为适合自身需求的产品。如何来确定自身需求呢?本人结合自身的项目经验，认为正确的做法是用户和厂商共同探讨制定《需求说明书》。
很多企业网络管理人员认为自己的网络管理自己最了解，这个观点其实是有着两面性的，既可以说他对，也可以说他不对。从日常的网络运维工作的角度来看，网管员确实是对本单位网络最为了解的人。但是企业网络的规模并不是一成不变的。摩卡业务服务管理软件认为：“从发展的角度来看，网管软件厂商凭借其项目经验，往往能够比较准确的估计出在近几年内用户网络的建设速度和规模，预见到客户的未来需求。功能单一且没有可扩展性的网管软件是适应不了当今信息化的发展速度的。”在这些潜在需求里面也许有厂商夸大的成分，但是最终的取决权还是在用户的手里，所以多听听厂商的意见并不是一件坏事。
 
如何成为网管软件中的收视冠军
一套综合性的 IT 管理软件，应该涵盖基础架构管理——包括网络监控管软件功能，具有网络拓扑、主机、流量分析、 IT 资产管理、 IT 变更管理等功能；应用管理——应用服务器、数据库、 Web 服务器等；端到端响应时间管理——应用响应时间管理，端到端监控；业务服务管理——从业务视角看待 IT ，进一步完善 IT 运维管理、往 IT 服务管理（ ITSM ）迈进； IT 运维管理——超越了网管理念，基于 ITIL 流程框架，满足对事故管理、问题管理、性能管理、变更管理、配置管理、发布管理、知识库等需求。
具体应该从一下几个方面来衡量：
 1 、较全的管理范围：对网络设备、主机、应用以及机房和业务，都可以做到统一管理，从而可以为关键业务的监控提供完整真实的数据。另外，还可以从多个角度对 IT 设备进行监控，除了监控 IT 设备的性能、可用性和变更外，还支持对网络流量进行协议层的分析，支持包括 Cisco 和 H3C 在内的当前主流的网络厂家设备；监控网络设备的配置文件的变更，并对其进行备份和恢复。
2 、监控指标多：好的网管软件提供的监控指标都是应该经过前期需求分析，筛选后的指标，是对用户真正有用的指标。同时，还应该将这些指标进行详细的分类，有的指标是和性能相关的，当这些指标超过一定门限值后会产生报警，而有些指标则不是，例如变更指标、物理内存、 IP 地址、虚拟内存 JVM 的大小等，这些指标都是如果发生变更就会对企业业务产生较大影响的数据，网管软件设立的大量的监控指标对于企业 IT 管理人员具有简单实用的意义。
3 、人性化的管理流程：  IT 管理员解决问题的过程，往往需要一整套运维管理方法来帮助 IT 运维团队有条理的完成工作，并且这个运维管理的建设应该是一个长期的建设过程，需要经历“计划、实施、检查、改进”这四个步骤来不断完善企业的运维管理流程。这个时候就需要 IT 运维管理软件来解决这个问题。好的 IT 运维管理软件提供了流程和表单的设计工具，并且按照 ITIL 的最佳实践方式为用户内置一整套的运维管理流程，各个流程环环相扣，帮助用户在长期的 IT 运维管理过程中不断完善提高我们的运维效率。
 
因此，一套好的网管软件可以帮助 IT 管理员提高工作效率，起到事半功倍的效果。也希望网管行业能够涌现更多的“收视冠军”。
