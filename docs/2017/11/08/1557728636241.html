　　1、首先按照图中显示的路径找到存放我们在本系列文章四中创建的Site的设置文件的文件夹。


　　2、找到并打开web.config文件，在图中标识的位置添加<connectionStrings>……</connectionStrings>节。
    1      < connectionStrings >
    2          < remove   name =" LocalSqlServer"  />
    3          < add   name =" LocalSqlServer"   connectionString =" Server=(local);Database=aspnetdb;Trusted_Connection=True"   providerName =" System.Data.SqlClient"  />
    4      </ connectionStrings >


　　3、重复第一步的过程，找到存放我们在本系列文章三中创建的Application的设置文件的文件夹。
