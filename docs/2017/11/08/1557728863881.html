10.VMware View 4.6安装与部署-view clint和view for ipad连接测试

Nov 8, 2017

        安装基于 Windows 的 View Client,最终用户需要从物理机打开 View Client 来连接其虚拟桌面。您可以运行基于 Windows 的安装程序文件来安装 View Client 的所有组件。如果 View 管理员启用了某些显示选项，那么除了通过 View Client 访问虚拟桌面外，最终用户还可以使用 ViewClient 配置这些显示选项。例如，最终用户可以选择显示协议或窗口大小，或者使用当前登录凭据进行 View身份验证。使用 View Client with Local Mode 时，最终用户会将虚拟桌面的副本下载到他们的本地计算机。之后，最终用户将可以在没有网络连接的情况下使用虚拟桌面。这会最大程度降低延迟并提高性能。

要启动 View Client 安装程序，请双击安装程序文件。

按照提示安装所需组件。

VMware View Client 服务会安装在 Windows 客户端计算机上。View Client 的服务名是 wsnm.exe。USB 组件的服务名是 wsnm_usbctrl.exe

安装后需要重新启动。

如果 View Client 在安装后没有自动启动，请双击桌面快捷方式或单击 [Start（开始）] &gt; [Programs（程序）] &gt; [VMware] &gt; [VMware View Client]。

输入有权使用至少一个桌面池的用户的凭据，选择域，然后单击 [Login（登录）]

要选择显示协议，请单击列表中桌面旁的向下箭头，再单击 [Display Protocol（显示协议）]，然后选择协议。

从桌面池列表中选择一个桌面，然后单击 [Connect（连接）]。View Client 将尝试连接到指定池中的桌面。

连接成功后，屏幕上将显示客户端窗口。

管理员指定root的桌面是vm-1-mabf

在Ipad上使用view

是首先进入apple 应用商店，搜索到vmware view for ipad。当然，也可以安装在iphone上。

这里我已经装过了，没有安装的可以选择安装，vmware view for ipad是免费的应用程序。

安装成功。

点击Add View Connection Server。

输入View Connection Server 的地址。连接。

之后输入用户名和密码。

可以看到win7 的桌面。

简单的鼠标操作技巧。

登入win7的桌面

使用Ipad键盘。

要使用桌面本地模式，vmware view必须配置View Transfer Server。才能使用View Client with Local Mode，将桌面下载到本地使用。然后通过同步的方式更新操作。在首次连接时，将要安装View Client with Local Mode，并连接服务器将服务器上的桌面下载下来。

View Client with Local Mode与View Client安装方式相同，登录方式也相同，安装好后连接服务器。

安装View Client with Local Mode不能安装在虚拟机里，也不能安装有vmware workstation的电脑中。

安装好后按照View Client的方式连接服务器。比View Client多了一个选项就是下载桌面到本地。

view clint和view for ipad连接测试.结束。目前vmware还有vmware view for MAC vmware view for AndRoid。
