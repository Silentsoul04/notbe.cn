四、新计算时代将来临
我们目前正处于第五代计算时代。前四代分别是：
1. 单机、
2. Client/Server(客户端-服务器)
3. Web
4. SOA等
下一代就是
5. 服务(Service)或者云计算
SOA和服务时代之间的不同就是SOA对扩展处理的不好:
  • SOA虽然它也能扩展，但是需要针对每个公司和项目手工进行调整。
  • 而对服务或云计算来说，扩展能力是其平台的本质。
