启用OWA提示用户密码过期工具

Nov 8, 2017

前不久和一个关系不错的朋友吃饭，聊到要是能用的Exchange2010的Web页面提示用户定期修改密码就好了。
于是找了很多资料，发现还真有这样的方法。
这些资料写的很详细，告诉大家如何去修改注册表，每一步都有截图。
推荐这篇： 配置 OWA 下次登录时更改密码 图文并茂写的很细致。

但是，很多朋友没有修改过注册表，又或者怕改错。
考虑到这一点，我导了个注册表出来，供大家使用。
系统环境：Exchange Server 2010 With SP1 
角色：CAS
使用方法：
1、双击“OWAPasswordEnable.reg”文件，点击“是”；

2、导入成功后，点击“确定”；
3、双击“IISReset.bat”文件，重新启动IIS服务让以上设置生效；
验证方法：
1、在Exchange管理控制台中创建一个下次登录需要修改密码的用户帐户；
2、使用该帐户在OWA中登录，结果如下图所示，便成功了。

从此您再也不用为用户不定期修改密码而发愁了。
