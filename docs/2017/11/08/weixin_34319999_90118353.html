ubuntu10.10自动挂载 win分区

Nov 8, 2017

我的系统ubuntu10.10,安装了ubuntu桌面，发现不能自动挂载win的分区。
1.首先 建立挂载目录用于自动挂载windows的分区
在终端运行命令：sudo mkdir /media/D 

2.运行 

sudo fdisk -l
查看磁盘分区情况, 下面以我的硬盘为例：
3.运行
sudo vi etc/fstab    ＃修改/etc/fstab配置文件，添加如下内容（橙色字体部分）
要注意分区格式！
# <file system> <mount point>   <type> <options>       <dump> <pass>
/dev/sda5       /media/D vfat      iocharset=utf8,umask=000 0 0
/dev/sda6       /media/E ntfs-3g iocharset=utf8,umask=000 0 0
/dev/sda7       /media/F ntfs-3g iocharset=utf8,umask=000 0 0 
保存退出reboot，一切顺利的话，在文件系统中你就可以访问windows分区了。
手动挂载fat时加编码
手动挂ntfs时不会加。只要安装了ntfs-3g就可以了

mount -t vfat -o iocharset=utf8 /dev/sdb /mnt/usb
