Windwows7 下安装mysql5

Nov 24, 2017

用二进制安装老出错，所以用绿色版安装

1、下载 mysql-noinstall-5.4.2-beta-win32.zip

2、解压到D:\Program Files\mysql5.4.2-beta（你可以设你自己的目录）

3、cmd打开命令行输入 echo %WINDIR%， 显示你的系统目录如：c:\Windows

4、在这个目录下新建my.ini文件，修改此文件权限为完全控制

5. 修改my.ini文件为如下内容

1
2
3
4
5
6
[mysqld]
# set basedir to your installation path
basedir=D:/Program Files/mysql5.4.2-beta
 
# set datadir to the location of your data directory
datadir=D:/Program Files/mysql5.4.2-beta/data

6. 在系统变量里添加D:\Program Files\mysql5.4.2-beta

7、安装mysql服务到windows service, 输入命令： mysqld –install

8、启动mysql: net start mysql

9、测试mysql: mysqlshow
