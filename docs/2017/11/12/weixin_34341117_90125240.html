无线局域网的应用

Nov 12, 2017

 

1、使用家长控制无线网络功能

　　很多无线路由器都有家长控制功能，能针对孩子在周一至周日不同的学习要求，设定不同的上网时段和网站限制，让孩子远离不良网站，从而实现健康上网。

　　下面举例说明家长控制功能的配置，例如：在周一到周五的允许上网时间1（20:00~21:00），仅允许孩子（MAC地址为00-1A-4D-31-2B-28）浏览中国学习在线、中国英语网、新浪、腾讯、百度等网站；在周六到周日的允许上网时间2（9:00~21:00），仅允许孩子浏览中国学习在线、中国英语网、新浪、腾讯、百度和迅雷等网站；家长（MAC地址为74-EA-3A-7B-93-99）在任何时间都可以上网。

　　使用家长控制无线局域网功能的设置步骤如下：

　　步骤一：进入无线路由器的管理界面，在左侧的导航栏中选择“上网控制  |    日程计划 ”，进入“日程计划设置”界面，单击“增加单个条目”。

　　步骤二：进入设置界面，添加周一至周五的允许上网时间1（即20:00~21：00），单击“保存”。

　　步骤三:单击“增加单个条目”，再添加一个条目，设置周六和周日的允许上网时间2（即9:00~21:00），单击“保存”。

　　步骤四:返回“日程计划设置”界面，即可看到所添加的日程计划。

　　步骤五：在左侧导航栏中选择“家长控制”功能，进入“家长控制设置”界面，选中”启用“单选按钮，填写家长电脑的MAC地址（如当前管理路由器的电脑即家长电脑，可以单击”设为家长PC“），然后单击”保存“。

　　步骤六:单击“增加单个条目”，进入“家长控制规则设置”界面，进行相应的设置，单击“保存”

　　步骤七：按与上一步相同的方法，再添加一个规则，单击“保存”

　　步骤八：返回到“家长控制设置”界面，即可看到所添加的规则，则家长控制功能已经开启。

2、配置Windows 7\8\10无线共享（系统自带）

　　通过设置Windows 8的无线共享，可以让两台或者多台笔记本通过其中一台笔记本的Internet网络无线上网，实现原理是借助笔记本的内置无线网卡建立虚拟AP设置一个无线覆盖热点。Windows 8的虚拟基站功能，能将一台能上网的电脑（前提是有无线网卡）虚拟成一个无线基站，共享网络连接。

　　步骤二：接下来搭建承载网络，输入“netsh wlan set hostednetwork mode=allow ssid =无线网络名称（比如hjq） key = 无线网络密码（密码至少8位）”，再按Enter键确认，提示承载网络设置成功。

　　　比如，  输入“netsh wlan set hostednetwork mode=allow ssid=xxxxx key=xxxxxxxx（8位或以上）”

　　注意：ssid号主要用来区分不同的网络，最多可以设置32个字符，只有为两台计算机的无线网卡设置了相同名称的ssid，才能实现互相通信。
