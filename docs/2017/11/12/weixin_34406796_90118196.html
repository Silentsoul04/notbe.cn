Eclipse/MyEclipse按任何键，都可以提示?（最强帮手）

Nov 12, 2017

说明：

　　一般在Eclipse ，MyEclipse代码里面，打个foreach，switch等这些，是无法得到代码提示的（不信自己试试），其他的就更不用说了，而在Microsoft Visual Studio 里面是得到非常友好的代码提示的。

     实际上，Eclipse ，MyEclipse代码里面的代码提示功能默认的一般是点“.”，一般是有了点“.”，才会有代码提示。原理：“Auto Activation triggers for java”这个选项就是指触发代码提示的的选项，把“.” 改成“zxcvbnmasdfghjklqwertyuiop.(,”的意思，就是指遇到26个字母和.，（这些符号就触发代码提示功能了。（具体后面有说，放心）增强Eclipse ，MyEclipse 的代码提示功能，具体怎么样来配置？

 

 

 

 

 

Eclipse如何做到按任何键，都可以提示?

 　　第一步：Windows  -> Perference 

 

 

　　第二步：输入ass

 

 

 

　　第三步：选择Content Assist

 

 

 

 　　第四步：可以看到，默认是.

 

 

 

 

　　第五步：现在，我将.改成zxcvbnmasdfghjklqwertyuiop.
