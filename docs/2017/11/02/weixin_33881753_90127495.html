网虫推荐-firefox 4.0插件

Nov 2, 2017

   firefox因其强大的插件扩展一直受到众多人亲赖，最近firefox又推出了4.0正式版本，相对3.6版本，页面加载速度上有不少提升，对于使用firefox4.0的网虫而言，哪些插件是必须的呢？

1. url拖拽打开：DragUrLink https://addons.mozilla.org/en-US/firefox/addon/dragurlink/

        推荐理由：很多常见浏览器都支持鼠标推拽url，搜狗，世界之窗，opera（url上下拖拽），对于这个功能，大家一定都不陌生，并且已经习惯了吧。这里我不多解释为什么要推荐了。

       本来先前在3.6版本有个功能更强大的dragit，不过很可惜，4.0上它不兼容。所以用DragUrLink来暂时代替吧。

2. 下载工具集成插件：flashgot https://addons.mozilla.org/en-US/firefox/addon/flashgot/

        推荐理由： 通过这个插件，可以在firefox中调用外部下载工具，例如迅雷，flashget等。本来在3.6的firefox中，迅雷是自带了个插件，安装好之后，是不需要在通过flashgot就能直接用迅雷下载东西的，不过在4.0的firefox版本中，迅雷自带的插件同样悲剧，不兼容了，因此flashgot还是很有必要安装。

3. 原生态下载工具：DownThemAll! https://addons.mozilla.org/en-US/firefox/addon/downthemall/

      推荐理由：提升下载速度到普通下载速度的4倍，并且支持批量下载。不过从官方的描述来看，没提到说支持断点续传功能，不过好在大多数情况下，下载大文件时，我们可以用迅雷等下载工具。

4.. 视频下载：net vedio hunter https://addons.mozilla.org/en-US/firefox/addon/netvideohunter-video-downloade/ 

        推荐理由：强大的网络视频探索功能，只要在页面上打开过一个视频，它就能获取到对应的视频下载地址。

5. 广告过滤：adblock-plus https://addons.mozilla.org/en-US/firefox/addon/adblock-plus/

与 人人网清爽浏览：https://addons.mozilla.org/zh-CN/firefox/addon/xiaonei-reformer/

        推荐理由：adblock-plus的强大的广告过滤功能，这个我就不多说了吧，浏览网页的神器啊。

       至于人人网清爽浏览么，现在人人网上的广告，也是挺烦人的，人人网注册用户这么多，里面像我这样成天上网的人也不在少数，因此也推荐下这个吧。这个插件的chrome版本用过，效果是人人网里面的广告，还有浏览器窗口底部的聊天工具条，都消失了，很不错。至于firefox版本的人人网广告过滤器，大家先试试吧，俺还没试呢。。。。

6.类似图片搜索：TinEye Reverse Image Search https://addons.mozilla.org/en-US/firefox/addon/tineye-reverse-image-search/?src=collection&collection_id=9c098dfd-d353-0183-46c0-f124fc245e12

      推荐理由：这个是个很神奇的插件，通过这个插件，可以去tineye访问某个图片类似的图片，怎么说呢，你在网上看到个MM的图片，不过这个图片大小不合适，你通过tineye插件可以查询出跟这个图片有类似特征的其它图片。虽然不一定必备，但确实是一个很有用的插件。

7.网页保存：Mozilla Archive Format https://addons.mozilla.org/en-US/firefox/addon/mozilla-archive-format/?src=collection&collection_id=9c098dfd-d353-0183-46c0-f124fc245e12

与 scrapbook https://addons.mozilla.org/zh-CN/firefox/addon/scrapbook/

      推荐理由：有时候可能会对一个网页上的内容很感兴趣，想把整个网页保存下来，但是平常会保存成一个html外加一个文件夹，里面再包含一堆零散文件，不好管理。使用这个插件，就可以将整个网页保存成为一个压缩文件，打开时候再用firefox打开就是了，而且使用这种方式保存网页，页面中的内容，基本上都能正常保存下来。

      而要抓取网站上某个目录下的多个文件时，就需要用到scrapbook了，其实是有很多抓取工具的，不过既然firefox有这样的插件，用用也不错啊。

8.计划安排，定时提醒：reminderfox https://addons.mozilla.org/en-US/firefox/addon/reminderfox/

      推荐理由：既然是网虫，那么盯着网页看的时间就比较长，因此在firefox中安装一个计划安排管理并且定时提醒功能的插件是不是有必要呢？

9.快速拨号：Fast Dial https://addons.mozilla.org/en-US/firefox/addon/fast-dial-5721/

      推荐理由：chrome,opera都有这个功能了，而且身边不少人也确实比较习惯这个功能，因此我也在这里推荐下吧。

10.兼容性浏览：IEtab2 https://addons.mozilla.org/en-US/firefox/addon/ie-tab-2-ff-36/

      推荐理由：浏览的网页多了，发现还是有些网页在firefox下显示有布局上的错误，后面想想，IE7不是也有兼容性视图吗，o(︶︿︶)o 唉，万恶的IE，搞这么多不兼容的东西出来。有了IEtab2，就能在浏览一些显示上有兼容错误的网站时，在firefox中调用ie内核来处理，就能正常显示了。
