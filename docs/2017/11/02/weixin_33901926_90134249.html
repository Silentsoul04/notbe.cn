Solaris下ftp配置(初稿-待补充)

Nov 2, 2017

1.自带ftp版本
    Version wu-2.6.2
2.ftp启动与停止
    启动并启用ftp:
        svcadm enable network/ftp
    停止并禁用ftp:
        svcadm disable network/ftp
3.使某个系统用户无法使用ftp或者恢复使用ftp
    vi /etc/ftpd/ftpusers
    向其中添加要禁止使用ftp的用户名,这样对应用户就无法使用ftp.
    默认情况下,root是禁止使用ftp的,在这个文件中,将root注释掉(在root那行前面加#),这样root也可用使用ftp.
4.警告信息与欢迎信息
    警告信息在 /etc/ftpd/banner.msg 文件中.
    向其中填写警告信息,内容自己定了.通常这个文件是不存在的.

    欢迎信息在 /etc/ftpd/welcome.msg 文件中.
    向其中填写欢迎信息,内容自己定.通常这个文件不存在.
5.查看当前使用ftp的人数与信息
    ftpcount用来查看每种访问方式的用户有几个登录到系统.例如:
-----------------------------------------------------------------------
-bash-3.00$ ftpcount
Service class realusers            -   2 users (no maximum)
Service class guestusers           -   0 users (no maximum)
Service class anonusers            -   0 users (no maximum)
-----------------------------------------------------------------------
    ftpwho用来查看谁已经登录了ftp.例如:
-----------------------------------------------------------------------
-bash-3.00$ ftpwho
Service class realusers: 
qa       18829  0.0  0.1 3484 2328 ?        S 16:43:52  0:00 /usr/sbin/in.ftpd -a
oracle   18908  0.0  0.1 3484 2328 ?        S 16:48:44  0:00 /usr/sbin/in.ftpd -a
   -   2 users (no maximum)
Service class guestusers: 
   -   0 users (no maximum)
Service class anonusers: 
   -   0 users (no maximum)
-----------------------------------------------------------------------
6.ftp停止与启动
ftpshut 在指定时间停止ftp服务,停止时会打印/etc/ftpd/shutdown.msg
ftprestart 重新启动先前被shut的ftp
7.匿名ftp配置
ftpconfig
8.添加新的ftp虚拟主机
ftpaddhost
