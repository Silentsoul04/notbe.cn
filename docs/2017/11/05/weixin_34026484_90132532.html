Nov 5, 2017

　然后

第一次进入web UI要进行kafka cluster的配置,都是一些比较简单的操作。

 

配置集群

　　点击【Cluster】>【Add Cluster】打开如下添加集群的配置界面：

 

 

 

 

 

 

 

 

 

 

 

　　它会自动识别出，当前，你所用的kafka版本

 

 

 

 

 　　当然，这里，大家后期还可以微调kafka的版本。当然是与自己集群的版本一致！

 

 

 

 

　　当然，大家也可以给输入集群的名字（如Kafka-Cluster-1）和 Zookeeper 服务器地址（如localhost:2181）。

 

 

　　我这里给

　　Cluster Nmae赋值为kafka

　　Cluster Zookeeper Hosts为master:2181,slave1:2181,slave2:2181

 

 

 　　kafka version，就选择我当前用的版本（它会默认识别到）。  

 

 

注意：　如果没有在 Kafka 中配置过 JMX_PORT，千万不要选择第一个复选框。

　　　　   Enable JMX Polling

　　　　     如果选择了该复选框，Kafka-manager 可能会无法启动。

 

　　我这里，是没有配置JMX_PORT
