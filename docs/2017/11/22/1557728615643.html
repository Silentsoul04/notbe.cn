在虚拟机上创建共享磁盘

Nov 22, 2017

 步骤：1.创建共享磁盘

     进入到在windows的cmd模式下进行vmware的安装目录(我的是d:\program files\vmware)

     CMD> vmware-vdiskmanager.exe -c -s 200Mb -a ide -t 0 disk.vmdk
    指创建了一个新的(-c参数)，200Mb大小(-s 200Mb)，IDE接口(-a ide) 名为'disk.vmdk'的磁盘（老版本共享磁盘文件的扩展名为.pln）。

这里我们创建一个共享磁盘

C:\Documents and Settings\Administrator>d:

D:\>cd "program files"

D:\Program Files>cd "vmware"

D:\Program Files\vmware>
D:\Program Files\vmware>vmware-vdiskmanager.exe -c -s 3Gb -a lsilogic -t 2  E:\disk\sharedisk.vmdk

Creating disk 'sharedisk.vmdk'

  Create: 100% done.

Virtual disk creation successful.

创建完毕后，E:\disk\目录下有两个个新文件

sharedisk.vmdk
sharedisk-flat.vmdk

2.配置虚拟机

每个虚拟的vmware机器都有一个vmx文件，编辑它，然后加上一下一段（两台虚拟机中都加）：
disk.locking="false" 
diskLib.dataCacheMaxSize="0"
diskLib.dataCacheMaxReadAheadSize="0"
diskLib.DataCacheMinReadAheadSize="0"
diskLib.dataCachePageSize="4096"
diskLib.maxUnsyncedWrites="0"
scsi1.present="TRUE" 
scsi1.virtualDev="lsilogic"
scsi1.sharedBus="virtual"
scsi1:1.present="TRUE" 
scsi1:1.mode="independent-persistent"
scsi1:1.filename="E:\disk\sharedisk.vmdk"
scsi1:1.deviceType="plainDisk"
