数据库 版本号是 661，打不开。此server支持 655 和更早的版本号。不支持降级路径...

Nov 22, 2017


“数据库 的版本号为 661，无法打开。此server支持 655 版及更低版本号。

不支持降级路径”

出现这种问题，通常是由于数据库版本号不同造成的。

我们能够用以下的语句查询数据库的版本号

use master

select @@VERSION

（1）661是sql2008 R2的版本

    Microsoft SQL Server 2008 R2 (RTM) - 10.50.1600.1 (Intel X86)   Apr  2 2010 15:53:02  Copyright (c) Microsoft

（2）655版本号也就是sql2008 sp1版本

   Microsoft SQL Server 2008 (SP1) - 10.0.2531.0 (Intel X86)   Mar 29 2009 10:27:29  Copyright (c) 1988-2008



假设后面是 1600 ，则表明仅仅安装了 sql2008r2 ，还要装一个 sql2008sp1 。这样才干保证后面是 2500.
