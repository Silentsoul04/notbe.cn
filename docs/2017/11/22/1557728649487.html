Windows Server 2008 自动部署操作系统----使用发现启动映像远程安装

Nov 22, 2017

12.4 实战：使用发现启动映像远程安装

发现映像是包含 Windows PE 2.0 并且已修改为强制 Setup.exe 在 Windows 部署服务模式下启动以查找 Windows 部署服务服务器的启动映像。通常，您将需要为未启用 PXE 的计算机创建发现映像。将发现映像保存到文件中，将其转换为 ISO 格式，然后刻录到 CD 或 DVD 上。

未启用 PXE 的网络适配器可以通过发现映像来访问 Windows 部署服务 PXE 服务器。

12.4.1在WDSServer上制作发现启动映像ISO

任务：

u 创建发现启动映像

u 安装Windows自动安装工具包（Windows AIK）

u 制作发现启动映像的ISO文件

步骤：

1. 如图12-115所示，在WDSServer上，打开Windows 部署服务，右击“Windows Microsoft LonghornSetup （x86）”，点击“创建发现启动映像”。

2. 如图12-116所示，在出现的发现映像元数据对话框，输入映像名称“Microsoft Windows Longhorn Discover Image(x86)”，映像说明“Microsoft Windows Longhorn Discover Image(x86)”

3. 位置和文件名，点击“浏览”。

 

图12-115 创建发现启动映像 图12-116 输入映像名和说明

4. 如图12-117所示，输入文件名WDSDiscover.wim，点击“打开”。

5. 如图12-118所示，要进行响应的Windows部署服务器，点击“浏览”，输入WDSServer.ess.com，点击“确定”。

 

图12-117 选择文件 图12-118 浏览到WDS服务器

6. 如图12-119所示，在发现映像元数据对话框，设置完毕后点击“下一步”。

7. 如图12-120所示，在出现的任务进度对话框，点击“完成”。

图12-119 发现映像数据 图12-120 完成

8. 如图12-121所示，右击“启动映像”，点击“添加启动映像”。

9. 如图12-122所示，在出现的映像文件对话框，点击“浏览”，选中WDSDiscover.wim映像，点击“打开”，点击“下一步”。

 

图12-121 添加启动映像 图12-122 指定映像位置

10. 如图12-123所示，在出现的映像元数据，输入映像名称“Microsoft Windows Longhorn Discover Image （x86）”，映像说明“Microsoft Windows Longhorn Discover Image （x86）”，点击“下一步”。

11. 如图12-124所示，在出现的摘要对话框，输入“下一步”。

 

图12-123 图12-124

12. 如图12-125所示，在出现的任务进度对话框，点击“完成”。

13. 如图12-126所示，点击“VM”à“Removable Devices”à“CD-ROM”à“Edit”。

 

图12-125 完成 图12-126 编辑CD-ROM

15. 如图12-128所示，插入光盘后会自动出现欢迎使用Windows自动安装工具包对话框，点击“Windows AIKido安装程序包”。如果没有出现该对话框，点击根目录下STARTCD.EXE。

 

图12-127 浏览到WindowsAIK.img 图12-128运行安装程序

16. 如图12-129所示，在出现的欢迎使用“Windows 自动安装工具包”安装向导，点击“下一步”。

17. 如图12-130所示，在出现的许可协议对话框，选择“我同意”，点击“下一步”。

 

图12-129 向导 图12-130 许可协议

18. 如图12-131所示。在出现的选择安装文件夹对话框，保持默认设置，点击“下一步”。

19. 如图12-132所示，在出现的确认安装对话框，点击“下一步”。

图12-131 选择安装文件夹 图12-132 确认安装

20. 如图12-133所示，在出现的安装完成对话框，点击“关闭”。

21. 如图12-134所示，点击“开始”à“程序”à“Microsoft Windows AIK”à“Windows PE工具命令提示”。

 

图12-133 安装完成 图12-134 Windows PE工具

22. 如图12-135所示，在命令提示符下输入copype.com x86 c:\winPE。

图12-135 制作发现启动映像

23. 运行c:\WinPE>copy /y c:\WDSDiscover.wim C:\WinPE\ISO\sources\boot.wim

25. 运行c:\Program Files\Windows AIK\Tools\x86>oscdimg.exe -n -bC:\WinPE\ISO\boot\etfsboot.com C:\WinPE\ISO c:\WDSDiscover.iso 制作ISO映像文件。

12.4.2使用发现启动映像ISO启动系统

为不支持PXE的计算机使用发现启动映像ISO启动系统，远程安装。

任务：

u 设置FileServer使用光驱启动操作系统。

步骤：

26. 如图12-137所示，将WDSServer计算机上将WDSDiscover.iso拖拽到物理机。

图12-137 拷贝发现启动映像

27. 如图12-138所示，在FileServer上配置CD-ROM使用WDSdiscover.iso文件，点击“OK”。

28. 如图12-139所示，设置FileServer启动顺序，将CD-ROM Drive设置到第一位。

 

图12-138 选择ISO文件 图12-139 使用光驱引导

29. 如图12-140所示，按F10，保存设置。

30. 如图12-141所示，启动时从光盘引导。

 

图12-140 确认 图12-141 加载安装映像

31. 如图12-142所示，在出现的Windows部署服务对话框，点击“下一步”。

32. 如图12-143所示，在出现的连接到WDSServer.ess.com对话框，输入域用户帐户和密码，点击“确定”。

 

图12-142 区域设置 图12-143 输入用户名和密码

33. 如图12-144所示，在出现的选择要安装的操作系统对话框，选择“win2k8 img”，点击“下一步”。

34. 如图12-145所示，在出现的您想将Windows安装在何处对话框，点击“下一步”。以后操作和以上实战远程安装步骤一样，这里就不在重述。
