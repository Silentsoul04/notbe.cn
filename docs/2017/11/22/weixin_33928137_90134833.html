如何转换鄚咖驼世界地图投影坐标

Nov 22, 2017

怎样将天地图转换为默卡托投影坐标
曾经我们讨论过怎样将谷歌卫星地图用ArcGIS转换为WGS84坐标系，今天我们来讲一下怎样将天地图的卫星地图转换为和谷歌卫星地图一样的墨卡托投影坐标系。
首先，我们用水经注天地图卫星地图下载器下载一个全球的TIF格式的卫星地图，然后再用ArcGIS进行投影转换。这里以全球的卫星地图为例，是由于仅仅有下载全球的卫星地图进行投影转换后，才干用肉眼看出非常明显的变化，也更能说明投影转换的效果，假设你是将天地图卫星地图的某一个城市或局部数据转换为墨卡托投影的话。除了下载的范围和级别不同外，操作步骤是全然同样的，你能够依照此文档的说明进行操作。
怎样下载天地图全球卫星地图数据？
下载安装水经注天地图卫星地图下载器。安装完毕启动软件后，软件界面例如以下图所看到的。



在工具栏中点击“框选下载区域”。然后在视图中框选须要下载的区域，例如以下图所看到的。

在选择区域中。双击鼠标左键。显示“新建任务”对话框。例如以下图所看到的。

因为这里我们是希望下载全球卫星地图数据以说明怎样将天地图卫星地图转换为墨卡托投影坐标系的方法，因此须要将最小坐标改动为“-180,-90”，最大坐标改动为“180,90”，但在实际应用中这里不须要作改动，除非你也要下载全球的卫星地图。

另外，这里我们将任务名称命名为“全球卫星地图数据下载”，在级别列表中我们选择第4级就可以。例如以下图所看到的。


在对话框中点击“确定”button，自己主动新建任务后并開始下载卫星地图数据，完毕下载后会询问是否须要导出，例如以下图所看到的。

点击“是”button显示“选择任务”对话框，在该对话框中，我们仅仅须要选择“全球卫星地图数据下载”任务就可以。“全球卫星地图数据下载_标签”任务仅仅有道路和名称等信息的透明图层，这里不用单独导出，因此能够不用选择，例如以下图所看到的。

点击“确定”button。显示“导出图片数据”对话框。例如以下图所看到的。

导出类型选择“拼接:默认”。图片保存类型选择“GeoTIF(*.tif)”，然后点击“输出”安钮，当询问是否须要叠加标签时，这里选择“否”。在实际应用中。假设你须要在卫星地图上叠加显示地名和道路就选择“是”。不须要叠加显示地名和道路就选择“否”。导出完毕后的下载结果在“D:\SGDownload\全球卫星地图数据下载_拼接[默认]\L04”文件夹下。例如以下图所看到的。



怎样将天地图进行墨卡托投影转换？
启动ArcMap软件，然后在工具栏上点击“AcToolbox”button，例如以下图所看到的。

在显示的“ArcToolbox”对话框中选择“Data Management Toos -> Projections and Transformations -> Raster -> Project Raster”，例如以下图所看到的。



在ArcToolbox中双击“Project Raster”显示“Project Raster”对话框。在该对话框中指定源数据路径为“D:\SGDownload\全球卫星地图数据下载_拼接[默认]\L04\全球卫星地图数据下载.tif”，指定输出路径和文件名称为“D:\SGDownload\全球卫星地图数据下载_拼接[默认]\L04\全球卫星地图数据下载_墨卡托.tif”，例如以下图所看到的。

点击“Output Coordinate System”相应的属性button，例如以下图所看到的。

在显示的“Spatial Reference Properties”对话框中。选择“Projected Coordinate System -> World -> WGS 1984 Web Mercator (auxiliary sphere)”。例如以下图所看到的。



单击“确定”button后，在“Ourtput Coordinate System”区域显示坐标系统为“WGS_1984_Web_Mercator_Auxiliary_Sphere”。例如以下图所看到的。

点击“OK”button，開始进行转换，转换完之后会在ArcMap中显示转换结果。且坐标已经显示为墨卡托投影坐标，例如以下图所看到的。



导出的结果目录，例如以下图所看到的。

我们分别打开转换前的“全球卫星地图数据下载”和转换后的“全球卫星地图数据下载_墨卡托”的TIF文件。例如以下图所看到的。





以上是世界地图的整个过程的卫星地图转换墨卡托坐标系。
