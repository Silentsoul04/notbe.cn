向网页设计师推荐15个很棒的网站

Nov 22, 2017

网络上各种各样的社区网站数不胜数，但专注于创意设计的很少。今天这篇文章收集了15个非常好的专注于设计的网站推荐给设计师们，不管是学习先进的设计理念，还是寻找免费资源与工具，这些网站都是很不错的去处，记得推荐和分享一下 ：）

DeviantArt

DeviantArt 是一个为艺术家展示各自作品、并交流讨论而设计的社交网站。它以分类系统为基础，辅以个人日志、新闻、社区论坛、每日作品、人气榜、最爱、收藏和可嵌套的评论与回复系统构建起具有自身特色的社交网络。至2010年8月DA已有1450万用户注册并收到超过1亿个各种作品的投稿。

Dribbble

Dribbble 是一个专门推发设计作品的网站，模式类似 Twitter，不同的是，网站分享的主要内容是设计师上传的 400x300 像素的设计图片。

Forrst

Forrst 是一个面向设计师和开发者的社区，这里是获得对你的作品的评价和建议的好地方。

LoveDsgn

LoveDsgn 也是一个面向设计师的社区，需要邀请注册，是获取作品犯反馈的好地方。

Drawar

Drawar 是一个论坛，可以讨论设计，开发，商业以及其它和设计师相关的一些话题。

Behance Network

Behance Network 是另外一个比较有名的关注设计的网站，有很多的作品示例，还有一个很大的社区。

DesignersCouch

DesignersCouch 也是一个设计社区，你可以在上面和其它的设计师交流设计理念，和他们进行合作。

DesignRelated

DesignRelated 也需要邀请注册，在这里你可以寻找到很多的设计灵感，可以创建自己的作品集。

Typophile

Typophile 是专注于字体的网站，可以到论坛中交流，阅读最新的文章，发现很多有用的资源。

SitePoint

SitePoint 不仅有一个很大的讨论区，还发布设计和开发方面的非常有用的文章，另外也发布和推介商业作品。

ColourLovers

ColourLovers 是设计师寻找配色灵感的最佳网站，你可以搜索其它设计师创建的调色板，也可以创建和分享自己的配色方案。

DesignBump

While DesignBump 是一个信息聚合网站，从那里可以得到了解到最新的设计和开发方面的文章和资讯。

The Web Blend

TheWebBlend 和 DesignBump 很像，可以每天关注到最新的设计和Web开发方面的文章和新闻。

CSSGlobe

CSSGlobe 是一个面向设计师和开发者，关注 CSS 和开发技术的社区网站， CSSGlobe 提供了丰富的开发资源链接。

Pixel Groovy
