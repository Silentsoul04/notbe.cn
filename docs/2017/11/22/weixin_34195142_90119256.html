数据库架构组件

Nov 22, 2017

1、 Oracle的公司的应用程序的广度和重要性。

2、 Oracle概述中。强调根的作用。也就是Oracle整体架构，掌握了。那么40%的Oracle知识就可以掌握。

3、 静态文件在数据库不启动是就是操作系统的文件，启动后则为数据库进程。

4、 动态文件，主要是SGA共享区和一些核心进程。

5、 User process主要就是client，而Server process主要是serverServer进程。Server进程的PGA不是共享内存来的。

6、 数据库文件（静态文件），主要是配置參数文件、超户文件、数据文件、控制文件、日志文件等。

7、 数据库实例（动态文件），主要是用来訪问数据库，包含内存结构和后台进程。

8、 连接数据库实例的方式有三种。包含1）User和Server同一台机器的直连方式；2）User和Server通过网络连接；3）浏览器和User通过网络连接，User和Server也通过网络连接。而User处于中间的应用server层。

9、 SGA中的Shared Pool的作用。主要是1）Library cache（库缓冲区）主要是用来对SQL语句等进行缓存；2）Data dictionary cache（数据字典缓冲区）主要用来对表等进行缓存。以上两种方法都是为了提高数据库操作的效率。

10、SGA中的Data buffer cache的作用，主要是用来缓存近期频繁使用的用户blocks。

11、 Server process的PGA（Program Global Area），是非共享的，仅仅能被Server process写，内存中排序区、Session信息、游标状态。
