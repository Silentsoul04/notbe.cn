Microsoft® Visual SourceSafe® 6.0 标准版 简介下载

Nov 27, 2017

今天在公司，看到公司用这种管理工具，现整理如下
 

 

Microsoft Visual SourceSafe 6.0 版本控制系统是微软最新获奖的版本控制系统，可管理软件和 Web 站点的开发。它可以同 Visual Basic、Visual C++、Visual J++、Visual InterDev、Visual FoxPro 开发环境以及 Microsoft Office 应用程序集成在一起，提供了方便易用、面向项目的版本控制功能。Visual SourceSafe 可以处理由各种开发语言、创作工具或应用程序所创建的任何文件类型。在提倡文件再使用的今天，用户可以同时在文件和项目级进行工作。Visual SourceSafe 面向项目的特性能更有效地管理工作组应用程序开发或 Web 站点开发工作中的日常任务。

功能

跨组共享文件

使用检出文件锁定、可视化合并和差别报告，Visual SourceSafe 能识别文件的变化和防止意外的代码覆盖。

支持所有文档格式

编程人员或非编程人员均可用 Visual SourceSafe 来管理文档的修改过程。Visual SourceSafe 提供了保存 Web 内容、源代码和企业文档的安全方法。

管理更容易

Visual SourceSafe 6.0 新的图形化存档向导使您可以方便而安全地保存不常用的文件。

Web 站点管理

Visual SourceSafe 通过站点映射、超链接和直接的 Web 配置，可方便地管理网络内容和 HTML，并检查超链结和直接在 Web 上部署。

节约时间

使用 RAS 检出文件的速度提高了十倍，而通过局域网则可提高了两倍。

与微软工具及第三方产品紧密结合

Visual SourceSafe 可以直接与 Visual Basic、Visual C++、Visual FoxPro、Visual InterDev、Visual J++、Access、FrontPage、Microsoft Office 开发版等微软开发工具及一些第三方产品直接集成在一起。

项目提示标签

项目提示标签可用于指示项目状态中的变化。

兼容 2000 年数据格式

所有报告和对话框都支持 2000 年数据格式。

增强的 Web 站点管理

Visual SourceSafe 6.0 中包括了站点映射、内外链接的检查和 Web 站点的部署。

熟悉的界面

Visual SourceSafe 6.0 与 Windows® Explorer 的用户界面在外观和功能上都很相似，可方便用户学习和使用。
