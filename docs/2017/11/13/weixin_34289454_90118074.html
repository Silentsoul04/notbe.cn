Ubuntu14.04 安装Android SDK（SDK tools only）

Nov 13, 2017

1、安装JDK（我安装的是Oracle的，而不是openjdk）

jdk目录：usr/lib/jvm/java-7-oracle/bin/java

2、下载Android-SDK，在下边的网页选择对应的版本，我下载的android-sdk_r24.2-linux.tgz，

http://developer.android.com/sdk/index.html#Other

3、cd到步骤2下载的目录，输入下面的指令，注意指令中的文件名和目录个根据自己的情况设置

:~$ tar -zxvf android-sdk_r24.2-linux.tgz

:~echo′exportANDROIDHOME="′echo′exportANDROIDHOME="′HOME'/下载/android-sdk-linux"' >> ~/.bashrc

:~echo′exportPATH="echo′exportPATH="PATH:ANDROIDHOME/tools:ANDROIDHOME/tools:ANDROID_HOME/platform-tools"' >> ~/.bashrc

:~$ echo 'export JAVA_CMD="/usr/lib/jvm/java-7-oracle/bin/java"' >> ~/.bashrc

4、关闭终端，再打开一个终端；

5、输入 ‘android’，启动Android SDK Manager

:~$ android

6、根据需要，选择Android SDK Platform-tools、Samples for SDK等等（大概2-3G）；

     注意：一定要选择 Android Support Repository，否则gradle会报错

 

至此，就安装完了~~~
