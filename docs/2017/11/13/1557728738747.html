在Windows Server 2008上创建 简单卷

Nov 13, 2017

简单卷

一个简单卷就是驻留在一个动态磁盘上的单一的卷。简单卷是物理磁盘的一部分，但它工作时就好像是物理上的一个独立单元。我们可以从动态磁盘的未分配空间来创建简单卷，但当只有一个动态磁盘时，简单卷是我们可以创建的唯一卷。简单卷与分区相似，与分区不同，简单卷既没有大小限制，也没有在一块磁盘上可创建卷的数目的限制。

在简单卷中可以使用的文件系统包括：NTFS、FAT、FAT32。我们可以通过将卷扩展到相同或不同磁盘上的未分配空间上可以增加现有简单卷的大小。要扩展简单卷，则该卷必须尚未格式化，或者通过 Windows 2003 或 Windows Server 2008 系列操作系统中使用的 NTFS 版本对卷进行格式化。在 Windows Server 2008 系列操作系统上，可以扩展简单卷，除非该卷是系统分区、启动分区或以前是基本磁盘上的分区的简单卷，这些基本磁盘已经通过 Windows 2008 转换为动态磁盘。扩展相同磁盘上的简单卷后，该卷仍为简单卷，且仍然可以镜像它。

只有 Windows 2000、Windows XP Professional 、Windows Server 2003和Windows Server 2008 家族操作系统才能访问简单卷。如果计算机还运行 MS-DOS、Windows 95、Windows 98、Windows Millennium Edition、Windows NT 4.0 或 Windows XP Home Edition，则应该在基本磁盘上创建基本卷，因为这些操作系统不能访问动态卷。

我们可以使用简单卷来实现所有的数据存储，直到在磁盘上我们需要更多的空间。为了获得更多的磁盘空间，我们可以创建一个扩展的、跨区的、带区的卷。

 

 

1.1.1 简单卷管理

简单卷可以在不删除数据的情况下，增加空间或缩小空间。

示例：创建简单卷

如图，右击动态磁盘1，点击“创建简单卷”。

输入大小2000M，点击“下一步”。

指定盘符，点击“下一步”。

选中“执行快速格式化”，文件系统选中“NTFS”，点击“下一步”，完成简单卷的创建。

示例：扩展简单卷

如下图，在创建D卷后又创建了F卷，后来发现动态磁盘上的简单卷D空间不够用了，您可以扩展其空间，简单卷的空间可以不连续。

右击D卷，点击“扩展卷”。

输入扩展的大小，点击“下一步”。

可以看到D卷的空间可以不连续，中间有F卷。

示例：收缩卷

如果卷的空间太大，可以收缩。

右击D卷，点击“压缩卷”，输入减少空间的大小，点击“确定”。

可以看到扩展出来的D卷右侧部分已经变为1000MB大小。

示例：添加镜像

简单卷可以添加镜像，和另外一块硬盘的空间形成镜像关系，这样就容错了。

右击D卷，点击“添加镜像”，在出现的对话框，选中磁盘2，点击“添加镜像”。

可以看到磁盘2上拿出和D卷一样大小的空间作为镜像。
