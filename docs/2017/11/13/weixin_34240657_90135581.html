　　解决办法1

[hadoop@master conf]$ cd $ZOOKEEPER_HOME
[hadoop@master zookeeper]$ bin/zkServer.sh stop
JMX enabled by default
Using config: /home/hadoop/app/zookeeper/bin/../conf/zoo.cfg
Stopping zookeeper ... STOPPED
[hadoop@master zookeeper]$ bin/zkServer.sh start
JMX enabled by default
Using config: /home/hadoop/app/zookeeper/bin/../conf/zoo.cfg
Starting zookeeper ... STARTED
[hadoop@master zookeeper]$

　　slave1和slave2都去配置。

 

　　因为，也许是你中间，更改了，导致没有及时切换反应过来，zookeeper那边还没读取到。

 

 

 

 

 

 

　　解决办法2

　　查看storm.yaml，是否缺少符号，有手误。

 

 　　一般以上定位，都可以解决了。
