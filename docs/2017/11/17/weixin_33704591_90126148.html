2.设置名字：


git config --global user.name "你的名字"

3.设置邮箱：
git config --global user.email "你的邮箱"

————————————————————————————————————————
4.输出更显目的设置
git config --global color.ui auto


5.如果忘记自己的配置，就用这个命令来查看一下


cat ~/.gitconfig

————————————————————————————————————————
[user]
name = 你的名字
email = 你的邮箱
[color]
ui = auto
