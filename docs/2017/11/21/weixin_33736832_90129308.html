Windows server 2008 64位 在双核CPU（32位）下的安装过程

Nov 21, 2017


1、    把光盘根目录下的 boot文件夹、 bootmgr文件、sources/boot.wim、32位光盘下的bootsect.exe 拷贝到C盘根目录下。

2、点击运行->输入cmd-> 切换到C盘根目录下（cd ../..）->输入bootsect  /nt60  C:

重新启动系统->进入windows server 2008安装->选择“修复计算机”->选择“命令提示符”->切换到C盘根目录下->删除boot文件夹、 bootmgr、sources/boot.wim、bootsect.exe（输入：del boot; del  sources; del bootmgr;; del bootsect.exe）->切换到WS2008安装目录下sources文件夹下的Setup.exe. 系统重新启动-〉进入WS2008安装-〉选择你要安装到的目录就可以了，安装过程需要自动重启几次

3、如果在Windows server 2008 上安装SQL server 2008，需要打SP2补丁

4、安装IIS7.0的过程中，不要漏掉安装IIS必须的组件，否则asp.net 程序不能运行

5、Web应用程序 要安装为应用程，而不是虚拟路径
