   6.代理(proxy)
   

   代理，非常重要，尤其在局域网横行的年头，没有代理，你在公司上不了QQ，没有代理，google不了网页快照，代理之威，可比七星碧玉刀，无刀，在局域网和开发当中，一切白搭：
   

   

    java 代码
   
 
   
 
    

      
    
 
    
    
     
  1. HttpClient httpclient=new HttpClient();

  2. httpClient.getHostConfiguration().setProxy("192.168.0.1", 9527);
