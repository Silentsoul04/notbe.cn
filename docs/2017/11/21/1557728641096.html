1、Win7下建立Wifi热点


其实就两条命令就ok了。这里同样给出建立方法，以管理员身份运行如下命令：


-- 建立热点，热点名=hotlei；密码=1234567890；

netsh wlan set hostednetwork mode=allow ssid=hotlei key=1234567890

netsh wlan start hostednetwork


-- 关闭热点（不用的时候关闭）

netsh wlan start hostednetwork
