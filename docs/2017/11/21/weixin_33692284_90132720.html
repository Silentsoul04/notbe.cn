数值的整数次方[快速幂运算]

Nov 21, 2017

/**
  * 1.全面考察指数的正负、底数是否为零等情况。
  * 2.写出指数的二进制表达，例如13表达为二进制1101。
  * 3.举例:10^1101 = 10^0001*10^0100*10^1000。
  * 4.通过&1和>>1来逐位读取1101，为1时将该位代表的乘数累乘到最终结果。
  */

（1）指数的正负符

（2）abs函数

C++

 1 class Solution {
 2 public:
 3     double Power(double b, int e) {
 4         int p = abs(e);
 5         double r = 1.0;
 6         while (p) {
 7             if (p & 1) r *= b;
 8             b *= b;
 9             p >>= 1;
10         }
11         return e > 0 ? r : 1 / r;
12     }
13 };
