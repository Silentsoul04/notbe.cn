Ubuntu10.10环境变量设置

Nov 23, 2017

a.下载jdk安装文件，这里下载bin格式，复制安装文件至/home/java下
    b.使用"chmod 777"或"chomd +x"向文件添加可执行权限
       sudo chomd +x jdk-6u23-linux-i586.bin
    c.运行该文件
      sudo ./jdk-6u23-linux-i586.bin
      运行后，程序会自动解压缩，并生成相应目录。到此，jdk安装已经完成.
     d. 环境变量的添加：
     打开 /etc/profile ,在 umask 022 ”前添加如下内容：
     export JAVA_HOME=/home/jack/Java/jdk1.6.0_23
     export PATH=$JAVA_HOME/bin:$JAVA_HOME/jre/bin:$PATH
     export CLASSPATH=$CLASSPATH:.:$JAVA_HOME/lib:$JAVA_HOME/jre/lib 
     保存文件并关闭当前窗口,使用source /etc/profile使刚才配置的信息生效
     在终端输入java -version进行测试，将看到如下内容：
     java version "1.6.0_23"
