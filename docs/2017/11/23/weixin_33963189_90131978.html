分享10款非常有用的 Ajax 插件

Nov 23, 2017

这篇文章与大家分享的是10款非常有用的 Ajax 插件，有用于图片的，用于分页的，还有用于导航的。这些作者的想法特别新颖，希望你能从中找到自己需要的插件。

1. AJAX-ZOOM

非常强大的一款插件，可用鼠标滚轮进行缩放，360° 旋转，全屏浏览等很多功能，在线演示 

2. VerticalSlider

用于显示长列表的很好的方式，以Ajax方式加载，在线演示

3. More plugin

一款实现更多功能的插件，可用于显示更多评论，也可用于显示后面多少篇博文等，在线演示 

4. HTML5 drag & drop image file uploader

这是一款实现HTML5拖放上传的插件，在线演示

5. jpaging

一款基于jQuery实现的AJAX分页插件，在线演示

6. jsLink

一个用于坏链检查的jQuery插件，在线演示

7. AJAX PayPal Cart

这款插件可以帮助你在网站中加入购物车功能，在线演示

8.jNavigate

这款插件用于动态加载网站内容，在线演示

9.Ajax Login Form

一个Ajax登陆表单插件，在线演示

10.Transform

一个实现跨浏览器的XML转换插件，在线演示
