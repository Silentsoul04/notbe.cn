IIS7下配置PHP5

Nov 23, 2017

通常使用IIS开发.net程序，IIS自带都支持，今天装个PHP，网上查阅了一下，记录下来。

首先开始IIS7(VISTA)，略过....

PHP配置：

1. 找到 ;extension=php_mbstring.dll, 前面的";"去掉,
    找到 ;extension=php_gd2.dll, 前面的";"去掉,
    找到 ;extension=php_mysql.dll, 前面的";"去掉.

2. 将     extension_dir = "./"
    改为   extension_dir = "d:/php/ext/"
注意后面一定要/结尾

3. 如果您的SESSION建立失败可以在C:"php目录下建立session

将    ;session.save_path = "/tmp"
改为   session.save_path = "d:/php/session"

保存为php.ini并复制到c:windows下

到刚才解压出来的PHP目录下的ext文件夹中，把其中的php_mysql.dll复制到System32文件夹下

打开IIS管理器，使用选中处理程序映射－－添加脚本映射，在弹出的对话框中，请求路径输入*.PHP，可执行文件为 php5isapi.dll(位于PHP目录下)

建立文档，加入<? phpInfo() ?>，查看一下，就可以了！！！
