VS2008开发Silverlight2程序：环境安装

Nov 23, 2017

微软在MIX08大会，发布了Silverlight 2 Beta1、Expression Blend 2.5 March 2008 Preview等一系列的产品。
(1). 如果你以前安装过silverlight相关插件，请删除silverlight的插件，silverlight for vs2008的模板，express blend等和silverlight相关的组件，只需要留下vs2008.
(2).下载最新的silverlight工具集
a. Microsoft Silverlight Tools Beta 1 for Visual Studio 2008

下载地址： http://www.microsoft.com/downloads/details.aspx?FamilyID=e0bae58e-9c0b-4090-a1db-f134d9f095fd&DisplayLang=en
这个安装包包括了如下组件：
Silverlight 2 Beta 1 
Silverlight 2 SDK Beta 1 
KB949325 for Visual Studio 2008 
Silverlight Tools Beta 1 for Visual Studio 2008
b. Microsoft Expression Blend 2.5 March 2008 Preview
下载地址： http://www.microsoft.com/downloads/details.aspx?FamilyID=32a3e916-e681-4955-bc9f-cfba49273c7c&DisplayLang=en

(3).安装上面两个安装包，注意：安装Microsoft Silverlight Tools Beta 1 for Visual Studio 2008提示要插入vs2008安装盘。
(4). 安装后我们就可以进行开发了。
