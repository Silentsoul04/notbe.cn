PowerDesigner 常用设置

Nov 23, 2017

1.使用 JDBC 方式连接 Oracle 逆向生成数据库 PDM

   使用 ODBC 方式连接 Oracle 数据库可以借鉴这位兄弟的博客：http://www.cnblogs.com/clivehua/articles/4593211.html

   个人感觉用 JDBC 的方式连接 Oracle 是比较简单。

    [Database]->[ Configure Connections]

   注意的几点：

   1. 如果出现无法初始化 JVM 时，说明你的 JDK 版本和 PowerDesigner 的 位数不一致，请调整。

   2. JDBC driver jar : 选择你项目中连接数据库的驱动 Jar 即可，并且将该 jar 的位置添加到系统变量 classPath 中。

   3. Connection profile: 连接名称，你随意

   4. Description: 对连接的描述，可以不填

   5. UserName/JDBC driver class/connecton URl 正确填写好点击测试连接。
