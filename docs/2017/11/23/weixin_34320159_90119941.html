XP硬盘安装win 7

Nov 23, 2017

  1. 用dt加载win7镜像，将文件拷贝到除系统分区外的任何分区，比如拷贝到 F:/win7/ 下。
  2. 把 F:/win7 目录下的bootmgr文件和boot目录拷贝到C盘（系统盘）的根目录下，然后在C盘根目录下建一个新的文件夹sources, 然后将 F:/win7/sources/ 目录下的 boot.wim拷贝到 C:/sources 目录下。
  3. 开始-运行-cmd- c:\boot\bootsect.exe /nt60 c:
  4. 重新启动系统，系统将进入安装win7界面，选择语言及键盘，下一步
  5. 当看到install now的时候，请点击左下角的 repair my computer
  6. 安装程序查找要修复的系统，请稍候，找不到，下一步
  7. 进入系统恢复选项，选择最后一项 command prompt, 进入命令行窗口 8.进入F:/win7/sources/, 运行setup.exe, 至此往后就可以正常安装了
