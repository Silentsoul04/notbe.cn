Autodesk Infrastructure Map Server 2012 – iViewer for iPad, iPhone, Andriod

Feb 6, 2017


 在我前面的文章Autodesk Infrastructure Map Server 2012(MapGuide 2012) 最新特性介绍中已经提到了，AIMS2012提供了适用于移动设备的浏览器--iViewer, 这样你可以在iPad、iPhone或者android手机上来浏览MapGuide地图，那你一定对这个很期待。如果你还没安装AIMS的话并且你手头有一部iPad、iPhone或者Android手机的话，你可以先用下的测试地址看看Autodesk MapGuide演示站点上的iViewer。不用下载任何软件到你的手机，直接用浏览器打开下面的地址就可以了。

 

http://sandbox.mapguide.com/iviewer/

 

 

 

 

其实呢，我也没有iPad，iPhone，为了写这个blog，我还特意到苹果店看了一下，美国这边iPhone4 299刀就可以买了，再想想中国，唉，一声叹息。我有个数据，美国San Francisco及Bey Area软件工程的平均年薪约7万$，而日常消费品却便宜的多，不少东西比中国还便宜，比如好多made in China的鞋子、衣服。macy‘s这样的大商场里一般的衣服也就20~60$，上百的就是很好的了。对中国人来说，年薪7万应该也不算太少了吧，但是你到中国的商场里看看，有几十块钱的东西吗？ 收入是中国人的5~6赔，物价还这么便宜，真是让中国人情何以堪啊。 

 

 

好了，回到正题，我这里用下面的网站模拟iPad  http://ipadpeek.com/， 其实就用Firefox也可以。  输入上面的地址 http://sandbox.mapguide.com/iviewer/  应该可以看到3个示例，如图

 

 点击任何一个即可打开iViewer，下面是第一个：

好了，其实很简单，大家自己玩儿一下吧。如果你安装了AIMS2012，并且安装了iViewer模块（这个可能要单独安装），那么你也可以非常容易的把你本地的地图发布到iViewer中。试试吧。
