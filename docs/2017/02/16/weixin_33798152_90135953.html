struts2配置i18n设置,点击链接生成国际化文本

Feb 16, 2017

配置的是工程级别的i18n设置如下：

1，配置两个i18n所用到的国际化的字符串属性文件，位于src目录下，例如为如下：

工程名_zh_CN.properties ,工程名_en_US.properties，在两个配置文件中写上对应的i18n国际化的字符串；

2，在struts.xml配置文件中声明一个常量，表明使用i18n(其实内部有一个拦截器的理解在内）

<constant name=”struts.custom.i18n.resources” value=”工程名”/>

3,前台访问中的字符串展示为<s:property value=”getText(‘字符串’)”/>

在超链接中写上如下链接action

地址：

actioname?request_locale=zh_CN

actionname?request_locale=en_US

注意其中的actionname里面需要写入任何的代码，只需要重写execute方法即可，然后让他正确返回的页面还是当前的前台的页面即可；
