Git 基本使用

Feb 20, 2017


# git 使用

# 安装后在自己电脑上首先要确认身份
$ git config --global user.name "test"
$ git config --global user.email "1764496637@qq.com"

# 创建版本库
$ git init  # 目录下会多了一个.git的目录，这个目录是Git来跟踪管理版本的

# 提交文件
# 先把文件添加到缓存区
$ git add test.txt
# 提交文件
$ git commit -m "test.txt"

# 查看是否有未提交的文件
$ git status

# 对比修改内容
$ git diff test.txt

# 查看历史记录
$ git log

# 版本回退的两种方式
# 1
$ git reset -hard HEAD^ (回退到上个版本，^^就是上上个版本，以此类推)
# 2
$ git resrt -hard HEAD~20 (回退到20版本)

# 撤销修改的两个方法
# 1：如果知道要删掉哪些内容，直接手动更改文件，然后add添加到暂存区，最后commit
# 2：使用 git reset -hard HEAD^ 回退到上个版本
# 3：直接执行撤销操作
$ git checkout -- test.txt

# 误删文件恢复(在 commit 前)
$ git checkout -- test.txt
