1. 用U盘安装Centos6.5 + Win7 双系统

准备工作：U盘(8G)、需要安装的Centos6.5系统(64bit)、EasyBCD(用来修复引导，否则开机只有一个系统)、
                   准备一块空闲的分区(里面的数据会丢失。提前做好备份)，UltraISO(用来制作U盘启动盘)

第一步：制作U盘启动盘
    1、  打开 UltraISO ，点击文件 -> 打开，找到你要安装的系统，然后点打开，加载系统

    2、  点击 启动 -> 写入硬盘映像 ，在硬盘驱动器那里选择你要安装到哪个U盘，然后点击写入，等待一会就好了

第二步：开始安装系统
    1、  开机按 DEl 进入主板设置，找到 boot 选择第一启动项为 U盘启动

    2、  重启电脑就进入了安装界面，这里可以看到四个选项：
            a、install or update an existing system (安装或者升级现有系统)，默认选择这个，有时候这个不能安装，可以尝试第二个
            b、install system with basic video driver (安装过程中采用基本的显卡驱动)
            c、rescue installed system (进入修复系统)
            d、boot from local drive (退出安装，从硬盘启动)
            e、memory test (内存检测)
            选择第一项回车，进入引导界面，点击 Next

    3、  出现语言选择界面，选择语言为English，然后点 Next

    4、  键盘选择 us，然后选择 Hard drive (硬盘驱动器)，选择最后一个 /dev/sdb8，因为我们的系统是在U盘上的，然后点 “OK”

    5、  这里提示：你要使用哪种设备安装，第一个为基本存储设备，第二个 可以自己指定存储设备，我们选择第一个，点击 Next

    6、  设置主机名，点击 Next，设置时区为 Asia/shanghai，点击 Next，设置管理员密码，这个密码最好设置比较好记的，以后经常
            要用到，这里会弹出：You hava provided a weak password:it is too simplistic/systematic ，选择 "Use Anyway" 就可以了点击 Next，

    7、  硬盘分区选项：
            a、Use All Space  (使用所有空间)，这个将会使用一整块硬盘，上面原来的数据和系统会丢失
            b、Replace Existing Linux System(s)  (替换当前的Linux系统)，删除原来设备上的 Linux分区
            c、Shrink Current System  (缩小当前系统)，相当于压缩卷来给要安装的系统留出空间
            d、Use Free Space  (使用剩余空间)，只使用所选设备中的未分区的空间，不会造成当前系统和数据的丢失
            e、Create Custom Layout  (创建自定义布局)，使用分区工具，手动配置分区
            这里我们选择最后一个，创建自定义布局，点击 Next

    8、  这里会出现你系统上的所有已使用分区和空闲分区，我们找到之前准备的未使用的分区，选中，然后点击 Create，
            选择默认的第一个选项，在 Mount Point中选择 /boot,文件格式选择ext3，Size选择500M，然后点击 "OK",重复刚才的操作，选中
            空闲分区，点击创建，选择文件系统的类型为 swap(交换分区)，Size为 (内存大小的 2倍)，然后点击 "OK"，最后继续上述操作，
            选择 / ，文件格式为 ext4，Size 选择 Additional Size  Options 中的 Fill to maximum  allowable size (将剩余的所有空间分配给 / )，然后点击 Next

    9、  选择 Write changes to disk (写入磁盘)

    10、这里很重要，这里是选择系统启动的引导，默认是安装在 U盘的，我们需要手动修改，点击 Change device，选择驱动器的顺序，第一个选择
            本地磁盘驱动器，第二个选择U盘驱动器点 "OK"

    11、下面就是安装的最后一步了，这里会比较久，等待一会，安装好后点击reboot，系统重启(拔掉U盘)

三、修复引导
        你会发现进入的是Windows的系统，不是Linux的，这里我们需要用前面准备的 EasyBCD软件来修复引导，打开软件，点击左侧的添加新条目，
        在右侧的 Linux/BCD 下，选择类型为：GRUB(Legacy) 、名称为 NeoSmart Linux，驱动器选择你安装的系统所在的盘符，然后点击添加条目，
        最后在编辑引导菜单中有一个开机默认时间，你可以自行修改，配置完点击保存就可以了，接下来重启电脑就可以看到界面会出现Windows7 和
        NeoSmart Linux 两个系统选择，到这里安装过程就结束了
