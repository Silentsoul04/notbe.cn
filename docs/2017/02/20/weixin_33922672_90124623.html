22. 使用 awk / grep / head / tail 命令进行文本 / 日志分析 (/home/D/acc.log)

一、awk
    
# 统计 a-read-file 接口 中，接口耗时 超过 0.007 秒的有多少个请求
    D@Demon ~]$ awk '$7>0.007' acc.log | wc -l   (-l 表示统计有多少行)
    404

二、head

1、head 命令参数

    -n            指定获取多少行
    -v            输出显示文件的文件名
    -c            按指定字节显示
    -q            隐藏文件名
