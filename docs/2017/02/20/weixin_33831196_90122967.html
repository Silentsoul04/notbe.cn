4. Linux 系统目录


/home            用户主目录的根节点，所有用户自己独有的文件一般放在这个目录下的用户目录下
/bin                二进制可执行命令
/tmp               存放临时文件
/usr                应用程序默认安装路径，一般的大文件都是放在这里，包括内核源代码，需要的空间是最大的
/lib                 存放一些库文件
/sys               sysfs 虚拟文件系统的挂载点，sysfs 类似于 proc ，现以取代 proc
/etc               系统的管理和配置文件
/proc              一种虚拟的基于内存的文件系统， 访问该文件可以获取系统信息     
/sbin             超级管理命令，系统管理员使用的管理程序
/var/log         存放系统日志
/var/lib          系统正常运行时要改变的数据
