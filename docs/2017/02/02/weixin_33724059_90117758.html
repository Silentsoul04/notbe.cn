免费的云技术平台--亚马逊WEB服务AWS提供免费方式

Feb 2, 2017


Amazon宣布，从11月1日他们会为新客户提供为期一年免费的AWS使用方式，所涉及到的产品包括EC2、负载均衡器、EBS和S3。 SimpleDB、SQS和SNS对新客户以及老客户的提供方式还没有确定。 客户可以在任何想要的地方使用服务，但有以下限制：

  • 对Amazon EC2 Linux微实例（内存613MB，支持32位和64位平台）750个小时的使用——这些时间足够在每个月中持续运行。
  • 对Elastic负载均衡器750个小时的使用，加上15GB的数据处理。
  • 10GB的Amazon Elastic Block Storage，加上1 million I/Os、1 GB的快照存储、10,000个快照的Get请求以及1000个快照的Put请求。
  • 5 GB的Amazon S3存储、20,000个Get请求以及2,000个Put请求。
  • 每月30GB的internet数据传输（在除Amazon CloudFront之外所有服务上15GB输入和15GB输出的数据传输）。
  • 针对Amazon SimpleDB Machine的25小时使用时间和1GB的存储。
  • Amazon简单队列服务（Simple Queue Service）的100,000个请求。
  • 针对Amazon简单通知服务（Simple Notification Service）的100,000个请求、100,000个HTTP通知以及1,000个邮件通知。

 

如果部署在免费账户上的应用程序使用的资源超过所分配的量，那么对使用的额外资源会按照标准的pay-as-you-go方式收费。

 

使用AWS的云计算平台，需要先注册，注册是当然是免费的，进入到http://aws.amazon.com/free/页面，点击“Sign up Now”按钮开始注册，然后按照提示操作很快即可注册成功。

注册成功后，你可以登录到AWS console。对于新用户，第一次登录会提示您注册EC2服务，即Elastic cloud computing service。注册需要您填写有效的信用卡信息。不过因为我们选择的是免费版，我们并不需要为此付费，如果你超过了免费服务提供的额度才会收费。

 

此外还需要注册S3服务，即Simple Storage service。

 

另外需要注意的是亚马逊采用电话验证方式，你需要填写你手边的有效电话号码，很快你就会接到亚马逊打来的电话，提醒你输入屏幕上显示的4位PIN码。如果输入正确则验证成功。

EC2服务和S3服务注册后貌似还需要一段时间才能用，我刚注册过，再等一段时间再看看效果尝尝鲜 ：）
