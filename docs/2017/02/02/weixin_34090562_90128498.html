前几天入手一大菠萝，写个初始化教程

Feb 2, 2017

一般情况，任何一个事物都是入门吓死人，其实并不是。。。

 

首先打开wifi共享的按钮（总共两个按钮，一个是充电供电功能，一个是WiFi共享）

一般刚入手都是有一个没有密码的WiFi == 》 Pineapple_xxx

手机连接一下，输入：http://172.16.42.1:1471/

就会有个向导模式，（老版本的用户名和密码是：root,pineapplesareyummy）

重新设置一下Root密码，避免黑吃黑，然后设置一下管理的WiFi（这个不是分享的WiFi）

你会发现这时候原来的WiFi变成了你设置的WiFi名了（其实还有一个隐藏WiFi）

来来来，我们把隐藏的WiFi开下来（去掉复选框 Hide Open AP）

这时候你会发现，又多了个不用输入密码的WiFi，这个就是我们的钓鱼WiFi了，之前第一步设置的是正常WiFi

等了会，发现一个小米2的手机上钩了，咳咳，我们这是普及安全知识，咳咳

怎么上网？额。。。2种，1就是你把他当路由器，2就是连接手机共享呗(当然了，它也可以当充电宝)

玩法千万总，你可以把Open AP换成诸如CMCC等等的名称，页面仿照一下，有些安全意识稍微高点的也会着了道（不是什么地方都有CMCC的，这个可以利用），也可以只是安全的“抓包”，到时候来个“重定向”。

里面所有的功能官方文档都写的很清楚了，以后有机会咱们再继续探讨，今儿就先到这，毕竟这不是我们专攻的方向，闲暇之时玩玩就行了

官方文档：http://wiki.wifipineapple.com/
