【号外】2011 Autodesk开发者日又来啦~~

Feb 9, 2017


今年的Autodesk开发者日将在上海（2011年11月11日）和北京（2011年11月14日）两地分别举行，会上除了会发布下一版本AutoCAD最新变化和API之外，还有Autodesk云技术及Autodesk针对基础设施行业（Infrastructure）的旗舰产品的最新开发信息。

 

 

 

摘自Autodesk网站
AutoCAD Civil 3D为一款Autodesk在基础设施行业的引领BIM设计潮流的旗舰产品，今年我们将继续深入地介绍 Civil 3D .NET API方面的扩展功能，包括可以实现多图档文件访问的API。
针 对Autodesk 地理信息产品线，AutoCAD Map 3D新增了许多新功能及API，有关于在布局中的多地图操作及设置动态指北针，还有针对FDO数据的清理工具增强等。对于任何Web类应用程序其服务器性 能表现是极其关键的指标，Autodesk Infrastructure Map Server（AIMS）作为Autodesk MapGuide Enterprise 与Autodesk Topobase的合并产品，在服务器性能方面的表现又如何呢？其新的UI创建工具以及API又有哪些呢？请来参加今年的开发者日，来现场获知更多关于下 一版本AutoCAD Map 3D以及 AIMS的新增功能及新增API吧。

 

 详情及报名请见：
