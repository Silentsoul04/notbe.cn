3. Root

首先需要下载以下几个文件：

  • TWRP recovery  ： https://twrp.me/Devices/
  • SuperSU Beta 2.49 ： 链接: http://pan.baidu.com/s/1nt3P6Nf 密码: 7bmv   或   https://download.chainfire.eu/740/SuperSU/BETA-SuperSU-v2.49.zip
  • Despair Kernel 内核：

            For Nexus5 链接: http://pan.baidu.com/s/1mguyemO 密码: w3u2  或 http://renderserver.net/devs/DespairFactor/WIP/M/hammerhead/
            For Nexus6  http://renderserver.net/devs/DespairFactor/WIP/M/Despair.M.Shamu.R2.4.zip

请先刷TWRP recovery，教程在此 Nexus 5 recovery 一键刷入工具发布 。

刷入TWRP recovery 后，需要进入TWRP，依次刷入

a. Despair.M.HH.R1.8.zip 内核
b. SuperSU Beta 2.49 



首先请把下载的Despair.M.HH.R1.8.zip 内核 和  SuperSU Beta 2.49 文件放在SD卡根目录

（1）重启手机进入recovery模式："电源键" 和 "音量-" 一起按进入bootloader，连续按两次 “音量- ” 选择recovery，按 “电源键” 确认进入recovery，手机将会重启进入TWRP

（2）由于这样刷机会 清空数据，所以请提前 备份保存


（3）刷入Despair.M.HH.R1.8.zip 内核：点击 install ，找到Despair.M.HH.R1.8.zip 内核文件并选择，滑动滑块确认刷入，成功后会返回主页。

话说这个图实在不知道怎么回事，总是显示有问题，大家请打开大图查看。


（4）刷入SuperSU: 同样方法点击install，选择下载的SuperSU2.49，滑动确认刷入。成功后，点击reboot -> system 重启系统


到此刷机和root完成。

4. 软件兼容

由于一些软件兼容性不好，导致大家不选择刷Android M，我尝试了大部分软件都兼容，也找到一些兼容版本

微信: 需要最新的 微信for Andrid 6.2.2官方内测体验版  链接: http://pan.baidu.com/s/1c1Ayy 密码: txh4

QQ ： QQ HD mini 2013，暂时只有这版可用  链接: http://pan.baidu.com/s/1nt5RvRj 密码: 5rjk

支付宝：其实可以用，只要不用支付宝自带的键盘就好，输入密码的时候现在其他记事本里写好，然后复制粘贴到支付宝密码框就可以登录了

fq路由器2 ： 这个比较麻烦，第一次打开软件时候不要同意root，也不要选择默认设 置，选择手动设置代理，然后打开软件后，进入设置，改变启动模式为Root，再去SuperSU里手动允许root权限，这时候退出fq路由器2重新打开 就可以了。如果一开始就允许root会闪退，无法进入软件。

nexus 5 Field test model 可以正常开启4G， Portal server修改 可以去感叹号

其他我装的软件基本都能正常使用。


关于耗电：我一开始装了绿色守护，正常使用，耗电确实有改善，后来把绿色守护关了发现不影响耗电，也就是说M系统本身耗电就控制的很好，因为在电量里有优化，只有系统级别的应用没有开优化。整体耗电满意~系统自带官方电源控制就好像内置了绿色守护~
