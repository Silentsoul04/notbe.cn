红蜻蜓截图工具在win7下保存截图为png失败

Oct 5, 2017

一直用红蜻蜓截图工具来做测试，一直用的好好的。前天公司换电脑，我很好运气换了一台，新机器，thinkpad E40，考虑到这机器性能比我先前的好很多，就在上面装了win7，而非先前使用的xp。
好在有台windows server2003服务器给我用，可以用上面的ie6来做测试。
不过今天在用使用红蜻蜓时候，在桌面上另存为我的截图为png，然后去打开这个截图，发现截图的大小是0字节，然后打开，windows照片管理器会提示说“windows照片查看器无法显示此图片，因为该文件是空的”。查看了下红蜻蜓的版本，是V2.02 build 20090728版本。

很奇怪的现象，先前在windows xp上也能正常跑来着。难道是因为系统级某些接口变了吗？或者说是win7的防护机制太严格了？
