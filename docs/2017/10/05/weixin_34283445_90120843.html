前后台互访

Oct 5, 2017

前台访问后台

 

 

1、<%=xxx%>访问后台的字段或方法。
-----------------<%=服务端语言的表达式可以为有返回值的方法，但返回值不能太大%>
-----------------<%可以执行服务器代码%>
-----------------<%#数据绑定%>

前台：
        <asp:HyperLink ID="HyperLink1" runat="server"><%=str() %></asp:HyperLink>
        <asp:HyperLink ID="HyperLink2" runat="server"><%=kaka %></asp:HyperLink>

后台：
    public string kaka = "kkkkkk";
    public string str()
    {
        return "hebeidaxue";
    }

2、通过页面上隐藏域访问,可以在后台把c#变量值保存到隐藏文本域当中。

　　<input id="xx" type="hidden" runat="server">

　　然后在前台javascript当中直接取隐藏文本域的值。

　　 document.getElementById('xx').value

3、在服务器端变量赋值后在页面注册脚本

　　Page.RegisterStartScript(" ","<script language='javascript'>var vary=" + value + "</script>");

　　value是后台变量，然后javascript中可以直接访问vary值,它的值就是后台变量value的值，这种方式只不过是能过一种间接的方式来访问c#变量。
