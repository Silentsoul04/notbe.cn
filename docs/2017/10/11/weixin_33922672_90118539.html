使用SecureCRT连接ubuntu

Oct 11, 2017

在虚拟机装好ubuntu之后。一直想用SecureCRT在物理机上连接一下，无奈总是失败，今天下定决心要把这个问题攻克了，晚上捣鼓一下，最终攻克了。

我是用的ubuntu版本号是14.04，装好之后默认是没有安装ssh服务的。须要手动安装： 
安装命令：

sudo apt-get install openssh-server

安装完毕之后，查看安装是否成功：

netstat -tlp

看到这个表示成功安装 

这个时候使用ifconfig查看linux IP地址：

然后在物理机上就能够使用SecureCRT来连接linux了。
