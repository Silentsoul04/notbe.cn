psl/sql本地与远程连接配置

Oct 11, 2017

一：下载Oracleclient

下载地址：http://www.oracle.com/technetwork/database/features/instant-client/index-097480.html，点击连接进入会出现下面页面。

选择第一个Instant Client for Microsoft Windows (32-bit) 进行下载就能够了。请不要下载64bit的，我试过。不能实现连接的。

这里须要注意的是下载client的时候。须要下载与数据库相应的版本号，你是oracle11G的版本号就下载11G的，版本号号要尽量精确。



二：配置Oracleclient

下载之后将其解压，不须要进行安装。比方解压到......\TOOLS_APP文件夹下。然后在instantclient_11_2文件夹下新建两层文件夹\NETWORK\ADMIN，再在ADMIN文件夹以下建一个tnsnames.ora文件,然后向文件里加入例如以下内容：

ORACLE =

  (DESCRIPTION =

    (ADDRESS = (PROTOCOL = TCP)(HOST = localhost)(PORT = 1521))

    (CONNECT_DATA =

      (SERVER = DEDICATED)

      (SERVICE_NAME = ORACLE) 

    )

  )

LISTENER_ORACLE =

  (ADDRESS = (PROTOCOL = TCP)(HOST = localhost)(PORT = 1521))

ORACLR_CONNECTION_DATA =

  (DESCRIPTION =

    (ADDRESS_LIST =

      (ADDRESS = (PROTOCOL = IPC)(KEY = EXTPROC1521))

    )

    (CONNECT_DATA =

      (SID = CLRExtProc)

      (PRESENTATION = RO)

    )

  )

以上的内容能够在这个地方进行拷贝：D:\instantclient_10_2\NETWORK\ADMIN\tnsnames.ora。


#上面的SERVICE_NAME =请填写你的数据库服务名

#localhost。假设是本地一般不须要改动，是远程的换成你的远程IP地址


三：进行安装PL/SQL Developer

将之前下载好的PL/SQL Developer（没有的在网上自行下载）进行安装，但须要强调一点：安装路径中不能有括号。不要安装在C:\Program Files (x86)文件夹以下。



四：进行PL/SQL Developer的配置：

安装完毕之后，执行PL/SQL Developer。此时出现的登录窗口不能进行登录，点击Calcelbutton，这时会在无登录状态下进入。然后配置对应信息。把之前的解压包信息配置上：点击Tools



最后验证妈可
