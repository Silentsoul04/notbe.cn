Basic4android - 使用Basic开发Android应用

Oct 19, 2017

Basic4android， 拥有独立的IDE，强大的libary覆盖android SDK，编译生成原生apk包及java代码。支持所见即所得的layout，让你只关注业务逻辑以快速开发。

环境安装

1) 安装Basic4android，官方网站有免费版下载，收费版本功能更全

2) 真机上安装B4A-Bridge，连接真机调试用，模拟器不需要

3) 安装JDK

4) 安装Android SDK (路径不要有空格)

    配置SDK，下载所需要的包 (我这里选择的是platform-tools,2.2 sdk,usb)

5) 配置Basic4android中的android sdk路径和jdk路径

Tools -> Configure Paths

模拟器配置

1) 配置AVD

可以New一个新的virtual device，选中，start启动模拟器

Hello word

1) 启动IDE，加下面一行代码，保存项目，project/package name设置名子(d4a.开头小写)

Sub Activity_Create(FirstTime As Boolean)
    Msgbox("hello,basic for android","test")
End Sub

2) F5启动，开始编译

部署到模拟器后运行效果

3) 代码文件物理结构

Files
 |----.bal(layout文件),资源文件等
Objects
 |----apk包及java代码，编译时才生成
hello.b4a 代码文件，一个module一个文件(类似一个windows窗体，但不一定是界面layout，一个panel也可以加载layout)

 IDE

1) 右下角4个选项

modules: 显示模块名及函数名
Files: files目录下的文件(资源)
Logs: log信息输出
Libs: 用到那些库就选择上，代码里就可以使用这些库了，不需要using或import之类。

2) 智能提示

3) 自动补全

界面可视化设计

菜单栏选择"designer"

Designer/Tools ->connect to device/emulator

连接到模拟器设计，这里一个是虚拟的可视化界面，一个是模拟器界面

菜单"add view"下选择你要添加的控件，更改其属性，如下图
