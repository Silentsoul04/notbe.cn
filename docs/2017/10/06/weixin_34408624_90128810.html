[游戏]近期期待作品一览

Oct 6, 2017

  马上总算要是9月了，9月将会有好几部大作值得期待一下，现在先列出来，等待发布。

 

一、无限试驾2

这款游戏是正统续作，原来在WII上玩过，虽然在WII上的画面不算好，但是其游戏性不错，期待一下，目已经有BETA版本了。

游戏名称：无限试驾2 
英文名称：Test Drive Unlimited 2 
开发厂商：Eden Games 
发行厂商：Atari 
游戏类型：竞速 
发行日期：2010年09月22日 
官方网站：http://www.testdriveunlimited2.com/

 

配置需求： 
最低配置: 
操作系统 Windows XP SP2, Vista SP2, Windows 7 
处理器 Intel Core 2 Duo 2.2GHz 或 AMD Athlon X2 4400+ 
内存 2GB 
显卡 Nvidia GeForce 8800 / ATI Radeon HD 3870 
声卡 DirectX 9.0c-compatible 
硬盘空间 14 GB 
输入设备 10-button controller 如同 Xbox 360 或 Logitech Dual-Action 
其他 网络连接, Microsoft .NET 3.5 required 
推荐配置: 
操作系统 Windows XP SP2, Vista SP2, Windows 7 
处理器 Intel Core 2 Duo 2.2GHz 或 AMD Athlon X2 4400+ 
内存 2GB 
显卡 Nvidia GeForce GTX 280 / ATI Radeon HD 4870 
声卡 DirectX 9.0c-compatible 
硬盘空间 14 GB 
输入设备 10-button controller 如同 Xbox 360 或 Logitech Dual-Action 
其他 Internet Broadband Connection, Microsoft .NET 3.5 required

 

二、暗黑血统：战神之怒

这款是原来发布于XBOX360上的，现在一直到今年9月才移植到PC上，很值得期待，感觉就像PS上的《战神》系列

本作的主角是末日四骑士之一。玩家将会带领着自己的军团，摧毁全世界，与饥荒之神、死神和瘟疫之神一起杀死地球上的最后一个人类。多数游戏凡是碰上此类题材都会在重要的时刻剧情一转，让人类最终免于被消灭，但是本作的目标是彻彻底底的、真正的摧毁全人类，但是摧毁人类只不过是本作的第一步，在游戏前半个小时的时间里就可以完成，然后就留下一个没有人类存在的世界，取而代之的是恶魔和堕落天使。 
最低配置 
操作系统 Windows XP SP3, Vista SP1, Windows 7 
处理器 AMD Athlon 64 Processor 3800+ 2.4Ghz, Intel Pentium 4 530 3.0Ghz Processor 
内存 1GB RAM (Windows XP), 2GB RAM (Vista / Windows 7) 
显卡 ATI Radeon X1900 o或 NVIDIA GeForce 8800/GeForce GT220 with 256MB of Video RAM 
DirectX®: DirectX 9.0c 
硬盘空间 12GB 以上剩余 
声卡 DirectX 9.0C 兼容声卡 
建议配置 
操作系统 Windows XP SP3, Vista SP1, Windows 7 
处理器 AMD Athlon 64 X2 5200+ Dual Core 2.60Ghz, Intel Core 2 Duo E6420 Dual Core 2.13Ghz 
内存 2GB RAM (Windows XP), 3GB RAM (Vista / Windows 7) 
显卡 ATI Radeon HD 3870 或 NVIDIA GeForce GTS 240 with 256MB of Video RAM 
DirectX®: DirectX 9.0c 
硬盘空间 12GB以上剩余 
声卡 DirectX 9.0C 兼容声卡

 

 

三、丧尸围城2

这款游戏是CAPCOM的动作冒险类大作《丧尸围城》的续作，由于第一代并没有在PC上发售，所以没玩过，不过由于CAPCOM的全平台计划，所以现在家用机上的大作都会移植到PC上，例如以前的《鬼泣4》，《生化危机5》、《失落的星球》（以上3款全部通关），现在就在等9月28日的正式发布了。

基本信息

发售日期：北美时间2010年9月28日 
多人连线：是，最多4人 
包含语种：英语，法语，德语，意大利语，西班牙语 
XBOX360手柄支持：是

基本配置

系统：XP，VISTA，Win7 
处理器：Intel Core 2 Duo 2.4 Ghz 或 AMD Athlon X2 2.2 Ghz 
内存2 GB RAM 
显卡：NVIDIA GeForce® 8800GTS 或 ATI Radeon HD 3850 
硬盘：8.5 GB free hard drive space


 

最近在玩的游戏——《燃烧僵尸》

前几天下了一款小游戏《燃烧僵尸》，是俯视射击类游戏，就是控制角色在大堆大堆的僵尸的围攻下使用各种各样的武器生存下来，比较有挑战性，有空的朋友可以试玩下。

中文名称：燃烧丧尸 
英文名称：Burn Zombie Burn 
游戏语言：英语 
开发厂商：Doublesix 
发行厂商：Kuju Entertainment Limited 
游戏类型：格斗 
发行日期：2010年08月13日 
官方网站：http://www.burnzombieburn.com/


游戏介绍： 
这款游戏的内容跟2009年所发售的PS3版内容完全一样，是一款采用了俯视视角的射击游戏，在游戏中你将要对抗成群的不同种类的僵尸：爆破僵尸、传染僵尸以及“放屁都会冒火的超级僵尸”。明白是款什么游戏了吧。在游戏中你将会有大量的枪械和各种武器来保护自己，共有六个不同关卡，三种游戏模式。PC版当然也会同Steam平台的成就系统、自动升级和排行榜等功能挂钩。

 

配置需求： 
操作系统 Windows XP (Service Pack 3), Windows Vista, Windows 7 
处理器 1.8 GHz, Dual Core 
内存 2 GB 
显卡 DirectX® 9 兼容 graphics adaptor. 512 MB, With Shader Model 3.0 
DirectX®: DirectX® 9 
硬盘空间 2 GB 
声卡 DirectX® 9 compliant sound card

 

 

此游戏目前已经有汉化版，想玩的朋友可以到这里下载：http://bbs.3dmgame.com/showtopic-1409686.html
