四、Html5 语音识别

Oct 6, 2017

Html5 实现语音识别So easy!!!!

一般的文本框

<input />

语音识别文本框

<input x-webkit-speech />

OK ... 就是这么简单！

 

其他参数：

1、lang　　强制输入框里的语言种类

<input type="text" x-webkit-speech lang="zh-CN"/>

2、x-webkit-grammar　　语音输入语法，"builtin:search”值使得语音输入的内容尽量靠近搜索内容，去除多余的字符。
例如自动删除写助词之类的，没有测试

<input type="text" x-webkit-speech x-webkit-grammar="bUIltin:search" />

3、onwebkitspeechchange 当语音识别成功，文本写入到文本框之后触发。

    <script type="text/javascript">
        function fun() {
            alert("OK");
        }
    </script>

<input x-webkit-speech  onwebkitspeechchange="fun();"/>

 

 

 

支持Html5的浏览器包括Firefox（火狐浏览器），IE9及其更高版本，Chrome（谷歌浏览器），Safari，Opera等；国内的傲游浏览器（Maxthon），以及基于IE或Chromium（Chrome的工程版或称实验版）所推出的360浏览器、搜狗浏览器、QQ浏览器、猎豹浏览器等国产浏览器同样具备支持HTML5的能力，所以此功能也只能在此条件下才OK！！
