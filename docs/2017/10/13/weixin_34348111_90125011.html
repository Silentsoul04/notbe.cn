windows下启动和关闭oracle数据库的bat脚本

Oct 13, 2017

1.启动脚本 
----------------------------------------------- 
net start OracleOraDb10g_home1TNSListener 
net start OracleServiceORCL 
rem net start OracleOraDb10g_home1iSQL*Plus 
rem net start OracleJobSchedulermyoracle 
net start OracleDBConsoleorcl 
-----------------------------------------------


2.关闭脚本 
----------------------------------------------- 
net stop OracleOraDb10g_home1TNSListener 
net stop OracleServiceORCL 
rem net stop OracleOraDb10g_home1iSQL*Plus 
rem net stop OracleJobSchedulermyoracle 
net stop OracleDBConsoleorcl 
----------------------------------------------- 
    net start/stop 分别用来启动网络服务,oracle数据库安装成功并且创建了对应的数据库实例后,会生成5个服务. 这5个服务名称可以在windows服务(service.msc)中看到. 
    关于为什么要把OracleOraDb10g_home1iSQL*Plus和OracleJobSchedulermyoracle这两个服务停掉,是因为通常这两个服务是被禁止运行的.而我平常也没用到.
