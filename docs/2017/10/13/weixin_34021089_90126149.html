Linux安装后更新系统

Oct 13, 2017

step1 先把/etc/apt/source.list做个备份,然后打开参照里面的原有的地址,改成国内的开源镜像的地址.

step2 sudo apt-get update 更新软件列表

sudo apt-get upgrade 更新已有软件

step3 sudo apt-get install 安装软件

以下是一些开源镜像站列表:

1.网易开源镜像站

http://mirrors.163.com

2.兰大开源镜像站

http://mirror.lzu.edu.cn

3.搜狐开源镜像站

http://mirrors.sohu.com

4.中科大开源镜像站

http://mirrors.ustc.edu.cn

5.上海交大开源镜像站

http://ftp.sjtu.edu.cn

6.北京交大镜像站

http://mirror.bjtu.edu.cn/cn

7.ubuntu中国内开源镜像站

http://mirror.bjtu.edu.cn/cn
