微软代码共享网站MSDN Code Gallery上线（转）

Oct 25, 2017

本文转自：http://www.infoq.com/cn/news/2008/02/CodeGallery。

微软创建了一个用于代码和应用程序共享的新站点，取名为Code Gallery。任何社区成员都可以在站点上发布一个应用或是源码，让其他人可以使用。微软员工也会通过这个站点张贴他们工作代码中的片段，揭示目前或是未来微软产品中的秘密。

Code Gallery，Code Plex和MSDN下载中心的区别是什么？Code Gallery是设计来用作一个共享空间，人们可以与他人共享演示C#关键特性的示例工程，或是任何看上去比较有吸引力值得共享的项目，也或者是会使社区受益的项目。Code Plex是用来进行协作式项目开发的。社区成员可以在Code Plex上通过资源库中的代码和资源共享、代码版本控制和版本发布来参与各种项目的开发。微软会继续把一些项目放到MSDN下载中心中，比如VS的试用版或是VS service packs。社区成员是无法使用MSDN下载中心的，它只对微软员工开发。由于GotDotNet即将被停用，所以GotDotNet上面的项目会被挪到Code Gallery或是Code Plex里面去。

Code Gallery全站使用的是一个客户许可。基本上，任何人都可以浏览、下载、编辑、重发布或是使用站上的所有资源——除了该许可中不允许的情况以外。

项目的拥有者可以给其他人——被称为协作者——编辑所发布的资源的权限。Code Gallery不需要源码控制程序。在每个资源页上都有对应的RSS种子。Code Gallery上上传的文件大小不能超过100 MB。
