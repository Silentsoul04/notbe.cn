iOS：集成ijkplayer视频直播

Oct 4, 2017

介绍：

ijkplayer 是一款做视频直播的框架, 基于ffmpeg, 支持 Android 和 iOS, 网上也有很多集成说明, 但是个人觉得还是不够详细, 在这里详细的讲一下在 iOS 中如何集成ijkplayer, 即便以前从没有接触过, 按着下面做也可以集成成功!

原地址 : http://www.open-open.com/lib/view/open1463123602429.html

对ijkplayer框架暴露头文件的简要介绍:

  • IJKMediaPlayback

    • IJKMediaPlayback 协议中定义了一系列视频播放相关操作的方法，当然还有使用property定义的一系列属性（你需要知道在协议中是不能定义属性的，因为属性属于类的，而是定义了与之相关的setter/getter方法，注意是方法。）你甚至会发现还定义了一系列的通知，包含播放是否已经准备好，视频加载状态改变，视频播放完成等。它们将来会用于你去监听视频的不同状态，根据不同的状态，你需要做出不同的响应。

    • 为什么说@protocol IJKMediaPlayback <NSObject>很重要呢？当你稍微了解ijkplayer框架后，你会发现框架中最重要的用于视频播放的IJKFFMoviePlayerController、IJKAVMoviePlayerControlle、IJKMPMoviePlayerController都是遵守该协议的。当你仔细的查看了它们的接口文件后，你发现只有在IJKFFMoviePlayerController类里或多或少的对外暴露了几个协议中的方法接口，其它的并没有？我们知道既然遵守了某协议，那么该协议中定义为@required的方法都必须实现，缺省是@required。你不用多想，IJKMediaPlayback协议中的方法都肯定被它们实现了，当你查看了它们对应的实现文件后，你会发现确实如此。

  • 视频播放器
    据我的了解，IJKFFMoviePlayerController、IJKAVMoviePlayerControlle、IJKMPMoviePlayerController都可以用于视频的点播与直播，IJKFFOptions可用于IJKFFMoviePlayerController播放器的选项设置。

一. 下载ijkplayer

二. 编译 ijkplayer

说是编译 ijkplayer, 其实是编译 ffmpeg, 在这里我们已经下载好了ijkplayer, 所以 github 上README.md中的Build iOS那一步中有一些步骤是不需要的.

下面开始一步一步编译:

1.打开终端, cd 到jkplayer-master文件夹中, 也就是下载完解压后的文件夹, 如下图:

 

2.执行命令行./init-ios.sh, 这一步是去下载 ffmpeg 的, 时间会久一点, 耐心等一下.如下图:

3.在第2步中下载完成后, 执行cd ios, 也就是进入到 ios目录中, 如下图:

4.进入 ios 文件夹后, 在终端依次执行./compile-ffmpeg.sh clean和./compile-ffmpeg.sh all命令, 编译 ffmpeg, 也就是README.md中这两步, 如下图:

编译时间较久, 耐心等待一下.
