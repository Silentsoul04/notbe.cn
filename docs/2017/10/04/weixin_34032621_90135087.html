烦死人的Vista和Longhron Server。

Oct 4, 2017

折腾了整整一天，很顺利的下载了Vista RC2并安装好，结果发现N多不爽！

  1. SQL Server 2005只能装客户端，虽然预料中的事情，但是还是烦。
  2. 内存消耗太恐怖了，稍开一下浏览器，内存消耗就到80％（总1G），然后就跳出内存不足的窗口（我不玩虚拟内存的，速度诶）。这样的内存消耗我怎么玩哦？只开了一下IE7而已，大部队（FW，PS，VS2005，SQL Server 2005，Live Messenger，QQ，FF）还没动呢，.NET开发环境都没配上，晕倒，估计开虚拟内存都不够。烦！
  3. 雅黑字体。为什么那么多人叫好？没搞懂，雅黑比宋体难看多了呀，扁扁的，横不横竖不竖，我是CRT 1280X1024，特别吃力。烦！
  4. 于是转念装Longhron Server，但是……LS据说还是没能解决声音问题，这可不行。而且据说最新的LS 5728不能开Aero，唉。烦！

挣扎了很久，看来11月的Vista是没指望了，我需要Server环境做开发，而LS，猴年马月去了。罢了罢了，不折腾了，老老实实用2003吧。其实2003速度还是一流的。

只不过我喜欢追新，可以通过新产品看见未来。设计的未来，技术的未来。特别是设计，Vista的各类软件界面，各种界面细节都有很多可嗅之处，毕竟MS是MS。我希望能从他的系统设计中捕捉到一些东西，捕捉到一些未来，然后吸收。再说了，我主攻设计，一个漂亮的设计环境可以让人很放松，很快乐，一个好的心情对设计来说是关键。

没法了，还是回到2003，不过现在设计和开发完全可以很好的分开进行，恩，那么以后设计集中时间在Vista中完成，Codding（css设计和.NET开发）到2003上去搞。……烦不烦……好像还是烦，馊主意。
