一个人的渺小与微不足道。

Oct 26, 2017

有一次在标准群中互相介绍自己所在领域时，一叶千鸟说他的方向是：

Web Interaction Design
Web Visual Design
Web Standards Design
Website User Researcher
Website Information Architect
Website User Experience……

昨天和没有耳多交流，然后了解了他的OPS框架；

今天无意中来到臭鱼的交互设计，阅读了他写的几篇关于交互设计分析的文章，并追随进入了UI花园……

还有很多很多人，在我成长的过程中让我一次次明白自身的渺小和微不足道，然后一次次的灰心和垂头丧气，然后一次次的化为斗志，奋起直追。是的，很多时候我的源动力就出于此。

在这个过程中，随着时间的推移，记忆开始模糊，所以决定建立一个分类，名为“博览”，为了牢记住曾经和即将鞭策我的高人们、为了让自己时时刻刻的被鞭策，记住：一个人是渺小和微不足道的！

居然在硬盘里找到了6年前我刚学网页设计时做的第一个个人网站，里面有一篇《全面发展就是全面平庸？——读《三重门》有感》，是自己在2000年8月31日写下，好吧，翻出来看看变质没有。


《全面发展就是全面平庸？——读《三重门》有感》【爆牙齿 2000 8 31】

前几日，小弟从家里过来，带来《三重门》。

小弟像我当年向他极力推荐《平凡的世界》一样推荐《三重门》。我满怀着希望翻开《三重门》。俗话说：希望越大，失望越大。刷刷刷，翻完它，就书中的故事而言，除了一些无里头式的幽默让我开怀一笑以外，我并没有太多的感受。到是后记中，作者的一篇关于人才的文章让我有所触动，这我们暂且放下，呆会儿再说。

作者韩寒在后记里写到：尽管情节不曲折，但小说里的人生存着，活着，这就是生活。不错，这本书是描写生活，一个中学生的生活。很普通，很平常，这样的生活我们每天都在看着，经历着，看着别人的生活，经历着自己的生活。但是作为好文章和小说，应该能让人感受到什么，领悟到什么，或者能让人有所向往也是不错的。《三重门》，它有出版的必要吗？或许它唯一的亮点是作者韩寒不到18岁，可是读者面对的是书，而不是作者。

现在的校园小说泛滥成灾，字都认不全的小孩都敢操起长篇，十几万字轻轻松松的就在他们笔下浪费掉。我很佩服他们的胆量，但一点都不认可。年龄的局限决定了他们人生经验的缺乏，也导致了题材的单一：几乎都是换个名字说自己和自己身边的故事。这样的小说目的何在？为了出版？没必要吧！为了记录自己的生活？那还不如坚持一天一篇日记。为了练笔？那就更不如多写几篇精辟的短文！ 回到《三重门》，后记里的一篇《穿着棉袄洗澡》让我有所感触。虽然其中的一些观点我不尽赞同，但但从文章来看，是篇好文章。韩寒的短文的确超越了和他同龄人的水平。但是……

这篇文章提出了两个让人深思的问题。一：全面发展最可能导致的结果是全面平庸。二：当今的教育制度是大便。全面发展很可能是全面平庸。这句话是对的。但是从韩寒这个中学生嘴里冒出来就错了。大多数大学生毕业后工作一段时间，也不一定就能看清自己的人生方向，何况一个中学生。敢问韩寒：你是否就肯定你会成为作家？一辈子的作家？我想韩寒是回答不出的，将来的事谁都说不清。那么，假如，我是说假如韩寒不能成为作家，他怎么办？以初中的文化水平（注意，文化不仅仅是语文或文学）去迎接这日行千里的世界？

人才是什么？专长于一类的，没错。然而这‘一类’并非全指爱好。社会上很多人都在干着他们并不喜欢的工作，而且干得很出色，这才是人才。曾无数次，在大学里我们常讨论，学习的知识究竟将来有什么用。毕业后，当我们面对一个完全陌生的新事物，我们无所畏惧，买书然后按各自的学习方法在短时间内搞懂它。这时，我们发现这就是大学里学到的，知识的另一种体现--能力。中学里打下的基础同样功不可没。韩寒有着很好的写作天赋，应该珍惜，可是当我知道他因为专攻写作而荒废了其他功课（高一就七科不及格，并留级），我深感遗憾，他在糟蹋他这种天赋。中学，绝对是应该全面发展的阶段！20岁以后，等到有了一定的知识作为基础，能力作信心，和一些必要的社会经验，再静下心来作到术业有专攻。

至于当今的教育制度，是很糟糕，但也不像韩寒说得那样糟糕透顶。毕竟他还是培养了很多人才，社会也并没有因此而退步。只是它还很不完善，培养人才的同时也埋没了不少人才。完善不是一天两天的事，我也没辙，只能寄望它快快的完善起来，好起来。

我不认为韩寒是个人才，更不是天才。他有写作的天赋，希望他珍惜，把其他科目补上来。如果因此形成一股跟风热，中学生们都各搞各，那就是社会的悲哀了。


好吧，最后一句：术业有专攻，博览群晓。
