我的四年建站故事（X）

Oct 12, 2017

今天版主们有在群里聊建站的事情了，似乎他们每个人都很热衷于搭建一个自己的网站或者博客，我突然之间觉得非常有必要做一些经验的分享。

 
       首先我想先介绍一下CMS（网站内容管理系统， 即 Content Management System ），网站很多著名的开源CMS，有PHP+MYSQL、ASP+SQL、.NET+SQL的。而大部分受欢迎的大都是PHP+MySQL的。我接触的大多也是PHP的开源建站系统。今天可以会介绍几个，门户的有dedecms 、帝国cms；论坛的有discuz、phpwind；博客的有wordpress等，这四年来，我大概用过超过十几种开源的建站系统，参与建设也将近十个网站，成功的应该是有3个的。一个珠宝门户，一个游戏网站，两个女性网，一个音乐网，一个招商网，然后还有 安卓巴士（http://www.apkbus.com/portal.php），坚持到现在。招商网现在应该算是最成功的了，记得大三那年就已经替政府融资超过一个亿了，收到当地政府各大招商局的大力支持，只不过我当时并没有拥有股权，算了废话少说吧，说说这几个cms吧 。
    
    今天主要讲讲dedecms、discuz、wordpress。这三个分别是门户，论坛，博客的最佳典范。
 
    dedecms 
    官网： http://www.dedecms.com/
   织梦内容管理系统(DedeCms) 以简单、实用、开源而闻名，是国内最知名的PHP开源网站管理系统，也是使用用户最多的PHP类CMS系统。记得前几天参加中国创业者大会的时候，因为安卓巴士在媒体席，然后我看到一些比较著名的科技博客居然也是用dedecms搭建的，比如那个快鲤鱼，对了虎嗅网之前是用discuz搭建的，不过最近他们改版了，估计也是用某开源建站系统定制了个模版吧。走题了好像，囧
 
    DedeCMS 基于 PHP和 MySQL技术开发，可同时使用于 Windows、 Linux、 Unix平台，环境需求如下：
1、Windows 平台
IIS/Apache + PHP4/PHP5 + MySQL3/4/5 如果在windows环境中使用，建议用DedeCMS提供的DedeAMPZ套件以达到最佳使用性能。
2、Linux/Unix 平台
Apache + PHP4/PHP5 + MySQL3/4/5 (PHP必须在非安全模式下运行) 建议使用平台： Linux+ Apache2.2 + PHP5.2 + MySQL5.0
3、PHP必须环境或启用的系统函数
allow_url_fopen GD扩展库 MySQL扩展库 系统函数 —— phpinfo、dir
其实上面说的PHP+MYSQL＋apache环境是常用的环境，建议有兴趣的同学可以在本地搭建一个环境，用于学习。我稍后也会出一个搭建环境的教程出来。
对于dedecms，因为有足够多的模版，在前几年的站长时代大受欢迎，很多非常牛逼的大网站都是用dedecms+discuz搭建的门户+论坛模式。记得我当年学习建站的时候真正深入了解的就是dedecms，还用他来做过了好多企业站和一个门户网站的。
 
discuz
官网： http://www.discuz.net/
discuz应该是迄今为止全球运用最大的论坛系统了。据说是有几百万个论坛都是用的discuz，现在你看到的国内的论坛，不论腾讯，网易等门户的论坛，还是机锋论坛，安智论坛，安卓巴士论坛等都是运营discuz搭建的。我们论坛前几天升级到了最新版的3.0哈，相信这个是大家不会陌生了哈。discuz论坛系统是戴志康（大C）在大学的时候一个人写的，有人记得06年的时候，80后创业四个创业新贵么？戴志康、茅侃侃、汽车之家泡泡网李想，mysee总裁高燃。
最后的结局是：
戴志康的康盛在2010年6000万美金被腾讯全资收购，现在在腾讯做O2O，现在微信上扫面商家打折或者送饮料那个，就是戴志康一帮人在腾讯做的；
混世魔王茅侃侃项目黄了，现在在做一个北京的堵车APP；
李想把泡泡网卖掉了，全力做汽车之家，现在据说年收入已经到达了9亿了，据说最近在想着上市的事情；
高燃的mysee项目失败，现在不知所踪；
 想着大一的时候经常看马云在赢在中国的讲话，一遍又一遍，然后经常跟同学说着上面四个80后财富新贵，现在5年过去了，又如何呢？似乎又换了一波人了，互联网就是这样，转瞬即逝，就像王兴说的：“传统行业创业好比登山，互联网创业好比冲浪。山总是在那里的，你准备好了就去登，永远有机会，登多高取决于实力。而浪是一个接着一个的，你只要踩上一个浪，保持住，它的高度就决定了你的高度。”有些人就踩到了浪花上，但是很快就又退去了。所以作为一名屌丝站长伪程序员就想对真正的程序员们说说，见识+技术能力才有可能创造点东西，况且技术能力那么不济呢？在我眼中，非常牛逼的技术和天才并没有两样，但不是你code就是天才了，就很酷了，完全不是这样子。不好意思再次走题，只有走题才有话说。o(︶︿︶)o 唉。
 
10年wordpress

        博客和内容管理系统 WordPress 迎来了它的10周年纪念日；10年来，WordPress 从最简单的博客服务一步步前进，并逐渐成为改变人们交流和帮助品牌走向社交化的一大工具。鉴于 WordPress 是一项开源服务，故而任何人都可以改写代码。通常来讲，博主们既可以为他们的网站申请主题以进行优化，同时还可以将第3方插件集成至博客中。目前，最新版的 WordPress 已拥有超过2100万次的下载量。

          相信有很有巴友都有尝试用wordpress搭建自己的个人博客，wordpress的确是非常棒的博客系统，拥有非常多个性化的模版，我自己也搭建了一个，但是很少使用 http://chino.apkbus.com/ 。

         突然不想写了，我不就是想介绍下一些开源的建站系统而已嘛。只要把一堆贴出来就行把。然后我也不想贴了，大家在网上搜索下CMS。或者建站系统。或者博客系统等等都能找到自己心仪的cms。话说最近对drupal非常感兴趣呢。
