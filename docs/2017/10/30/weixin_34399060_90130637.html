继续研究 SO_KEEPALIVE 问题

Oct 30, 2017

借助于网上他人文章，研究 SO_KEEPLIVE 问题：http://www.iteye.com/topic/1118643

client端加了 setsockopt 函数调用。看看能否在服务器端未启动时，或下线时候，一直等待两小时左右。

   int sockfd;

char dict(char temp){                
   if(temp == 'Z'){                
         return 'A';                
   }

}

      struct sockaddr_in client_address;                
      int i,btye;                
      char char_recv,char_send;

      }
