Windows 8 metro 真机调试程序

Oct 23, 2017

首先从Microsoft下载一个工具 Remote Tools for Visual Studio 2012 RC

利用 Remote Tools for Visual Studio 2012 RC，可以在不具有 Visual Studio 的计算机上进行远程调试、远程测试和性能探查。利用此组工具，还可以在运行 Windows 8 Release Preview 的计算机上安装开发人员许可证并部署 Windows Metro 风格应用程序。

1. 首先在有触摸屏的真机上或者是Win8平板上 安装这个调试工具。

2.右键工程的属性，点击调试(Debug)选项， 在远程调试机器里面输入目标机器的 IP 地址，或者点击发现(Find)按钮自动寻找目标机器。

 

3. 如果目标真机上没有安装Windows 8 开发者许可证，此时会出现一个安装开发者证书的对话框，点击安装.（我的电脑已经安装了开发者许可证）

 

4.输入你注册的微软开发者帐号和密码进行登录。

输入成功后会出现一个到期日期(到期后还可以续订)。

 

点击 远程机器调试（Remote Machine）进行真机调试。

 

最后要注意下目标真机要用微软的Live ID 进行登录才能连接，否则无法连接到目标机器。

 

希望能给后面的开发者带来帮助!~~.
