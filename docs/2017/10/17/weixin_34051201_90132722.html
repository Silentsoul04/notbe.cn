Ubuntu14.04安装JDK

Oct 17, 2017

下载oracle jdk包

从oracle官网下载jdk包，请选择Linux的tar包：

如果想使用命令行下载工具进行下载，可以先获得下载地址，然后运行curl进行下载：

curl -L -O -H "Cookie: oraclelicense=accept-securebackup-cookie" -k "http://download.oracle.com/otn-pub/java/jdk/8u91-b14/jdk-8u91-linux-x64.tar.gz"

解压缩

把jdk包解压缩到/usr/lib目录下：

sudo tar -zxf jdk-8u91-linux-x64.tar.gz -C /usr/lib/

到/usr/lib目录下，查看一下解压缩后的目录：

设置环境变量

打开 /etc/profile文件，在末尾添加下面的内容：

export JAVA_HOME=/usr/lib/jdk1.8.0_91

export JRE_HOME=${JAVA_HOME}/jre

export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib

export PATH=${JAVA_HOME}/bin:$PATH

保存后重新加载 /etc/profile文件：

source /etc/profile

检查安装结果

现在运行java –version命令看看安装的结果：

好了，安装完成。
