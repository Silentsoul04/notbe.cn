HTML：链接标签的使用

Oct 18, 2017

介绍：

链接分为两种：
　　第一种：链接到本网页其他位置 例如：从网页顶部链接到底部
　　　　<a href="#somewhere">link</a>
　　　　<p id="sonewhere">...</p>
　　第二种：链接到其他网页 例如：从百度链接到搜狐 
　　　　<a href="adv.html">adv</a>
　　　　<a href="http://souhu.com">souhu</a>
