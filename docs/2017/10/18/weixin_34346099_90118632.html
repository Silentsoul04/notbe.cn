ylbtech-jQuery:jQuery学习

jQuery语法实例

$(this).hide()
jQuery 的 hide() 函数，隐藏当前的 HTML 元素。
$("p").hide()
jQuery 的 hide() 函数，隐藏所有 <p> 元素。
$(".test").hide()
jQuery 的 hide() 函数，隐藏所有 class="test" 的元素。
$("#test").hide()
jQuery 的 hide() 函数，隐藏 id="test" 的元素。
