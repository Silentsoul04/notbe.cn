前言

　　　Tableau作为一款BI报表工具，它与传统的BI工具有什么不同呢。今天我带大家先来认识一下Tableau。

　　   Tableau家族产品包括Tableau Desktop、Tableau Server、Tableauonline、Tableau Public 和 Tableau Reader。

Tableau家族产品有哪些？（全网最详解）

 

 

 

　　Tableau 的所有产品都支持 Unicode，并兼容用任何语言存储的数据。用户界面和支持文档的语言有英语、法语、德语、西班牙语、巴西葡萄牙语、日语、韩语和简体中文。

　　Windows 环境和 MacOS 环境都支持 Tableau Desktop。此外，在配置了适当的底层 Windows 操作系统，并满足最低硬件要求的情况下，所有 Tableau 产品都可在虚拟化环境中运行。这些虚拟解决方案包括 Citrix 环境、Parallels、Microsoft Virtual PC 和 VMware。


Read more at https://www.tableau.com/zh-cn/products/desktop/download#lSO14cH0kFuW5cZo.99
