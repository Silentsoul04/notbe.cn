连接到远程的Autodesk Vault Server

Mar 27, 2017


除开发环境下，在实际应用环境中多数情况下Autodesk Vault Server不会和Client安装在一台机器上，因为大多数情况下一台Autodesk Vault Server会为多个Vault客户端服务。下面是Vault的架构图：

 

从Vault client连接到Vault Server其实也非常简单，因为Vault客户端是通过Web Service与Vault服务器通讯的，也就是说他们是通过HTTP协议（默认80端口) 通讯。既然这样，保证Vault服务器的80端口畅通即可。如果你用Windows Server，为服务器添加Web Role。如果用的Win7，那么80端口默认是关闭的，需要手动开一下：

进入到控制面板--》系统和安全—windows 防火墙，运行程序穿透防火墙

 

找到World Wide Services(HTTP）万维网服务，后面打钩即可。

如果找不到这个条目或者你是用的其他端口，那么在防火墙设置那里点“高级设置”，然后添加新的Inbound规则，TCP 80端口（或者你用的其他端口号）

这样Vault client就可以登录到远程的Vault服务器了：
