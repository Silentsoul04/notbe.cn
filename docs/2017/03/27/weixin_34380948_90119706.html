MagicHouse- 智能家居管理机器人 在线演示

Mar 27, 2017

你是否想过家里有一个帮你打点一切家务的机器人？如今我们的工作，在一定程度上满足了你的愿望。MagicHouse是一个可以帮你管理家庭电器，传感器数据，陪你聊天解闷，能和你在微博，微信，手机，平板和浏览器上与你无缝交流的机器小人，具备一定的情感。让我们拭目以待。

一. 在线演示客户端

 

二. 基本功能

     1. 完整的社会属性：与新浪微博完整对接，未来将实现对微信，WordPress,人人网和其他社交媒体的完整功能对接。

     2. 提供智能的传感器管理和查询控制功能。您可以通过语音和文字指令，通过微博和其他方式，例如“开灯”，“打开窗帘”，“我觉得有点冷，把空调温度稍微调高一点”,“好热啊”，系统会自动帮您控制家庭中的智能设备。

     3. 自动应答和学习功能：它可以根据您的调侃进行智能应答，风趣幽默，更重要的是，它可以通过各种方式学习您的说话和各种各样的知识。

     4. 完整的用户管理权限：不同用户有着完整的评级分数和等级，高级权限者方能使用更高级的功能，包括对家电的控制等。当等级达到一定程度后，它会慢慢的依赖您，并向您推荐您感兴趣的东西。

     5. 手机和平板端的语音识别和合成功能：打字太麻烦？用手机直接和智能小屋对话吧，它可以顺畅的和您通过语音聊天，目前已经提供Windows Phone和Win8应用程序。

     6. 智能家居的领域专家：它会自动收集智能家居领域的技术博客和微博，您可以问它该领域的不同知识和问题，它会在对应的微博中@您。

     6. 其他高级功能：

        (1)它能猜测和分析某用户的心情和兴趣

        (2)能知道全国范围内的天气情况

        (3)您能控制它发微博（甚至可以带图片），转发微博等

        (4)智能推荐系统

        更多高级功能正在开发中。

 

三. 自然语言使用示例

     1.  今天天气怎么样？  北京明天的天气如何？ 石家庄后天的天气？

     2. 我想知道”热情的沙漠-BUPT”最近的心情如何？

     3. 帮我开灯。  打开空调。 设置空调温度27度。 我有点冷。 

     4. 教你“我漂亮不？” ，“哇，主人，您是最最漂亮的美女！”

     5. （在某条微博的评论中）帮我转微博 “这些孩子真聪明！”

     6. 发微博 “今天好热啊，神奇小屋都热的受不了啦！”
