Visual Studio 2005连接My SQL,可视化数据库连接,数据管理

Sep 13, 2017

1、下载安装ADO.NET Driver for MySQL (Connector/NET)
最新版本5.1.3，下载地址： http://www.mysql.com/products/connector/
下载完整版( MySql.Data.msi )安装后会在Visual Studio 2005中安装相应连接驱动。
2、打开Visual Studio 2005，新建数据库连接

选择数据源列表中的MySQL DataBase


输入用户名密码等，连接.....


连接成功，可以在Visual Studio 2005中管理MySQL数据库中的数据了；
也可以在数据源控件中选择刚新建的连接，以方便绑定到GridView等控件。
