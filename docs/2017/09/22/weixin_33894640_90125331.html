第19届Jolt大奖揭晓（转载）

Sep 22, 2017

原文链接：http://www.infoq.com/cn/news/2009/03/19th-jolt-award。

据Jolt大奖官方网站消息，第19届获奖名单已于近日揭晓。在13个奖项中，每个奖项均各有一个Jolt大奖，两个生产效率奖。这次评选共有300多个提名，65项入围。最终获奖名单如下：

  • Jolt荣誉堂：Compuware DevPartner
  • 一般图书类
    • Jolt大奖：Adrenaline Junkies and Template Zombies: Understanding Patterns of Project Behavior（本书中文版由人民邮电出版社图灵公司引进）
    • 生产效率奖：Emergent Design: The Evolutionary Nature of Professional Software Development（本书中文版由人民邮电出版社图灵公司引进）；Intellectual Property and Open Source: A Practical Guide to Protecting Code
  • 技术图书类
    • Jolt大奖：Real World Haskell（由清华大学出版社引进）
    • 生产效率奖：High Performance MySQL: Optimization, Backups, Replication, and More；Programming in Scala
  • 变更管理和配置管理工具
    • Jolt大奖：OpenMake Meister
    • 生产效率奖：AccuRev；Change Manager
  • 数据库引擎、数据工具
    • Jolt大奖：Postgres Plus
    • 生产效率奖：Aqua Data Studio；Vertica Analytic Database
  • 设计与建模工具
    • Jolt大奖：Blueprint Requirements Center
    • 生产效率奖：RAVEN；Screen Architect
  • 开发环境
    • Jolt大奖：Altova MissionKit
    • 生产效率奖：JBuilder 2008；ReSharper 4.1
  • 企业工具
    • Jolt大奖：MindTouch Deki
    • 生产效率奖：Agile Platform；Zenoss Core
  • 类库、框架、组件
    • Jolt大奖：ComponentOne Studio for Silverlight
    • 生产效率奖：Essential Studio Enterprise Edition；Threading Building Blocks Library 2.1
  • 移动设备与Web开发工具
    • Jolt大奖：MITE
    • 生产效率奖：Adobe Device Central；NetBeans IDE
  • 项目管理工具
    • Jolt大奖：Pivotal Tracker
    • 生产效率奖：Rally Enterprise；ScrumWorks Pro
  • 安全工具
    • Jolt大奖：Parasoft Application Security Solution
    • 生产效率奖：GuardIT；InTrust for Databases

    • 生产效率大奖：DotTrace；JavaRebel

图灵公司的总编辑刘江在博客上对Jolt大奖评论说：

Jolt这个奖项自称并被很多其他媒体称为“软件界的奥斯卡奖”，有一定道理，因为软件界好像也没有什么其他类似的通用奖项有广泛的影响，有较长历史，而且一直坚持到现在了。
不过，必须注意，这个奖项是一个评选委员会评出的，而且要参选是要交钱的。商业因素和“委员会设计”问题（一个反模式，参见图灵出版的《反模式》一书）都存在。
事实上，这个奖到今天为止，总体上看真正有价值的就是图书奖。（其次是为数不多的几次网站奖和名人堂奖。）如果刨去近几年最后的得奖品种稍有偏差，十多年来基本上没有漏掉太多真正的经典书（早年的评选尤为精准），参考价值还是非常大的，可以作为技术图书界的风向标来看。

Jolt大奖官方网站上介绍了从提名、入围直到最终评选的简要流程：

在提名的时候需要提交哪些信息？

联系方式、分类、子类、补充产品信息（可选）、竞争产品相比的优势所在、费用。

入围名单是怎么定下来的？

评委们会根据申请表、提交的补充材料、产品的影响力、他们对产品的熟悉程度、在线评审、同行评审、网站上提供的信息等等作出评价。我们强烈建议你在提名的时候同时提交一个5分钟的Flash视频，或是PowerPoint，或者webinar，对产品的核心功能做重点介绍。

那最终获奖名单又是怎么定的？

评委们不单单会考虑一些典型标准（生产力、创新性、投资回报、质量等等），也是在努力寻找业界领军的产品。获奖的产品都是应用相当广泛的，简单而又功能丰富，对行业领域造成了影响，或者解决了某个存在已久，而且其他产品或者图书一直回避的问题。
