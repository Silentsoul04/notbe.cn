魔兽争霸在win7下的分辨率解决方法

Sep 15, 2017

我的电脑是宽屏,从xp换到win7之后,玩魔兽争霸,屏幕两侧总是有两条比较宽的黑色带,去设置游戏的分辨率为1024*768*32,发现不管用.虽然我的电脑分辨率是1366*768的分辨率,但之前在xp中,游戏分辨率1024*768*32时,还是全屏没有左右两侧的黑条的. 
上网找了下,发现在新浪博客上已经有人给出解决方法了: 
建立一个注册表导入文件war3revolution.reg,给里面写下如下内容:

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Blizzard Entertainment] 
[HKEY_CURRENT_USER\Software\Blizzard Entertainment\Warcraft III] 
[HKEY_CURRENT_USER\Software\Blizzard Entertainment\Warcraft III\Video] 
"reswidth"=dword:00000556 
"resheight"=dword:00000320 
"colordepth"=dword:00000020

这个是1024*768*32分辨率的解决办法. 
其中reswidth是长度,resheight是高度,colordepth是颜色深度.上面填写的都是deword的值.

一些常用的值对应的deword值: 
800         00000320 
600         00000258 
768         00000300 
1280       00000500

对于不常用的分辨率的值,怎么计算它的dword值呢? 
打开windows计算器(win键+R,输入calc就能打开),在菜单view中选择 

比如要屏幕分辨率是1366,选择Dec(10进制)和Dword,在数值框输入1366,然后将Dec那块选择为Hex(16进制),
