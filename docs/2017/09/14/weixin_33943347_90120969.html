久违的IE6。

Sep 14, 2017

自从去年初安装了vista后，我就再也没有见过IE6。在前端开发这个面上，接近一年的时间我基本处于无IE6的环境下，不知道还有没有人和我一样，后半年的盲调试、意念中想像中调试完全是种折磨，再好的盲调试始终会有盲点。海词2008在对IE6的支持上始终不如FF和IE7，甚至不如从来没考虑过的safari和opera，极其勉强。值得庆幸的是依靠尽可能的回避hack使用产生差异最少的布局属性再加上团队为我截IE6图的指引，基本上使得IE6下没有出现无法容忍的纰漏。

今天我终于艰难的把我的vista替换成了Windows server 2008 + Windows server 2003。终于有了IE6，好了，调试方便了，海词2008的所有IE6用户有福了，团队也不用再喂我IE6截图了。但是Windows server 2008的IIS7我还没配置好对shtml的支持，IE7又成问题了……不过这个问题不算大。

硬盘越来越大，系统中的东西越来越多，习惯配置越来越多，不能轻易的format，每一次更换系统就像搬一次家，真是麻烦，备份过来备份过去。尤其是我这么一个守旧的人，一张图片丢失都舍不得。尤其是我这么一个懒人，不喜欢使用光盘。

死了几个月的word也可以安装了！死了几个月的TM也能登录了！也可以装Visual Studio 2008了！也可以装Adobe CS3系列了！……

o yeah. 新年新系统新气象！
