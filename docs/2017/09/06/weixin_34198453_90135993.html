Visual Studio 2008 Beta 2 初步体验

Sep 6, 2017

我选择的是Visual Studio 2008 Standard Edition Beta 2 (722.7MB)，初步体验了一下，感觉还不错。
 
开始安装...


安装中ing....


安装成功，启动visual studio 2008


起始页


关于


新建网站，可以选择.net版本，这个很有用的噢。发布后的dll为随机命名。


新建工程，可以创建web application工程、WCP,WPF等工程，可以选择.net版本。发布后dll文件为固定命名。


除了设计，代码视图，还多了一个Spli 视图状态，这个也挺好的。


工具箱多了两个组，ajax,reporting；

新建的基于.net3.5的web项目都默认带有ajax，部份web.config如下：

对javascript的支持更好，智能感知比2005强大了。

             
强大的CSS可视化管理功能,右边除了解决方案，属性视图，还多了几个：Apply Styles,Manage Styles,Styles Properties

总体感觉：
比2005方便了，强大了，对新技术的支持，可以创建wpf,wcf等工程，可以创建基于不同.net版本的项目；
对css的支持与管理，对页面布局方面确实比2005改进了不少，页面放个div,设计视图就可看到，有显示提示，有点像dw, javascript的支持也更好了。安装也比较顺利，不像2005那么"蜗牛"，安装了2008后会对先前的.net fx打上一个sp1的补丁。
感觉还是比较满意的，但愿正式版不要和beta2差别太大，呵呵。 
注：目前还没发现和vs2005有冲突，一切正常。
