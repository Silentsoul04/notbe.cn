[CCTV] 聚焦80后

Sep 5, 2017

有人说：80后是缺乏历史认知、缺乏自省精神、缺乏责任感、是精神缺钙的一代。有一知名作家甚至公开讲：“80后基本是被港台文化洗脑的一代，他们只知有港台，他们以为中国生下来就是世界强国，他们很可笑。” 面对社会的诸多误解和质疑，《我们》特邀一群最普通的80后们到场，与前辈们进行对话，谈谈他们如何看待和理解改革开放三十年的历史发展。 他们现场的内心自省又将给我们带来哪些对这一代独生子女们的全新认识？
