也聊聊Web 2.0

Sep 5, 2017

有段时间没有更新Blog了，工作很忙、似乎感觉工作与生活之间没有明显的分界线了、怕怕 ，利用周末好好补一补、呵呵。
一、Web 2.0的概念
 虽然听惯了web 2.0，对它非常“熟悉”，但如果静下心来仔细想想，却又不知道web 2.0到底是干什么的，又会感到“陌生”。那就聊聊web 2.0吧。 (http://www.cnblogs.com/chy710)
web2.0是当前互联网发展中的一项变革，与早期的web1.0(传统的以网站主动发布信息为主)相比，更多的是让访问者(用户)参与到互联网中，由网络用户构建互联网的内容。web1.0侧重于门户网站的发展壮大，而web2.0则重视网络用户的参与及个性化的展示；web1.0侧重于封闭集中，web2.0则重视开放联合；web1.0侧重于一对多，而web2.0重视多对多。web2.0的概念是不断发展的，目前普遍的一个认识是web2.0是具有交互性等特点的理念与技术的集合。web2.0所倡导的是一种开放、参与、分享、创造的精神。
在web2.0环境下的互联网，网络用户从信息的接受者逐渐转变成信息的制造者和传播者，逐渐从受众群体转向具有自主话语权的社会群体，逐渐从单个个体转向共同关注点的团体，互联网的服务模式也逐渐发生的变革。 (http://www.cnblogs.com/chy710)

二、Web 2.0的核心范畴
１、博客----Blog
这个可能是最常见、最典型的了，从形式上看，博客网站的出现，成为一个新的网络用户聚集中心。博客之间的交流主要是通过标签(Tag)、反向引用(TrackBack)、评论(Comment)等方式来进行。从而很方便的是用户可以发表观点、分享、交流、交友等。前几天在一个证券大师的博客看到，他总结到2007年投资股票的人都不看股评改看博客了，呵呵，我觉得这很有道理。记得早些年人人都喜欢做个人主页，时髦呀，那时也没啥做的，全Html静态的，写一点个人介绍放几张照片，找个免费空间，就这样搞定了，再把地址告诉朋友，那感觉呀，自豪呀，总以为别人不知道咱是搞互联网的，后来都慢慢＂死＂掉了，基本上不更新的，再说人家免费空间也不是终身免费呀。再看看现在，都写Blog了， 什么QQ空间、Msn Space,新浪博客等等....，明星写、大师写、专家写、草根写，这也方便了、互动性强。

2、移动博客----Moblog
以移动终端设备为用户载体，以移动互联网为信息交互渠道的博客系统，特点是可以随时随地体验博客带来的乐趣，主要是年轻时常一族常用。

3、播客----Podcast (http://www.cnblogs.com/chy710)
不同于博客，用户在互联网上发布音频、视频等多媒体信息而不仅仅是文字和图片，以得到更好的用户体验。

4、标签----Tag
web 2.0时代的信息分类方式，可以简单地理解为关键词与传统的网站内容分类最大的不同就在于网民能根据自己的个性化需求，自己定制标签，从而实现准确快速的发现信息。增加了信息之间的相关性与用户之间的交互性。

5、内容聚合----Rss
现在大部分网站都能看到如：RSS,XML,Feed等标记，可以很方便的收集自己相要的信息，只需要一个RSS浏览器就可以看到自己所关注的所有网站的内容，及时了解最新的信息，而不用再去一个个访问网站了。Google的个性化主页也提供此服务。RSS主要通过一套标准协议把数据汇集到一起，是一种更为直接的互动方式。

6、对等互联网----P2P
 网民所获得的资源是由其它网民提供的，同时也为其它网民提供资源，即“人人为我，我为人人”思想在互联网中的具体体现。常见的就是P2P共享下载，P2P是一种网络技术(点对占的网络应用)，但更多的是一种思想的体现。

7、维基----Wiki (http://www.cnblogs.com/chy710)
体现开放、合作、共享的网络文化，可以让网民参与到互联网内容的编辑过程中。有人认为，wiki系统属于人类知识的网络系统，可以在web上对其自由的浏览、创建、更改。

8、阿贾克斯----Ajax
 这个对开发人员来说是最熟悉的了，简单的说是就是页面无刷新实现客户端对服务的调用，获取数据更新数据，从而让用户感觉到一种很好的用户体验。

9、Web服务----Web Service
 提供在线服务供其它用户使用，达到网络分享。提供都开放接口，使用都就可以把需要的功能集成到自己的应用系统中。

10、社会化网络软件----SNS (http://www.cnblogs.com/chy710)
也称朋友圈，是虚拟的网络活动与现实的社会活动的有机结合，使“人机对话”逐渐转变成“人与人对话”。依据六度理论 ，以认识朋友的朋友为基础，扩展自己的社会人脉。
