现场研讨会：把你的.NET技能从桌面应用到Android和iOS

Sep 20, 2017

今天登记 加入我们即将举行的研讨会，Xamarin大学介绍： 在星期二将指导桌面开始者使用Visual Studio for Xamarin开发移动应用，8月29日， 你将看到将你的桌面经验扩展到移动世界。对于有长时间桌面开发者，我将分享我的个人经验和技术的最佳实践，给您一步一步的指导，使你进阶到原生Android和iOS开发。

i注册
 
在会议期间，您将学习如何：
  • 使用Use Visual Studio Tools for Xamarin 和C#来为大量的设备和平台设计和构建本地应用
  • 将桌面经验对应到移动开发上，包括对比桌面和移动开发的APIs
  • 在将你的桌面应用移植到平板电脑或者移动设备时，优化用户体验

你也会看到示例代码，得到提示和技巧，我们将Xamarin高校辅导员和移动专家为可以在线回答你的问题。在8月29日上午9点 登记注册并加入我们！ 无论你是一个构建商业桌面应用程序，或应用程序内部使用，本次会议将给你的移动事业发展。
