Xamarin Live Player Preview 2: 连续运行和调试应用程序

Sep 16, 2017

我们介绍Xamarion现场播放器，对于开发都来说，是一个在Visual Studio、iOS或者Android设备上快速开发Xamarin的一种全新的方式，它在今年的Build开发者大会上公布出来。基于你的反馈，从最初的预览，我们很高兴地宣布一个更新的Xamarin现场播放预览的包装完整的新功能和优化。

关于Xamarin现场播放器

Xamarin现场播放器应用使开发者易于通过本地WiFi直接在设备上发布，测试和调试他们的应用。通过修改Xamarin.Forms开发者现在可以通过它来实时修改他们的XAML，并在它们的设备上查看到变更影响，不必通过编译或者等待一个完整的发布。

入门

在最近发布的 Visual Studio 2017 version 15.4 Preview 和Visual Studio for Mac version 7.2 (Alpha) 中，包含了Xamarin现场播放器，并且通过在设备中启用一个单一的复选框来启用这个功能。 在Visual Studio 2017中通过工具 -> 选项 -> Xamarin -> 其它，选中、“Enable Xamarin Live Player”. 在Visual Studio for Mac, 到 preferences中你会发现Xamarin Live Player 一节，包含一个简单的CheckBox：

下载现场播放器

我们为你创建了一个简单版本的Xamarin现场播放器的Alpha应用，你可以从苹果测试飞行或者Google应用商店下载。所以你可以快速的访问Xamarin现场播放器最新的强大的功能。简单注册 Xamarin Live Player Alpha program, 几天之后你就会收到一封电子邮件告诉你如何开始测试的最后版本。

What’s New?

Xamarin现场播放器简化了应用的发布处理。通过彻底的重新设计，开发人员现在得到了一个完整的介绍和第一次建立现场播放器的经验。

此外，当Live播放器打开并将代码部署到其上时，VisualStudio和Live Player现在对配对状态给予实时反馈。

我们还添加了高级网络连接测试、调试日志，并简化了设置，这样您就可以更快找到你需要的地方。除了用户界面的改进外，引擎盖也有了巨大的改进，以帮助复杂的应用程序。你也可以通过浏览发行说明（ release notes here），查看所有令人兴奋的改进之处。
