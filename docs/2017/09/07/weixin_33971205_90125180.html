如何用WordPress搭建自己的博客（转）

Sep 7, 2017

wordpress搭建博客，其实蛮简单的，只需要下面几个步骤：

1、首先需要注册一个域名

一个域名每年的费用大约需要50-60元人民币，最好注册.com域名，不要注册.cn结尾的域名，像我的网站这样的.cn后缀的域名在很多方面不好，过段时间我可能也会换个.com的域名。另外，尽量在Godaddy这样比较大的域名注册商注册域名，不要在国内注册。这样做的原因请看月光博客的两篇文章：不要在国内注册域名和在国外进行域名注册。
2、购买一个主机空间

用Wordpress搭建博客，网站空间需要支持php，并且有MySQL数据库。Linux + Apache+MySql+PHP主机空间最好。空间每年的费用根据不同的配置一百多到几百多都有，访问量不是很大的话，买个便宜的就足够用了，但也不要买太便宜的。买空间最好也买国外的，国外的空间价格比国内的便宜，而稳定性比国内的要好很多倍，而且不需要备案。但是使用国外的空间也有个问题：一定不能在自己的网站或博客上写党不愿意听的话，否则，很快你的网站在中国大陆就不能访问了，网站所在的IP就被屏蔽了。普通网站空间一般都是多个空间共享一个IP，因此，为了防止被别人连累，购买国外空间的时候最好再多花十几美元购买一个独立的IP。我的网站现在就是用的美国的主机空间，独立IP。

3、设置域名和空间

在主机空间上绑定域名，将域名指向所购买的主机空间的IP，等待域名解释生效。

4、下载安装Wordpress

安装Wordpress五分钟就可以搞定，传说中的Famous 5-Minute Install。
英文版从这里下载：http://wordpress.org/download/
中文版从这里下载：http://code.google.com/p/wpcn/downloads/list
下载并解压。
把wp-config-sample.php这个文件重命名为 wp-config.php，用记事本或其它文本编辑工具打开这个文件，填上相关的Mysql数据库信息。这一步不做也可以，在下面的安装过程中，程序会提示填写相关配置信息。

将所有文件上传到主机空间中，在浏览器中打开：http://example.com/wp-admin/install.php(将example.com换成相应的域名) 就看到安装界面了，根据页面提示填上相关内容（主要是数据库用户名和密码要填写正确，其它的信息都可以随便填，之后在后台可以更改），然后提交就OK了，Wordpress就搭建完成了。
