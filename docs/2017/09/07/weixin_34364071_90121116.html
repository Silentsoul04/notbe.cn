爆牙齿的世界杯日记（阿根疼啦）

Sep 7, 2017

我在楼上，死死抓着栏杆，俯视着斜下方墙上的投影屏幕，从一开始就憋着这四个字和拳头，等待爆发和挥舞。从0:1等到0:3，从阿根廷必胜等到你就进一个好歹让我把这如鲠在喉的刺一吐而快嘛。最后0:4的时候，算了算了，别进了，我，纯爷们儿，咽！

球赛完了，我趴在栏杆上对着屏幕发了许久呆。楼下热闹的在抽奖，回头一望，楼上一片狼藉已空无一人，带着鲠在喉的刺，还看个屁的西班牙，撤吧。快速下楼穿过喧闹的人群走出门。

刚闪到门口，一个凄凉的白色影子，独自拧着一瓶酒，在空旷的大街上，沉寂的夜色中，惨白的月光下……噢，不好意思，写高兴了，貌似没有月亮，咱得尊重事实，重来。

刚闪到门口，一个凄凉的白色影子，独自拧着一瓶酒，在空旷的大街上，沉寂的夜色中，昏暗的路灯下，坐在花台边缘，叉开着双腿，耷拉着脑袋，游离的眼神，微颤的肩膀，眼看着就要“呜呜”叫两声了……嗯，好了，够了够了……总之，我刚走出喧闹的咖啡厅就走入一个鬼撑出来的熟悉气场。知音啊，我立刻返回叫服务员开了一瓶酒再走出来：白鸦，干！

咕噜咕噜，把那刺活活的，活-活-的咽了下去。疼啊，又将是个四年。

半夜，QQ一闪，小鱼儿说：等着看你的悲情日记，这场最郁闷，喝酒的兴致都没了。悲情悲情，其实我挺不想把这年复四年的悲情又拿出来娱乐一番，其实我挺想反其道写成一篇欢快的文章，可是我又明显没法去入欢快的戏，会显得不专业的。最关键的是，要真是悲情，那完全符合我的气质，我这号悲剧人物写悲情，那是手到擒来行云流水的专业。可是，这次不悲情啊，这可如何是好？向左？向右？

听见，冬天，的离开，我在某年某月醒过来；我想，我等，我期待，未来却不能理智安排；阴天，傍晚，车窗外，未来有一个人在等待；向左，向右，向前看，还要拐几个弯才来……自打我爱上你，你就开始拐弯，亲爱的你都拐了六个弯了，连中国都进过世界杯了，太不厚道了！好吧好吧，向前看，还要拐几个弯才来。还要拐几个弯才来？话说山路十八弯，莫非还有十二弯，12X4=48，我32，一加80……锤子，太危险了，老子都耄耋了。

悲情悲情，我提醒自己略微有一点点跑题，对不起，酝酿一下情绪，严肃，专业。

之后，一路去吃了宵夜，啃了很多龙虾，心情好些了，就把西班牙看了，后面还有四场，依旧会看，不过爆牙的世界杯日记就到此为止，不再写了。原因很简单，我，是个感性的人，所谓感性的人，就是有感才有性。我只对阿根廷有感，所以阿根廷在，我就有很多很多的性，就乐意写很多很多的屁话。但是德国、意大利、荷兰、西班牙、葡萄牙等等甚至巴西，这些队打得好，我欢呼，打得臭，我唾骂，然而无论欢呼还是唾骂都不带有感情色彩，没有感自然就没有性，清心寡欲还写个屁，不写了。

但凡和感性沾边，都是个奇怪的东西，没法以常理来解释。为什么热爱阿根廷呢？这场比赛阿根廷前场几乎就没有威胁的进攻，中场几乎就没流畅的配合，而球一到阿根廷的后场我就揪心的胆颤，每一次！德国的锋线真就是刀一样，肆意而轻松的撕割着阿根廷撕割着死死抓着栏杆，大气不敢出一声的我。马拉多纳赛后说0:4不能表现出场上的真实情况，他没说错，德国仁慈于是阿根廷幸运，只输了4个。

且不说德国无数次的威胁，就这四个球。第一个落点极佳的任意球和灵性的一蹭；第二个大范围团队穿插调动破密集防守，倒地传向空档，刺穿防线，不贪功的推给空门；第三个快速突破下底，再一拨还下，小传门前，倒地抢点；第四个精准的刚好挑传过两个防守，凌空推射。四个毫无争议的球，四个本应该都属于阿根廷的方式。尤其是第二个球，你可曾想起四年前那24脚撕开密集防守的经典和老马倒地的世纪之传？当被以其人之道还其人之身时，是怎样的痛？

是的，作为阿迷，输了六个弯二十四年，绝对是最输得起球的球迷。可是怎么也预料不到会以这种毫无争议，话都说不出来只能咕噜咕噜喝酒的方式离去。世界杯以来的几场比赛，我一直认为阿根廷是没认真的吊儿郎当的，而打德国这场，阿根廷真的已经尽了全力了，全场都非常的认真，结果……

这不是一场所期待的传世之战、复仇之战，只有德国的华丽，阿根廷甚至连英格兰都不如，英格兰赢得了尊重，而阿根廷只有无奈和屈辱。很多人把矛头指向马拉多纳，指向球员，如果怎么怎么，如果怎么怎么，TMD有意思吗？还看不出来吗？真的是无法弥补的差距，可以说任何如果都不存在，这是一场必输的对决。

90年，阿根廷拼掉四个核心主力几乎无替补可用的打入决赛，半残队伍射门1:24的顽强逼平德国又遇见裁判偏向德国的争议点球，阿根廷输了，我们痛骂裁判，但我们没有输人，我们输得起。我仍记得那时淘汰意大利而付出的恐怖代价，半残队伍挺入决赛的悲壮。

94年，药检风波，阿根廷输了。我们恨马拉多纳不小心也极度怀疑足联的栽赃扼杀，老马还需要兴奋剂？但老马坐到观众席上，我们没输人，我们输得起。

98年，和崛起的强大荷兰打得难分难解，可惜最后被博格坎普绝杀的阿根廷，那是虽败犹荣的不服气！

02年，豪华阵容的阿根廷落入历史最死的死亡之组，还未热身便出局了，随之出局告别的风之子、战神、雷东多，平添许多愁。

06年，这大概是第一次团队闪耀的一届阿根廷。四年前的点球败给德国，显然是耿耿于怀的。

10年了，该怎么说呢？

马拉多纳带着我们回86年，于是整个阿根廷我只认识了一个人：梅西。于是我们都坠入了马拉多纳给我们编织的幻想中，复制了上帝之手和世纪进球的梅西，会复制86年的大力神杯。从第二场贝隆不再首发后，梅西连位置都完全的回到了马拉多纳的位置。变动是成功的，梅西确如马拉多纳一样，盘活了阿根廷的中场；变动也是失败的，10年不是86年，整体协防的现代足球留给巨星的空间越来越小，竖给巨星的门槛也越来越高。第一场比赛后我曾说过，此后梅西必定将遭到围剿，而梅西要成为新一代的马拉多纳，必须摔倒然后一个跟头翻起来，战胜围剿穿越围剿。梅西摔倒了，没能爬起来。最最重要的是：作为前锋，无人传球的梅西和作为前卫，无人可传的梅西是等同的。阿根廷VS德国，这是一场时代之间的较量，是典型的古典球星足球VS现代整体足球，结果就是：1986VS2010=0:4。

90年开始，从来没有哪一届阿根廷像本届世界杯这样全队只看见了1个人，即便是90，94年马拉多纳仍闪耀的时代，也还有卡尼吉亚、巴蒂、戈耶切亚等众星捧月，而10年的阿根廷除了梅西还有谁？过于热捧梅西的马拉多纳，真是在不知不觉的丢掉了团队。没有批评马拉多纳的意思，我接受他的做法，如果马拉多纳不这么做，他就不是马拉多纳了，而这届阿根廷就不是马拉多纳的阿根廷了。就像他脆弱的伏在女婿肩上，甚至没用完换人名额。如果是个理性冷静的教练哪怕换上一个年轻球员也是有好处的，可那就不是马拉多纳了，就不是感性面对失败只为冠军而生的马拉多纳了。所以，我接受，我全盘接受，还换狗屁人，还进狗屁球，要争狗屁尊严，拿不到冠军一切都没意义。

马拉多纳吹出一个美丽的气泡，把我，把我们包裹起来，我们心甘情愿。当五彩斑斓的泡沫破灭时，一身湿透的我们终于看清，任何人都无法阻止时代的前进。那么，让冰冷的我们为华丽而完美的德国鼓掌吧，他们的每一个进球，每一次进攻，每一次防守都无可挑剔，他们是真正的时代天骄，同时具备了南美的细腻和欧洲的硬朗，还有德国传统的团队精神钢铁意志，还有最难能可贵的：无可限量的年轻与朝气！

只是我多么的希望，我看着德国用熟悉而华美的阿根廷方式击败阿根廷的每一分钟，我竟然是多么的希望，他们身上穿着的是蓝白相间的队服，我多么的希望，我能够在直播的时候为每一个纯粹的阿根廷式进球而欢呼。可是我不能，因为他们穿着德国队服，他们的球进的是阿根廷的门，纵然再华丽我都不忍多看一眼。我只能，死死抓住栏杆，盯着屏幕，盯着这两群最熟悉的陌生人，在似幻非幻的内心迷茫和疼痛中，沉默不语。我，我们难以接受德国用阿根廷的方式，高高在上的，把所有期待击碎得找不到一丝残骸。

我终于明白了，我爱的不是足球，我爱的是阿根廷，我爱的是野性的长发和蓝白的相间。

四年后，你若又转弯，我继续陪你。你若还要转十二个弯，我就一定会奋力活到耄耋之年。

“只是因为在人群中多看了你一眼”，那一抹蓝和一抹白就再也挥之不去。

End.
