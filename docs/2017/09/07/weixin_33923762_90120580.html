struts2编辑框中自动填充已经有的列表对应的值

Sep 7, 2017

你返回原来的页面是如何返回的？是forward还是redirect？
先在action里面name，password写get set
如果是forward的话，直接jsp页面里写上
<input name="name" value=" name"/><inputname="password"value="name"/><inputname="password"value=" {password}"/> 
就可以
如果是redirect的话，在struts.xml里要写上
<result name="redirect" ......>***.jsp?name= {name}&password={name}&password= {password}</result>
jsp页面则还是
<input name="name" value=" name"/><inputname="password"value="

name"/><inputname="password"value="{password}"/>
